# 글쓰기 수정 — Edit Writing (Stage 1: 골격)

글의 **뼈대**를 다듬습니다. 내용 배치, 문장 구조, 용어 일관성, 간결함.

## 파이프라인 위치

```
/equalize-baseline → ▶ /edit-writing → /tighten-structure → /clarify-writing → /refine-style → /smooth-flow → /verify-facts → /deep-review → /align-reference
```

## 인자
- `$ARGUMENTS`: 수정할 파일 경로

## 적용 axis (memory)
- 1순위: **통사론** (문장 구조·어순·절 중첩·문장 호흡)
- 2순위: **의미론** (용어 일관·과장·기피 어휘)

## 점검 항목

### 1. 용어 일관성 (의미론 5)
- 같은 개념에 같은 용어, 새 용어는 첫 등장 시 정의
- 같은 개념을 두 표현으로 혼용 금지 (예: Visual Rebuild ≠ 메쉬 재생성 동시 사용)

### 2. 기피 어휘 (의미론 1·4)
매핑표는 `linguistic_semantics.md` 참조. 핵심:
- 구어 동사: 쪼개·가르·섞어두·떼어놓·몰아넣 → 형식 기술 용어
- 모호 대명사: "쪽만/쪽은" → 정확한 명사
- 과장 부사: 매우·훨씬·완전히·완벽하게·획기적·차원이 다른 → 구체 비교

### 3. 문장 구조 (통사론 1·5·7)
- 한국어 자연 어순 (배경→주제, 조건→귀결)
- 절 중첩 3단계 이내, 한 문장 쉼표 4개 이내
- 이중 피동·번역투 회피

### 4. 간결함
- 불필요 반복 제거
- 단, **구체 정보(수치·예시·메커니즘) 절대 보존** (process 7)

### 5. 단계 표지 (통사론 10)
2단계+ 프로세스는 "먼저 ~ 그다음 ~" 명시적 표지
메타 announcement ("두 단계로 이루어집니다") 단독 문장 X
