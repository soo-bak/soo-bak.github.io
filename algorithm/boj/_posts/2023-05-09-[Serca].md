## 문제 링크
  [26766번 - Serca](https://www.acmicpc.net/problem/26766)

## 설명
입력으로 주어지는 숫자 `n` 개 만큼 하트를 ASCII 아트로 출력하는 문제입니다. <br>

단, 세로로 출력해야 합니다. <br>

`C#` 의 경우, `StringBuilder` 를 사용하지 않으면, 시간 초과가 됩니다. <br>

- - -

## Code
<b>[ C# ] </b>
<br>

  ```c#
namespace Solution {

  using System.Text;
  class Program {
    static void Main(string[] args) {

      string[] heart = {
        " @@@   @@@ ",
        "@   @ @   @",
        "@    @    @",
        "@         @",
        " @       @ ",
        "  @     @  ",
        "   @   @   ",
        "    @ @    ",
        "     @     "
      };

      StringBuilder sb = new StringBuilder();

      var n = int.Parse(Console.ReadLine()!);
      for (int i = 0; i < n; i++) {
        for (int j = 0; j < 9; j++)
          sb.Append(heart[j] + "\n");
      }

      Console.Write(sb.ToString());

    }
  }
}
  ```

  ```c++
#include <bits/stdc++.h>

using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  const string heart[] = {
    " @@@   @@@ ",
    "@   @ @   @",
    "@    @    @",
    "@         @",
    " @       @ ",
    "  @     @  ",
    "   @   @   ",
    "    @ @    ",
    "     @     "
  };

  int n; cin >> n;
  for (int i = 0; i < n; i++) {
    for (int j = 0; j < 9; j++)
      cout << heart[j] << "\n";
  }

  return 0;
}
  ```
