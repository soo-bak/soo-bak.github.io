## 문제 링크
  [10171번 - 고양이](https://www.acmicpc.net/problem/10171)

## 설명
ASCII 그림으로 고양이를 출력하는 문제입니다. <br>

`\` 문자는 이스케이프 문자로도 사용된다는 점에 주의해야 합니다. <br>

- - -

## Code
<b>[ C# ] </b>
<br>

  ```c#
namespace Solution {
  class Program {
    static void Main(string[] args) {

      Console.WriteLine("\\    /\\");
      Console.WriteLine(" )  ( ')");
      Console.WriteLine("(  /  )");
      Console.WriteLine(" \\(__)|");

    }
  }
}
  ```
<br><br>
<b>[ C++ ] </b>
<br>

  ```c++
#include <bits/stdc++.h>

using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  cout << "\\    /\\\n" <<
          " )  ( ')\n" <<
          "(  /  )\n" <<
          " \\(__)|\n";

  return 0;
}
  ```
