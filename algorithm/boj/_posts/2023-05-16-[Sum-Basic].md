## 문제 링크
  [8393번 - 합](https://www.acmicpc.net/problem/8393)

## 설명
`1` 부터 입력으로 주어지는 숫자 `n` 까지의 합을 구하는 단순한 문제입니다. <br>

반복문을 이용하여 간단히 구현합니다. <br>

- - -

## Code
<b>[ C# ] </b>
<br>

  ```c#
namespace Solution {
  class Program {
    static void Main(string[] args) {

      var n = int.Parse(Console.ReadLine()!);

      int sum = 0;
      for (int i = 1; i <= n; i++)
        sum += i;

      Console.WriteLine(sum);

    }
  }
}
  ```
<br><br>
<b>[ C++ ] </b>
<br>

  ```c++
#include <bits/stdc++.h>

using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;

  int sum = 0;
  for (int i = 1; i <= n; i++)
    sum += i;

  cout << sum << "\n";

  return 0;
}
  ```
