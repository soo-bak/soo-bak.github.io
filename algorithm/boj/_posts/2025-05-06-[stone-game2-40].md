## Code

### C#

```csharp
using System;

class Program {
  static void Main() {
    int n = int.Parse(Console.ReadLine());
    Console.WriteLine(n % 2 == 0 ? "SK" : "CY");
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  if (n % 2 == 0) cout << "SK\n";
  else cout << "CY\n";

  return 0;
}
```
