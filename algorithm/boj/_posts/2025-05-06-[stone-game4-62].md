## Code

### C#

```csharp
using System;

class Program {
  static void Main() {
    int n = int.Parse(Console.ReadLine());
    var dp = new bool[1001];
    dp[1] = true;
    dp[3] = true;

    for (int i = 4; i <= n; i++)
      dp[i] = !dp[i - 1] && !dp[i - 3] && !dp[i - 4];

    Console.WriteLine(dp[n] ? "CY" : "SK");
  }
}
```

### C++

```cpp

}
```
