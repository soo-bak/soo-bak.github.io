## 문제 링크
[15727번 - 조별과제를 하려는데 조장이 사라졌다](https://www.acmicpc.net/problem/15727)

## 설명

성우가 조장을 찾으러 가는데, 1분에 최대 5만큼 이동할 수 있습니다.

조장까지의 거리가 `L`일 때, 조장을 찾는 데 걸리는 최소 시간을 구해야 합니다.

<br>

## 접근법

거리 `L`을 이동하는 데 필요한 최소 시간은 `L`을 5로 나눈 값을 올림한 것입니다.

예를 들어 거리가 12라면 5씩 2번 이동하고 나머지 2를 이동하기 위해 1번 더 필요하므로 총 3분입니다.

<br>
올림 계산은 두 가지 방법이 있습니다.

<br>
첫 번째는 `L / 5`를 구한 후 나머지가 있으면 1을 더하는 방법입니다.

<br>
두 번째는 `(L + 4) / 5`로 계산하는 방법인데, 이는 나머지가 1 이상이면 자동으로 올림되는 수학적 트릭입니다.

예를 들어 `L = 12`일 때 `(12 + 4) / 5 = 16 / 5 = 3`이 됩니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var L = int.Parse(Console.ReadLine()!);

      var time = L / 5;
      if (L % 5 != 0) time++;

      Console.WriteLine(time);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int L; cin >> L;

  cout << (L + 4) / 5 << "\n";

  return 0;
}
```
