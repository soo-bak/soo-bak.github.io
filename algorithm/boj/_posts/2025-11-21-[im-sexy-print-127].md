## 문제 링크
[15733번 - 나는 누구인가](https://www.acmicpc.net/problem/15733)

## 설명

입력 없이 `"I'm Sexy"`를 출력합니다.

<br>

## 접근법

문장을 그대로 출력하면 됩니다.

<br>

---

## Code

### C#

```csharp
using System;

class Program {
  static void Main() {
    Console.WriteLine("I'm Sexy");
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  cout << "I'm Sexy";
  
  return 0;
}
```
