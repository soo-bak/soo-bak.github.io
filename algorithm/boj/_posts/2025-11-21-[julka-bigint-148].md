## 문제 링크
[8437번 - Julka](https://www.acmicpc.net/problem/8437)

## 설명

두 사람이 가진 사과의 총합 `T`와 클라우디아가 나탈리아보다 더 많이 가진 개수 `D`가 주어질 때, 각자가 가진 사과의 개수를 구해야 합니다.

클라우디아가 가진 사과는 `(T + D) / 2`이고, 나탈리아가 가진 사과는 `T - 클라우디아`입니다.

입력 값은 최대 10¹⁰⁰까지 커질 수 있으므로, 일반적인 정수 타입으로는 표현할 수 없어 임의 정밀도 산술이 필요합니다.

<br>

## 접근법

C#의 경우 `System.Numerics.BigInteger` 타입을 사용하면 큰 수를 직접 처리할 수 있습니다. 입력을 `BigInteger`로 파싱한 후 덧셈, 뺄셈, 나눗셈 연산자를 사용하여 결과를 계산합니다.

C++의 경우 큰 수 연산을 문자열 기반으로 직접 구현해야 합니다.

덧셈은 낮은 자릿수부터 더하며 올림을 처리합니다.

뺄셈은 큰 수에서 작은 수를 빼되 받아내림을 처리합니다.

2로 나누기는 가장 높은 자릿수부터 순회하며 각 자리를 2로 나눈 몫을 결과에 추가하고 나머지는 다음 자리로 전달합니다.

<br>
모든 연산 후 앞자리 0을 제거하여 출력합니다.

<br>

## Code

### C#

```csharp
using System;
using System.Numerics;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var total = BigInteger.Parse(Console.ReadLine()!);
      var diff = BigInteger.Parse(Console.ReadLine()!);

      var klaudia = (total + diff) / 2;
      var natalia = total - klaudia;

      Console.WriteLine(klaudia);
      Console.WriteLine(natalia);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

string trimZero(const string& s) {
  size_t pos = s.find_first_not_of('0');
  return (pos == string::npos) ? "0" : s.substr(pos);
}

string addStrings(const string& a, const string& b) {
  string result;
  int carry = 0, i = a.size() - 1, j = b.size() - 1;

  while (i >= 0 || j >= 0 || carry) {
    int sum = carry;
    if (i >= 0) sum += a[i--] - '0';
    if (j >= 0) sum += b[j--] - '0';
    result.push_back(char('0' + (sum % 10)));
    carry = sum / 10;
  }

  reverse(result.begin(), result.end());
  return trimZero(result);
}

string subStrings(const string& a, const string& b) {
  string result;
  int borrow = 0, i = a.size() - 1, j = b.size() - 1;

  while (i >= 0) {
    int diff = (a[i] - '0') - borrow;
    if (j >= 0) diff -= (b[j] - '0');

    if (diff < 0)
      diff += 10, borrow = 1;
    else
      borrow = 0;

    result.push_back(char('0' + diff));
    --i; --j;
  }

  while (result.size() > 1 && result.back() == '0') result.pop_back();

  reverse(result.begin(), result.end());
  return trimZero(result);
}

string divByTwo(const string& s) {
  string result;
  int carry = 0;

  for (char ch : s) {
    int cur = carry * 10 + (ch - '0');
    result.push_back(char('0' + cur / 2));
    carry = cur % 2;
  }

  return trimZero(result);
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string total, diff; cin >> total >> diff;

  string sumTD = addStrings(total, diff);
  string klaudia = divByTwo(sumTD);
  string natalia = subStrings(total, klaudia);

  cout << klaudia << "\n" << natalia << "\n";

  return 0;
}
```
