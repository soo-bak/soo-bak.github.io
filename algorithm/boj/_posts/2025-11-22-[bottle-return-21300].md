## 문제 링크
[21300번 - Bottle Return](https://www.acmicpc.net/problem/21300)

## 설명

뉴욕주에서는 6종류의 음료 용기(병·캔)마다 5센트의 보증금을 환불해 줍니다.

6개 품목의 반환 개수가 입력으로 주어질 때, 총 환불 금액을 센트 단위로 계산하는 문제입니다.

<br>

## 접근법

6개의 정수를 입력받아 각각에 5를 곱한 후 합산하면 됩니다.

모든 품목의 환불 가격이 동일하므로 단순히 전체 개수 합계에 5를 곱해도 결과는 같습니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var tokens = Console.ReadLine()!.Split();
      var refund = 0;

      foreach (var token in tokens)
        refund += int.Parse(token) * 5;

      Console.WriteLine(refund);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int refund = 0;
  for (int i = 0; i < 6; ++i) {
    int count; cin >> count;
    refund += count * 5;
  }

  cout << refund << "\n";

  return 0;
}
```
