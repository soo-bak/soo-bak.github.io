## 문제 링크
[15721번 - 번데기](https://www.acmicpc.net/problem/15721)

## 설명

A명이 원으로 앉아 게임을 진행하는 상황에서, 라운드별로 패턴이 달라지는 구호를 외치며 T번째로 특정 구호를 말하는 사람을 찾는 문제입니다.

각 라운드는 고정 부분 "뻔 데기 뻔 데기"로 시작하고, 그 다음 "뻔"을 라운드 번호 + 1번, "데기"를 라운드 번호 + 1번 반복합니다.

예를 들어 1라운드는 "뻔 데기 뻔 데기 뻔 뻔 데기 데기", 2라운드는 "뻔 데기 뻔 데기 뻔 뻔 뻔 데기 데기 데기" 형태입니다.

각 구호를 외칠 때마다 차례가 한 사람씩 넘어가며, T는 10,000 이하입니다.

<br>

## 접근법

라운드를 1번부터 시작하여 각 라운드의 패턴을 순서대로 생성합니다.

매 라운드마다 고정 패턴 "뻔 데기 뻔 데기" 4개를 먼저 처리하고, 이어서 "뻔"을 라운드 번호 + 1번, "데기"를 라운드 번호 + 1번 반복합니다.

<br>
각 구호를 외칠 때마다 차례를 다음 사람으로 넘깁니다.

원형 배치이므로 마지막 사람 다음은 첫 번째 사람이 됩니다.

찾으려는 구호가 나올 때마다 카운트를 증가시키고, 카운트가 T에 도달하면 그때의 사람 번호를 출력합니다.

<br>
라운드가 진행될수록 각 라운드의 길이가 늘어나지만, T가 10,000 이하이므로 시뮬레이션 방식으로 충분히 처리할 수 있습니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var a = int.Parse(Console.ReadLine()!);
      var t = int.Parse(Console.ReadLine()!);
      var target = int.Parse(Console.ReadLine()!);

      var idx = 0;
      var count = 0;
      var round = 1;

      while (true) {
        int[] fixedPattern = {0, 1, 0, 1};

        foreach (var chant in fixedPattern) {
          if (chant == target) count++;
          if (count == t) {
            Console.WriteLine(idx);
            return;
          }
          idx = (idx + 1) % a;
        }

        for (var i = 0; i < round + 1; i++) {
          if (0 == target) count++;
          if (count == t) {
            Console.WriteLine(idx);
            return;
          }
          idx = (idx + 1) % a;
        }

        for (var i = 0; i < round + 1; i++) {
          if (1 == target) count++;
          if (count == t) {
            Console.WriteLine(idx);
            return;
          }
          idx = (idx + 1) % a;
        }

        round++;
      }
    }
  }
}
```

### C++

#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, t, target; cin >> a >> t >> target;
  int idx = 0, cnt = 0, round = 1;

  while (true) {
    int fixedPattern[4] = {0, 1, 0, 1};

    for (int chant : fixedPattern) {
      if (chant == target) ++cnt;
      if (cnt == t) {
        cout << idx << "\n";
        return 0;
      }
      idx = (idx + 1) % a;
    }

    for (int i = 0; i < round + 1; i++) {
      if (0 == target) ++cnt;
      if (cnt == t) {
        cout << idx << "\n";
        return 0;
      }
      idx = (idx + 1) % a;
    }

    for (int i = 0; i < round + 1; i++) {
      if (1 == target) ++cnt;
      if (cnt == t) {
        cout << idx << "\n";
        return 0;
      }
      idx = (idx + 1) % a;
    }

    ++round;
  }

  return 0;
}
