## 문제 링크
[7891번 - Can you add this?](https://www.acmicpc.net/problem/7891)

## 설명

여러 테스트 케이스가 주어지고, 각 케이스마다 두 정수가 입력됩니다.

각 케이스의 두 정수 합을 순서대로 출력하는 문제입니다.

<br>

## 접근법

첫 줄에서 테스트 케이스 개수를 입력받은 후, 그 횟수만큼 반복하며 두 정수를 읽고 합을 출력합니다.

<br>
입력 범위가 `±10⁹`이므로 32비트 정수로 처리할 수 있습니다.

합 또한 오버플로우가 발생하지 않으므로 일반적인 정수 타입으로 충분합니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var t = int.Parse(Console.ReadLine()!);

      for (int i = 0; i < t; i++) {
        var tokens = Console.ReadLine()!.Split();
        var x = long.Parse(tokens[0]);
        var y = long.Parse(tokens[1]);

        Console.WriteLine(x + y);
      }
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;

  while (t--) {
    ll x, y; cin >> x >> y;
    cout << x + y << "\n";
  }

  return 0;
}
```
