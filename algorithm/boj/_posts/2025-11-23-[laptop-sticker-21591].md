## 문제 링크
[21591번 - Laptop Sticker](https://www.acmicpc.net/problem/21591)

## 설명

노트북 본체 크기 `(wc, hc)`와 스티커 크기 `(ws, hs)`가 주어집니다.

스티커를 회전 없이 붙이되, 모든 변에서 1cm 여유를 두어야 한다면 스티커가 붙을 수 있는지 판정하여 맞으면 `1`, 아니면 `0`을 출력합니다.

<br>

## 접근법

사방 1cm씩 비워야 하므로 가로 기준으로 `ws <= wc - 2`, 세로 기준으로 `hs <= hc - 2`를 동시에 만족해야만 스티커를 붙일 수 있습니다.

입력 범위가 작으므로 단순 비교 후 조건이 모두 참이면 `1`, 하나라도 거짓이면 `0`을 출력하면 됩니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var tokens = Console.ReadLine()!.Split();
      int wc = int.Parse(tokens[0]);
      int hc = int.Parse(tokens[1]);
      int ws = int.Parse(tokens[2]);
      int hs = int.Parse(tokens[3]);

      bool fit = (ws <= wc - 2) && (hs <= hc - 2);
      Console.WriteLine(fit ? 1 : 0);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int wc, hc, ws, hs; cin >> wc >> hc >> ws >> hs;
  bool fit = (ws <= wc - 2) && (hs <= hc - 2);
  cout << (fit ? 1 : 0) << "\n";

  return 0;
}
```
