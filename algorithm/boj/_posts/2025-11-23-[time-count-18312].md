## 문제 링크
[18312번 - 시각](https://www.acmicpc.net/problem/18312)

## 설명

0시 0분 0초부터 N시 59분 59초까지의 모든 시각에서 특정 숫자가 시·분·초 표기에 하나라도 포함되는 경우의 수를 구하는 문제입니다.

시각은 항상 두 자리로 표현되며, N은 0 이상 23 이하, 찾을 숫자는 0 이상 9 이하입니다.

<br>

## 접근법

0시부터 주어진 시까지, 0분부터 59분까지, 0초부터 59초까지 모든 시각을 삼중 반복으로 순회합니다.

각 시각마다 시·분·초의 십의 자리와 일의 자리를 나눗셈과 나머지 연산으로 추출하여 찾을 숫자와 비교합니다.

<br>
하나라도 일치하면 카운트를 증가시킵니다.

총 경우의 수는 최대 24 × 60 × 60 = 86,400개로 충분히 작아 전체 순회로 처리합니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static bool HasDigit(int h, int m, int s, int k) {
      return h / 10 == k || h % 10 == k ||
             m / 10 == k || m % 10 == k ||
             s / 10 == k || s % 10 == k;
    }

    static void Main(string[] args) {
      var tokens = Console.ReadLine()!.Split();
      var n = int.Parse(tokens[0]);
      var k = int.Parse(tokens[1]);

      var count = 0;

      for (var h = 0; h <= n; h++) {
        for (var m = 0; m < 60; m++) {
          for (var s = 0; s < 60; s++) {
            if (HasDigit(h, m, s, k)) count++;
          }
        }
      }

      Console.WriteLine(count);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

bool hasDigit(int h, int m, int s, int k) {
  return h / 10 == k || h % 10 == k ||
         m / 10 == k || m % 10 == k ||
         s / 10 == k || s % 10 == k;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, k; cin >> n >> k;
  int count = 0;

  for (int h = 0; h <= n; h++)
    for (int m = 0; m < 60; m++)
      for (int s = 0; s < 60; s++)
        if (hasDigit(h, m, s, k)) count++;

  cout << count << "\n";

  return 0;
}
```
