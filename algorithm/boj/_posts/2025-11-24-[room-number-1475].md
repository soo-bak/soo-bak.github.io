## 문제 링크
[1475번 - 방 번호](https://www.acmicpc.net/problem/1475)

## 설명

방 번호가 주어질 때, 0부터 9까지의 숫자가 각각 하나씩 들어있는 세트를 최소 몇 개 구매해야 방 번호를 만들 수 있는지 구하는 문제입니다.

6과 9는 뒤집어서 서로 대신 사용할 수 있습니다.

예를 들어 방 번호가 9999인 경우, 6 네 개를 뒤집어 사용하거나 9 두 개와 6 두 개를 적절히 섞어 사용할 수 있습니다.

<br>

## 접근법

방 번호의 각 자리 숫자가 몇 번씩 등장하는지 세어 배열에 저장합니다.

<br>
6과 9는 서로 대신 사용할 수 있으므로, 두 숫자의 등장 횟수를 합친 후 2로 나눈 값(올림)이 실제로 필요한 세트 개수가 됩니다.

예를 들어 6이 3개, 9가 2개 필요하면 총 5개를 두 세트에서 나눠 쓸 수 있으므로 3세트가 필요합니다.

<br>
나머지 숫자들은 각 숫자의 등장 횟수만큼 세트가 필요합니다.

예를 들어 1이 4번 등장하면 최소 4세트가 필요합니다.

<br>
모든 숫자를 확인했을 때 가장 많이 필요한 세트 개수가 전체 답이 됩니다.

각 숫자별로 필요한 세트 개수 중 최댓값을 구하여 출력합니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var roomNumber = Console.ReadLine()!;
      var cnt = new int[10];

      foreach (var digit in roomNumber) cnt[digit - '0']++;

      var reversibleTotal = cnt[6] + cnt[9];
      var reversibleRequired = (reversibleTotal + 1) / 2;

      cnt[6] = cnt[9] = reversibleRequired;

      var maxSets = 0;

      for (var i = 0; i < 10; i++) {
        if (cnt[i] > maxSets) maxSets = cnt[i];
      }

      Console.WriteLine(maxSets);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string roomNumber; cin >> roomNumber;
  int cnt[10] = {0,};

  for (char digit : roomNumber) cnt[digit - '0']++;

  int reversibleTotal = cnt[6] + cnt[9];
  int reversibleRequired = (reversibleTotal + 1) / 2;

  cnt[6] = cnt[9] = reversibleRequired;

  int maxSets = 0;

  for (int i = 0; i < 10; i++)
    maxSets = max(maxSets, cnt[i]);

  cout << maxSets << "\n";

  return 0;
}
```
