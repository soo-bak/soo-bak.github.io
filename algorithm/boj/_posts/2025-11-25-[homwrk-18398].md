## 문제 링크
[18398번 - HOMWRK](https://www.acmicpc.net/problem/18398)

## 설명

여러 개의 테스트 케이스가 주어지고, 각 테스트 케이스마다 여러 쌍의 두 수가 주어집니다.

테스트 케이스 개수와 각 테스트 케이스의 숫자 쌍 개수, 그리고 숫자 쌍들이 주어질 때, 각 숫자 쌍의 합과 곱을 출력하는 문제입니다.

합과 곱은 공백으로 구분하여 한 줄에 출력합니다.

<br>

## 접근법

테스트 케이스 개수를 먼저 읽고, 각 테스트 케이스에 대해 숫자 쌍 개수를 읽은 후 해당 개수만큼 숫자 쌍을 처리합니다.

각 숫자 쌍을 읽으면 즉시 합과 곱을 계산하여 출력합니다.

예를 들어, 3과 4가 주어지면 합은 7, 곱은 12이므로 "7 12"를 출력합니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var t = int.Parse(Console.ReadLine()!);

      for (var i = 0; i < t; i++) {
        var n = int.Parse(Console.ReadLine()!);

        for (var j = 0; j < n; j++) {
          var parts = Console.ReadLine()!.Split();
          var a = int.Parse(parts[0]);
          var b = int.Parse(parts[1]);

          Console.WriteLine($"{a + b} {a * b}");
        }
      }
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;

  while (t--) {
    int n; cin >> n;

    for (int i = 0; i < n; i++) {
      ll a, b; cin >> a >> b;

      cout << a + b << " " << a * b << "\n";
    }
  }

  return 0;
}
```
