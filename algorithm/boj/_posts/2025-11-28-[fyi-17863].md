## 문제 링크
[17863번 - FYI](https://www.acmicpc.net/problem/17863)

## 설명

7자리 전화번호 (1000000 ≤ N ≤ 9999999)가 주어지는 상황에서, 앞 세 자리가 555이면 YES를, 아니면 NO를 출력하는 문제입니다.

<br>

## 접근법

전화번호의 앞 세 자리가 555인지 확인하면 됩니다.

입력을 문자열로 받아 처음 세 문자를 확인하는 방법을 사용합니다.

문자열의 앞 세 자리가 "555"와 일치하는지 비교하여 결과를 출력합니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var s = Console.ReadLine()!;

      if (s.StartsWith("555")) Console.WriteLine("YES");
      else Console.WriteLine("NO");
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;

  if (s.substr(0, 3) == "555") cout << "YES\n";
  else cout << "NO\n";

  return 0;
}
```
