## 문제 링크
[24262번 - 알고리즘의 수행 시간 1](https://www.acmicpc.net/problem/24262)

## 설명

특정 알고리즘의 수행 시간이 주어지는 상황에서, 입력 크기 n (1 ≤ n ≤ 500,000)이 주어질 때, 알고리즘의 수행 횟수와 시간 복잡도의 최고차항 차수를 구하는 문제입니다.

제시된 MenOfPassion 알고리즘은 배열의 중간 인덱스 값을 한 번 반환하는 단순한 구조로, 입력 크기와 무관하게 항상 1회 실행됩니다.

<br>

## 접근법

알고리즘이 입력 크기 n에 관계없이 항상 동일한 연산을 수행하므로, 수행 횟수는 항상 1입니다.

입력값 n을 받되, 계산 없이 항상 1과 0을 출력하면 됩니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      Console.ReadLine();
      Console.WriteLine(1);
      Console.WriteLine(0);
    }
  }
}
```

```cpp
#include <bits/stdc++.h>

typedef long long ll;
```
