## 문제 링크
[15963번 - CASIO](https://www.acmicpc.net/problem/15963)

## 설명

송찬이가 계산기에 필요한 배터리 번호 N과 선생님이 가져온 배터리 번호 M (1 ≤ N, M ≤ 10¹⁰)이 주어지는 상황에서, 두 배터리가 같은지 판별하는 문제입니다.

두 번호가 같으면 1을, 다르면 0을 출력합니다.

<br>

## 접근법

두 정수를 입력받아 같은지 비교하면 됩니다.

배터리 번호는 최대 10¹⁰으로 10자리 정수이므로, 64비트 정수형으로 충분히 표현할 수 있습니다.

두 값이 동일한지 비교한 후, 같으면 1을, 다르면 0을 출력합니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var input = Console.ReadLine()!.Split();
      var n = long.Parse(input[0]);
      var m = long.Parse(input[1]);
      Console.WriteLine(n == m ? 1 : 0);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll n, m; cin >> n >> m;
  cout << (n == m ? 1 : 0) << '\n';
  return 0;
}
```
