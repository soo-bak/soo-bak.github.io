## 문제 링크
[14581번 - 팬들에게 둘러싸인 홍준](https://www.acmicpc.net/problem/14581)

## 설명

홍준의 아이디가 주어지는 상황에서, 아이디 문자열(최대 20글자)이 주어질 때, 홍준을 팬들이 둘러싼 모습을 3x3 형태로 출력하는 문제입니다.

가운데 위치에는 `:아이디:` 형식으로 홍준의 아이디를 배치하고, 나머지 8칸은 모두 `:fan:` 이모티콘으로 채웁니다.

<br>

## 접근법

첫 번째 줄과 세 번째 줄에는 `:fan:`을 세 번 출력합니다.

두 번째 줄에는 `:fan:`과 `:아이디:` 형식의 홍준 아이디, 그리고 다시 `:fan:`을 순서대로 출력합니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var id = Console.ReadLine()!;
      var fan = ":fan:";

      Console.WriteLine(fan + fan + fan);
      Console.WriteLine(fan + ":" + id + ":" + fan);
      Console.WriteLine(fan + fan + fan);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string id; cin >> id;
  string fan = ":fan:";

  cout << fan << fan << fan << "\n";
  cout << fan << ":" << id << ":" << fan << "\n";
  cout << fan << fan << fan << "\n";

  return 0;
}
```
