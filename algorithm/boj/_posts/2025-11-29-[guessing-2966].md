## 문제 링크
[2966번 - 찍기](https://www.acmicpc.net/problem/2966)

## 설명

객관식 문제의 정답과 세 사람의 찍기 패턴이 주어지는 상황에서, 문제 개수 N (1 ≤ N ≤ 100)과 정답 문자열(A, B, C로만 구성)이 주어질 때, 가장 많은 문제를 맞춘 사람과 그 개수를 구하는 문제입니다.

세 사람의 찍기 패턴은 다음과 같이 반복됩니다:
- Adrian: `ABC` 반복 (3글자 주기)
- Bruno: `BABC` 반복 (4글자 주기)
- Goran: `CCAABB` 반복 (6글자 주기)

최고 점수를 출력한 후, 해당 점수를 받은 사람의 이름을 Adrian, Bruno, Goran 순서로 출력합니다. 동점자가 여러 명이면 모두 출력합니다.

<br>

## 접근법

각 사람의 패턴을 정답 문자열과 비교하여 점수를 계산합니다.

<br>
정답 문자열을 순회하며 각 위치의 문자를 세 사람의 패턴과 비교합니다. 패턴은 계속 반복되므로 나머지 연산을 사용하여 현재 위치에 해당하는 패턴 문자를 찾습니다. 일치하면 해당 사람의 점수를 1 증가시킵니다.

모든 비교가 끝난 후 세 사람 중 최고 점수를 구하고, 이 점수와 같은 점수를 받은 사람의 이름을 순서대로 출력합니다.

<br>
각 문제를 한 번씩만 확인하므로 시간 복잡도는 O(N)입니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var n = int.Parse(Console.ReadLine()!);
      var answer = Console.ReadLine()!;

      var adrianPattern = "ABC";
      var brunoPattern = "BABC";
      var goranPattern = "CCAABB";

      var adrianScore = 0;
      var brunoScore = 0;
      var goranScore = 0;
      
      for (var i = 0; i < n; i++) {
        if (answer[i] == adrianPattern[i % adrianPattern.Length])
          adrianScore++;
        if (answer[i] == brunoPattern[i % brunoPattern.Length])
          brunoScore++;
        if (answer[i] == goranPattern[i % goranPattern.Length])
          goranScore++;
      }

      var maxScore = Math.Max(adrianScore, Math.Max(brunoScore, goranScore));
      Console.WriteLine(maxScore);
      if (maxScore == adrianScore) Console.WriteLine("Adrian");
      if (maxScore == brunoScore) Console.WriteLine("Bruno");
      if (maxScore == goranScore) Console.WriteLine("Goran");
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string answer; cin >> answer;

  string adrianPattern = "ABC";
  string brunoPattern = "BABC";
  string goranPattern = "CCAABB";

  int adrianScore = 0;
  int brunoScore = 0;
  int goranScore = 0;
  
  for (int i = 0; i < n; i++) {
    if (answer[i] == adrianPattern[i % adrianPattern.size()])
      adrianScore++;
    if (answer[i] == brunoPattern[i % brunoPattern.size()])
      brunoScore++;
    if (answer[i] == goranPattern[i % goranPattern.size()])
      goranScore++;
  }

  int maxScore = max({adrianScore, brunoScore, goranScore});
  cout << maxScore << "\n";
  if (maxScore == adrianScore) cout << "Adrian\n";
  if (maxScore == brunoScore) cout << "Bruno\n";
  if (maxScore == goranScore) cout << "Goran\n";
  
  return 0;
}
```
