## 문제 링크
[10815번 - 숫자 카드](https://www.acmicpc.net/problem/10815)

## 설명

상근이가 가진 숫자 카드들과 확인할 숫자들이 주어지는 상황에서, N (1 ≤ N ≤ 500,000)장의 카드에 적힌 정수와 M (1 ≤ M ≤ 500,000)개의 정수가 주어질 때, 각 정수가 카드에 존재하면 1을, 존재하지 않으면 0을 출력하는 문제입니다.

숫자는 -10,000,000 이상 10,000,000 이하이며, 상근이가 가진 카드에 같은 수가 적힌 것은 없습니다.

<br>

## 접근법

N개의 모든 카드에 대해 M개의 질의를 직접 비교하면 O(N×M) 시간이 걸려 최악의 경우 시간 초과가 발생합니다.

효율적으로 해결하기 위해 카드 목록을 먼저 정렬한 후, 각 질의에 대해 이진 탐색을 사용합니다.

이진 탐색은 정렬된 배열에서 특정 값의 존재 여부를 O(log N) 시간에 확인할 수 있으므로, 전체 시간 복잡도는 정렬 O(N log N) + 질의 처리 O(M log N) = O((N + M) log N)이 됩니다.

<br>

## Code

### C#

```csharp
using System;
using System.Text;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var n = int.Parse(Console.ReadLine()!);
      var cards = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
      Array.Sort(cards);

      var m = int.Parse(Console.ReadLine()!);
      var queries = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
      var sb = new StringBuilder();

      foreach (var num in queries) {
        var exists = Array.BinarySearch(cards, num) >= 0;
        sb.Append(exists ? '1' : '0').Append(' ');
      }

      Console.WriteLine(sb.ToString().TrimEnd());
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi cards(n);
  for (int i = 0; i < n; i++)
    cin >> cards[i];
  sort(cards.begin(), cards.end());

  int m; cin >> m;
  ostringstream out;
  for (int i = 0; i < m; i++) {
    int num; cin >> num;
    bool exists = binary_search(cards.begin(), cards.end(), num);
    out << (exists ? 1 : 0) << (i + 1 == m ? '\n' : ' ');
  }
  cout << out.str();
  
  return 0;
}
```
