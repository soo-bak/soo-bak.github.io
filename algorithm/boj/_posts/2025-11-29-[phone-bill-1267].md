## 문제 링크
[1267번 - 핸드폰 요금](https://www.acmicpc.net/problem/1267)

## 설명

여러 통화의 시간 정보가 주어지는 상황에서, N (1 ≤ N ≤ 20)개의 통화 시간(초)이 주어질 때, 영식 요금제와 민식 요금제 중 더 저렴한 요금제를 출력하는 문제입니다.

두 요금제의 규칙은 다음과 같습니다:
- 영식 요금제(Y): 30초마다 10원 (0~29초는 10원, 30~59초는 20원, ...)
- 민식 요금제(M): 60초마다 15원 (0~59초는 15원, 60~119초는 30원, ...)

두 요금제의 비용이 같으면 "Y M"과 요금을 함께 출력합니다.

<br>

## 접근법

각 통화에 대해 두 요금제의 비용을 각각 계산합니다.

영식 요금제는 30초 단위로 올림하여 10원씩 부과되고, 민식 요금제는 60초 단위로 올림하여 15원씩 부과됩니다.

통화 시간을 각 단위로 나눈 몫에 1을 더하면 필요한 단위 수가 됩니다.

모든 통화에 대해 요금을 합산한 후, 두 요금제의 총액을 비교하여 더 저렴한 쪽을 출력합니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var n = int.Parse(Console.ReadLine()!);
      var times = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

      var youngshik = 0;
      var minshik = 0;
      
      for (var i = 0; i < n; i++) {
        var time = times[i];
        youngshik += (time / 30 + 1) * 10;
        minshik += (time / 60 + 1) * 15;
      }

      if (youngshik < minshik) Console.WriteLine($"Y {youngshik}");
      else if (youngshik > minshik) Console.WriteLine($"M {minshik}");
      else Console.WriteLine($"Y M {youngshik}");
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi times(n);
  for (int i = 0; i < n; i++)
    cin >> times[i];

  int youngshik = 0, minshik = 0;
  for (int time : times) {
    youngshik += (time / 30 + 1) * 10;
    minshik += (time / 60 + 1) * 15;
  }

  if (youngshik < minshik) cout << "Y " << youngshik << "\n";
  else if (youngshik > minshik) cout << "M " << minshik << "\n";
  else cout << "Y M " << youngshik << "\n";
  
  return 0;
}
```
