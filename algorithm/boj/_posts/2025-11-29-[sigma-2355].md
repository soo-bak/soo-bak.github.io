## 문제 링크
[2355번 - 시그마](https://www.acmicpc.net/problem/2355)

## 설명

두 정수가 주어지는 상황에서, A와 B (-10^9 ≤ A, B ≤ 10^9)가 주어질 때, A부터 B까지의 모든 정수의 합을 구하는 문제입니다.

두 수의 순서는 정해져 있지 않으며, A가 B보다 클 수도 있습니다.

<br>

## 접근법

두 수가 어떤 순서로 주어지든, 먼저 작은 수와 큰 수를 구분해야 합니다.

1부터 n까지의 합은 n × (n + 1) / 2 공식으로 계산할 수 있습니다.

A부터 B까지의 합은 1부터 B까지의 합에서 1부터 (A - 1)까지의 합을 빼면 됩니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var input = Array.ConvertAll(Console.ReadLine()!.Split(), long.Parse);
      long a = input[0], b = input[1];
      if (a > b) (a, b) = (b, a);

      long sumB = b * (b + 1) / 2;
      long sumA = (a - 1) * a / 2;
      Console.WriteLine(sumB - sumA);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll a, b; cin >> a >> b;
  if (a > b) swap(a, b);

  ll sumB = b * (b + 1) / 2;
  ll sumA = (a - 1) * a / 2;
  cout << sumB - sumA << "\n";
  
  return 0;
}
```
