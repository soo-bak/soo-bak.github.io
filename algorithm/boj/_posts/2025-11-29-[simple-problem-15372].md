## 문제 링크
[15372번 - A Simple Problem.](https://www.acmicpc.net/problem/15372)

## 설명

테스트 케이스 개수 T와 각 테스트 케이스마다 양의 정수 N (1 ≤ N ≤ 200,000)이 주어지는 상황에서, N²의 배수 중 가장 작은 양의 정수를 구하는 문제입니다.

<br>

## 접근법

어떤 수 M의 배수는 M × k (k = 1, 2, 3, ...)로 표현할 수 있습니다.

N²의 배수는 N² × 1, N² × 2, N² × 3, ... 형태입니다.

<br>
이 중 가장 작은 양의 정수는 N² × 1 = N²입니다.

따라서 각 테스트 케이스마다 N을 입력받아 N²을 계산하여 출력합니다.

<br>

## Code

### C#

```csharp
using System;
using System.Text;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var t = int.Parse(Console.ReadLine()!);
      var sb = new StringBuilder();

      for (var i = 0; i < t; i++) {
        var n = long.Parse(Console.ReadLine()!);
        sb.Append(n * n).Append('\n');
      }

      Console.Write(sb.ToString());
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;

  while (t--) {
    ll n; cin >> n;
    cout << n * n << "\n";
  }

  return 0;
}
```
