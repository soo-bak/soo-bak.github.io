## 문제 링크
[20361번 - 일우는 야바위꾼](https://www.acmicpc.net/problem/20361)

## 설명

야바위 게임에서 공의 위치를 추적하는 상황에서, 컵의 개수 N (1 ≤ N ≤ 1,000,000), 처음 공이 들어있는 컵의 번호 X, 그리고 교환 횟수 K (1 ≤ K ≤ 1,000,000)가 주어지고, 각 교환마다 두 컵의 번호가 주어질 때, 모든 교환이 끝난 후 공이 들어있는 컵의 번호를 구하는 문제입니다.

두 컵을 교환할 때 공이 들어있는 컵도 함께 이동합니다.

<br>

## 접근법

공의 현재 위치를 변수로 유지하며 교환을 추적합니다.

각 교환마다 두 컵의 번호를 입력받아, 공이 현재 그 중 하나에 있다면 공의 위치를 다른 쪽으로 갱신합니다. 공이 교환되는 두 컵 중 어느 것에도 없다면 공의 위치는 변하지 않습니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var input = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
      var n = input[0];
      var ballPos = input[1];
      var k = input[2];

      for (var i = 0; i < k; i++) {
        var swap = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
        var cup1 = swap[0];
        var cup2 = swap[1];
        
        if (ballPos == cup1) ballPos = cup2;
        else if (ballPos == cup2) ballPos = cup1;
      }

      Console.WriteLine(ballPos);
    }
  }
}
```

### C++

  for (int i = 0; i < k; i++) {
    int cup1, cup2; cin >> cup1 >> cup2;
    
    if (ballPos == cup1) ballPos = cup2;
    else if (ballPos == cup2) ballPos = cup1;
  }

  cout << ballPos << "\n";

  return 0;
}
