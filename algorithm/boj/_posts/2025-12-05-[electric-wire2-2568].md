## 문제 링크
[2568번 - 전깃줄 - 2](https://www.acmicpc.net/problem/2568)

## 설명

전깃줄 (A, B)가 주어질 때, 교차하는 전선이 없도록 제거해야 할 최소 전선의 개수와 그 A 좌표들을 출력하는 문제입니다.

전선을 A 좌표 기준으로 정렬하면, 교차 없이 남길 수 있는 최대 전선 집합은 B 좌표의 가장 긴 증가하는 부분 수열(LIS)에 해당합니다. LIS에 포함되지 않는 전선들이 제거 대상이 됩니다.

<br>

## 접근법

전선을 A 좌표 기준으로 정렬하면, 두 전선이 교차하는 조건은 B 좌표가 역전되는 경우입니다. 따라서 교차 없이 남길 수 있는 최대 전선은 B 좌표가 증가하는 순서를 유지하는 것, 즉 LIS가 됩니다.

<br>
먼저 전선들을 A 좌표 기준으로 정렬합니다.

다음으로 B 좌표들에 대해 LIS를 구하면서, 각 전선이 어떤 전선을 이어받는지 기록합니다. 이렇게 하면 나중에 LIS에 포함된 전선들을 역추적할 수 있습니다.

이후 LIS에 포함되지 않은 전선들의 A 좌표를 오름차순으로 출력합니다.

<br>
시간 복잡도는 O(N log N)입니다.

<br>

## Code

### C#

```csharp
using System;
using System.Collections.Generic;
using System.IO;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var sr = new StreamReader(Console.OpenStandardInput());
      var sw = new StreamWriter(Console.OpenStandardOutput());

      var n = int.Parse(sr.ReadLine()!);
      var wires = new (int a, int b)[n];
      for (var i = 0; i < n; i++) {
        var parts = Array.ConvertAll(sr.ReadLine()!.Split(), int.Parse);
        wires[i] = (parts[0], parts[1]);
      }
      Array.Sort(wires, (x, y) => x.a.CompareTo(y.a));

      var lisVal = new int[n];
      var lisPos = new int[n];
      var prev = new int[n];
      var len = 0;

      for (var i = 0; i < n; i++) {
        var x = wires[i].b;
        var idx = Array.BinarySearch(lisVal, 0, len, x);
        if (idx < 0)
          idx = ~idx;

        prev[i] = (idx == 0) ? -1 : lisPos[idx - 1];
        lisVal[idx] = x;
        lisPos[idx] = i;
        if (idx == len)
          len++;
      }

      sw.WriteLine(n - len);

      var keepA = new HashSet<int>();
      for (var cur = lisPos[len - 1]; cur != -1; cur = prev[cur])
        keepA.Add(wires[cur].a);

      var toRemove = new List<int>();
      for (var i = 0; i < n; i++) {
        if (!keepA.Contains(wires[i].a))
          toRemove.Add(wires[i].a);
      }
      toRemove.Sort();
      foreach (var a in toRemove)
        sw.WriteLine(a);

      sw.Flush();
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;
typedef pair<int, int> pii;
typedef vector<pii> vp;
typedef vector<bool> vb;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vp w(n);
  for (int i = 0; i < n; i++)
    cin >> w[i].first >> w[i].second;
  sort(w.begin(), w.end());

  vi lisVal(n), lisPos(n), prev(n);
  int len = 0;

  for (int i = 0; i < n; i++) {
    int x = w[i].second;
    auto it = lower_bound(lisVal.begin(), lisVal.begin() + len, x);
    int idx = it - lisVal.begin();

    prev[i] = (idx == 0) ? -1 : lisPos[idx - 1];
    lisVal[idx] = x;
    lisPos[idx] = i;
    if (idx == len)
      len++;
  }

  cout << n - len << "\n";

  vb keep(n, false);
  for (int cur = lisPos[len - 1]; cur != -1; cur = prev[cur])
    keep[cur] = true;

  for (int i = 0; i < n; i++) {
    if (!keep[i])
      cout << w[i].first << "\n";
  }

  return 0;
}
```
