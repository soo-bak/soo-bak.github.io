## 문제 링크
[10372번 - Alarm Clock](https://www.acmicpc.net/problem/10372)

## 설명
디지털 시계의 네 자리가 점등한 7-세그먼트 LED 수의 합이 n과 같을 때의 시각을 찾는 문제입니다.

가능한 시각이 여러 개면 아무거나 출력하고, 없으면 Impossible을 출력합니다.

<br>

## 접근법
먼저, 각 숫자 0부터 9까지의 점등 수를 배열로 준비합니다. 0은 6개, 1은 2개, 2는 5개 순입니다.

다음으로, 모든 시각을 완전 탐색합니다. 시는 0부터 23까지, 분은 0부터 59까지입니다. 네 자리의 점등 수 합이 n과 같으면 해당 시각을 출력하고 종료합니다.

끝까지 찾지 못하면 Impossible을 출력합니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var seg = new int[] {6, 2, 5, 5, 4, 5, 6, 3, 7, 6};
      var n = int.Parse(Console.ReadLine()!);
      for (var h = 0; h < 24; h++) {
        for (var m = 0; m < 60; m++) {
          var a = h / 10;
          var b = h % 10;
          var c = m / 10;
          var d = m % 10;
          var cnt = seg[a] + seg[b] + seg[c] + seg[d];
          if (cnt == n) {
            Console.WriteLine($"{a}{b}:{c}{d}");
            return;
          }
        }
      }
      Console.WriteLine("Impossible");
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int seg[10] = {6, 2, 5, 5, 4, 5, 6, 3, 7, 6};
  int n; cin >> n;
  for (int h = 0; h < 24; h++) {
    for (int m = 0; m < 60; m++) {
      int a = h / 10, b = h % 10, c = m / 10, d = m % 10;
      int cnt = seg[a] + seg[b] + seg[c] + seg[d];
      if (cnt == n) {
        cout << setw(2) << setfill('0') << h << ":"
             << setw(2) << setfill('0') << m << "\n";
        return 0;
      }
    }
  }
  cout << "Impossible\n";

  return 0;
}
```
