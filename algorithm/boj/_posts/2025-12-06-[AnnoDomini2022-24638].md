## 문제 링크
[24638번 - Anno Domini 2022](https://www.acmicpc.net/problem/24638)

## 설명
연도가 AD 또는 BC로 주어집니다. 연도 0이 없으므로 1 BC 바로 다음이 AD 1입니다.

두 연도 사이의 경과 연수를 구하는 문제입니다.

<br>

## 접근법
먼저, 연도를 하나의 정수 축으로 변환합니다. AD는 양수로, BC는 1을 빼서 음수로 변환합니다. 이렇게 하면 1 BC가 0이 되어 연도 0 부재를 보정할 수 있습니다.

다음으로, 이렇게 변환하면 두 시점의 차이는 절댓값으로 구할 수 있습니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static int ToVal(string line) {
      var p = line.Split(' ');
      if (p[0] == "AD")
        return int.Parse(p[1]);
      return -(int.Parse(p[0]) - 1);
    }

    static void Main(string[] args) {
      var y1 = ToVal(Console.ReadLine()!);
      var y2 = ToVal(Console.ReadLine()!);
      Console.WriteLine(Math.Abs(y1 - y2));
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int toVal(const string& a, const string& b) {
  if (a == "AD")
    return stoi(b);
  return -(stoi(a) - 1);
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s1a, s1b, s2a, s2b; cin >> s1a >> s1b >> s2a >> s2b;

  int y1 = toVal(s1a, s1b);
  int y2 = toVal(s2a, s2b);

  cout << abs(y1 - y2) << "\n";

  return 0;
}
```
