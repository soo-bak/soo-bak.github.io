## 문제 링크
[24577번 - Espresso!](https://www.acmicpc.net/problem/24577)

## 설명
주문은 1~4샷 에스프레소 또는 라떼입니다. 에스프레소는 샷 수만큼 물을 소비하고, 라떼는 스팀 우유 추가로 샷 수에 1을 더한 만큼 물을 소비합니다.

탱크 용량이 주어지고 시작 시 가득 차 있습니다. 다음 주문을 수행할 물이 부족하면 그 직전에 리필하고 주문을 처리합니다. 모든 주문 처리 후 리필 횟수를 구하는 문제입니다.

<br>

## 접근법
먼저, 현재 물을 탱크 용량으로 시작합니다.

다음으로, 각 주문을 읽어 필요한 물을 계산합니다. 라떼인 경우 1을 더합니다. 필요한 물이 현재 물보다 크면 리필 횟수를 증가시키고 탱크를 채운 뒤 주문을 처리합니다.

이후, 모든 주문을 처리한 뒤 리필 횟수를 출력합니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var first = Console.ReadLine()!.Split();
      var n = int.Parse(first[0]);
      var s = int.Parse(first[1]);

      var cur = s;
      var ans = 0;
      for (var i = 0; i < n; i++) {
        var order = Console.ReadLine()!;
        var need = order[0] - '0';
        if (order.Length > 1)
          need += 1;

        if (need > cur) {
          ans++;
          cur = s - need;
        } else cur -= need;
      }

      Console.WriteLine(ans);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, s; cin >> n >> s;
  int cur = s, ans = 0;
  for (int i = 0; i < n; i++) {
    string order; cin >> order;
    int need = order[0] - '0';
    if (order.size() > 1)
      need += 1;

    if (need > cur) {
      ans++;
      cur = s - need;
    } else cur -= need;
  }

  cout << ans << "\n";

  return 0;
}
```
