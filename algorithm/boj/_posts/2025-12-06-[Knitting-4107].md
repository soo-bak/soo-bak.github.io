## 문제 링크
[4107번 - Knitting](https://www.acmicpc.net/problem/4107)

## 설명
첫 행에 N코가 있고 총 M행을 뜹니다. 길이 K의 패턴 배열이 주어지며, 각 행마다 해당 패턴 값을 더해 다음 행의 코 수를 만듭니다.

패턴은 순환하며, 각 행의 코 수는 항상 양수임이 보장됩니다. 모든 행의 코 수 합을 구하는 문제입니다.

<br>

## 접근법
먼저, 현재 코 수를 N으로 시작합니다.

다음으로, 각 행마다 현재 코 수를 총합에 더합니다. 그리고 다음 행의 코 수는 패턴 배열에서 순환 인덱스로 가져온 값을 더해 갱신합니다.

이후, 모든 행을 처리한 뒤 총합을 출력합니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      while (true) {
        var first = Console.ReadLine()!.Split();
        var N = int.Parse(first[0]);
        var M = int.Parse(first[1]);
        var K = int.Parse(first[2]);
        if (N == 0 && M == 0 && K == 0)
          break;

        var line = Console.ReadLine()!.Split();
        var pat = new int[K];
        for (var i = 0; i < K; i++)
          pat[i] = int.Parse(line[i]);

        var st = N;
        var total = 0;
        for (var i = 0; i < M; i++) {
          total += st;
          st += pat[i % K];
        }
        Console.WriteLine(total);
      }
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  while (true) {
    int N, M, K; cin >> N >> M >> K;
    if (N == 0 && M == 0 && K == 0)
      break;

    vi pat(K);
    for (int i = 0; i < K; i++)
      cin >> pat[i];

    int st = N;
    int total = 0;
    for (int i = 0; i < M; i++) {
      total += st;
      st += pat[i % K];
    }
    cout << total << "\n";
  }

  return 0;
}
```
