## 문제 링크
[16408번 - Poker Hand](https://www.acmicpc.net/problem/16408)

## 설명
5장의 카드가 주어질 때, 같은 랭크가 몇 장까지 겹치는지 최댓값을 구합니다. 카드는 서로 다른 2글자 문자열로 주어지며, 첫 글자가 랭크입니다.

<br>

## 접근법
먼저, 각 카드의 첫 글자인 랭크를 확인합니다.

다음으로, 각 랭크의 빈도를 센 뒤 최댓값을 출력합니다.

<br>

---

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var freq = new int[128];
      var tokens = Console.ReadLine()!.Split();
      for (var i = 0; i < 5; i++) {
        var rank = tokens[i][0];
        freq[rank]++;
      }
      var ans = 0;
      for (var i = 0; i < freq.Length; i++) {
        if (freq[i] > ans)
          ans = freq[i];
      }
      Console.WriteLine(ans);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int freq[128] = {0};
  for (int i = 0; i < 5; i++) {
    string card; cin >> card;
    freq[card[0]]++;
  }

  int ans = 0;
  for (int i = 0; i < 128; i++)
    ans = max(ans, freq[i]);

  cout << ans << "\n";

  return 0;
}
```
