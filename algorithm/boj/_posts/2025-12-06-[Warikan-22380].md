## 문제 링크
[22380번 - 割り勘](https://www.acmicpc.net/problem/22380)

## 설명
총 비용을 인원수로 나눈 1인당 부담액이 있습니다. 어떤 사람의 보유액이 부족하면 가진 전액만 받고, 나머지는 추후에 받습니다.

오늘 당장 모을 수 있는 금액의 합을 구하는 문제입니다.

<br>

## 접근법
먼저, 1인당 부담액을 계산합니다. 문제에서 나누어떨어짐이 보장됩니다.

다음으로, 각 사람에 대해 보유액과 부담액 중 작은 값을 더합니다. 보유액이 충분하면 부담액 전액을, 부족하면 가진 전액만 받습니다.

이후, 모든 사람을 처리한 뒤 합계를 출력합니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      while (true) {
        var first = Console.ReadLine()!.Split();
        var n = int.Parse(first[0]);
        var m = int.Parse(first[1]);
        if (n == 0 && m == 0)
          break;

        var cost = m / n;
        var parts = Console.ReadLine()!.Split();
        var sum = 0;
        for (var i = 0; i < n; i++) {
          var a = int.Parse(parts[i]);
          sum += a >= cost ? cost : a;
        }
        Console.WriteLine(sum);
      }
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  while (true) {
    int n, m; cin >> n >> m;
    if (n == 0 && m == 0)
      break;

    int cost = m / n;
    int sum = 0;
    for (int i = 0; i < n; i++) {
      int a; cin >> a;
      sum += (a >= cost) ? cost : a;
    }
    cout << sum << "\n";
  }

  return 0;
}
```
