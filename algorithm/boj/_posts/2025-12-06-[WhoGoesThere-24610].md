## 문제 링크
[24610번 - Who Goes There?](https://www.acmicpc.net/problem/24610)

## 설명
대회 수용 인원, 학교 수, 그리고 각 학교가 등록한 팀 수가 주어집니다. 배정 규칙은 웨이브 방식으로, 각 라운드마다 모든 학교의 1팀씩 배정합니다.

어떤 라운드에서든 수용 인원이 먼저 소진되면 중단합니다. 모든 라운드를 돌았는데도 자리가 남으면 등록 순서대로 추가 배정을 반복합니다.

<br>

## 접근법
먼저, 각 학교 등록 팀 수와 배정된 팀 수를 저장할 배열을 준비합니다.

다음으로, 수용 인원이 남아 있을 때 한 바퀴를 돌며 아직 등록 팀 수보다 적게 배정받은 학교에 1팀씩 배정합니다.

이후, 한 바퀴 동안 아무도 배정하지 못했다면 더 이상 늘릴 수 없으므로 종료합니다. 최종 결과를 입력 순서대로 출력합니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var first = Console.ReadLine()!.Split();
      var n = int.Parse(first[0]);
      var m = int.Parse(first[1]);

      var wish = new int[m];
      var ans = new int[m];
      for (var i = 0; i < m; i++)
        wish[i] = int.Parse(Console.ReadLine()!);

      while (n > 0) {
        var granted = false;
        for (var i = 0; i < m && n > 0; i++) {
          if (ans[i] < wish[i]) {
            ans[i]++;
            n--;
            granted = true;
          }
        }
        if (!granted) break;
      }

      for (var i = 0; i < m; i++)
        Console.WriteLine(ans[i]);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;
  vi wish(m), ans(m, 0);
  for (int i = 0; i < m; i++)
    cin >> wish[i];

  while (n > 0) {
    bool granted = false;
    for (int i = 0; i < m && n > 0; i++) {
      if (ans[i] < wish[i]) {
        ans[i]++;
        n--;
        granted = true;
      }
    }
    if (!granted) break;
  }

  for (int i = 0; i < m; i++)
    cout << ans[i] << "\n";

  return 0;
}
```
