## 문제 링크
[24378번 - КАСТИНГ](https://www.acmicpc.net/problem/24378)

## 설명
전체 n명 배우 중 키가 큰 사람이 a명, 파란 눈이 b명, 금발이 c명입니다.

세 조건을 모두 만족하는 배우 수의 최소 또는 최대를 구하는 문제입니다. 입력 첫 줄이 1이면 최소, 2이면 최대를 출력합니다.

<br>

## 접근법
먼저, 최대 교집합은 세 집합 중 가장 작은 크기입니다. 세 집합 모두에 속해야 하므로 a, b, c 중 최솟값이 됩니다.

다음으로, 최소 교집합은 포함-배제 원리로 구합니다. a + b + c - 2n이 하한이며, 음수가 되면 0입니다.

이후, 요청에 따라 최소 또는 최대를 출력합니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var req = int.Parse(Console.ReadLine()!);
      var arr = Console.ReadLine()!.Split();
      var n = int.Parse(arr[0]);
      var a = int.Parse(arr[1]);
      var b = int.Parse(arr[2]);
      var c = int.Parse(arr[3]);

      var minInt = Math.Max(a + b + c - 2 * n, 0);
      var maxInt = Math.Min(a, Math.Min(b, c));

      Console.WriteLine(req == 1 ? minInt : maxInt);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int req; cin >> req;
  int n, a, b, c; cin >> n >> a >> b >> c;

  int minInt = max(a + b + c - 2 * n, 0);
  int maxInt = min(a, min(b, c));

  cout << (req == 1 ? minInt : maxInt) << "\n";

  return 0;
}
```
