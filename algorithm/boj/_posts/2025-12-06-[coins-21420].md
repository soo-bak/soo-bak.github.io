## 문제 링크
[21420번 - Монетки](https://www.acmicpc.net/problem/21420)

## 설명
n개의 동전이 앞면(0) 또는 뒷면(1)으로 놓여 있습니다.

모든 동전을 같은 면으로 맞추기 위해 뒤집어야 할 최소 동전 수를 구하는 문제입니다.

<br>

## 접근법
먼저, 앞면 개수와 뒷면 개수를 각각 셉니다.

다음으로, 두 값 중 더 작은 쪽을 뒤집으면 최소 횟수가 됩니다. 앞면이 적으면 앞면을 모두 뒤집고, 뒷면이 적으면 뒷면을 모두 뒤집습니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var n = int.Parse(Console.ReadLine()!);
      var cnt0 = 0;
      var cnt1 = 0;
      for (var i = 0; i < n; i++) {
        var v = int.Parse(Console.ReadLine()!);
        if (v == 0) cnt0++;
        else cnt1++;
      }
      Console.WriteLine(Math.Min(cnt0, cnt1));
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int cnt0 = 0, cnt1 = 0;
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    if (v == 0) cnt0++;
    else cnt1++;
  }
  cout << min(cnt0, cnt1) << "\n";

  return 0;
}
```
