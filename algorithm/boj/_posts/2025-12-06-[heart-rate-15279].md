## 문제 링크
[15279번 - Heart Rate](https://www.acmicpc.net/problem/15279)

## 설명
심장 박동이 일정할 때, p초 동안 b회를 측정했다면 분당 박동수를 계산할 수 있습니다.

추정 분당 박동수는 60b/p이고, 오차 1박동을 고려한 최소/최대는 60(b-1)/p와 60(b+1)/p입니다. 세 값을 출력하는 문제입니다.

<br>

## 접근법
먼저, 한 박동 주기는 p/b초이므로 분당 박동수는 60을 주기로 나눈 60b/p가 됩니다.

다음으로, 오차를 고려하면 실제 박동 수가 b-1이거나 b+1일 수 있으므로 최소는 60(b-1)/p, 최대는 60(b+1)/p입니다.

소수점 4자리까지 출력합니다.

<br>

## Code

### C#

```csharp
using System;
using System.Globalization;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var T = int.Parse(Console.ReadLine()!);
      for (var t = 0; t < T; t++) {
        var parts = Console.ReadLine()!.Split();
        var b = double.Parse(parts[0], CultureInfo.InvariantCulture);
        var p = double.Parse(parts[1], CultureInfo.InvariantCulture);
        var min = 60.0 * (b - 1.0) / p;
        var bpm = 60.0 * b / p;
        var max = 60.0 * (b + 1.0) / p;
        Console.WriteLine($"{min:F4} {bpm:F4} {max:F4}");
      }
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int T; cin >> T;
  cout << fixed << setprecision(4);
  while (T--) {
    double b, p; cin >> b >> p;
    double mn = 60.0 * (b - 1.0) / p;
    double bpm = 60.0 * b / p;
    double mx = 60.0 * (b + 1.0) / p;
    cout << mn << " " << bpm << " " << mx << "\n";
  }

  return 0;
}
```
