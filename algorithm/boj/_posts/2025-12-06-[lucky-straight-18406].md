## 문제 링크
[18406번 - 럭키 스트레이트](https://www.acmicpc.net/problem/18406)

## 설명

점수의 자릿수가 짝수일 때, 왼쪽 절반 자릿수 합과 오른쪽 절반 자릿수 합이 같으면 럭키 스트레이트를 사용할 수 있습니다. 사용 가능 여부를 판별하는 문제입니다.

<br>

## 접근법

입력을 문자열로 받아 길이의 절반을 기준으로 왼쪽과 오른쪽 각 자릿수의 합을 구합니다. 두 합이 같으면 LUCKY, 다르면 READY를 출력합니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var s = Console.ReadLine()!;
      var half = s.Length / 2;
      var sumL = 0;
      var sumR = 0;
      for (var i = 0; i < half; i++)
        sumL += s[i] - '0';
      for (var i = half; i < s.Length; i++)
        sumR += s[i] - '0';
      Console.WriteLine(sumL == sumR ? "LUCKY" : "READY");
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  int half = s.size() / 2;
  int sumL = 0;
  int sumR = 0;
  for (int i = 0; i < half; i++)
    sumL += s[i] - '0';
  for (int i = half; i < (int)s.size(); i++)
    sumR += s[i] - '0';
  cout << (sumL == sumR ? "LUCKY" : "READY") << "\n";

  return 0;
}
```
