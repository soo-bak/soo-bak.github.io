## 문제 링크
[17874번 - Piece of Cake!](https://www.acmicpc.net/problem/17874)

## 설명

정사각형 케이크를 가로 한 번, 세로 한 번 잘라서 네 조각으로 나눕니다. 네 조각 중 가장 큰 조각의 부피를 구하는 문제입니다.

<br>

## 접근법

가로로 자르면 위쪽과 아래쪽 두 부분으로 나뉘고, 세로로 자르면 왼쪽과 오른쪽으로 나뉩니다. 가장 큰 조각을 얻으려면 가로 방향에서 더 긴 쪽과 세로 방향에서 더 긴 쪽을 선택하면 됩니다.

케이크 두께가 4cm이므로, 가로 방향 최대 길이와 세로 방향 최대 길이를 곱한 뒤 4를 곱하면 답입니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var nums = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
      var n = nums[0];
      var h = nums[1];
      var v = nums[2];
      var hMax = Math.Max(h, n - h);
      var vMax = Math.Max(v, n - v);
      Console.WriteLine((long)hMax * vMax * 4);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll n, h, v; cin >> n >> h >> v;
  ll hMax = max(h, n - h);
  ll vMax = max(v, n - v);
  cout << hMax * vMax * 4 << "\n";

  return 0;
}
```
