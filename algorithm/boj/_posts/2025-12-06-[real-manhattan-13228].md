## 문제 링크
[13228번 - The REAL Manhattan distance](https://www.acmicpc.net/problem/13228)

## 설명
두 지점이 각각 좌표와 층수로 주어질 때, 첫 번째 빌딩에서 내려오고 도로를 이동한 뒤 두 번째 빌딩으로 올라가는 총 이동거리를 구하는 문제입니다.

층간 거리는 1이며, 도로 이동은 맨해튼 거리를 사용합니다.

<br>

## 접근법
먼저, 첫 번째 빌딩에서 1층까지 내려오는 거리는 첫 번째 층수와 같습니다.

다음으로, 도로에서의 이동 거리는 두 좌표 간 맨해튼 거리입니다. x좌표 차이의 절댓값과 y좌표 차이의 절댓값을 더합니다.

이후, 두 번째 빌딩에서 목적지 층까지 올라가는 거리는 두 번째 층수와 같습니다. 세 값을 모두 더하면 답이 됩니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var T = int.Parse(Console.ReadLine()!);
      for (var t = 0; t < T; t++) {
        var parts = Console.ReadLine()!.Split();
        var x1 = long.Parse(parts[0]);
        var y1 = long.Parse(parts[1]);
        var f1 = long.Parse(parts[2]);
        var x2 = long.Parse(parts[3]);
        var y2 = long.Parse(parts[4]);
        var f2 = long.Parse(parts[5]);
        var ans = f1 + f2 + Math.Abs(x1 - x2) + Math.Abs(y1 - y2);
        Console.WriteLine(ans);
      }
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int T; cin >> T;
  while (T--) {
    ll x1, y1, f1, x2, y2, f2;
    cin >> x1 >> y1 >> f1 >> x2 >> y2 >> f2;
    ll ans = f1 + f2 + llabs(x1 - x2) + llabs(y1 - y2);
    cout << ans << "\n";
  }

  return 0;
}
```
