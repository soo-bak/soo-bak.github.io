## 문제 링크
[20674번 - Statistics](https://www.acmicpc.net/problem/20674)

## 설명
향후 n일의 확진자 수를 발표할 때, 발표치는 비증가(non-increasing) 해야 합니다. 발표치는 실제보다 클 수 없으므로 필요하면 값을 줄입니다.

줄인 총합을 최소화하는 문제입니다.

<br>

## 접근법
먼저, 첫 번째 값을 현재 기준으로 설정합니다.

다음으로, 두 번째 값부터 순서대로 확인합니다. 현재 값이 기준보다 크면 기준으로 맞추기 위해 초과분을 줄여야 하므로 답에 더합니다. 현재 값이 기준보다 작으면 그 값이 새 기준이 됩니다.

이후, 모든 값을 확인한 후 줄인 총합을 출력합니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var n = int.Parse(Console.ReadLine()!);
      var cur = int.Parse(Console.ReadLine()!);
      var ans = 0;
      for (var i = 1; i < n; i++) {
        var p = int.Parse(Console.ReadLine()!);
        if (p > cur) ans += p - cur;
        else cur = p;
      }
      Console.WriteLine(ans);
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int cur; cin >> cur;
  int ans = 0;
  for (int i = 1; i < n; i++) {
    int p; cin >> p;
    if (p > cur) ans += p - cur;
    else cur = p;
  }
  cout << ans << "\n";

  return 0;
}
```
