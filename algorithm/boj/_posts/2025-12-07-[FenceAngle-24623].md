## 문제 링크
[24623번 - Изгороди](https://www.acmicpc.net/problem/24623)

## 설명
도랑의 중간 지점에서 두 울타리가 연결되어 있을 때, 두 울타리 사이의 각도를 구하는 문제입니다.

<br>

## 접근법
도랑의 길이가 2n이고 중간 지점에서 길이 n인 울타리가 연결되어 있습니다. 중간 지점에서 양 끝점까지의 거리가 모두 n이므로 세 점이 이루는 삼각형은 세 변의 길이가 같습니다. 이 경우 두 울타리가 이루는 각도는 입력에 관계없이 항상 90도입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    Console.ReadLine();
    Console.ReadLine();
    Console.WriteLine(90);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, alpha;
  cin >> n >> alpha;
  cout << 90 << "\n";

  return 0;
}
```
