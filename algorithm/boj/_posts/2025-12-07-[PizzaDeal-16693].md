## 문제 링크
[16693번 - Pizza Deal](https://www.acmicpc.net/problem/16693)

## 설명
조각 피자의 면적과 가격, 원형 피자의 반지름과 가격이 주어질 때, 가격 대비 면적이 더 큰 피자를 선택하는 문제입니다.

<br>

## 접근법
먼저, 각 피자의 가격 대비 면적을 계산합니다. 조각 피자는 면적을 가격으로 나누고, 원형 피자는 원의 넓이 공식으로 면적을 구한 뒤 가격으로 나눕니다.

다음으로, 두 값을 비교하여 조각 피자의 값이 더 크면 Slice of pizza를, 그렇지 않으면 Whole pizza를 출력합니다. 문제에서 두 값이 같지 않음이 보장되므로 동률 처리는 필요하지 않습니다.

<br>

## Code

### C#

```csharp
using System;

namespace Solution {
  class Program {
    static void Main(string[] args) {
      var l1 = Console.ReadLine()!.Split();
      var l2 = Console.ReadLine()!.Split();
      var a1 = double.Parse(l1[0]);
      var p1 = double.Parse(l1[1]);
      var r1 = double.Parse(l2[0]);
      var p2 = double.Parse(l2[1]);

      var slice = a1 / p1;
      var whole = Math.PI * r1 * r1 / p2;

      if (slice > whole) Console.WriteLine("Slice of pizza");
      else Console.WriteLine("Whole pizza");
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);

  double a1, p1, r1, p2;
  cin >> a1 >> p1 >> r1 >> p2;

  double slice = a1 / p1;
  double whole = M_PI * r1 * r1 / p2;

  if (slice > whole) cout << "Slice of pizza\n";
  else cout << "Whole pizza\n";

  return 0;
}
```
