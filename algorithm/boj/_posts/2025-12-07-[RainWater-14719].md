## 문제 링크
[14719번 - 빗물](https://www.acmicpc.net/problem/14719)

## 설명
2차원 세계에 블록이 쌓여 있을 때, 비가 충분히 온 후 고이는 빗물의 총량을 구하는 문제입니다.

<br>

## 접근법
각 칸에 고이는 물의 양은 그 칸의 왼쪽과 오른쪽에서 가장 높은 블록 중 낮은 쪽의 높이에서 현재 블록 높이를 뺀 값입니다. 이 값이 양수일 때만 물이 고입니다.

각 위치마다 왼쪽의 최대 높이와 오른쪽의 최대 높이를 구합니다. 두 값 중 작은 값이 해당 위치에서 물이 찰 수 있는 수위가 됩니다. 현재 블록 높이보다 수위가 높으면 그 차이만큼 물이 고입니다.

모든 칸에 대해 고이는 물의 양을 합산하면 답이 됩니다.

<br>

## Code

### C#

```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var h = int.Parse(first[0]);
    var w = int.Parse(first[1]);
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var total = 0;
    for (var i = 1; i < w - 1; i++) {
      var lmax = 0;
      var rmax = 0;
      for (var l = 0; l < i; l++) if (arr[l] > lmax) lmax = arr[l];
      for (var r = i + 1; r < w; r++) if (arr[r] > rmax) rmax = arr[r];
      var water = Math.Min(lmax, rmax) - arr[i];
      if (water > 0) total += water;
    }

    Console.WriteLine(total);
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int h, w; cin >> h >> w;
  vi a(w);
  for (int i = 0; i < w; i++) cin >> a[i];

  int total = 0;
  for (int i = 1; i < w - 1; i++) {
    int lmax = 0, rmax = 0;
    for (int l = 0; l < i; l++) lmax = max(lmax, a[l]);
    for (int r = i + 1; r < w; r++) rmax = max(rmax, a[r]);
    int water = min(lmax, rmax) - a[i];
    if (water > 0) total += water;
  }

  cout << total << "\n";

  return 0;
}
```
