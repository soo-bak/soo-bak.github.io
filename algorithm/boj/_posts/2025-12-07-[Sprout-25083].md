## 문제 링크
[25083번 - 새싹](https://www.acmicpc.net/problem/25083)

## 설명
새싹 모양의 아스키 아트를 출력하는 문제입니다.

<br>

## 접근법
주어진 문자열을 줄바꿈을 포함하여 그대로 출력합니다. 역슬래시나 큰따옴표 같은 특수 문자는 이스케이프 처리가 필요합니다.

<br>

- - -

## Code

### C#

```csharp
using System;

class Program {
  static void Main() {
    Console.WriteLine("         ,r'\"7");
    Console.WriteLine("r`-_   ,'  ,/");
    Console.WriteLine(" \\. \". L_r'");
    Console.WriteLine("   `~\\/");
    Console.WriteLine("      |");
    Console.WriteLine("      |");
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>

  return 0;
}
```
