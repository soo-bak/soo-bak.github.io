## 문제 링크
[13416번 - 주식 투자](https://www.acmicpc.net/problem/13416)

## 설명
N일 동안 세 회사 중 하루에 한 곳만 주식을 살 수 있을 때, 최대로 벌 수 있는 이익의 합을 구하는 문제입니다. 손해가 예상되면 아무 것도 사지 않아도 됩니다.

<br>

## 접근법
각 날마다 세 회사의 당일 손익이 주어지므로, 그날 선택 가능한 최선은 세 값 중 최댓값입니다. 이 값이 양수일 때만 이익에 더하고, 0 이하라면 주식을 사지 않습니다.

모든 날에 대해 이를 반복하여 합산하면 최대 이익이 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    while (t-- > 0) {
      var n = int.Parse(Console.ReadLine()!);
      var ans = 0L;
      for (var i = 0; i < n; i++) {
        var parts = Console.ReadLine()!.Split();
        var a = int.Parse(parts[0]);
        var b = int.Parse(parts[1]);
        var c = int.Parse(parts[2]);
        var p = Math.Max(a, Math.Max(b, c));
        if (p > 0) ans += p;
      }
      Console.WriteLine(ans);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int T; cin >> T;
  while (T--) {
    int n; cin >> n;
    ll ans = 0;
    for (int i = 0; i < n; i++) {
      int a, b, c; cin >> a >> b >> c;
      int p = max({a, b, c});
      if (p > 0) ans += p;
    }
    cout << ans << "\n";
  }

  return 0;
}
```
