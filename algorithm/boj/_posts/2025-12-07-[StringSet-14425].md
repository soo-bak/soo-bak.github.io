## 문제 링크
[14425번 - 문자열 집합](https://www.acmicpc.net/problem/14425)

## 설명
N개의 문자열로 이루어진 집합과 M개의 질의 문자열이 주어질 때, 질의 문자열 중 집합에 포함된 것의 개수를 구하는 문제입니다.

<br>

## 접근법
N개의 문자열을 해시 집합에 저장합니다. 해시 집합은 문자열의 존재 여부를 평균 상수 시간에 확인할 수 있습니다.

M개의 질의 문자열을 순회하며 집합에 포함되어 있는지 확인하고, 포함된 경우 개수를 증가시킵니다.

<br>

- - -

## Code

### C#

```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);

    var set = new HashSet<string>();
    for (var i = 0; i < n; i++)
      set.Add(Console.ReadLine()!);

    var ans = 0;
    for (var i = 0; i < m; i++)
      if (set.Contains(Console.ReadLine()!)) ans++;

    Console.WriteLine(ans);
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m;
  cin >> n >> m;
  unordered_set<string> s;
  s.reserve(n * 2);
  for (int i = 0; i < n; i++) {
    string str; cin >> str;
    s.insert(str);
  }

  int ans = 0;
  for (int i = 0; i < m; i++) {
    string str; cin >> str;
    if (s.find(str) != s.end()) ans++;
  }
  cout << ans << "\n";

  return 0;
}
```
