## 문제 링크
[15501번 - 부당한 퍼즐](https://www.acmicpc.net/problem/15501)

## 설명
두 순열이 주어질 때, 회전과 뒤집기 연산만으로 첫 번째 순열을 두 번째 순열과 같게 만들 수 있는지 판단하는 문제입니다.

<br>

## 접근법
회전은 시작 위치를 옮기는 것과 같습니다. 따라서 첫 번째 순열의 첫 원소가 두 번째 순열의 어느 위치에 있는지 찾은 후, 그 위치부터 순서대로 비교하면 됩니다.

먼저 두 번째 순열에서 첫 번째 순열의 첫 원소와 같은 값의 위치를 찾습니다. 그 위치를 시작점으로 하여 순환하며 모든 원소가 일치하면 가능한 경우입니다.

일치하지 않으면 두 번째 순열을 뒤집은 후 같은 방식으로 다시 비교합니다. 이것도 일치하지 않으면 불가능한 경우입니다.

<br>

## Code

### C#

```csharp
using System;

class Program {
  static bool Check(int[] a, int[] b, int n) {
    var pos = Array.IndexOf(b, a[0]);
    if (pos == -1) return false;
    for (var i = 0; i < n; i++)
      if (a[i] != b[(pos + i) % n]) return false;
    return true;
  }

  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var a = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var b = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var ok = Check(a, b, n);
    if (!ok) {
      Array.Reverse(b);
      ok = Check(a, b, n);
    }

    Console.WriteLine(ok ? "good puzzle" : "bad puzzle");
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

bool check(const vi& a, const vi& b, int n) {
  int pos = -1;
  for (int i = 0; i < n; i++) if (b[i] == a[0]) { pos = i; break; }
  if (pos == -1) return false;
  for (int i = 0; i < n; i++)
    if (a[i] != b[(pos + i) % n]) return false;
  return true;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi a(n), b(n);
  for (int i = 0; i < n; i++) cin >> a[i];
  for (int i = 0; i < n; i++) cin >> b[i];

  bool ok = check(a, b, n);
  if (!ok) {
    reverse(b.begin(), b.end());
    ok = check(a, b, n);
  }

  cout << (ok ? "good puzzle" : "bad puzzle") << "\n";

  return 0;
}
```
