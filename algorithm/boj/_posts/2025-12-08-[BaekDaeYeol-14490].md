## 문제 링크
[14490번 - 백대열](https://www.acmicpc.net/problem/14490)

## 설명
콜론으로 구분된 두 수의 비율을 기약분수 형태로 출력하는 문제입니다.

<br>

## 접근법
입력 문자열에서 콜론을 기준으로 두 정수를 분리합니다. 두 수의 최대공약수를 유클리드 호제법으로 구한 뒤, 각각을 최대공약수로 나누어 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static int Gcd(int a, int b) => b == 0 ? a : Gcd(b, a % b);

  static void Main() {
    var parts = Console.ReadLine()!.Split(':');
    var n = int.Parse(parts[0]);
    var m = int.Parse(parts[1]);
    var g = Gcd(n, m);
    Console.WriteLine($"{n / g}:{m / g}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int gcdInt(int a, int b) {
  while (b) {
    int t = a % b;
    a = b;
    b = t;
  }
  return a;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  int pos = s.find(':');
  int n = stoi(s.substr(0, pos));
  int m = stoi(s.substr(pos + 1));
  int g = gcdInt(n, m);
  cout << n / g << ":" << m / g << "\n";

  return 0;
}
```
