## 문제 링크
[25314번 - 코딩은 체육과목 입니다](https://www.acmicpc.net/problem/25314)

## 설명
N바이트 정수를 저장할 수 있는 자료형 이름을 출력하는 문제입니다. 4바이트마다 long을 하나씩 붙이고 마지막에 int를 붙입니다.

<br>

## 접근법
입력을 4로 나눈 몫만큼 long과 공백을 반복해서 출력하고, 마지막에 int를 붙입니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var cnt = n / 4;
    var sb = new StringBuilder();
    for (var i = 0; i < cnt; i++) sb.Append("long ");
    sb.Append("int");
    Console.WriteLine(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int cnt = n / 4;
  for (int i = 0; i < cnt; i++) cout << "long ";
  cout << "int\n";

  return 0;
}
```
