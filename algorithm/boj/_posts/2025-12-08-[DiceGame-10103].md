## 문제 링크
[10103번 - 주사위 게임](https://www.acmicpc.net/problem/10103)

## 설명
두 사람이 주사위를 굴려서 게임을 합니다. 매 라운드 더 낮은 숫자를 던진 사람이 상대의 주사위 눈만큼 점수를 잃습니다. 같은 숫자가 나오면 변화가 없습니다. 모든 라운드가 끝난 후 두 사람의 점수를 출력하는 문제입니다.

<br>

## 접근법
두 사람 모두 100점에서 시작합니다. 매 라운드마다 두 주사위 눈을 비교하여 진 사람의 점수를 이긴 사람의 주사위 눈만큼 뺍니다. 비기면 아무 일도 일어나지 않습니다. 모든 라운드를 처리한 뒤 각자의 점수를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sa = 100;
    var sb = 100;
    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = int.Parse(parts[0]);
      var b = int.Parse(parts[1]);
      if (a > b) sb -= a;
      else if (a < b) sa -= b;
    }
    Console.WriteLine(sa);
    Console.WriteLine(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int sa = 100, sb = 100;
  for (int i = 0; i < n; i++) {
    int a, b; cin >> a >> b;
    if (a > b) sb -= a;
    else if (a < b) sa -= b;
  }
  cout << sa << "\n" << sb << "\n";

  return 0;
}
```
