## 문제 링크
[2566번 - 최댓값](https://www.acmicpc.net/problem/2566)

## 설명
9×9 격자에 적힌 수들 중 최댓값과 그 위치를 찾는 문제입니다.

<br>

## 접근법
격자의 모든 칸을 순서대로 읽으면서 현재까지의 최댓값보다 큰 값이 나오면 최댓값과 해당 위치를 갱신합니다.

입력이 모두 0일 수도 있으므로 초기 최댓값을 0으로 설정합니다. 최댓값이 여러 곳에 있더라도 한 곳만 출력하면 되므로 처음 등장한 위치를 유지하면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var max = 0;
    var rPos = 1;
    var cPos = 1;
    for (var r = 1; r <= 9; r++) {
      var parts = Console.ReadLine()!.Split();
      for (var c = 1; c <= 9; c++) {
        var v = int.Parse(parts[c - 1]);
        if (v > max) {
          max = v;
          rPos = r;
          cPos = c;
        }
      }
    }
    Console.WriteLine(max);
    Console.WriteLine($"{rPos} {cPos}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int maxVal = 0, rPos = 1, cPos = 1;
  for (int r = 1; r <= 9; r++) {
    for (int c = 1; c <= 9; c++) {
      int v; cin >> v;
      if (v > maxVal) {
        maxVal = v;
        rPos = r;
        cPos = c;
      }
    }
  }
  cout << maxVal << "\n" << rPos << " " << cPos << "\n";

  return 0;
}
```
