## 문제 링크
[15953번 - 상금 헌터](https://www.acmicpc.net/problem/15953)

## 설명
두 대회의 등수가 주어질 때, 각 대회에서 받는 상금의 합을 구하는 문제입니다. 각 대회는 등수 구간별로 상금이 다르게 지급됩니다.

<br>

## 접근법
상금 지급 구간이 정해져 있으므로 등수별 상금을 미리 배열에 저장해둡니다.

2017년 대회는 1등부터 21등까지 상금을 받고, 2018년 대회는 1등부터 31등까지 상금을 받습니다. 각 등수에 해당하는 상금을 배열에 넣어두면 등수로 바로 상금을 찾을 수 있습니다.

두 대회의 등수가 입력되면 각각 배열에서 상금을 찾아 더합니다. 등수가 0이거나 상금 범위를 벗어나면 상금이 없습니다. 합계에 10000을 곱해 원 단위로 출력합니다.

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var prize17 = new int[22];
    var idx = 1;
    int[] val17 = {500, 300, 200, 50, 30, 10};
    int[] cnt17 = {1, 2, 3, 4, 5, 6};
    for (var g = 0; g < val17.Length; g++) {
      for (var c = 0; c < cnt17[g]; c++) {
        if (idx < prize17.Length) prize17[idx] = val17[g];
        idx++;
      }
    }

    var prize18 = new int[33];
    idx = 1;
    int[] val18 = {512, 256, 128, 64, 32};
    int[] cnt18 = {1, 2, 4, 8, 16};
    for (var g = 0; g < val18.Length; g++) {
      for (var c = 0; c < cnt18[g]; c++) {
        if (idx < prize18.Length) prize18[idx] = val18[g];
        idx++;
      }
    }

    var t = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < t; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = int.Parse(parts[0]);
      var b = int.Parse(parts[1]);
      var sum = 0;
      if (a > 0 && a < prize17.Length) sum += prize17[a];
      if (b > 0 && b < prize18.Length) sum += prize18[b];
      Console.WriteLine(sum * 10000);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  vi p17(22, 0);
  int val17[] = {500, 300, 200, 50, 30, 10};
  int cnt17[] = {1, 2, 3, 4, 5, 6};
  int idx = 1;
  for (int g = 0; g < 6; g++) {
    for (int c = 0; c < cnt17[g]; c++) {
      if (idx < 22) p17[idx] = val17[g];
      idx++;
    }
  }

  vi p18(33, 0);
  int val18[] = {512, 256, 128, 64, 32};
  int cnt18[] = {1, 2, 4, 8, 16};
  idx = 1;
  for (int g = 0; g < 5; g++) {
    for (int c = 0; c < cnt18[g]; c++) {
      if (idx < 33) p18[idx] = val18[g];
      idx++;
    }
  }

  int t; cin >> t;
  while (t--) {
    int a, b; cin >> a >> b;
    int sum = 0;
    if (a > 0 && a < 22) sum += p17[a];
    if (b > 0 && b < 33) sum += p18[b];
    cout << sum * 10000 << "\n";
  }

  return 0;
}
```
