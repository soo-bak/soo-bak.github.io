## 문제 링크
[9610번 - 사분면](https://www.acmicpc.net/problem/9610)

## 설명
여러 점이 주어질 때, 각 점이 어느 사분면에 있는지 또는 축 위에 있는지 세어 출력하는 문제입니다.

<br>

## 접근법
각 점의 좌표를 확인하여 분류합니다. x 또는 y가 0이면 축 위에 있습니다. 둘 다 0이 아니면 부호에 따라 사분면을 결정합니다. x와 y가 모두 양수면 1사분면, x가 음수이고 y가 양수면 2사분면, 둘 다 음수면 3사분면, x가 양수이고 y가 음수면 4사분면입니다. 각 경우의 개수를 세어 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var cnt = new int[5];
    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var x = int.Parse(parts[0]);
      var y = int.Parse(parts[1]);
      if (x == 0 || y == 0) cnt[0]++;
      else if (x > 0 && y > 0) cnt[1]++;
      else if (x < 0 && y > 0) cnt[2]++;
      else if (x < 0 && y < 0) cnt[3]++;
      else cnt[4]++;
    }
    for (var i = 1; i <= 4; i++)
      Console.WriteLine($"Q{i}: {cnt[i]}");
    Console.WriteLine($"AXIS: {cnt[0]}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int cnt[5] = {0};
  for (int i = 0; i < n; i++) {
    int x, y; cin >> x >> y;
    if (x == 0 || y == 0) cnt[0]++;
    else if (x > 0 && y > 0) cnt[1]++;
    else if (x < 0 && y > 0) cnt[2]++;
    else if (x < 0 && y < 0) cnt[3]++;
    else cnt[4]++;
  }
  for (int i = 1; i <= 4; i++)
    cout << "Q" << i << ": " << cnt[i] << "\n";
  cout << "AXIS: " << cnt[0] << "\n";

  return 0;
}
```
