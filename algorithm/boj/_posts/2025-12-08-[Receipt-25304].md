## 문제 링크
[25304번 - 영수증](https://www.acmicpc.net/problem/25304)

## 설명
영수증에 적힌 총액과 각 물건의 가격, 개수가 주어질 때, 물건별 금액의 합이 총액과 일치하는지 확인하는 문제입니다.

<br>

## 접근법
각 물건의 가격과 개수를 곱한 값을 모두 더합니다. 이 합이 영수증의 총액과 같으면 Yes를, 다르면 No를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var x = int.Parse(Console.ReadLine()!);
    var n = int.Parse(Console.ReadLine()!);
    var sum = 0;
    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = int.Parse(parts[0]);
      var b = int.Parse(parts[1]);
      sum += a * b;
    }
    Console.WriteLine(sum == x ? "Yes" : "No");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int x, n; cin >> x >> n;
  int sum = 0;
  for (int i = 0; i < n; i++) {
    int a, b; cin >> a >> b;
    sum += a * b;
  }
  cout << (sum == x ? "Yes" : "No") << "\n";

  return 0;
}
```
