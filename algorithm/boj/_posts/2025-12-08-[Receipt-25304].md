## 문제 링크
[25304번 - 영수증](https://www.acmicpc.net/problem/25304)

## 설명
영수증 총액 X와 N개 항목의 가격·개수가 주어질 때, 항목 가격×개수 합이 X와 일치하면 "Yes", 아니면 "No"를 출력합니다. 단순 합산으로 해결됩니다.

<br>

## 접근법
- 총액 X, 항목 수 N 입력.
- N번 반복하며 `sum += price * count`.
- `sum == X`면 Yes, 아니면 No.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    int X = int.Parse(Console.ReadLine()!);
    int N = int.Parse(Console.ReadLine()!);
    int sum = 0;
    for (int i = 0; i < N; i++) {
      var parts = Console.ReadLine()!.Split();
      int a = int.Parse(parts[0]);
      int b = int.Parse(parts[1]);
      sum += a * b;
    }
    Console.WriteLine(sum == X ? "Yes" : "No");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int X, N; cin >> X >> N;
  int sum = 0;
  for (int i = 0; i < N; i++) {
    int a, b; cin >> a >> b;
    sum += a * b;
  }
  cout << (sum == X ? "Yes" : "No") << "\n";
  return 0;
}
```
