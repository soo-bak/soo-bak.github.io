## 문제 링크
[1247번 - 부호](https://www.acmicpc.net/problem/1247)

## 설명
여러 정수의 합이 양수인지, 음수인지, 0인지 판별하는 문제입니다. 정수의 개수가 최대 10만 개이고 각 정수가 64비트 범위이므로 합이 64비트를 넘을 수 있습니다. C#에서는 BigInteger를 사용하고, C++에서는 128비트 정수형을 사용하여 오버플로 없이 합을 구합니다.

<br>

## 접근법
각 테스트 케이스마다 모든 정수를 더한 뒤 부호를 확인합니다. 합이 0이면 0을, 양수면 +를, 음수면 -를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Numerics;

class Program {
  static void Main() {
    for (var t = 0; t < 3; t++) {
      var n = int.Parse(Console.ReadLine()!);
      var sum = new BigInteger(0);
      for (var i = 0; i < n; i++) {
        var x = long.Parse(Console.ReadLine()!);
        sum += x;
      }
      if (sum == 0) Console.WriteLine("0");
      else Console.WriteLine(sum > 0 ? "+" : "-");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef __int128_t i128;
typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  for (int t = 0; t < 3; t++) {
    int n; cin >> n;
    i128 sum = 0;
    for (int i = 0; i < n; i++) {
      ll x; cin >> x;
      sum += (i128)x;
    }
    if (sum == 0) cout << "0\n";
    else cout << (sum > 0 ? "+\n" : "-\n");
  }

  return 0;
}
```
