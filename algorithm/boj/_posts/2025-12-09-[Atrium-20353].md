## 문제 링크
[20353번 - Atrium](https://www.acmicpc.net/problem/20353)

## 설명
정사각형 중정의 넓이가 주어질 때, 중정을 둘러싸는 벽의 총 길이를 구하는 문제입니다.

<br>

## 접근법
정사각형의 넓이가 a이면 한 변의 길이는 a의 제곱근입니다.

둘레는 한 변의 네 배이므로, 제곱근을 구한 뒤 4를 곱하면 됩니다.

결과는 소수점 아래 여섯 자리까지 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var a = double.Parse(Console.ReadLine()!);
    var ans = 4 * Math.Sqrt(a);
    Console.WriteLine($"{ans:F6}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  double a; cin >> a;
  double ans = 4.0 * sqrt(a);
  cout.setf(ios::fixed);
  cout.precision(6);
  cout << ans << "\n";

  return 0;
}
```
