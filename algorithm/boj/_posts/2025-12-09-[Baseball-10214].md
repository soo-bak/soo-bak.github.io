## 문제 링크
[10214번 - Baseball](https://www.acmicpc.net/problem/10214)

## 설명
9이닝 동안 연세대와 고려대의 득점이 주어질 때, 총점을 비교해 승자를 판정하는 문제입니다.

<br>

## 접근법
각 이닝의 점수를 읽으면서 두 팀의 총점을 누적합니다.

9이닝이 끝나면 총점을 비교해서, 연세대가 높으면 Yonsei, 고려대가 높으면 Korea, 같으면 Draw를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var _ = 0; _ < t; _++) {
      var y = 0;
      var k = 0;
      for (var i = 0; i < 9; i++) {
        var parts = Console.ReadLine()!.Split();
        y += int.Parse(parts[0]);
        k += int.Parse(parts[1]);
      }
      if (y > k) Console.WriteLine("Yonsei");
      else if (y < k) Console.WriteLine("Korea");
      else Console.WriteLine("Draw");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int T; cin >> T;
  while (T--) {
    int y = 0, k = 0;
    for (int i = 0; i < 9; i++) {
      int sy, sk; cin >> sy >> sk;
      y += sy; k += sk;
    }
    if (y > k) cout << "Yonsei\n";
    else if (y < k) cout << "Korea\n";
    else cout << "Draw\n";
  }

  return 0;
}
```
