## 문제 링크
[1233번 - 주사위](https://www.acmicpc.net/problem/1233)

## 설명
면 수가 다른 세 주사위를 동시에 던질 때 나오는 눈의 합 중, 가장 자주 등장하는 합을 구합니다. 빈도가 같은 합이 여러 개면 더 작은 합을 출력합니다.

<br>

## 접근법
첫 번째 주사위가 1부터 S1까지, 두 번째가 1부터 S2까지, 세 번째가 1부터 S3까지 나올 수 있으므로 삼중 반복문으로 모든 경우를 확인합니다. 

각 경우의 합이 몇 번 나오는지 배열에 기록한 뒤, 가장 많이 나온 합을 출력합니다.

빈도가 같은 합이 여러 개면 작은 합을 출력해야 하므로, 작은 합부터 확인하면서 현재까지의 최대 빈도보다 클 때만 답을 갱신합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var s1 = int.Parse(parts[0]);
    var s2 = int.Parse(parts[1]);
    var s3 = int.Parse(parts[2]);

    var freq = new int[81];
    for (var a = 1; a <= s1; a++)
      for (var b = 1; b <= s2; b++)
        for (var c = 1; c <= s3; c++)
          freq[a + b + c]++;

    var bestCnt = 0;
    var bestSum = 0;
    for (var sum = 1; sum < freq.Length; sum++) {
      if (freq[sum] > bestCnt) {
        bestCnt = freq[sum];
        bestSum = sum;
      }
    }

    Console.WriteLine(bestSum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int s1, s2, s3; cin >> s1 >> s2 >> s3;
  vi freq(81, 0);
  for (int a = 1; a <= s1; a++)
    for (int b = 1; b <= s2; b++)
      for (int c = 1; c <= s3; c++)
        freq[a + b + c]++;

  int bestCnt = 0, bestSum = 0;
  for (int sum = 1; sum < 81; sum++) {
    if (freq[sum] > bestCnt) {
      bestCnt = freq[sum];
      bestSum = sum;
    }
  }

  cout << bestSum << "\n";

  return 0;
}
```
