## 문제 링크
[1402번 - 아무래도이문제는A번난이도인것같다](https://www.acmicpc.net/problem/1402)

## 설명
정수 A가 B로 변할 수 있는지 묻는 문제입니다. A를 여러 정수의 곱으로 표현하고 그 합을 새로운 값으로 삼는 과정을 반복할 수 있습니다.

<br>

## 접근법
임의의 정수 A는 A × 1로 표현할 수 있고, 이 합은 A + 1입니다. 또한 A × 1 × 1로 표현하면 합은 A + 2가 됩니다. 이런 식으로 1을 원하는 만큼 곱하면 합을 얼마든지 늘릴 수 있습니다. 반대로 A × (-1) × (-1)로 표현하면 합은 A - 2가 됩니다. 따라서 어떤 A든 어떤 B로든 변할 수 있으므로 항상 yes를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < t; i++) {
      Console.ReadLine();
      Console.WriteLine("yes");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    ll a, b; cin >> a >> b;
    cout << "yes\n";
  }

  return 0;
}
```
