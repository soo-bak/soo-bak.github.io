## 문제 링크
[15873번 - 공백 없는 A+B](https://www.acmicpc.net/problem/15873)

## 설명
공백 없이 붙어 있는 두 자연수의 합을 구하는 문제입니다. 각 수는 1부터 10 사이이므로 입력 길이는 2자리에서 4자리까지 가능합니다.

<br>

## 접근법
입력 길이에 따라 경우를 나눕니다. 길이가 2이면 두 수 모두 한 자리 수이므로 각 자릿수를 더합니다. 길이가 4이면 두 수 모두 10이므로 합은 20입니다. 길이가 3이면 한 쪽이 10이고 다른 쪽이 한 자리 수인데, 두 번째 문자가 0이면 앞의 수가 10이고 아니면 뒤의 수가 10입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var ans = 0;
    if (s.Length == 2) {
      ans = (s[0] - '0') + (s[1] - '0');
    } else if (s.Length == 3) {
      if (s[1] == '0') ans = 10 + (s[2] - '0');
      else ans = (s[0] - '0') + 10;
    } else ans = 20;
    Console.WriteLine(ans);
  }
}
```
