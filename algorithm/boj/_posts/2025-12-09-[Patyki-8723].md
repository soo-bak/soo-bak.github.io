## 문제 링크
[8723번 - Patyki](https://www.acmicpc.net/problem/8723)

## 설명
세 막대 길이가 주어질 때, 어떤 삼각형을 만들 수 있는지 판별하는 문제입니다.

정삼각형이면 2, 직각삼각형이면 1, 둘 다 아니면 0을 출력합니다.

<br>

## 접근법
먼저 세 길이를 오름차순으로 정렬합니다.

세 길이가 모두 같으면 정삼각형입니다.

그렇지 않다면 가장 짧은 두 변의 제곱 합이 가장 긴 변의 제곱과 같은지 확인합니다. 같으면 직각삼각형입니다.

둘 다 아니면 0을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var len = new int[3];
    for (var i = 0; i < 3; i++)
      len[i] = int.Parse(parts[i]);
    Array.Sort(len);

    int ans;
    if (len[0] == len[1] && len[1] == len[2]) ans = 2;
    else if (len[0] * len[0] + len[1] * len[1] == len[2] * len[2]) ans = 1;
    else ans = 0;

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a[3];
  for (int i = 0; i < 3; i++)
    cin >> a[i];
  sort(a, a + 3);

  int ans = 0;
  if (a[0] == a[1] && a[1] == a[2]) ans = 2;
  else if (a[0] * a[0] + a[1] * a[1] == a[2] * a[2]) ans = 1;

  cout << ans << "\n";

  return 0;
}
```
