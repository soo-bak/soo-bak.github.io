## 문제 링크
[25495번 - 에어팟](https://www.acmicpc.net/problem/25495)

## 설명
에어팟의 배터리 소모 규칙에 따라 n회 연결 시도 후 누적 소모량을 구하는 문제입니다.

다른 기기에 연결하면 2%, 같은 기기를 연속 연결하면 직전 소모의 두 배를 사용합니다.

누적이 100% 이상이 되면 연결이 실패하고 배터리가 0%로 초기화됩니다.

<br>

## 접근법
이전 기기 번호와 직전 소모량을 저장하며 시뮬레이션합니다.

연결 시 소모량을 계산하고, 누적이 100 이상이면 모든 값을 초기화합니다.

그렇지 않으면 누적에 더하고 상태를 갱신합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var prev = -1;
    var lastCost = 0;
    var used = 0;
    foreach (var x in arr) {
      var cost = (x == prev) ? lastCost * 2 : 2;
      if (used + cost >= 100) {
        used = 0;
        prev = -1;
        lastCost = 0;
        continue;
      }
      used += cost;
      lastCost = cost;
      prev = x;
    }

    Console.WriteLine(used);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi a(n);
  for (int i = 0; i < n; i++) cin >> a[i];

  int prev = -1, lastCost = 0, used = 0;
  for (int x : a) {
    int cost = (x == prev) ? lastCost * 2 : 2;
    if (used + cost >= 100) {
      used = 0;
      prev = -1;
      lastCost = 0;
      continue;
    }
    used += cost;
    lastCost = cost;
    prev = x;
  }

  cout << used << "\n";

  return 0;
}
```
