## 문제 링크
[24265번 - 알고리즘 수업 - 알고리즘의 수행 시간 4](https://www.acmicpc.net/problem/24265)

## 설명
이중 루프의 수행 횟수와 최고차항의 차수를 구하는 문제입니다.

i < j인 쌍의 개수만큼 수행되므로 n(n-1)/2회이고, 최고차항은 n²이므로 차수는 2입니다.

<br>

## 접근법
수행 횟수는 n(n-1)/2로 계산합니다.

차수는 항상 2이므로 그대로 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = long.Parse(Console.ReadLine()!);
    Console.WriteLine(n * (n - 1) / 2);
    Console.WriteLine(2);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll n; cin >> n;
  cout << n * (n - 1) / 2 << "\n";
  cout << 2 << "\n";

  return 0;
}
```
