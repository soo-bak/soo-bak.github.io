## 문제 링크
[24266번 - 알고리즘 수업 - 알고리즘의 수행 시간 5](https://www.acmicpc.net/problem/24266)

## 설명
삼중 루프의 수행 횟수와 최고차항의 차수를 구하는 문제입니다.

세 개의 루프가 각각 1부터 n까지 순회하므로 총 n³번 실행되고, 차수는 3입니다.

<br>

## 접근법
수행 횟수는 n × n × n으로 계산합니다.

차수는 항상 3이므로 그대로 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = long.Parse(Console.ReadLine()!);
    Console.WriteLine(n * n * n);
    Console.WriteLine(3);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll n; cin >> n;
  cout << n * n * n << "\n";
  cout << 3 << "\n";

  return 0;
}
```
