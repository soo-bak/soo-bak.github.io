## 문제 링크
[24263번 - 알고리즘 수업 - 알고리즘의 수행 시간 2](https://www.acmicpc.net/problem/24263)

## 설명
주어진 알고리즘의 수행 횟수와 시간 복잡도의 최고차항 차수를 구하는 문제입니다.

<br>

## 접근법
함수는 1부터 n까지 한 번씩 반복하므로 수행 횟수는 n입니다.

수행 시간은 n에 선형으로 비례하므로 최고차항 차수는 1입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    Console.WriteLine(n);
    Console.WriteLine(1);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  cout << n << "\n" << 1 << "\n";

  return 0;
}
```
