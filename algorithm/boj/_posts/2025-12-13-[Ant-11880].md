## 문제 링크
[11880번 - 개미](https://www.acmicpc.net/problem/11880)

## 설명
직육면체의 서로 반대편 꼭짓점을 잇는 최단 경로의 제곱을 구하는 문제입니다.

<br>

## 접근법
개미가 직육면체 표면을 따라 이동할 때, 한 면을 펼치면 최단 경로는 직선이 됩니다.

가장 긴 변을 그대로 두고 나머지 두 변을 합친 방향으로 펼치는 것이 최단입니다.

최댓값을 m, 나머지 두 변의 합을 s라 하면 최단 거리의 제곱은 m의 제곱과 s의 제곱을 더한 값입니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.IO;

class Program {
  static void Main() {
    using var reader = new StreamReader(Console.OpenStandardInput());
    using var writer = new StreamWriter(Console.OpenStandardOutput());

    var t = int.Parse(reader.ReadLine()!);
    for (var i = 0; i < t; i++) {
      var line = reader.ReadLine()!.Split();
      var a = long.Parse(line[0]);
      var b = long.Parse(line[1]);
      var c = long.Parse(line[2]);
      var m = Math.Max(a, Math.Max(b, c));
      var s = a + b + c - m;
      writer.WriteLine(m * m + s * s);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    ll a, b, c; cin >> a >> b >> c;
    ll mx = max(a, max(b, c));
    ll sum = a + b + c - mx;
    cout << mx * mx + sum * sum << "\n";
  }

  return 0;
}
```
