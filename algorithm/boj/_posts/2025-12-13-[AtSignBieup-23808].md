## 문제 링크
[23808번 - 골뱅이 찍기 - ㅂ](https://www.acmicpc.net/problem/23808)

## 설명
ㅂ자 모양의 골뱅이 패턴을 출력하는 문제입니다.

<br>

## 접근법
전체 크기는 5n x 5n이 됩니다.

양쪽 세로줄은 왼쪽 n칸과 오른쪽 n칸에 해당합니다.

가운데 가로줄은 2n번째 행부터 3n번째 행 직전까지, 아래 가로줄은 4n번째 행부터 끝까지입니다.

이후, 위 조건 중 하나라도 만족하면 골뱅이를, 아니면 공백을 출력합니다.

<br>

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var size = 5 * n;
    var sb = new StringBuilder();
    for (var i = 0; i < size; i++) {
      for (var j = 0; j < size; j++) {
        var vertical = (j < n) || (j >= 4 * n);
        var midRow = (i >= 2 * n && i < 3 * n);
        var bottomRow = (i >= 4 * n);
        sb.Append(vertical || midRow || bottomRow ? '@' : ' ');
      }
      sb.Append('\n');
    }
    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  int size = 5 * n;
  for (int i = 0; i < size; i++) {
    for (int j = 0; j < size; j++) {
      bool vertical = (j < n) || (j >= 4 * n);
      bool midRow = (i >= 2 * n && i < 3 * n);
      bool bottomRow = (i >= 4 * n);
      cout << (vertical || midRow || bottomRow ? '@' : ' ');
    }
    cout << "\n";
  }

  return 0;
}
```
