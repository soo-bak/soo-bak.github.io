## 문제 링크
[23804번 - 골뱅이 찍기 - ㄷ](https://www.acmicpc.net/problem/23804)

## 설명
ㄷ자 모양의 골뱅이 패턴을 출력하는 문제입니다.

<br>

## 접근법
전체 높이는 5n이 됩니다.

첫 n줄과 마지막 n줄은 5n개의 골뱅이를 출력합니다.

가운데 3n줄은 왼쪽 n칸만 골뱅이로 채웁니다.

<br>

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();

    var full = new string('@', 5 * n);
    var left = new string('@', n);

    for (var i = 0; i < n; i++)
      sb.AppendLine(full);
    for (var i = 0; i < 3 * n; i++)
      sb.AppendLine(left);
    for (var i = 0; i < n; i++)
      sb.AppendLine(full);

    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  string full(5 * n, '@');
  string left(n, '@');

  for (int i = 0; i < n; i++)
    cout << full << "\n";
  for (int i = 0; i < 3 * n; i++)
    cout << left << "\n";
  for (int i = 0; i < n; i++)
    cout << full << "\n";

  return 0;
}
```
