## 문제 링크
[23803번 - 골뱅이 찍기 - ㄴ](https://www.acmicpc.net/problem/23803)

## 설명
ㄴ자 모양의 골뱅이 패턴을 출력하는 문제입니다.

<br>

## 접근법
전체 높이는 5n이 됩니다.

처음 4n줄은 왼쪽 n칸만 골뱅이로 채우고 줄바꿈합니다.

마지막 n줄은 5n개의 골뱅이를 모두 출력합니다.

<br>

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();

    var col = new string('@', n);
    var row = new string('@', 5 * n);

    for (var i = 0; i < 5 * n - n; i++)
      sb.AppendLine(col);

    for (var i = 0; i < n; i++)
      sb.AppendLine(row);

    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  string col(n, '@');
  string row(5 * n, '@');

  for (int i = 0; i < 5 * n - n; i++)
    cout << col << "\n";
  for (int i = 0; i < n; i++)
    cout << row << "\n";

  return 0;
}
```
