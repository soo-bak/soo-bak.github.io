## 문제 링크
[23806번 - 골뱅이 찍기 - ㅁ](https://www.acmicpc.net/problem/23806)

## 설명
ㅁ자 모양의 골뱅이 패턴을 출력하는 문제입니다.

<br>

## 접근법
전체 크기는 5n x 5n이 됩니다.

윗줄과 아랫줄의 두께 n만큼은 모두 골뱅이로 채웁니다.

가운데 3n줄은 앞뒤 n개만 골뱅이로, 가운데 3n개는 공백으로 출력합니다.

<br>

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();

    for (var i = 0; i < n; i++)
      sb.AppendLine(new string('@', 5 * n));

    var side = new string('@', n);
    var mid = new string(' ', 3 * n);
    for (var i = 0; i < 3 * n; i++)
      sb.AppendLine(side + mid + side);

    for (var i = 0; i < n; i++)
      sb.AppendLine(new string('@', 5 * n));

    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; 
  if (!(cin >> n)) return 0;
  string full(5 * n, '@');
  string side(n, '@');
  string mid(3 * n, ' ');

  for (int i = 0; i < n; i++)
    cout << full << "\n";
  for (int i = 0; i < 3 * n; i++)
    cout << side << mid << side << "\n";
  for (int i = 0; i < n; i++)
    cout << full << "\n";

  return 0;
}
```
