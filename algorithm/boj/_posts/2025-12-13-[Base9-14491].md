## 문제 링크
[14491번 - 9진수](https://www.acmicpc.net/problem/14491)

## 설명
10진수를 9진수로 변환하는 문제입니다.

<br>

## 접근법
9로 나눈 나머지를 구해 자릿수를 얻고, 몫이 0이 될 때까지 반복합니다.

얻은 나머지들을 역순으로 이어붙이면 9진수 표현이 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();
    while (n > 0) {
      sb.Append(n % 9);
      n /= 9;
    }
    for (var i = sb.Length - 1; i >= 0; i--)
      Console.Write(sb[i]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  string s;
  while (n > 0) {
    s.push_back(char('0' + (n % 9)));
    n /= 9;
  }
  reverse(s.begin(), s.end());
  cout << s << "\n";

  return 0;
}
```
