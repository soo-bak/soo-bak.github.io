## 문제 링크
[24365번 - ПЧЕЛИЧКАТА МАЯ](https://www.acmicpc.net/problem/24365)

## 설명
세 꽃에 앉은 벌을 옮겨 모두 같은 수가 되도록 할 때 최소 이동 거리를 구하는 문제입니다.

<br>

## 접근법
목표 개수는 전체 벌 수를 3으로 나눈 값입니다.

첫 번째 꽃에서 목표까지의 차이와, 앞 두 꽃의 합에서 목표 두 배까지의 차이를 더합니다.

이 두 절댓값의 합이 최소 이동 거리가 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var a = arr[0]; var b = arr[1]; var c = arr[2];
    var t = (a + b + c) / 3;
    var dist = Math.Abs(a - t) + Math.Abs(a + b - 2 * t);
    Console.WriteLine(dist);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, c;
  if (!(cin >> a >> b >> c)) return 0;
  int t = (a + b + c) / 3;
  int dist = abs(a - t) + abs(a + b - 2 * t);
  cout << dist << "\n";

  return 0;
}
```
