## 문제 링크
[2765번 - 자전거 속도](https://www.acmicpc.net/problem/2765)

## 설명
바퀴 지름과 회전수, 시간으로 이동 거리와 평균 속도를 구하는 문제입니다.

회전수가 0이면 입력이 종료됩니다.

<br>

## 접근법
이동 거리는 원주(π × 지름) × 회전수로 계산하고, 인치를 마일로 변환합니다.

평균 속도는 거리를 시간(시간 단위로 변환)으로 나눕니다.

결과를 소수 둘째 자리까지 출력합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var pi = 3.1415927;
    var inchPerMile = 12.0 * 5280.0;
    var secPerHour = 3600.0;

    var idx = 1;
    while (true) {
      var line = Console.ReadLine()!.Split();
      var dia = double.Parse(line[0]);
      var rev = double.Parse(line[1]);
      var time = double.Parse(line[2]);
      if (rev == 0) break;

      var dist = pi * dia * rev / inchPerMile;
      var mph = dist / (time / secPerHour);
      Console.WriteLine($"Trip #{idx}: {dist:F2} {mph:F2}");
      idx++;
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  const double PI = 3.1415927;
  const double INCH_PER_MILE = 12.0 * 5280.0;
  const double SEC_PER_HOUR = 3600.0;

  int idx = 1;
  double dia, time;
  double rev;
  while (cin >> dia >> rev >> time) {
    if (rev == 0) break;
    double dist = PI * dia * rev / INCH_PER_MILE;
    double mph = dist / (time / SEC_PER_HOUR);
    cout.setf(ios::fixed);
    cout << setprecision(2);
    cout << "Trip #" << idx << ": " << dist << " " << mph << "\n";
    idx++;
  }

  return 0;
}
```
