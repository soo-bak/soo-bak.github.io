## 문제 링크
[21631번 - Checkers](https://www.acmicpc.net/problem/21631)

## 설명
흰색과 검은색 말을 일렬로 쌓을 때 검은 연속 구간의 최대 개수를 구하는 문제입니다.

<br>

## 접근법
검은 구간을 최대화하려면 흰색과 검은색을 교차시켜 배치합니다.

흰색이 검은색 이상이면 모든 검은 말을 분리할 수 있으므로 답은 검은색 개수입니다.

흰색이 더 적으면 흰색으로 나눌 수 있는 구간은 흰색 개수에 1을 더한 값이 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Console.ReadLine()!.Split();
    var w = long.Parse(line[0]);
    var b = long.Parse(line[1]);

    var ans = (w >= b) ? b : w + 1;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll a, b;
  if (!(cin >> a >> b)) return 0;
  ll ans = (a >= b) ? b : a + 1;
  cout << ans << "\n";

  return 0;
}
```
