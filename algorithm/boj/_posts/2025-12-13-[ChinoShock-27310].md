## 문제 링크
[27310번 - :chino_shock:](https://www.acmicpc.net/problem/27310)

## 설명
이모지 문자열이 주어질 때 입력 난이도를 계산하는 문제입니다.

난이도는 전체 길이 + 콜론 개수 × 2 + 언더바 개수 × 5 로 정의됩니다.

문자열은 항상 콜론으로 시작하고 끝나므로 콜론 개수는 항상 2개입니다.

<br>

## 접근법
문자열의 길이를 구하고, 언더바 개수를 센 뒤 공식에 대입합니다.

콜론 개수는 항상 2이므로 전체 길이 + 2 + 언더바 개수 × 5 를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var cnt = 0;
    foreach (var c in s) if (c == '_') cnt++;
    var ans = s.Length + 2 + cnt * 5;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  int cnt = 0;
  for (char c : s) if (c == '_') ++cnt;
  cout << s.size() + 2 + cnt * 5 << "\n";

  return 0;
}
```
