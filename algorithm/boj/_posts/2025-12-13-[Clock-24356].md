## 문제 링크
[24356번 - ЧАСОВНИК](https://www.acmicpc.net/problem/24356)

## 설명
시작 시각과 종료 시각이 주어질 때 경과 시간과 30분 단위 완주 횟수를 구하는 문제입니다.

<br>

## 접근법
먼저 두 시각을 분 단위로 변환하여 차이를 구합니다.

종료 시각이 시작 시각보다 빠르면 하루를 더해줍니다.

이후 경과 시간을 30으로 나눈 몫이 완주 횟수가 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var t1 = line[0]; var m1 = line[1]; var t2 = line[2]; var m2 = line[3];

    var start = t1 * 60 + m1;
    var end = t2 * 60 + m2;
    if (end < start) end += 24 * 60;

    var diff = end - start;
    Console.WriteLine($"{diff} {diff / 30}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t1, m1, t2, m2;
  if (!(cin >> t1 >> m1 >> t2 >> m2)) return 0;
  int start = t1 * 60 + m1;
  int end = t2 * 60 + m2;
  if (end < start) end += 24 * 60;
  int diff = end - start;
  cout << diff << ' ' << diff / 30 << "\n";

  return 0;
}
```
