## 문제 링크
[21866번 - 추첨을 통해 커피를 받자](https://www.acmicpc.net/problem/21866)

## 설명
9문제 점수를 보고 해커인지, 추첨 대상인지, 아닌지를 판정하는 문제입니다.

<br>

## 접근법
먼저 각 문제의 배점을 초과했는지 확인합니다.

하나라도 초과하면 해커로 판정합니다.

해커가 아니라면 총점이 100 이상이면 draw, 아니면 none을 출력합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Console.ReadLine()!.Split();
    var max = new[] {100, 100, 200, 200, 300, 300, 400, 400, 500};
    var hacker = false;
    var sum = 0;
    for (var i = 0; i < 9; i++) {
      var v = int.Parse(line[i]);
      sum += v;
      if (v > max[i]) hacker = true;
    }

    if (hacker) Console.WriteLine("hacker");
    else if (sum >= 100) Console.WriteLine("draw");
    else Console.WriteLine("none");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  vi maxScore = {100, 100, 200, 200, 300, 300, 400, 400, 500};
  vi v(9);
  for (int i = 0; i < 9; i++)
    cin >> v[i];

  bool hacker = false;
  int sum = 0;
  for (int i = 0; i < 9; i++) {
    sum += v[i];
    if (v[i] > maxScore[i]) hacker = true;
  }

  if (hacker) cout << "hacker\n";
  else if (sum >= 100) cout << "draw\n";
  else cout << "none\n";

  return 0;
}
```
