## 문제 링크
[15051번 - Máquina de café](https://www.acmicpc.net/problem/15051)

## 설명
커피머신을 어느 층에 두어야 전체 이동 시간이 최소인지 구하는 문제입니다.

<br>

## 접근법
각 층에 머신을 뒀을 때의 총 이동 시간을 계산합니다.

모든 직원이 왕복하므로 층 차이에 2를 곱해 비용을 계산합니다.

세 경우 중 최솟값을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var a = new int[3];
    for (var i = 0; i < 3; i++)
      a[i] = int.Parse(Console.ReadLine()!);

    var cost1 = 0 * a[0] + 2 * a[1] + 4 * a[2];
    var cost2 = 2 * a[0] + 0 * a[1] + 2 * a[2];
    var cost3 = 4 * a[0] + 2 * a[1] + 0 * a[2];

    var ans = Math.Min(cost1, Math.Min(cost2, cost3));
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a1, a2, a3;
  if (!(cin >> a1 >> a2 >> a3)) return 0;

  int cost1 = 0 * a1 + 2 * a2 + 4 * a3;
  int cost2 = 2 * a1 + 0 * a2 + 2 * a3;
  int cost3 = 4 * a1 + 2 * a2 + 0 * a3;

  int ans = min(cost1, min(cost2, cost3));
  cout << ans << "\n";

  return 0;
}
```
