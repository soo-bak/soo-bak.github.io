## 문제 링크
[17945번 - 통학의 신](https://www.acmicpc.net/problem/17945)

## 설명
이차방정식의 두 정수근을 오름차순으로 구하는 문제입니다.

<br>

## 접근법
근의 공식을 적용하면 판별식은 a의 제곱에서 b를 뺀 값입니다.

두 근은 -a에서 판별식의 제곱근을 더하거나 뺀 값이 됩니다.

두 근이 같으면 하나만, 다르면 오름차순으로 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var a = line[0]; var b = line[1];
    var d = a * a - b;
    var r1 = -a - (int)Math.Sqrt(d);
    var r2 = -a + (int)Math.Sqrt(d);
    if (r1 == r2) Console.WriteLine(r1);
    else {
      if (r1 < r2) Console.WriteLine($"{r1} {r2}");
      else Console.WriteLine($"{r2} {r1}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b;
  if (!(cin >> a >> b)) return 0;
  int d = a * a - b;
  int r1 = -a - static_cast<int>(sqrt(d));
  int r2 = -a + static_cast<int>(sqrt(d));
  if (r1 == r2) cout << r1 << "\n";
  else {
    if (r1 > r2) swap(r1, r2);
    cout << r1 << ' ' << r2 << "\n";
  }

  return 0;
}
```
