## 문제 링크
[25828번 - Corona Virus Testing](https://www.acmicpc.net/problem/25828)

## 설명
개인 검사와 그룹 검사 중 키트를 더 적게 사용하는 방식을 구하는 문제입니다.

<br>

## 접근법
개인 검사는 그룹 수와 그룹당 인원을 곱한 값입니다.

그룹 검사는 그룹 수에 양성 그룹에서 추가로 필요한 키트를 더한 값입니다.

두 값을 비교하여 개인이 적으면 1, 그룹이 적으면 2, 같으면 0을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var g = line[0]; var p = line[1]; var t = line[2];

    var indiv = g * p;
    var group = g + t * p;

    if (indiv < group) Console.WriteLine(1);
    else if (indiv > group) Console.WriteLine(2);
    else Console.WriteLine(0);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>

  else cout << 0 << "\n";

```
