## 문제 링크
[9517번 - 아이 러브 크로아티아](https://www.acmicpc.net/problem/9517)

## 설명
8명이 원으로 앉아 폭탄을 돌리며 퀴즈를 풀 때, 210초가 되는 순간 폭탄을 들고 있는 사람을 구하는 문제입니다.

정답이면 다음 사람에게 넘기고, 오답이나 패스면 넘기지 않습니다.

<br>

## 접근법
경과 시간과 현재 폭탄 보유자를 추적하며 시뮬레이션합니다.

각 질문 시간만큼 더해 210초 이상이 되면 즉시 종료합니다.

아직 210초 미만이고 정답이면 다음 사람으로 넘깁니다.

8번 다음은 1번으로 순환합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var cur = int.Parse(Console.ReadLine()!);
    var n = int.Parse(Console.ReadLine()!);
    var elapsed = 0;
    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!.Split();
      var t = int.Parse(line[0]);
      var res = line[1][0];
      elapsed += t;
      if (elapsed >= 210) break;
      if (res == 'T') {
        cur++;
        if (cur == 9) cur = 1;
      }
    }
    Console.WriteLine(cur);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {

  int cur, n;
  if (!(cin >> cur)) return 0;
  cin >> n;
  int elapsed = 0;
  for (int i = 0; i < n; i++) {
    int t; char res;
    cin >> t >> res;
    elapsed += t;
    if (elapsed >= 210) break;
    if (res == 'T') {
      cur++;
      if (cur == 9) cur = 1;
    }
  }
  cout << cur << "\n";

  return 0;
}
```
