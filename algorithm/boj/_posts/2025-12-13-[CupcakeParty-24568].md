## 문제 링크
[24568번 - Cupcake Party](https://www.acmicpc.net/problem/24568)

## 설명
컵케이크를 28명의 학생에게 나눠준 뒤 남는 개수를 구하는 문제입니다.

<br>

## 접근법
일반 박스에는 8개, 작은 박스에는 3개의 컵케이크가 들어 있습니다.

총 컵케이크 수에서 28을 빼면 남는 개수가 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var r = int.Parse(Console.ReadLine()!);
    var s = int.Parse(Console.ReadLine()!);
    Console.WriteLine(8 * r + 3 * s - 28);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int r, s; cin >> r >> s;
  cout << 8 * r + 3 * s - 28 << "\n";

  return 0;
}
```
