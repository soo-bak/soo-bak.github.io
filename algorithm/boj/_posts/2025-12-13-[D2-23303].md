## 문제 링크
[23303번 - 이 문제는 D2 입니다.](https://www.acmicpc.net/problem/23303)

## 설명
문자열에 D2 또는 d2가 포함되어 있는지 확인하는 문제입니다.

<br>

## 접근법
문자열에서 두 패턴을 검사합니다.

하나라도 포함되면 D2, 아니면 unrated를 출력합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var ok = s.Contains("D2") || s.Contains("d2");
    Console.WriteLine(ok ? "D2" : "unrated");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s;
  if (!getline(cin, s)) return 0;
  if (s.find("D2") != string::npos || s.find("d2") != string::npos)
    cout << "D2\n";
  else cout << "unrated\n";

  return 0;
}
```
