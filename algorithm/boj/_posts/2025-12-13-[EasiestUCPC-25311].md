## 문제 링크
[25311번 - UCPC에서 가장 쉬운 문제 번호는?](https://www.acmicpc.net/problem/25311)

## 설명
UCPC 예선에서 가장 쉬운 문제 번호를 출력하는 문제입니다.

<br>

## 접근법
어떤 연도든 가장 쉬운 문제는 항상 A번입니다.

연도와 상관없이 A를 출력하면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    Console.ReadLine();
    Console.WriteLine("A");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int y; cin >> y;
  cout << "A\n";

  return 0;
}
```
