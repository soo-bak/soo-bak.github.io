## 문제 링크
[18411번 - 試験 (Exam)](https://www.acmicpc.net/problem/18411)

## 설명
세 시험 점수 중 가장 높은 두 점수의 합을 구하는 문제입니다.

<br>

## 접근법
세 값을 정렬한 뒤 상위 두 개를 더합니다.

또는 총합에서 최솟값을 빼도 같은 결과를 얻을 수 있습니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    Array.Sort(arr);
    Console.WriteLine(arr[1] + arr[2]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  vi v(3);
  for (int i = 0; i < 3; i++)
    cin >> v[i];
  sort(v.begin(), v.end());
  cout << v[1] + v[2] << "\n";

  return 0;
}
```
