## 문제 링크
[15633번 - Fan Death](https://www.acmicpc.net/problem/15633)

## 설명
자연수 n의 모든 약수 합에 5를 곱하고 24를 뺀 값을 구하는 문제입니다.

<br>

## 접근법
약수 합을 구할 때 n의 제곱근까지만 순회하면 효율적입니다.

i가 n의 약수이면 n을 i로 나눈 값도 약수이므로, 두 값이 같지 않으면 둘 다 더합니다.

약수 합을 구한 뒤 5를 곱하고 24를 빼면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sum = 0;
    for (var i = 1; i * i <= n; i++) {
      if (n % i != 0) continue;
      var pair = n / i;
      if (i == pair) sum += i;
      else sum += i + pair;
    }
    var ans = 5 * sum - 24;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp

  }
```
