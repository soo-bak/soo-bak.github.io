## 문제 링크
[14761번 - FizzBuzz](https://www.acmicpc.net/problem/14761)

## 설명
1부터 n까지 순회하며 조건에 따라 Fizz, Buzz, FizzBuzz 또는 숫자를 출력하는 문제입니다.

<br>

## 접근법
x와 y 모두의 배수인지 먼저 확인해 FizzBuzz를 출력하고, 그렇지 않으면 x나 y 각각의 배수인지 확인합니다.

어떤 조건에도 해당하지 않으면 숫자를 그대로 출력합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Console.ReadLine()!.Split();
    var x = int.Parse(line[0]);
    var y = int.Parse(line[1]);
    var n = int.Parse(line[2]);

    for (var i = 1; i <= n; i++) {
      var fx = (i % x == 0);
      var fy = (i % y == 0);
      if (fx && fy) Console.WriteLine("FizzBuzz");
      else if (fx) Console.WriteLine("Fizz");
      else if (fy) Console.WriteLine("Buzz");
      else Console.WriteLine(i);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int x, y, n;
  if (!(cin >> x >> y >> n)) return 0;

  for (int i = 1; i <= n; i++) {
    bool fx = (i % x == 0);
    bool fy = (i % y == 0);
    if (fx && fy) cout << "FizzBuzz\n";
    else if (fx) cout << "Fizz\n";
    else if (fy) cout << "Buzz\n";
    else cout << i << "\n";
  }

  return 0;
}
```
