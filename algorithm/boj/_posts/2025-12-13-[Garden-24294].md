## 문제 링크
[24294번 - ГРАДИНА](https://www.acmicpc.net/problem/24294)

## 설명
두 직사각형 화단의 외곽 타일 개수를 구하는 문제입니다.

<br>

## 접근법
두 직사각형을 위아래로 붙이면 전체 높이는 두 높이의 합이 됩니다.

전체 너비는 둘 중 큰 값이고, 꺾이는 부분이 추가로 생깁니다.

둘레를 계산하고 꺾이는 부분까지 더해 타일 수를 구합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var v = new int[4];
    for (var i = 0; i < 4; i++)
      v[i] = int.Parse(Console.ReadLine()!);

    var ans = 2 * Math.Max(v[0], v[2]) + 2 * (v[1] + v[3]) + 4;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int w1, h1, w2, h2;
  if (!(cin >> w1 >> h1 >> w2 >> h2)) return 0;

  int ans = 2 * max(w1, w2) + 2 * (h1 + h2) + 4;
  cout << ans << "\n";

  return 0;
}
```
