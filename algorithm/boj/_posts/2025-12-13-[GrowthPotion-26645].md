## 문제 링크
[26645번 - 성장의 비약 선택권](https://www.acmicpc.net/problem/26645)

## 설명
현재 레벨에서 사용 가능한 포션 중 가장 높은 레벨에 도달할 수 있는 포션을 선택하는 문제입니다.

<br>

## 접근법
각 포션의 사용 가능 범위와 개수를 고려하여 최적의 포션을 선택합니다.

1번 포션은 5개로 209까지 사용 가능하므로, 레벨이 205 이하면 1번이 최적입니다.

마찬가지로 2번은 217 이하, 3번은 228 이하, 4번은 239 이하일 때 최적이 됩니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var lev = int.Parse(Console.ReadLine()!);
    var ans = 0;
    if (lev <= 209 - 4) ans = 1;
    else if (lev <= 219 - 2) ans = 2;
    else if (lev <= 229 - 1) ans = 3;
    else if (lev <= 239) ans = 4;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int lev;
  if (!(cin >> lev)) return 0;
  int ans = 0;
  if (lev <= 209 - 4) ans = 1;
  else if (lev <= 219 - 2) ans = 2;
  else if (lev <= 229 - 1) ans = 3;
  else if (lev <= 239) ans = 4;
  cout << ans << "\n";

  return 0;
}
```
