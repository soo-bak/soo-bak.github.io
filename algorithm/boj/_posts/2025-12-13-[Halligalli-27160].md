## 문제 링크
[27160번 - 할리갈리](https://www.acmicpc.net/problem/27160)

## 설명
과일별 개수 합이 정확히 5인 것이 있는지 판단하는 문제입니다.

<br>

## 접근법
각 과일 이름의 첫 글자가 모두 다르므로 이를 기준으로 종류를 구분합니다.

종류별로 개수를 누적한 뒤, 합이 5인 과일이 있으면 YES, 없으면 NO를 출력합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sum = new int[4];
    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!.Split();
      var c = line[0][0];
      var x = int.Parse(line[1]);
      switch (c) {
        case 'S': sum[0] += x; break;
        case 'B': sum[1] += x; break;
        case 'L': sum[2] += x; break;
        case 'P': sum[3] += x; break;
      }
    }
    var ok = false;
    foreach (var v in sum)
      if (v == 5) ok = true;
    Console.WriteLine(ok ? "YES" : "NO");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  int sum[4] = {0, 0, 0, 0};
  for (int i = 0; i < n; i++) {
    string s; int x;
    cin >> s >> x;
    char c = s[0];
    if (c == 'S') sum[0] += x;
    else if (c == 'B') sum[1] += x;
    else if (c == 'L') sum[2] += x;
    else if (c == 'P') sum[3] += x;
  }
  bool ok = false;
  for (int v : sum) if (v == 5) ok = true;
  cout << (ok ? "YES" : "NO") << "\n";

  return 0;
}
```
