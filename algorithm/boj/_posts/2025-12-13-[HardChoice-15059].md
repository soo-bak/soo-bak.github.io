## 문제 링크
[15059번 - Hard choice](https://www.acmicpc.net/problem/15059)

## 설명
세 종류 식사의 보유량과 요청량이 주어질 때, 부족한 식사 수의 합을 구하는 문제입니다.

<br>

## 접근법
각 종류에 대해 요청량에서 보유량을 뺀 값이 양수이면 그만큼 부족합니다.

세 종류 모두에 대해 부족한 수를 합산하여 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var avail = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var need = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var lack = 0;
    for (var i = 0; i < 3; i++)
      if (avail[i] < need[i]) lack += need[i] - avail[i];
    Console.WriteLine(lack);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a[3], b[3];
  for (int i = 0; i < 3; i++)
    cin >> a[i];
  for (int i = 0; i < 3; i++)
    cin >> b[i];
  int lack = 0;
  for (int i = 0; i < 3; i++)
    if (a[i] < b[i]) lack += b[i] - a[i];
  cout << lack << "\n";

  return 0;
}
```
