## 문제 링크
[24086번 - 身長 (Height)](https://www.acmicpc.net/problem/24086)

## 설명
작년 키와 현재 키가 주어질 때, 키가 얼마나 자랐는지 구하는 문제입니다.

<br>

## 접근법
현재 키에서 작년 키를 빼면 증가분이 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var a = int.Parse(Console.ReadLine()!);
    var b = int.Parse(Console.ReadLine()!);
    Console.WriteLine(b - a);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b; cin >> a >> b;
  cout << (b - a) << "\n";

  return 0;
}
```
