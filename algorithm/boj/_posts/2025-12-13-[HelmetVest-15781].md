## 문제 링크
[15781번 - 헬멧과 조끼](https://www.acmicpc.net/problem/15781)

## 설명
헬멧과 조끼를 각각 하나씩 착용할 때, 얻을 수 있는 최대 방어력 합을 구하는 문제입니다.

<br>

## 접근법
헬멧 중 방어력이 가장 높은 것과 조끼 중 방어력이 가장 높은 것을 선택하면 됩니다.

각각의 최댓값을 구해서 더한 값이 정답입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var nm = Console.ReadLine()!.Split();
    var n = int.Parse(nm[0]);
    var m = int.Parse(nm[1]);

    var lineH = Console.ReadLine()!.Split();
    var maxH = 0;
    for (var i = 0; i < n; i++) {
      var h = int.Parse(lineH[i]);
      if (h > maxH) maxH = h;
    }

    var lineV = Console.ReadLine()!.Split();
    var maxV = 0;
    for (var i = 0; i < m; i++) {
      var v = int.Parse(lineV[i]);
      if (v > maxV) maxV = v;
    }

    Console.WriteLine(maxH + maxV);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;
  int maxH = 0, maxV = 0;
  for (int i = 0; i < n; i++) {
    int h; cin >> h;
    if (h > maxH) maxH = h;
  }
  for (int i = 0; i < m; i++) {
    int v; cin >> v;
    if (v > maxV) maxV = v;
  }
  cout << maxH + maxV << "\n";

  return 0;
}
```
