## 문제 링크
[24072번 - 帰省 (Homecoming)](https://www.acmicpc.net/problem/24072)

## 설명
귀성 기간과 방문 날짜를 비교하여 만날 수 있는지 판단하는 문제입니다.

<br>

## 접근법
비타로는 a일 후 오전에 도착해서 b일 후 오전에 떠납니다.

비바코는 c일 후 오후에 방문하므로, c가 a 이상이고 b 미만이면 만날 수 있습니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var a = arr[0]; var b = arr[1]; var c = arr[2];
    var ans = (a <= c && c < b) ? 1 : 0;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, c;
  if (!(cin >> a >> b >> c)) return 0;
  int ans = (a <= c && c < b) ? 1 : 0;
  cout << ans << "\n";

  return 0;
}
```
