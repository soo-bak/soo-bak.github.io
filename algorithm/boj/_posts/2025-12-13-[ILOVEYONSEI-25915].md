## 문제 링크
[25915번 - 연세여 사랑한다](https://www.acmicpc.net/problem/25915)

## 설명
알파벳 석판 위에서 비밀번호를 순서대로 밟을 때 이동 거리의 최소 합을 구하는 문제입니다.

<br>

## 접근법
현재 위치에서 시작하여 비밀번호 문자열을 순회합니다.

각 문자로 이동할 때 두 문자 사이의 알파벳 인덱스 차이를 누적합니다.

이후, 현재 위치를 갱신하며 반복합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var cur = Console.ReadLine()![0];
    var target = "ILOVEYONSEI";
    var dist = 0;
    foreach (var ch in target) {
      dist += Math.Abs(ch - cur);
      cur = ch;
    }
    Console.WriteLine(dist);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  char cur; 
  if (!(cin >> cur)) return 0;
  const string target = "ILOVEYONSEI";
  int dist = 0;
  for (char ch : target) {
    dist += abs(ch - cur);
    cur = ch;
  }
  cout << dist << "\n";

  return 0;
}
```
