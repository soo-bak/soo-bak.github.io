## 문제 링크
[11257번 - IT Passport Examination](https://www.acmicpc.net/problem/11257)

## 설명
IT Passport 시험의 합격 여부를 판정하는 문제입니다.

<br>

## 접근법
총점이 55점 이상이고, 각 영역이 해당 만점의 30% 이상이면 합격입니다.

30% 비교는 정수 연산으로 처리하여 실수 오차를 피합니다.

모든 조건을 만족하면 PASS, 아니면 FAIL을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    const int maxS = 35, maxM = 25, maxT = 40;
    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!.Split();
      var id = line[0];
      var s = int.Parse(line[1]);
      var m = int.Parse(line[2]);
      var t = int.Parse(line[3]);
      var sum = s + m + t;

      var pass = sum >= 55;
      pass &= s * 10 >= maxS * 3;
      pass &= m * 10 >= maxM * 3;
      pass &= t * 10 >= maxT * 3;

      Console.WriteLine($"{id} {sum} {(pass ? "PASS" : "FAIL")}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  const int maxS = 35, maxM = 25, maxT = 40;
  int n; 
  if (!(cin >> n)) return 0;
  for (int i = 0; i < n; i++) {
    string id; int s, m, t;
    cin >> id >> s >> m >> t;
    int sum = s + m + t;
    bool pass = sum >= 55;
    pass &= s * 10 >= maxS * 3;
    pass &= m * 10 >= maxM * 3;
    pass &= t * 10 >= maxT * 3;
    cout << id << ' ' << sum << ' ' << (pass ? "PASS" : "FAIL") << "\n";
  }

  return 0;
}
```
