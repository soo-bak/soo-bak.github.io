## 문제 링크
[20540번 - 연길이의 이상형](https://www.acmicpc.net/problem/20540)

## 설명
주어진 MBTI의 각 지표를 반대 지표로 바꾸어 출력하는 문제입니다.

<br>

## 접근법
각 자리의 문자를 반대 문자로 변환합니다.

E는 I로, S는 N으로, T는 F로, J는 P로 바꾸고 그 반대도 동일합니다.

길이가 4로 고정이므로 각 위치별로 조건문을 사용하면 됩니다.

<br>

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var sb = new StringBuilder();
    sb.Append(s[0] == 'E' ? 'I' : 'E');
    sb.Append(s[1] == 'S' ? 'N' : 'S');
    sb.Append(s[2] == 'T' ? 'F' : 'T');
    sb.Append(s[3] == 'J' ? 'P' : 'J');
    Console.WriteLine(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {

  string s;

  string ans;
  ans.push_back(s[0] == 'E' ? 'I' : 'E');
  ans.push_back(s[1] == 'S' ? 'N' : 'S');
  ans.push_back(s[2] == 'T' ? 'F' : 'T');
  ans.push_back(s[3] == 'J' ? 'P' : 'J');
  cout << ans << "\n";

  return 0;
}
```
