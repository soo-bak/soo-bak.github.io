## 문제 링크
[24623번 - Изгороди](https://www.acmicpc.net/problem/24623)

## 설명
두 도랑의 끝을 울타리로 이었을 때, 울타리들이 이루는 각도를 구하는 문제입니다.

<br>

## 접근법
기하학적으로 두 울타리는 항상 직각을 이룹니다.

따라서 어떤 입력이 들어와도 정답은 90입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    Console.ReadLine();
    Console.ReadLine();
    Console.WriteLine(90);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, alpha; cin >> n >> alpha;
  cout << 90 << "\n";

  return 0;
}
```
