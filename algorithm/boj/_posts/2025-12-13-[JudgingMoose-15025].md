## 문제 링크
[15025번 - Judging Moose](https://www.acmicpc.net/problem/15025)

## 설명
무스 뿔의 왼쪽과 오른쪽 타인 수가 주어질 때, 종류와 포인트를 출력하는 문제입니다.

<br>

## 접근법
둘 다 0이면 Not a moose를 출력합니다.

두 수가 같으면 Even과 함께 두 수의 합을 출력합니다.

두 수가 다르면 Odd와 함께 큰 값의 2배를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Console.ReadLine()!.Split();
    var l = int.Parse(line[0]);
    var r = int.Parse(line[1]);

    if (l == 0 && r == 0) {
      Console.WriteLine("Not a moose");
      return;
    }

    if (l == r) Console.WriteLine($"Even {l + r}");
    else Console.WriteLine($"Odd {2 * Math.Max(l, r)}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int l, r; cin >> l >> r;
  if (l == 0 && r == 0) {
    cout << "Not a moose\n";
    return 0;
  }
  if (l == r) cout << "Even " << (l + r) << "\n";
  else cout << "Odd " << 2 * max(l, r) << "\n";

  return 0;
}
```
