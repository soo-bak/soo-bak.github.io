## 문제 링크
[13311번 - 행운의 편지](https://www.acmicpc.net/problem/13311)

## 설명
2부터 1000까지 모든 a에 대해 특정 나머지 조건을 만족하는 정수를 구하는 문제입니다.

<br>

## 접근법
-1은 어떤 양수 a로 나누어도 나머지가 a-1이 됩니다.

따라서 -1을 출력하면 모든 조건을 만족합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    Console.WriteLine("-1");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  cout << "-1\n";

  return 0;
}
```
