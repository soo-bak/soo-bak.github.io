## 문제 링크
[15921번 - 수찬은 마린보이야!!](https://www.acmicpc.net/problem/15921)

## 설명
연습 기록이 주어질 때 확률은 `(평균) / (기댓값)`이다. 기댓값 정의상 동일한 데이터에서 평균과 기댓값은 항상 같으므로, 데이터가 존재하면 값은 `1`. 단, N=0이면 데이터가 없어 "divide by zero"를 출력해야 한다.

## 접근법
N=0이면 바로 `"divide by zero"`. 그렇지 않으면 평균과 기댓값이 같으므로 결과는 항상 `1.00`이다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    int n = int.Parse(Console.ReadLine()!);
    if (n == 0) {
      Console.WriteLine("divide by zero");
      return;
    }
    Console.ReadLine(); // 기록들은 읽기만 하면 됨
    Console.WriteLine("1.00");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  if (n == 0) {
    cout << "divide by zero\n";
    return 0;
  }
  // 남은 n개는 읽어서 버린다
  for (int i = 0; i < n; i++) {
    int x; cin >> x;
  }
  cout << "1.00\n";
  return 0;
}
```
