## 문제 링크
[14720번 - 우유 축제](https://www.acmicpc.net/problem/14720)

## 설명
우유 가게가 일렬로 있을 때, 딸기→초코→바나나→딸기 순서로만 우유를 마실 수 있습니다.

앞에서부터 지나가며 최대 몇 팩을 마실 수 있는지 구하는 문제입니다.

<br>

## 접근법
다음에 마셔야 할 우유 종류를 기록해 둡니다.

가게를 순서대로 지나가면서 현재 가게의 우유가 마셔야 할 종류와 같으면 마시고, 다음 종류로 넘어갑니다.

종류는 0, 1, 2가 순환하므로 3으로 나눈 나머지를 사용합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var line = Console.ReadLine()!.Split();
    var need = 0;
    var cnt = 0;
    for (var i = 0; i < n; i++) {
      var milk = int.Parse(line[i]);
      if (milk == need) {
        cnt++;
        need = (need + 1) % 3;
      }
    }
    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi shop(n);
  for (int i = 0; i < n; i++)
    cin >> shop[i];

  int need = 0, cnt = 0;
  for (int x : shop) {
    if (x == need) {
      ++cnt;
      need = (need + 1) % 3;
    }
  }

  cout << cnt << "\n";

  return 0;
}
```
