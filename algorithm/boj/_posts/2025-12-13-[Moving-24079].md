## 문제 링크
[24079번 - 移動 (Moving)](https://www.acmicpc.net/problem/24079)

## 설명
A에서 B를 거쳐 C까지 이동할 때, 허용 시간 내에 도착 가능한지 판정하는 문제입니다.

<br>

## 접근법
총 이동 시간은 X + Y 시간이고, 허용 시간은 Z시간 30분입니다.

이동 시간이 정수 단위이므로, X + Y가 Z 이하이면 무조건 가능합니다.

X + Y가 Z + 1 이상이면 30분을 더해도 부족하므로 불가능합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var x = int.Parse(Console.ReadLine()!);
    var y = int.Parse(Console.ReadLine()!);
    var z = int.Parse(Console.ReadLine()!);

    Console.WriteLine(x + y <= z ? 1 : 0);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int x, y, z; cin >> x >> y >> z;
  cout << (x + y <= z ? 1 : 0) << "\n";

  return 0;
}
```
