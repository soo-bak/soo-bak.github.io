## 문제 링크
[5893번 - 17배](https://www.acmicpc.net/problem/5893)

## 설명
이진수가 주어질 때, 17배한 결과를 이진수로 출력하는 문제입니다.

<br>

## 접근법
17은 16 더하기 1이므로, 원래 수에 16을 곱한 값과 원래 수를 더하면 됩니다.

이진수에서 16을 곱하는 것은 뒤에 0을 네 개 붙이는 것과 같습니다.

두 이진수 문자열을 뒤에서부터 한 자리씩 더하면서 올림을 처리하면 결과를 구할 수 있습니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static string Add(string a, string b) {
    var i = a.Length - 1;
    var j = b.Length - 1;
    var carry = 0;
    var sb = new StringBuilder(Math.Max(a.Length, b.Length) + 1);
    while (i >= 0 || j >= 0 || carry > 0) {
      var sum = carry;
      if (i >= 0) sum += a[i--] - '0';
      if (j >= 0) sum += b[j--] - '0';
      sb.Append((char)('0' + (sum & 1)));
      carry = sum >> 1;
    }
    var arr = sb.ToString().ToCharArray();
    Array.Reverse(arr);
    return new string(arr);
  }

  static void Main() {
    var n = Console.ReadLine()!;
    var shifted = n + "0000";
    var ans = Add(n, shifted);
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

string addBin(const string& a, const string& b) {
  int i = (int)a.size() - 1, j = (int)b.size() - 1, carry = 0;
  string res;
  res.reserve(max(a.size(), b.size()) + 1);
  while (i >= 0 || j >= 0 || carry) {
    int sum = carry;
    if (i >= 0) sum += a[i--] - '0';
    if (j >= 0) sum += b[j--] - '0';
    res.push_back(char('0' + (sum & 1)));
    carry = sum >> 1;
  }
  reverse(res.begin(), res.end());
  return res;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string n; cin >> n;
  string shifted = n + "0000";
  cout << addBin(n, shifted) << "\n";

  return 0;
}
```
