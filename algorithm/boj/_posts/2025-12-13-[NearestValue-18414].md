## 문제 링크
[18414번 - X に最も近い値 (The Nearest Value)](https://www.acmicpc.net/problem/18414)

## 설명
구간 [L, R] 안에서 X에 가장 가까운 값을 구하는 문제입니다.

<br>

## 접근법
X가 구간 안에 있으면 X가 답입니다.

X가 L보다 작으면 L이, X가 R보다 크면 R이 답입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var p = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var x = p[0]; var l = p[1]; var r = p[2];
    var ans = 0;
    if (x < l) ans = l;
    else if (x > r) ans = r;
    else ans = x;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int x, l, r; cin >> x >> l >> r;
  int ans;
  if (x < l) ans = l;
  else if (x > r) ans = r;
  else ans = x;
  cout << ans << "\n";

  return 0;
}
```
