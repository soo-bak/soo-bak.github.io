## 문제 링크
[27159번 - 노 땡스!](https://www.acmicpc.net/problem/27159)

## 설명
연속된 카드를 하나의 그룹으로 묶고 각 그룹의 최소값 합을 구하는 문제입니다.

<br>

## 접근법
카드가 오름차순이므로 각 그룹의 첫 카드가 항상 최소값입니다.

연속이 끊기는 지점마다 새 그룹이 시작되므로, 해당 카드를 점수에 더합니다.

예를 들어 [3, 4, 5, 8, 9]는 [3, 4, 5]와 [8, 9] 두 그룹이고, 최소값 3과 8을 더해 총점 11이 됩니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var ans = arr[0];
    for (var i = 1; i < n; i++) {
      if (arr[i] - arr[i - 1] != 1) ans += arr[i];
    }
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  vi v(n);
  for (int i = 0; i < n; i++)
    cin >> v[i];

  int ans = v[0];
  for (int i = 1; i < n; i++) {
    if (v[i] - v[i - 1] != 1) ans += v[i];
  }
  cout << ans << "\n";

  return 0;
}
```
