## 문제 링크
[27465번 - 소수가 아닌 수](https://www.acmicpc.net/problem/27465)

## 설명
정수 n이 주어질 때, n 이상이면서 소수가 아닌 수를 하나 찾는 문제입니다.

답은 10⁹ 이하여야 하며, 여러 답 중 아무거나 출력해도 됩니다.

<br>

## 접근법
짝수는 2를 제외하면 모두 소수가 아닙니다.

따라서 n 이상의 가장 작은 짝수를 찾으면 됩니다.

<br>

먼저 n이 1 이하이면 1을 출력합니다. 1은 소수가 아니므로 조건을 만족합니다.

다음으로 n이 4 이하이면 4를 출력합니다. 2와 3은 소수이므로, 2 이상에서 가장 작은 합성수인 4를 선택합니다.

이후 n이 5 이상이면, n이 짝수일 때 n을, 홀수일 때 n + 1을 출력합니다.

<br>

이 방법은 소수 판별 없이 O(1)에 답을 구할 수 있습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var ans = n <= 1 ? 1 : n <= 4 ? 4 : n % 2 == 0 ? n : n + 1;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int ans = n <= 1 ? 1 : n <= 4 ? 4 : n % 2 == 0 ? n : n + 1;
  cout << ans << "\n";

  return 0;
}
```
