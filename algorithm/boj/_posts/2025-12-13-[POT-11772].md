## 문제 링크
[11772번 - POT](https://www.acmicpc.net/problem/11772)

## 설명
각 수를 밑과 지수로 분리하여 거듭제곱의 합을 구하는 문제입니다.

<br>

## 접근법
각 수의 마지막 자리는 지수이고, 나머지 앞부분은 밑입니다.

10으로 나눈 몫과 나머지로 밑과 지수를 분리합니다.

거듭제곱 값을 모두 더해 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sum = 0L;
    for (var i = 0; i < n; i++) {
      var p = int.Parse(Console.ReadLine()!);
      var baseVal = p / 10;
      var exp = p % 10;
      sum += (long)Math.Pow(baseVal, exp);
    }
    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  ll sum = 0;
  for (int i = 0; i < n; i++) {
    int p; cin >> p;
    int baseVal = p / 10;
    int exp = p % 10;
    sum += (ll)pow(baseVal, exp);
  }
  cout << sum << "\n";

  return 0;
}
```
