## 문제 링크
[16504번 - 종이접기](https://www.acmicpc.net/problem/16504)

## 설명
종이를 끝까지 접었을 때 최종적으로 남는 수를 구하는 문제입니다.

반씩 접으며 겹치는 수를 더하면 결국 모든 칸의 합이 됩니다.

<br>

## 접근법
모든 칸의 값을 더한 합이 답입니다.

합이 커질 수 있으므로 64비트 정수를 사용합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sum = 0L;
    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!.Split();
      for (var j = 0; j < n; j++)
        sum += long.Parse(line[j]);
    }
    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  ll sum = 0;
  for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
      ll x; cin >> x;
      sum += x;
    }
  }
  cout << sum << "\n";

  return 0;
}
```
