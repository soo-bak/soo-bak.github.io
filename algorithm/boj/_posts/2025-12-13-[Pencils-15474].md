## 문제 링크
[15474번 - 鉛筆 (Pencils)](https://www.acmicpc.net/problem/15474)

## 설명
N개 이상의 연필을 구매할 때 필요한 최소 금액을 구하는 문제입니다.

<br>

## 접근법
두 종류의 세트 중 하나만 선택하여 구매합니다.

각 세트별로 N개 이상을 확보하기 위한 세트 수는 올림 나눗셈으로 계산합니다.

두 세트의 비용을 각각 계산한 뒤 더 작은 값을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var p = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var n = p[0]; var a = p[1]; var b = p[2]; var c = p[3]; var d = p[4];
    var cntX = (n + a - 1) / a;
    var cntY = (n + c - 1) / c;
    var costX = cntX * b;
    var costY = cntY * d;
    Console.WriteLine(Math.Min(costX, costY));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll n, a, b, c, d; cin >> n >> a >> b >> c >> d;
  ll cntX = (n + a - 1) / a;
  ll cntY = (n + c - 1) / c;
  ll costX = cntX * b;
  ll costY = cntY * d;
  cout << min(costX, costY) << "\n";

  return 0;
}
```
