## 문제 링크
[25703번 - 포인터 공부](https://www.acmicpc.net/problem/25703)

## 설명
N중 포인터 선언 코드를 규칙에 맞게 출력하는 문제입니다.

<br>

## 접근법
첫 줄은 int a를, 둘째 줄은 첫 포인터 선언을 고정으로 출력합니다.

이후 2부터 N까지 반복하며 포인터 개수만큼 별표를 붙이고 직전 포인터의 주소를 참조합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);

    Console.WriteLine("int a;");
    Console.WriteLine("int *ptr = &a;");

    for (var i = 2; i <= n; i++) {
      Console.Write("int ");
      Console.Write(new string('*', i));
      if (i == 2) Console.WriteLine("ptr2 = &ptr;");
      else Console.WriteLine($"ptr{i} = &ptr{i - 1};");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; 
  if (!(cin >> n)) return 0;

  cout << "int a;\n";
  cout << "int *ptr = &a;\n";
  for (int i = 2; i <= n; i++) {
    cout << "int " << string(i, '*');
    if (i == 2) cout << "ptr2 = &ptr;\n";
    else cout << "ptr" << i << " = &ptr" << i - 1 << ";\n";
  }

  return 0;
}
```
