## 문제 링크
[18883번 - N M 찍기](https://www.acmicpc.net/problem/18883)

## 설명
1부터 n×m까지의 수를 n행 m열로 출력하는 문제입니다.

<br>

## 접근법
1부터 시작하는 카운터를 두고 이중 반복으로 출력합니다.

같은 행의 숫자는 공백으로 구분하고, 행이 끝나면 개행합니다.

<br>

## Code

### C#
```csharp
using System;
using System.IO;
using System.Text;

class Program {
  static void Main() {
    var reader = new StreamReader(Console.OpenStandardInput());
    var writer = new StreamWriter(Console.OpenStandardOutput());
    var sb = new StringBuilder();

    var nm = Array.ConvertAll(reader.ReadLine()!.Split(), int.Parse);
    var n = nm[0];
    var m = nm[1];
    var cur = 1;
    for (var i = 0; i < n; i++) {
      for (var j = 0; j < m; j++) {
        if (j > 0) sb.Append(' ');
        sb.Append(cur++);
      }
      sb.AppendLine();
    }

    writer.Write(sb);
    writer.Flush();
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; 
  if (!(cin >> n >> m)) return 0;
  int cur = 1;
  for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
      if (j) cout << ' ';
      cout << cur++;
    }
    cout << "\n";
  }

  return 0;
}
```
