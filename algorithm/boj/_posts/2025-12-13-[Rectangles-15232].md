## 문제 링크
[15232번 - Rectangles](https://www.acmicpc.net/problem/15232)

## 설명
R행 C열의 별 사각형을 출력하는 문제입니다.

<br>

## 접근법
각 줄마다 C개의 별로 이루어진 문자열을 만듭니다.

이를 R번 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var r = int.Parse(Console.ReadLine()!);
    var c = int.Parse(Console.ReadLine()!);
    var line = new string('*', c);
    for (var i = 0; i < r; i++)
      Console.WriteLine(line);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int r, c; cin >> r >> c;
  string line(c, '*');
  for (int i = 0; i < r; i++)
    cout << line << "\n";

  return 0;
}
```
