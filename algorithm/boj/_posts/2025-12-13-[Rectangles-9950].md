## 문제 링크
[9950번 - Rectangles](https://www.acmicpc.net/problem/9950)

## 설명
직사각형의 길이, 너비, 넓이 중 하나가 0으로 주어질 때 나머지 값을 구하는 문제입니다.

<br>

## 접근법
넓이는 길이와 너비의 곱이므로, 0인 값을 나머지 두 값으로 계산합니다.

길이가 0이면 넓이를 너비로 나누고, 너비가 0이면 넓이를 길이로 나눕니다.

넓이가 0이면 길이와 너비를 곱합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    while (true) {
      var line = Console.ReadLine()!.Split();
      var l = int.Parse(line[0]);
      var w = int.Parse(line[1]);
      var a = int.Parse(line[2]);
      if (l == 0 && w == 0 && a == 0) break;

      if (l == 0) l = a / w;
      else if (w == 0) w = a / l;
      else if (a == 0) a = l * w;

      Console.WriteLine($"{l} {w} {a}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>

  while (true) {
    int l, w, a;
    if (!(cin >> l >> w >> a)) break;
    if (l == 0 && w == 0 && a == 0) break;

    if (l == 0) l = a / w;
    else if (w == 0) w = a / l;
    else if (a == 0) a = l * w;

    cout << l << " " << w << " " << a << "\n";
  }

  return 0;
}
```
