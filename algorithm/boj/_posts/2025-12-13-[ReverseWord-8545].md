## 문제 링크
[8545번 - Zadanie próbne](https://www.acmicpc.net/problem/8545)

## 설명
세 글자 단어를 뒤집어 출력하는 문제입니다.

<br>

## 접근법
문자열을 읽어서 뒤집은 뒤 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var arr = s.ToCharArray();
    Array.Reverse(arr);
    Console.WriteLine(new string(arr));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  reverse(s.begin(), s.end());
  cout << s << "\n";

  return 0;
}
```
