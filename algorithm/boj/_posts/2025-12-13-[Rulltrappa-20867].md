## 문제 링크
[20867번 - Rulltrappa](https://www.acmicpc.net/problem/20867)

## 설명
걷기와 서기 중 더 빠른 에스컬레이터 이용 방법을 구하는 문제입니다.

<br>

## 접근법
각 줄의 대기 시간과 에스컬레이터 이동 시간을 더해 총 소요 시간을 계산합니다.

왼쪽은 걷기로 대기 시간과 이동 시간을 더하고, 오른쪽은 서기로 계산합니다.

총 시간이 더 작은 쪽을 선택하여 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line1 = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var m = line1[0]; var s = line1[1]; var g = line1[2];

    var line2 = Array.ConvertAll(Console.ReadLine()!.Split(), double.Parse);
    var a = line2[0]; var b = line2[1];

    var line3 = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var l = line3[0]; var r = line3[1];

    var tL = l / a + (double)m / g;
    var tR = r / b + (double)m / s;

    Console.WriteLine(tL < tR ? "friskus" : "latmask");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int m, s, g;
  if (!(cin >> m >> s >> g)) return 0;
  double a, b; cin >> a >> b;
  int l, r; cin >> l >> r;

  double tL = l / a + (double)m / g;
  double tR = r / b + (double)m / s;

  cout << (tL < tR ? "friskus" : "latmask") << "\n";

  return 0;
}
