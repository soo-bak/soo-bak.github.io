## 문제 링크
[8871번 - Zadanie próbne 2](https://www.acmicpc.net/problem/8871)

## 설명
본선 라운드 수가 주어질 때, 전체 문제 수의 최솟값과 최댓값을 구하는 문제입니다.

<br>

## 접근법
연습 라운드 1개를 포함하면 총 라운드는 n+1개입니다.

각 라운드에 최소 2문제, 최대 3문제가 출제되므로, 전체 최솟값은 2를 곱한 값이고 최댓값은 3을 곱한 값입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var minVal = 2 * (n + 1);
    var maxVal = 3 * (n + 1);
    Console.WriteLine($"{minVal} {maxVal}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  cout << 2 * (n + 1) << " " << 3 * (n + 1) << "\n";

  return 0;
}
```
