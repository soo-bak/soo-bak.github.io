## 문제 링크
[14656번 - 조교는 새디스트야!!](https://www.acmicpc.net/problem/14656)

## 설명
학생들이 줄을 서 있을 때, 자기 자리에 서 있지 않은 학생의 수를 구하는 문제입니다.

<br>

## 접근법
i번 위치에는 i번 학생이 서 있어야 합니다.

순서대로 확인하면서 위치와 번호가 다른 학생의 수를 세면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var line = Console.ReadLine()!.Split();
    var cnt = 0;
    for (var i = 0; i < n; i++) {
      var num = int.Parse(line[i]);
      if (num != i + 1) cnt++;
    }
    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int cnt = 0;
  for (int i = 1; i <= n; i++) {
    int num; cin >> num;
    if (num != i) ++cnt;
  }
  cout << cnt << "\n";

  return 0;
}
```
