## 문제 링크
[23080번 - 스키테일 암호](https://www.acmicpc.net/problem/23080)

## 설명
스키테일 암호로 암호화된 문자열을 복호화하는 문제입니다.

<br>

## 접근법
암호문에서 k칸마다 한 글자씩 원본 메시지가 들어있습니다.

인덱스 0에서 시작해 k씩 증가시키며 해당 문자를 모읍니다.

모은 문자들을 이어붙이면 복호화된 메시지가 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var k = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;

    var sb = new StringBuilder();
    for (var i = 0; i < s.Length; i += k)
      sb.Append(s[i]);

    Console.WriteLine(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int k;
  if (!(cin >> k)) return 0;
  string s; cin >> s;
  string ans;
  for (int i = 0; i < (int)s.size(); i += k)
    ans.push_back(s[i]);
  cout << ans << "\n";

  return 0;
}
```
