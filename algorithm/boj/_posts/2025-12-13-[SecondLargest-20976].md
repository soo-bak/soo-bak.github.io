## 문제 링크
[20976번 - 2 番目に大きい整数 (The Second Largest Integer)](https://www.acmicpc.net/problem/20976)

## 설명
세 정수 중 두 번째로 큰 값을 구하는 문제입니다.

<br>

## 접근법
세 수를 정렬합니다.

정렬된 배열에서 가운데 값이 두 번째로 큰 값입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    Array.Sort(arr);
    Console.WriteLine(arr[1]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  vi v(3);
  for (int i = 0; i < 3; i++)
    cin >> v[i];
  sort(v.begin(), v.end());
  cout << v[1] << "\n";

  return 0;
}
```
