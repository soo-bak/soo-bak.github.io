## 문제 링크
[25625번 - 샤틀버스](https://www.acmicpc.net/problem/25625)

## 설명
셔틀버스를 탈 수 있는 최소 대기 시간을 구하는 문제입니다.

<br>

## 접근법
버스가 제2공학관까지 y분이 남았고 편도 시간이 x분이라면, y가 x보다 작으면 버스가 아직 서울대입구를 출발하지 않은 것입니다.

이 경우 제2공학관 도착 후 돌아오는 시간까지 기다려야 하므로 x + y분입니다.

반대로 y가 x 이상이면 버스가 이미 돌아오는 중이므로 y - x분만 기다리면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var p = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var x = p[0]; var y = p[1];
    var ans = y < x ? x + y : y - x;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int x, y; cin >> x >> y;
  int ans = (y < x) ? (x + y) : (y - x);
  cout << ans << "\n";

  return 0;
}
```
