## 문제 링크
[14782번 - Bedtime Reading, I](https://www.acmicpc.net/problem/14782)

## 설명
주어진 정수의 모든 약수의 합을 구하는 문제입니다.

<br>

## 접근법
1부터 n까지 순회하며 n을 나누어 떨어지게 하는 수를 찾아 합산합니다.

n이 100만 이하이므로 단순 순회로 충분합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sum = 0;
    for (var i = 1; i <= n; i++) {
      if (n % i == 0) sum += i;
    }
    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  int sum = 0;
  for (int i = 1; i <= n; i++) {
    if (n % i == 0) sum += i;
  }
  cout << sum << "\n";

  return 0;
}
```
