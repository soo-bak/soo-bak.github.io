## 문제 링크
[8558번 - Silnia](https://www.acmicpc.net/problem/8558)

## 설명
n!의 일의 자릿수를 구하는 문제입니다.

<br>

## 접근법
1부터 n까지 곱하면서 매번 10으로 나머지를 취해 일의 자리만 유지합니다.

n이 5 이상이면 2와 5가 곱해져 결과는 항상 0이 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var last = 1;
    for (var i = 1; i <= n; i++)
      last = (last * i) % 10;
    Console.WriteLine(last);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int last = 1;
  for (int i = 1; i <= n; i++)
    last = (last * i) % 10;
  cout << last << "\n";

  return 0;
}
```
