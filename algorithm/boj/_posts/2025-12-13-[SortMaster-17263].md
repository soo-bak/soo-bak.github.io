## 문제 링크
[17263번 - Sort 마스터 배지훈](https://www.acmicpc.net/problem/17263)

## 설명
배열을 정렬했을 때 마지막 원소를 구하는 문제입니다.

<br>

## 접근법
정렬된 배열의 마지막 원소는 최댓값입니다.

전체 정렬 없이 한 번 순회하며 최댓값만 찾으면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var line = Console.ReadLine()!.Split();
    var mx = -1;
    for (var i = 0; i < n; i++) {
      var x = int.Parse(line[i]);
      if (x > mx) mx = x;
    }
    Console.WriteLine(mx);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  int mx = -1;
  for (int i = 0; i < n; i++) {
    int x; cin >> x;
    if (x > mx) mx = x;
  }
  cout << mx << "\n";

  return 0;
}
```
