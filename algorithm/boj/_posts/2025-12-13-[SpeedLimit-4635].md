## 문제 링크
[4635번 - Speed Limit](https://www.acmicpc.net/problem/4635)

## 설명
속도와 누적 시간 기록으로 총 이동 거리를 구하는 문제입니다.

n이 -1이면 입력이 종료됩니다.

<br>

## 접근법
각 구간의 시간은 현재 누적 시간에서 이전 누적 시간을 뺀 값입니다.

구간 시간에 속도를 곱해 거리를 누적합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    while (true) {
      var n = int.Parse(Console.ReadLine()!);
      if (n == -1) break;

      var prev = 0;
      var dist = 0;
      for (var i = 0; i < n; i++) {
        var line = Console.ReadLine()!.Split();
        var s = int.Parse(line[0]);
        var t = int.Parse(line[1]);
        dist += s * (t - prev);
        prev = t;
      }
      Console.WriteLine($"{dist} miles");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  while (true) {
    int n;
    if (!(cin >> n)) break;
    if (n == -1) break;
    int prev = 0;
    int dist = 0;
    for (int i = 0; i < n; i++) {
      int s, t; cin >> s >> t;
      dist += s * (t - prev);
      prev = t;
    }
    cout << dist << " miles\n";
  }

  return 0;
}
```
