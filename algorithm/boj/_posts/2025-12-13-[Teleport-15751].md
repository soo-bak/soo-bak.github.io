## 문제 링크
[15751번 - Teleportation](https://www.acmicpc.net/problem/15751)

## 설명
직선 상에서 a에서 b로 이동할 때의 최소 거리를 구하는 문제입니다.

텔레포터를 통해 x와 y 사이를 즉시 이동할 수 있습니다.

<br>

## 접근법
직접 이동하는 경우와 텔레포터를 이용하는 경우를 비교합니다.

텔레포터를 이용하면 a에서 x로 이동 후 y로 순간이동하거나, a에서 y로 이동 후 x로 순간이동할 수 있습니다.

세 경로의 거리를 모두 계산해 최솟값을 출력합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var p = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var a = p[0];
    var b = p[1];
    var x = p[2];
    var y = p[3];

    var direct = Math.Abs(a - b);
    var viaXY = Math.Abs(a - x) + Math.Abs(y - b);
    var viaYX = Math.Abs(a - y) + Math.Abs(x - b);

    var ans = Math.Min(direct, Math.Min(viaXY, viaYX));
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, x, y;
  if (!(cin >> a >> b >> x >> y)) return 0;

  int direct = abs(a - b);
  int viaXY = abs(a - x) + abs(y - b);
  int viaYX = abs(a - y) + abs(x - b);
  int ans = min(direct, min(viaXY, viaYX));

  cout << ans << "\n";

  return 0;
}
```
