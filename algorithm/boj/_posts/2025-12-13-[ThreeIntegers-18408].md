## 문제 링크
[18408번 - 3つの整数 (Three Integers)](https://www.acmicpc.net/problem/18408)

## 설명
세 정수 A,B,C가 주어지고 모두 1 또는 2입니다. 등장 횟수가 더 많은 값(1 또는 2)을 출력합니다. 동점은 입력 조건상 없습니다.

<br>

## 접근법
세 수를 읽어 1의 개수와 2의 개수를 센 뒤, 더 큰 쪽을 출력합니다. 간단한 카운팅으로 해결됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var count1 = 0; var count2 = 0;
    foreach (var p in parts) {
      if (p == "1") count1++; else count2++;
    }
    Console.WriteLine(count1 > count2 ? 1 : 2);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, c; cin >> a >> b >> c;
  int cnt1 = (a == 1) + (b == 1) + (c == 1);
  int cnt2 = 3 - cnt1;
  cout << (cnt1 > cnt2 ? 1 : 2) << "\n";

  return 0;
}
```
