## 문제 링크
[13580번 - Andando no tempo](https://www.acmicpc.net/problem/13580)

## 설명
시간 이동 크레딧을 사용해 현재로 돌아올 수 있는지 판단하는 문제입니다.

<br>

## 접근법
이동 시간은 양수이므로 한 번 이동으로는 돌아올 수 없습니다.

두 값이 같다면 서로 반대 방향으로 이동하여 상쇄할 수 있습니다.

또한 두 값의 합이 나머지 한 값과 같다면 세 번 이동으로 상쇄할 수 있습니다.

이 두 조건 중 하나라도 만족하면 S, 아니면 N을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var p = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var a = p[0]; var b = p[1]; var c = p[2];

    var ok = (a == b) || (a == c) || (b == c) || (a + b == c) || (a + c == b) || (b + c == a);
    Console.WriteLine(ok ? "S" : "N");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>

  cout << (ok ? 'S' : 'N') << "\n";

  return 0;
}
```
