## 문제 링크
[25175번 - 두~~부 두부 두부](https://www.acmicpc.net/problem/25175)

## 설명
두부 게임에서 다음 차례인 사람의 번호를 구하는 문제입니다.

<br>

## 접근법
현재 차례인 사람이 3모를 외쳤으므로, 목표 모 수까지의 차이만큼 이동합니다.

원형 배치이므로 나머지 연산으로 위치를 구합니다.

음수 나머지 보정을 위해 n을 한 번 더해준 뒤 나머지를 구합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var n = line[0]; var m = line[1]; var k = line[2];
    var diff = k - 3;
    var idx = ((m - 1 + diff) % n + n) % n;
    Console.WriteLine(idx + 1);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m, k;
  if (!(cin >> n >> m >> k)) return 0;
  int diff = k - 3;
  int idx = ((m - 1 + diff) % n + n) % n;
  cout << idx + 1 << "\n";

  return 0;
}
```
