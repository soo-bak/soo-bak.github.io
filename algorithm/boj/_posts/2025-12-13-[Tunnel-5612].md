## 문제 링크
[5612번 - 터널의 입구와 출구](https://www.acmicpc.net/problem/5612)

## 설명
터널의 초기 차량 수와 매 분 유입·유출 차량 수가 주어질 때, 터널 내 최대 차량 수를 구하는 문제입니다.

단, 차량 수가 음수가 되면 0을 출력합니다.

<br>

## 접근법
초기 차량 수로 시작해서 매 분 유입 차량을 더하고 유출 차량을 뺍니다.

차량 수가 음수가 되면 잘못된 데이터이므로 0을 출력합니다.

음수가 되지 않으면 매 분 최댓값을 갱신하여 마지막에 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var cur = int.Parse(Console.ReadLine()!);
    var mx = cur;
    var valid = true;
    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!.Split();
      var enter = int.Parse(line[0]);
      var exit = int.Parse(line[1]);
      cur += enter - exit;
      if (cur < 0) { valid = false; break; }
      if (cur > mx) mx = cur;
    }
    Console.WriteLine(valid ? mx : 0);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int cur; cin >> cur;
  int mx = cur;
  bool ok = true;
  for (int i = 0; i < n; i++) {
    int a, b; cin >> a >> b;
    cur += a - b;
    if (cur < 0) { ok = false; break; }
    if (cur > mx) mx = cur;
  }
  cout << (ok ? mx : 0) << "\n";

  return 0;
}
```
