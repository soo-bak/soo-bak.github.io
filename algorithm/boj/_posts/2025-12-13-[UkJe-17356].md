## 문제 링크
[17356번 - 욱 제](https://www.acmicpc.net/problem/17356)

## 설명
욱 제 능력치가 A,B일 때 M=(B−A)/400이고, 욱이 제를 이길 확률은 `1/(1+10^M)`입니다. A,B가 주어지면 이 값을 출력합니다. 오차 1e-4 이내면 정답입니다.

<br>

## 접근법
공식을 그대로 double로 계산해 출력하면 됩니다. A,B는 0~5000이라 오버플로 우려가 없습니다. 충분한 자릿수를 출력하도록 포맷을 지정해도 되고 기본 실수 출력으로도 허용 오차를 만족합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);
    double m = (b - a) / 400.0;
    double ans = 1.0 / (1.0 + Math.Pow(10.0, m));
    Console.WriteLine(ans.ToString("F20"));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  double a, b; cin >> a >> b;
  double m = (b - a) / 400.0;
  double ans = 1.0 / (1.0 + pow(10.0, m));
  cout.setf(ios::fixed);
  cout.precision(20);
  cout << ans << "\n";

  return 0;
}
```
