## 문제 링크
[13610번 - Volta](https://www.acmicpc.net/problem/13610)

## 설명
빠른 차가 느린 차를 한 바퀴 앞서는 첫 랩 번호를 구하는 문제입니다.

<br>

## 접근법
빠른 차가 k랩을 돌 때 느린 차는 더 적은 랩을 돕니다.

빠른 차가 느린 차보다 정확히 한 바퀴 앞서는 조건을 수식으로 세웁니다.

조건을 만족하는 최소 k를 반복문으로 찾아 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var x = line[0]; var y = line[1];
    var k = 1;
    while ((y - x) * k < y)
      k++;
    Console.WriteLine(k);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int x, y;
  if (!(cin >> x >> y)) return 0;
  int k = 1;
  while ((y - x) * k < y)
    k++;
  cout << k << "\n";

  return 0;
}
```
