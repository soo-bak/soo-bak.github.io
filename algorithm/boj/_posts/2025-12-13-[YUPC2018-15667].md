## 문제 링크
[15667번 - 2018 연세대학교 프로그래밍 경진대회](https://www.acmicpc.net/problem/15667)

## 설명
불꽃의 총 개수가 주어질 때, 중형 불꽃의 개수 K를 구하는 문제입니다.

<br>

## 접근법
대형 1개, 중형 K개, 소형 K의 제곱 개이므로 총 개수는 K의 제곱에 K를 더하고 1을 더한 값입니다.

K를 1부터 늘려가며 총 개수가 N과 같아지는 값을 찾으면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    for (var k = 1; ; k++) {
      var total = k * k + k + 1;
      if (total == n) { Console.WriteLine(k); break; }
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  for (int k = 1;; k++) {
    int total = k * k + k + 1;
    if (total == n) { cout << k << "\n"; break; }
  }

  return 0;
}
```
