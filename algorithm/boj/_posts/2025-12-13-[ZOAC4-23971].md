## 문제 링크
[23971번 - ZOAC 4](https://www.acmicpc.net/problem/23971)

## 설명
테이블에서 일정 간격을 두고 앉을 때 최대 인원을 구하는 문제입니다.

<br>

## 접근법
세로로 n칸, 가로로 m칸 이상 비워야 하므로 한 블록의 크기는 (n+1) x (m+1)이 됩니다.

각 방향으로 앉을 수 있는 인원은 올림 나눗셈으로 계산합니다.

두 값의 곱이 최대 수용 인원이 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var line = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var h = line[0]; var w = line[1]; var n = line[2]; var m = line[3];
    var rows = (h + n) / (n + 1);
    var cols = (w + m) / (m + 1);
    Console.WriteLine(rows * cols);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int h, w, n, m;
  if (!(cin >> h >> w >> n >> m)) return 0;
  int rows = (h + n) / (n + 1);
  int cols = (w + m) / (m + 1);
  cout << rows * cols << "\n";

  return 0;
}
```
