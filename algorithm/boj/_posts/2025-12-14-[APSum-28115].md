## 문제 링크
[28115번 - 등차수열의 합](https://www.acmicpc.net/problem/28115)

## 설명
수열 a가 주어질 때, 두 등차수열 b와 c의 합으로 나타낼 수 있는지 판별하는 문제입니다.

두 등차수열의 합은 등차수열이므로, a가 등차수열이면 가능하고 아니면 불가능합니다.

<br>

## 접근법
먼저 a가 등차수열인지 확인합니다. 길이가 2 이하이면 항상 등차수열입니다.

길이가 3 이상이면 인접한 항의 차이가 모두 같은지 확인합니다.

등차수열이면 b를 a와 같게, c를 모두 0으로 구성하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var a = Array.ConvertAll(Console.ReadLine()!.Split(), long.Parse);

    var ok = true;
    if (n > 2) {
      var diff = a[1] - a[0];
      for (var i = 2; i < n; i++) {
        if (a[i] - a[i - 1] != diff) { ok = false; break; }
      }
    }

    if (!ok) {
      Console.WriteLine("NO");
      return;
    }

    Console.WriteLine("YES");
    Console.WriteLine(string.Join(" ", a));
    Console.WriteLine(string.Join(" ", new long[n]));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef vector<ll> vll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vll a(n);
  for (int i = 0; i < n; i++) cin >> a[i];

  bool ok = true;
  if (n > 2) {
    ll diff = a[1] - a[0];
    for (int i = 2; i < n; i++) {
      if (a[i] - a[i - 1] != diff) { ok = false; break; }
    }
  }

  if (!ok) {
    cout << "NO\n";
    return 0;
  }

  cout << "YES\n";
  for (int i = 0; i < n; i++) {
    if (i) cout << ' ';
    cout << a[i];
  }
  cout << "\n";
  for (int i = 0; i < n; i++) {
    if (i) cout << ' ';
    cout << 0;
  }
  cout << "\n";

  return 0;
}
```
