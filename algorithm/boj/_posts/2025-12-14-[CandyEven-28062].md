## 문제 링크
[28062번 - 준석이의 사탕 사기](https://www.acmicpc.net/problem/28062)

## 설명
n개의 사탕 묶음 중 일부를 선택하여 총 개수가 짝수가 되도록 할 때, 최대 개수를 구하는 문제입니다.

총합이 짝수면 모두 선택하면 되고, 홀수면 가장 작은 홀수 묶음을 제외합니다.

<br>

## 접근법
먼저 모든 묶음의 합과 최소 홀수 값을 구합니다.

합이 짝수면 그대로 출력하고, 홀수면 최소 홀수를 뺀 값을 출력합니다.

홀수 묶음이 없는데 합이 홀수인 경우는 존재하지 않으므로 별도 처리가 필요 없습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var sum = 0;
    var minOdd = 1001;
    foreach (var x in arr) {
      sum += x;
      if ((x & 1) == 1 && x < minOdd) minOdd = x;
    }
    if ((sum & 1) == 0) Console.WriteLine(sum);
    else if (minOdd == 1001) Console.WriteLine(0);
    else Console.WriteLine(sum - minOdd);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int sum = 0, minOdd = 1001;
  for (int i = 0; i < n; i++) {
    int x; cin >> x;
    sum += x;
    if (x % 2 == 1) minOdd = min(minOdd, x);
  }
  if (sum % 2 == 0) cout << sum << "\n";
  else if (minOdd == 1001) cout << 0 << "\n";
  else cout << sum - minOdd << "\n";

  return 0;
}
```
