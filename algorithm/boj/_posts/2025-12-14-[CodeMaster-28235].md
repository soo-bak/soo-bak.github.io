## 문제 링크
[28235번 - 코드마스터 2023](https://www.acmicpc.net/problem/28235)

## 설명
입력된 구호에 대응하는 응원 문구를 출력하는 문제입니다.

<br>

## 접근법
각 구호와 응원 문구의 쌍을 저장해두고, 입력에 따라 대응하는 문구를 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var map = new Dictionary<string, string> {
      ["SONGDO"] = "HIGHSCHOOL",
      ["CODE"] = "MASTER",
      ["2023"] = "0611",
      ["ALGORITHM"] = "CONTEST"
    };
    var s = Console.ReadLine()!;
    Console.WriteLine(map[s]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  unordered_map<string, string> mp = {
    {"SONGDO", "HIGHSCHOOL"},
    {"CODE", "MASTER"},
    {"2023", "0611"},
    {"ALGORITHM", "CONTEST"}
  };
  string s; cin >> s;
  cout << mp[s] << "\n";

  return 0;
}
```
