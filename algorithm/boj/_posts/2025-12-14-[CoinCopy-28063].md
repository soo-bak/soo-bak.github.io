## 문제 링크
[28063번 - 동전 복사](https://www.acmicpc.net/problem/28063)

## 설명
n × n 격자에서 동전을 복사하여 모든 칸을 채우는 최소 조작 횟수를 구하는 문제입니다.

한 번 조작하면 동전이 한 방향으로 이동하며 지나간 모든 칸이 채워집니다.

<br>

## 접근법
초기 위치에 따라 답이 결정됩니다.

n이 1이면 이미 채워져 있으므로 0입니다.

꼭짓점에 있으면 가로와 세로 한 번씩으로 2회, 테두리에 있으면 3회, 내부에 있으면 4회가 필요합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var x = parts[0];
    var y = parts[1];

    var ans = 0;
    if (n == 1) ans = 0;
    else if ((x == 1 || x == n) && (y == 1 || y == n)) ans = 2;
    else if (x == 1 || x == n || y == 1 || y == n) ans = 3;
    else ans = 4;

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp

  if (n == 1) ans = 0;
  else if ((x == 1 || x == n) && (y == 1 || y == n)) ans = 2;
  else if (x == 1 || x == n || y == 1 || y == n) ans = 3;
  else ans = 4;

  cout << ans << "\n";

  return 0;
}
```
