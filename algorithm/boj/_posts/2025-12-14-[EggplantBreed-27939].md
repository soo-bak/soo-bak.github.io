## 문제 링크
[27939번 - 가지 교배](https://www.acmicpc.net/problem/27939)

## 설명
조수들과 키위가 서로 다른 교배 규칙을 따를 때, 최종 결과를 구하는 문제입니다.

조수들은 P-우선 교배를 하므로, 모든 가지가 W일 때만 W를 만들고 하나라도 P가 있으면 P를 만듭니다.

키위는 W-우선 교배를 하므로, 조수 결과 중 하나라도 W가 있으면 최종 W를 만듭니다.

<br>

## 접근법
각 조수별로 받은 품종이 모두 W인지 확인합니다.

하나라도 모두 W인 조수가 있으면 W, 아니면 P를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var colors = Console.ReadLine()!.Split();

    var mk = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var m = mk[0];
    var k = mk[1];

    var hasAllW = false;
    for (var i = 0; i < m; i++) {
      var parts = Console.ReadLine()!.Split();
      var allW = true;
      for (var j = 0; j < k; j++) {
        var idx = int.Parse(parts[j]) - 1;
        if (colors[idx] == "P") { allW = false; break; }
      }
      if (allW) hasAllW = true;
    }

    Console.WriteLine(hasAllW ? "W" : "P");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vector<char> color(n);
  for (int i = 0; i < n; i++) cin >> color[i];

  int m, k; cin >> m >> k;
  bool hasAllW = false;
  for (int i = 0; i < m; i++) {
    bool allW = true;
    for (int j = 0; j < k; j++) {
      int idx; cin >> idx;
      if (color[idx - 1] == 'P') allW = false;
    }
    if (allW) hasAllW = true;
  }

  cout << (hasAllW ? "W" : "P") << "\n";

  return 0;
}
```
