## 문제 링크
[28097번 - 모범생 포닉스](https://www.acmicpc.net/problem/28097)

## 설명
공부 계획 N개를 순서대로 수행하며, 각 사이에 8시간 휴식을 한다. 총 소요 시간을 일(24시간 단위)과 시간으로 출력한다. 마지막 계획 뒤에는 휴식이 없다.

<br>

## 접근법
모든 계획 시간 합에 (N-1)개의 8시간 휴식을 더해 총 시간을 구하고, `days = total/24`, `hours = total%24`를 출력한다. O(N) 합산이다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    int n = int.Parse(Console.ReadLine()!);
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    int total = 0;
    for (int i = 0; i < n; i++) {
      total += arr[i];
      if (i != n - 1) total += 8;
    }
    Console.WriteLine($"{total / 24} {total % 24}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; 
  if (!(cin >> n)) return 0;
  int total = 0;
  for (int i = 0; i < n; i++) {
    int t; cin >> t;
    total += t;
    if (i != n - 1) total += 8;
  }
  cout << total / 24 << " " << total % 24 << "\n";
  return 0;
}
```
