## 문제 링크
[27918번 - 탁구 경기](https://www.acmicpc.net/problem/27918)

## 설명
라운드 승자 예측이 주어졌을 때, 순서대로 점수를 누적하며 어느 한 쪽이 2점 앞서거나 N라운드를 모두 소화하면 종료합니다. 종료 시점의 점수를 `X:Y` 형식으로 출력합니다.

<br>

## 접근법
예측된 승자를 한 줄씩 읽어 D면 달구 점수, P면 포닉스 점수를 1씩 올립니다. 매 라운드 후 두 점수 차가 2 이상이면 루프를 종료합니다. O(N) 시뮬레이션으로 충분합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    int N = int.Parse(Console.ReadLine()!);
    int d = 0, p = 0;
    for (int i = 0; i < N; i++) {
      char c = Console.ReadLine()![0];
      if (c == 'D') d++; else p++;
      if (Math.Abs(d - p) >= 2) break;
    }
    Console.WriteLine($"{d}:{p}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int N;
  if (!(cin >> N)) return 0;
  int d = 0, p = 0;
  for (int i = 0; i < N; i++) {
    char c; cin >> c;
    if (c == 'D') ++d;
    else ++p;
    if (abs(d - p) >= 2) break;
  }
  cout << d << ":" << p << "\n";
  return 0;
}
```
