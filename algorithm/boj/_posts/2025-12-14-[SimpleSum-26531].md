## 문제 링크
[26531번 - Simple Sum](https://www.acmicpc.net/problem/26531)

## 설명
a + b = c 형식의 수식이 주어질 때, 등식이 맞는지 검증하는 문제입니다.

a, b, c는 한 자리 정수이며, 맞으면 YES, 아니면 NO를 출력합니다.

<br>

## 접근법
입력을 파싱하여 a + b와 c를 비교합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split(' ');
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[2]);
    var c = int.Parse(parts[4]);
    Console.WriteLine(a + b == c ? "YES" : "NO");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, c; char op, eq;
  cin >> a >> op >> b >> eq >> c;
  cout << (a + b == c ? "YES" : "NO") << "\n";

  return 0;
}
```
