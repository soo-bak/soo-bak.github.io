## 문제 링크
[28135번 - Since 1973](https://www.acmicpc.net/problem/28135)

## 설명
숫자를 1부터 세되, 숫자에 "50"이 포함되어 있으면 한 번 더 센다고 할 때, 숫자 N이 처음 등장하는 순서를 구합니다. N 이전까지 등장한 "50"을 포함한 수의 개수만큼 N의 위치가 뒤로 밀립니다.

<br>

## 접근법
1부터 N-1까지 문자열로 변환해 `"50"`이 포함되는지 확인하고 개수를 센 뒤, `답 = N + 개수`를 출력합니다. N ≤ 1,000,000이므로 단순 확인으로 충분합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    int n = int.Parse(Console.ReadLine()!);
    int extra = 0;
    for (int i = 1; i < n; i++) {
      if (i.ToString().Contains("50")) extra++;
    }
    Console.WriteLine(n + extra);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  int extra = 0;
  for (int i = 1; i < n; i++) {
    string s = to_string(i);
    if (s.find("50") != string::npos) ++extra;
  }
  cout << n + extra << "\n";
  return 0;
}
```
