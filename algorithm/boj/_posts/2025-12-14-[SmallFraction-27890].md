## 문제 링크
[27890번 - 특별한 작은 분수](https://www.acmicpc.net/problem/27890)

## 설명
시작 값에서 n번 반복하며 값을 갱신한 결과를 구하는 문제입니다.

짝수이면 (x / 2) ⊕ 6, 홀수이면 (2 × x) ⊕ 6으로 갱신합니다.

<br>

## 접근법
n번 반복하며 규칙에 따라 x를 갱신합니다.

최종 값을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static int Next(int x) => (x % 2 == 0) ? ((x / 2) ^ 6) : ((2 * x) ^ 6);

  static void Main() {
    var parts = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var x = parts[0];
    var n = parts[1];
    for (var i = 0; i < n; i++) x = Next(x);
    Console.WriteLine(x);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int x, n; cin >> x >> n;
  for (int i = 0; i < n; i++) {
    if (x % 2 == 0) x = (x / 2) ^ 6;
    else x = (2 * x) ^ 6;
  }
  cout << x << "\n";

  return 0;
}
```
