## 문제 링크
[6887번 - Squares](https://www.acmicpc.net/problem/6887)

## 설명
같은 크기의 정사각형 타일 n개로 만들 수 있는 가장 큰 정사각형의 한 변 길이를 구하는 문제입니다.

<br>

## 접근법
k × k ≤ n을 만족하는 최대 정수 k를 구하면 됩니다.

n의 제곱근을 구한 뒤 내림하면 답이 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var k = (int)Math.Sqrt(n);
    Console.WriteLine($"The largest square has side length {k}.");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int k = (int)sqrt(n);
  cout << "The largest square has side length " << k << ".\n";

  return 0;
}
```
