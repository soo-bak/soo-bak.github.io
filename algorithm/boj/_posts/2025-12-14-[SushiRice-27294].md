## 문제 링크
[27294번 - 몇개고?](https://www.acmicpc.net/problem/27294)

## 설명
시간과 술 여부가 주어질 때, 초밥에 올릴 밥알 개수를 구하는 문제입니다.

점심 시간(12~16시)이고 술을 마시지 않으면 320개, 그 외에는 280개입니다.

<br>

## 접근법
시간이 12 이상 16 이하이고 술을 마시지 않으면 320, 아니면 280을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var t = parts[0];
    var s = parts[1];
    var ans = (12 <= t && t <= 16 && s == 0) ? 320 : 280;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t, s; cin >> t >> s;
  int ans = (12 <= t && t <= 16 && s == 0) ? 320 : 280;
  cout << ans << "\n";

  return 0;
}
```
