## 문제 링크
[26594번 - ZOAC 5](https://www.acmicpc.net/problem/26594)

## 설명
고장난 키보드가 한 번 누른 문자를 n번 반복해서 입력하는 상황에서, 입력된 문자열을 보고 n을 구하는 문제입니다.

<br>

## 접근법
키보드는 같은 문자를 n번 연속으로 입력하므로, 문자열의 맨 앞에서 첫 문자가 몇 번 반복되는지 세면 됩니다.

첫 문자와 다른 문자가 나타나면 바로 종료합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var n = 1;
    for (var i = 1; i < s.Length; i++) {
      if (s[i] == s[0]) n++;
      else break;
    }
    Console.WriteLine(n);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  int n = 1;
  for (int i = 1; i < (int)s.size(); i++) {
    if (s[i] == s[0]) ++n;
    else break;
  }
  cout << n << "\n";

  return 0;
}
```
