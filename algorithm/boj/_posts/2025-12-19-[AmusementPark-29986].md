## 문제 링크
[29986번 - Amusement Park Adventure](https://www.acmicpc.net/problem/29986)

## 설명
놀이기구 수 n과 카를리토스의 키 h가 주어지고, 각 놀이기구의 최소 키 제한이 주어질 때, 탈 수 있는 놀이기구의 개수를 출력하는 문제입니다.

<br>

## 접근법
각 놀이기구의 키 제한이 h 이하인지 확인하며 개수를 셉니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var h = int.Parse(first[1]);

    var arr = Console.ReadLine()!.Split();
    var cnt = 0;
    for (var i = 0; i < n; i++) {
      var a = int.Parse(arr[i]);
      if (h >= a) cnt++;
    }

    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, h; cin >> n >> h;
  int cnt = 0;
  for (int i = 0; i < n; i++) {
    int a; cin >> a;
    if (h >= a) cnt++;
  }
  cout << cnt << "\n";

  return 0;
}
```
