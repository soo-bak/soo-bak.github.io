## 문제 링크
[17598번 - Animal King Election](https://www.acmicpc.net/problem/17598)

## 설명
9개의 투표 결과가 주어질 때, 더 많은 표를 받은 동물을 출력하는 문제입니다.

과반을 받은 쪽이 항상 존재합니다.

<br>

## 접근법
9줄을 읽으면서 Tiger와 Lion의 개수를 각각 셉니다.

이후 더 많은 쪽을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var tiger = 0;
    var lion = 0;
    for (var i = 0; i < 9; i++) {
      var s = Console.ReadLine()!;
      if (s == "Tiger") tiger++;
      else lion++;
    }

    Console.WriteLine(tiger > lion ? "Tiger" : "Lion");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int tiger = 0, lion = 0;
  for (int i = 0; i < 9; i++) {
    string s; cin >> s;
    if (s == "Tiger") tiger++;
    else lion++;
  }

  cout << (tiger > lion ? "Tiger" : "Lion") << "\n";

  return 0;
}
```
