## 문제 링크
[15096번 - Batter Up](https://www.acmicpc.net/problem/15096)

## 설명
n개의 타석 결과가 주어질 때, 슬러깅 퍼센티지를 구하는 문제입니다.

-1은 볼넷으로 공식 타석에서 제외하고, 나머지 값의 합을 공식 타석 수로 나눕니다.

<br>

## 접근법
-1이 아닌 값만 합산하고 개수를 셉니다.

이후 합을 개수로 나눈 값을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var sum = 0;
    var cnt = 0;
    for (var i = 0; i < n; i++) {
      if (parts[i] != -1) {
        sum += parts[i];
        cnt++;
      }
    }

    Console.WriteLine((double)sum / cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;

  int sum = 0, cnt = 0;
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    if (v != -1) {
      sum += v;
      cnt++;
    }
  }

  cout << fixed << setprecision(10) << (double)sum / cnt << "\n";

  return 0;
}
```
