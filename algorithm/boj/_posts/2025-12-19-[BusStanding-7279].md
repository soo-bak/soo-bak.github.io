## 문제 링크
[7279번 - Autobusas](https://www.acmicpc.net/problem/7279)

## 설명
n개의 정류장과 k개의 좌석이 있는 버스에서, 서 있는 승객의 최대 인원을 구하는 문제입니다.

각 정류장마다 탑승과 하차가 이루어지며, 좌석이 부족하면 서서 이동합니다.

<br>

## 접근법
현재 승객 수를 관리하며 정류장을 순회합니다.

각 정류장에서 하차 후 탑승을 처리하고, 좌석 수를 초과하는 인원이 서 있는 승객입니다.

최대값을 갱신하며 진행합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var k = int.Parse(first[1]);

    var cur = 0;
    var best = 0;
    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!.Split();
      var a = int.Parse(line[0]);
      var b = int.Parse(line[1]);

      cur -= b;
      cur += a;

      var standing = Math.Max(0, cur - k);
      if (standing > best) best = standing;
    }

    Console.WriteLine(best);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, k; cin >> n >> k;

  int cur = 0, best = 0;
  for (int i = 0; i < n; i++) {
    int a, b; cin >> a >> b;

    cur -= b;
    cur += a;

    int standing = max(0, cur - k);
    best = max(best, standing);
  }

  cout << best << "\n";

  return 0;
}
```
