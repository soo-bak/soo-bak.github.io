## 문제 링크
[28938번 - Конвейер](https://www.acmicpc.net/problem/28938)

## 설명
n개의 이동 명령이 주어질 때, 최종 이동 방향을 출력하는 문제입니다.

합계가 음수면 왼쪽, 양수면 오른쪽, 0이면 제자리입니다.

<br>

## 접근법
명령 값을 모두 더한 후 부호에 따라 결과를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Console.ReadLine()!.Split();

    var sum = 0;
    for (var i = 0; i < n; i++)
      sum += int.Parse(parts[i]);

    if (sum < 0) Console.WriteLine("Left");
    else if (sum > 0) Console.WriteLine("Right");
    else Console.WriteLine("Stay");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;

  int sum = 0;
  for (int i = 0; i < n; i++) {
    int x; cin >> x;
    sum += x;
  }

  if (sum < 0) cout << "Left\n";
  else if (sum > 0) cout << "Right\n";
  else cout << "Stay\n";

  return 0;
}
```
