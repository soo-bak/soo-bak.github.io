## 문제 링크
[29766번 - DKSH 찾기](https://www.acmicpc.net/problem/29766)

## 설명
D, K, S, H로만 이루어진 문자열이 주어질 때, DKSH가 몇 번 등장하는지 세는 문제입니다.

<br>

## 접근법
문자열의 처음부터 끝에서 4번째 위치까지 순회하며, 현재 위치부터 4글자가 DKSH인지 확인합니다. 일치할 때마다 개수를 증가시킵니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var cnt = 0;
    for (var i = 0; i + 3 < s.Length; i++) {
      if (s[i] == 'D' && s[i + 1] == 'K' && s[i + 2] == 'S' && s[i + 3] == 'H')
        cnt++;
    }
    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  int cnt = 0;
  for (size_t i = 0; i + 3 < s.size(); i++) {
    if (s[i] == 'D' && s[i + 1] == 'K' && s[i + 2] == 'S' && s[i + 3] == 'H')
      cnt++;
  }

  cout << cnt << "\n";

  return 0;
}
```
