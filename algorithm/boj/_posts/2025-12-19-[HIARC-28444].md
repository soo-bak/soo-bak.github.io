## 문제 링크
[28444번 - HI-ARC=?](https://www.acmicpc.net/problem/28444)

## 설명
다섯 정수 h, i, a, r, c가 주어질 때, h × i - a × r × c를 계산하여 출력하는 문제입니다.

<br>

## 접근법
두 곱의 차를 계산하여 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var p = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var h = p[0];
    var i = p[1];
    var a = p[2];
    var r = p[3];
    var c = p[4];
    Console.WriteLine(h * i - a * r * c);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int h, i, a, r, c; cin >> h >> i >> a >> r >> c;
  cout << h * i - a * r * c << "\n";

  return 0;
}
```
