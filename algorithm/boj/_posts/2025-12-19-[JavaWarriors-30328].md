## 문제 링크
[30328번 - Java Warriors](https://www.acmicpc.net/problem/30328)

## 설명
팀 수 n이 주어질 때, 팀당 등록비 4000달러를 곱해 총액을 출력하는 문제입니다.

<br>

## 접근법
n에 4000을 곱한 값을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    Console.WriteLine(n * 4000);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  cout << n * 4000 << "\n";

  return 0;
}
```
