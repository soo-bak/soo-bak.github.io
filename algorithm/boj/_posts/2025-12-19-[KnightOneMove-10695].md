## 문제 링크
[10695번 - Dalia](https://www.acmicpc.net/problem/10695)

## 설명
나이트의 시작 좌표와 목표 좌표가 주어질 때, 정확히 한 번 이동으로 도달 가능한지 판별하는 문제입니다.

<br>

## 접근법
나이트의 한 번 이동은 가로 1칸 세로 2칸, 또는 가로 2칸 세로 1칸입니다.

두 좌표의 차이가 이 조합에 해당하면 가능, 아니면 불가능합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);

    for (var tc = 1; tc <= t; tc++) {
      var parts = Console.ReadLine()!.Split();
      var r1 = int.Parse(parts[1]);
      var c1 = int.Parse(parts[2]);
      var r2 = int.Parse(parts[3]);
      var c2 = int.Parse(parts[4]);

      var dx = Math.Abs(r1 - r2);
      var dy = Math.Abs(c1 - c2);

      var ok = (dx == 1 && dy == 2) || (dx == 2 && dy == 1);
      Console.WriteLine($"Case {tc}: {(ok ? "YES" : "NO")}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;

  for (int tc = 1; tc <= t; tc++) {
    int n, r1, c1, r2, c2;
    cin >> n >> r1 >> c1 >> r2 >> c2;

    int dx = abs(r1 - r2);
    int dy = abs(c1 - c2);

    bool ok = (dx == 1 && dy == 2) || (dx == 2 && dy == 1);
    cout << "Case " << tc << ": " << (ok ? "YES" : "NO") << "\n";
  }

  return 0;
}
```
