## 문제 링크
[28236번 - 점심시간 레이스](https://www.acmicpc.net/problem/28236)

## 설명
각 반의 위치가 주어질 때, 급식실까지 가장 빨리 도착하는 반을 구하는 문제입니다.

급식실은 1층 오른쪽 끝에 있으므로, f층에서 왼쪽 계단 기준 d칸 떨어진 교실의 이동 시간은 (m - d) + (f - 1)입니다.

동률이면 번호가 작은 반이 우승합니다.

<br>

## 접근법
각 반에 대해 이동 시간을 계산하고, 최솟값을 가진 반 번호를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var n = first[0];
    var m = first[1];
    var k = first[2];
    var bestTime = int.MaxValue;
    var winner = -1;
    for (var i = 1; i <= k; i++) {
      var fd = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
      var f = fd[0];
      var d = fd[1];
      var time = (m - d) + (f - 1);
      if (time < bestTime) {
        bestTime = time;
        winner = i;
      }
    }
    Console.WriteLine(winner);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m, k; cin >> n >> m >> k;
  int bestTime = INT_MAX, winner = -1;
  for (int i = 1; i <= k; i++) {
    int f, d; cin >> f >> d;
    int time = (m - d) + (f - 1);
    if (time < bestTime) {
      bestTime = time;
      winner = i;
    }
  }
  cout << winner << "\n";

  return 0;
}
```
