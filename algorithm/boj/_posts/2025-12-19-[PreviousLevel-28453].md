## 문제 링크
[28453번 - Previous Level](https://www.acmicpc.net/problem/28453)

## 설명
n개의 레벨이 주어질 때, 각 레벨을 구간에 따라 번호로 변환하여 출력하는 문제입니다.

300이면 1, 275 이상이면 2, 250 이상이면 3, 그 외에는 4를 출력합니다.

<br>

## 접근법
각 레벨에 대해 조건을 확인하여 해당하는 번호를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    for (var i = 0; i < n; i++) {
      var lev = parts[i];
      var grp = (lev == 300) ? 1 : (lev >= 275 ? 2 : (lev >= 250 ? 3 : 4));
      Console.Write(grp);
      if (i + 1 < n) Console.Write(" ");
    }
    Console.WriteLine();
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  for (int i = 0; i < n; i++) {
    int lev; cin >> lev;
    int grp = (lev == 300) ? 1 : (lev >= 275 ? 2 : (lev >= 250 ? 3 : 4));
    cout << grp;
    if (i + 1 < n) cout << ' ';
  }
  cout << "\n";

  return 0;
}
```
