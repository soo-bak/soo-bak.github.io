## 문제 링크
[28417번 - 스케이트보드](https://www.acmicpc.net/problem/28417)

## 설명
각 선수의 런 2회, 트릭 5회 점수가 주어질 때, 가장 높은 최종 점수를 구하는 문제입니다.

최종 점수는 런 중 높은 점수와 트릭 상위 2개의 합입니다.

<br>

## 접근법
각 선수에 대해 런 점수 중 최댓값을 구합니다.

트릭 점수를 정렬하여 상위 2개를 더합니다.

모든 선수의 최종 점수 중 최댓값을 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var best = 0;
    for (var i = 0; i < n; i++) {
      var nums = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
      var runMax = Math.Max(nums[0], nums[1]);
      var tricks = nums.Skip(2).OrderByDescending(x => x).ToArray();
      var total = runMax + tricks[0] + tricks[1];
      if (total > best) best = total;
    }
    Console.WriteLine(best);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int best = 0;
  for (int i = 0; i < n; i++) {
    int run1, run2; cin >> run1 >> run2;
    vi t(5);
    for (int j = 0; j < 5; j++) cin >> t[j];
    sort(t.begin(), t.end(), greater<int>());
    int total = max(run1, run2) + t[0] + t[1];
    best = max(best, total);
  }
  cout << best << "\n";

  return 0;
}
```
