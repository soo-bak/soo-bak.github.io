## 문제 링크
[7240번 - Rogutės](https://www.acmicpc.net/problem/7240)

## 설명
n개의 구간을 순서대로 지나갈 때, 최종 속도를 구하는 문제입니다.

각 구간을 통과하면 속도가 변하고, 마지막 구간을 제외한 경계에서 속도가 한계를 넘으면 1만큼 감속합니다.

<br>

## 접근법
속도 0에서 시작해 각 구간마다 속도를 갱신합니다.

마지막 구간이 아닐 때 속도가 한계를 넘으면 1 감속합니다.

모든 구간을 지난 후 최종 속도를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var s = int.Parse(first[1]);

    var speed = 0;
    for (var i = 0; i < n; i++) {
      var delta = int.Parse(Console.ReadLine()!);
      speed += delta;
      if (i != n - 1 && speed > s) speed--;
    }

    Console.WriteLine(speed);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, s; cin >> n >> s;

  int speed = 0;
  for (int i = 0; i < n; i++) {
    int delta; cin >> delta;
    speed += delta;
    if (i != n - 1 && speed > s) speed--;
  }

  cout << speed << "\n";

  return 0;
}
```
