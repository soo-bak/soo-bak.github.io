## 문제 링크
[29684번 - Which Team Should Receive the Sponsor Prize?](https://www.acmicpc.net/problem/29684)

## 설명
여러 팀의 정답 시간이 주어질 때, 2023초에 가장 가까운 시간을 기록한 팀 번호를 출력하는 문제입니다.

가장 가까운 팀이 유일함이 보장되며, 0이 입력되면 종료합니다.

<br>

## 접근법
각 데이터셋에서 팀별 시간을 순회하며 2023과의 차이를 계산합니다.

최소 차이를 기록한 팀 번호를 저장하고 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    while (true) {
      var n = int.Parse(Console.ReadLine()!);
      if (n == 0) break;

      var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

      var bestDiff = int.MaxValue;
      var bestIdx = -1;
      for (var i = 0; i < n; i++) {
        var diff = Math.Abs(arr[i] - 2023);
        if (diff < bestDiff) {
          bestDiff = diff;
          bestIdx = i + 1;
        }
      }

      Console.WriteLine(bestIdx);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  while (cin >> n && n != 0) {
    vi a(n);
    for (int i = 0; i < n; i++)
      cin >> a[i];

    int bestDiff = INT_MAX;
    int bestIdx = -1;
    for (int i = 0; i < n; i++) {
      int diff = abs(a[i] - 2023);
      if (diff < bestDiff) {
        bestDiff = diff;
        bestIdx = i + 1;
      }
    }

    cout << bestIdx << "\n";
  }

  return 0;
}
```
