## 문제 링크
[13670번 - Alarme Despertador](https://www.acmicpc.net/problem/13670)

## 설명
현재 시각과 알람 시각이 주어질 때, 알람이 울리기까지 남은 분을 계산하는 문제입니다.

<br>
자정을 넘어갈 수 있으며, 입력 끝에 0 0 0 0이 주어집니다.

<br>

## 접근법
시각을 분 단위로 변환하여 차이를 구합니다. 알람 시각에서 현재 시각을 뺀 값이 음수이면 자정을 넘긴 것이므로 1440분을 더합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    while (true) {
      var line = Console.ReadLine();
      if (line == null) break;
      var p = line.Split(' ');
      var h1 = int.Parse(p[0]);
      var m1 = int.Parse(p[1]);
      var h2 = int.Parse(p[2]);
      var m2 = int.Parse(p[3]);
      if (h1 == 0 && m1 == 0 && h2 == 0 && m2 == 0) break;

      var cur = h1 * 60 + m1;
      var alarm = h2 * 60 + m2;
      var diff = alarm - cur;
      if (diff < 0) diff += 1440;

      Console.WriteLine(diff);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int h1, m1, h2, m2;
  while (cin >> h1 >> m1 >> h2 >> m2) {
    if (h1 == 0 && m1 == 0 && h2 == 0 && m2 == 0) break;

    int cur = h1 * 60 + m1;
    int alarm = h2 * 60 + m2;
    int diff = alarm - cur;
    if (diff < 0) diff += 1440;

    cout << diff << "\n";
  }

  return 0;
}
```
