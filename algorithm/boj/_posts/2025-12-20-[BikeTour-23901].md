## 문제 링크
[23901번 - Bike Tour](https://www.acmicpc.net/problem/23901)

## 설명
N개의 높이 중 양 끝을 제외하고, 좌우보다 높은 위치의 개수를 세는 문제입니다.

<br>

## 접근법
양 끝을 제외한 각 위치에서 왼쪽과 오른쪽 높이보다 현재 높이가 크면 피크입니다. 모든 위치를 확인하며 피크의 개수를 셉니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);

    for (var tc = 1; tc <= t; tc++) {
      var n = int.Parse(Console.ReadLine()!);
      var h = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

      var cnt = 0;
      for (var i = 1; i + 1 < n; i++) {
        if (h[i] > h[i - 1] && h[i] > h[i + 1]) cnt++;
      }

      Console.WriteLine($"Case #{tc}: {cnt}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;

  for (int tc = 1; tc <= t; tc++) {
    int n; cin >> n;
    vi h(n);
    for (int i = 0; i < n; i++) cin >> h[i];

    int cnt = 0;
    for (int i = 1; i + 1 < n; i++) {
      if (h[i] > h[i - 1] && h[i] > h[i + 1]) cnt++;
    }

    cout << "Case #" << tc << ": " << cnt << "\n";
  }

  return 0;
}
```
