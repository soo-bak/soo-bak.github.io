## 문제 링크
[26940번 - Chokladkartongen](https://www.acmicpc.net/problem/26940)

## 설명
초콜릿 상자의 남은 개수를 시간 순으로 관찰한 값이 주어질 때, 최소 구매 횟수를 구하는 문제입니다. 관찰값이 증가했다면 새 상자를 개봉한 것입니다.

<br>

## 접근법
인접한 관찰값을 비교해 뒤값이 더 크면 새 상자를 산 것으로 간주합니다. 모든 증가 횟수를 세면 최소 구매 횟수가 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var cnt = 0;
    for (var i = 1; i < n; i++) {
      if (arr[i] > arr[i - 1]) cnt++;
    }

    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi a(n);
  for (int i = 0; i < n; i++)
    cin >> a[i];

  int cnt = 0;
  for (int i = 1; i < n; i++) {
    if (a[i] > a[i - 1]) cnt++;
  }

  cout << cnt << "\n";

  return 0;
}
```
