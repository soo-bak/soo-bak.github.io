## 문제 링크
[31561번 - 시계탑](https://www.acmicpc.net/problem/31561)

## 설명
비정상 속도로 도는 시계의 분침이 가리키는 값을 실제 분으로 환산하는 문제입니다.

<br>

## 접근법
실제 0~15분 동안 분침은 2배 속도로 돌아 0~30 위치를 가리킵니다.

실제 15~60분 동안은 2/3배 속도로 돌아 30~60 위치를 가리킵니다.

표시된 분이 30 이하이면 2로 나누고, 30 초과이면 역산하여 (3 × 표시분 - 60) / 2로 실제 분을 구합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var m = int.Parse(Console.ReadLine()!);
    var t = m <= 30 ? m / 2.0 : (3 * m - 60) / 2.0;
    Console.WriteLine(t.ToString("F1"));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int m; cin >> m;
  double t = m <= 30 ? m / 2.0 : (3 * m - 60) / 2.0;
  cout << fixed << setprecision(1) << t << "\n";

  return 0;
}
```
