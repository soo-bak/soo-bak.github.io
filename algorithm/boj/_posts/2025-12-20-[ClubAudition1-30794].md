## 문제 링크
[30794번 - 가희와 클럽 오디션 1](https://www.acmicpc.net/problem/30794)

## 설명
키 노트 레벨과 판정 문자열이 주어질 때, 해당하는 점수를 계산하여 출력하는 문제입니다.

<br>

## 접근법
먼저 판정 문자열에 따라 기본 점수 배율을 결정합니다.

이후 레벨과 배율을 곱하여 최종 점수를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var lv = int.Parse(parts[0]);
    var judge = parts[1];

    var score = judge switch {
      "miss" => 0,
      "bad" => 200 * lv,
      "cool" => 400 * lv,
      "great" => 600 * lv,
      "perfect" => 1000 * lv,
      _ => 0
    };

    Console.WriteLine(score);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int lv; string judge;
  cin >> lv >> judge;

  int score = 0;
  if (judge == "miss") score = 0;
  else if (judge == "bad") score = 200 * lv;
  else if (judge == "cool") score = 400 * lv;
  else if (judge == "great") score = 600 * lv;
  else if (judge == "perfect") score = 1000 * lv;

  cout << score << "\n";

  return 0;
}
```
