## 문제 링크
[30031번 - 지폐 세기](https://www.acmicpc.net/problem/30031)

## 설명
n장의 지폐에 대해 가로/세로 길이가 주어질 때, 총액을 구하는 문제입니다.

<br>
대한민국 지폐는 가로 길이에 따라 136mm는 1000원, 142mm는 5000원, 148mm는 10000원, 154mm는 50000원입니다.

<br>

## 접근법
가로 길이를 금액으로 변환하는 표를 만들어 각 지폐의 금액을 더합니다. 세로 길이는 모두 동일하므로 사용하지 않습니다.

- - -

## Code

### C#
```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var value = new Dictionary<int, int> {
      {136, 1000},
      {142, 5000},
      {148, 10000},
      {154, 50000}
    };

    var sum = 0;
    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var w = int.Parse(parts[0]);
      sum += value[w];
    }

    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  unordered_map<int, int> val = {
    {136, 1000},
    {142, 5000},
    {148, 10000},
    {154, 50000}
  };

  int sum = 0;
  for (int i = 0; i < n; i++) {
    int w, h; cin >> w >> h;
    sum += val[w];
  }

  cout << sum << "\n";

  return 0;
}
```
