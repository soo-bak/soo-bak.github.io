## 문제 링크
[31472번 - 갈래의 색종이 자르기](https://www.acmicpc.net/problem/31472)

## 설명
짝수 길이의 정사각형을 가로로 반 잘랐을 때 한 조각의 넓이가 주어지면, 원래 정사각형의 둘레를 구하는 문제입니다.

<br>

## 접근법
한 조각의 넓이가 W이면 원래 정사각형의 넓이는 2W입니다. 한 변의 길이는 2W의 제곱근이고, 문제 조건상 항상 정수가 됩니다. 둘레는 한 변의 길이에 4를 곱하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var w = int.Parse(Console.ReadLine()!);
    var side = (int)Math.Sqrt(2 * w);

    Console.WriteLine(4 * side);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int w; cin >> w;
  int side = (int)round(sqrt(2 * w));

  cout << 4 * side << "\n";

  return 0;
}
```
