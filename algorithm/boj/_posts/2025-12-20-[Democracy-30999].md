## 문제 링크
[30999번 - 민주주의](https://www.acmicpc.net/problem/30999)

## 설명
문제 후보별로 출제위원들의 찬반 투표가 주어질 때, 과반수 찬성을 받은 문제의 개수를 구하는 문제입니다.

<br>

## 접근법
각 문제 후보에 대해 찬성표(O)의 개수를 세고, 출제위원 수의 절반을 초과하면 출제될 문제로 카운트합니다. 출제위원 수가 홀수이므로 동률은 발생하지 않습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);
    var need = m / 2;
    var ans = 0;

    for (var i = 0; i < n; i++) {
      var s = Console.ReadLine()!;
      var cnt = 0;
      foreach (var ch in s) if (ch == 'O') cnt++;
      if (cnt > need) ans++;
    }

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;
  int need = m / 2;
  int ans = 0;

  for (int i = 0; i < n; i++) {
    string s; cin >> s;
    int cnt = 0;
    for (char c : s) if (c == 'O') cnt++;
    if (cnt > need) ans++;
  }

  cout << ans << "\n";

  return 0;
}
```
