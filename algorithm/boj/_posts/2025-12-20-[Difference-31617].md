## 문제 링크
[31617번 - 差 (Difference)](https://www.acmicpc.net/problem/31617)

## 설명
두 수열 A와 B가 주어질 때, A의 원소에 K를 더한 값이 B에 존재하는 쌍의 개수를 구하는 문제입니다.

<br>

## 접근법
A의 각 원소와 B의 각 원소를 모두 비교합니다. A의 원소에 K를 더한 값이 B의 원소와 같으면 카운트를 증가시킵니다. N, M이 작으므로 2중 루프로 충분합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var k = int.Parse(Console.ReadLine()!);
    var n = int.Parse(Console.ReadLine()!);
    var va = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var m = int.Parse(Console.ReadLine()!);
    var vb = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var cnt = 0;
    for (var i = 0; i < n; i++) {
      for (var j = 0; j < m; j++) {
        if (va[i] + k == vb[j]) cnt++;
      }
    }

    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int k, n; cin >> k >> n;
  vi va(n);
  for (int &a : va) cin >> a;

  int m; cin >> m;
  vi vb(m);
  for (int &b : vb) cin >> b;

  int cnt = 0;
  for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
      if (va[i] + k == vb[j]) cnt++;
    }
  }

  cout << cnt << "\n";

  return 0;
}
```
