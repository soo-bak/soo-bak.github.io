## 문제 링크
[29729번 - 가변 배열](https://www.acmicpc.net/problem/29729)

## 설명
초기 최대 크기가 주어진 가변 배열에 삽입과 삭제 명령을 처리한 뒤 최종 최대 크기를 구하는 문제입니다. 삽입 시 사용량이 용량과 같으면 용량을 두 배로 확장합니다.

<br>

## 접근법
현재 용량과 사용량 두 변수를 유지하면서 명령을 순서대로 처리합니다. 삽입 명령이 들어오면 사용량이 용량과 같은지 확인하고, 같으면 용량을 두 배로 늘린 뒤 사용량을 증가시킵니다. 삭제 명령은 단순히 사용량만 감소시킵니다.

용량은 늘어나기만 하고 줄어들지 않으므로 모든 명령 처리 후 용량이 최종 최대 크기가 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var s = int.Parse(first[0]);
    var n = int.Parse(first[1]);
    var m = int.Parse(first[2]);

    var u = 0;
    var total = n + m;
    for (var i = 0; i < total; i++) {
      var cmd = int.Parse(Console.ReadLine()!);
      if (cmd == 1) {
        if (u == s) s *= 2;
        u++;
      } else u--;
    }

    Console.WriteLine(s);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int s, n, m; cin >> s >> n >> m;

  int u = 0;
  int total = n + m;
  for (int i = 0; i < total; i++) {
    int cmd; cin >> cmd;
    if (cmd == 1) {
      if (u == s) s *= 2;
      u++;
    } else u--;
  }

  cout << s << "\n";

  return 0;
}
```
