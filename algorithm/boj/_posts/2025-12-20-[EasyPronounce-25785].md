## 문제 링크
[25785번 - Easy-to-Pronounce Words](https://www.acmicpc.net/problem/25785)

## 설명
주어진 단어가 발음하기 쉬운지 판정하는 문제입니다.

모음과 자음이 번갈아가며 등장하면 발음하기 쉬운 단어로 간주하며, 시작 문자는 모음이든 자음이든 상관없습니다.

<br>

## 접근법
문자열을 앞에서부터 순회하면서 인접한 두 문자가 같은 종류(둘 다 모음 또는 둘 다 자음)인지 확인합니다.

같은 종류가 연속으로 나타나면 모음-자음 번갈아 나오는 조건을 위반한 것이므로 즉시 발음하기 어려운 단어로 판정하고 탐색을 종료합니다. 끝까지 위반이 없으면 발음하기 쉬운 단어입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static bool IsVowel(char c) =>
    c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u';

  static void Main() {
    var s = Console.ReadLine()!;
    var ok = true;
    for (var i = 1; i < s.Length; i++) {
      var prevV = IsVowel(s[i - 1]);
      var curV = IsVowel(s[i]);
      if (prevV == curV) { ok = false; break; }
    }
    Console.WriteLine(ok ? 1 : 0);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

bool isVowel(char c) {
  return c=='a'||c=='e'||c=='i'||c=='o'||c=='u';
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;

  bool ok = true;
  for (int i = 1; i < (int)s.size(); i++) {
    bool prevV = isVowel(s[i - 1]);
    bool curV = isVowel(s[i]);
    if (prevV == curV) { ok = false; break; }
  }

  cout << (ok ? 1 : 0) << "\n";

  return 0;
}
```
