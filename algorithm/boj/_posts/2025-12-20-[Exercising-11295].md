## 문제 링크
[11295번 - Exercising](https://www.acmicpc.net/problem/11295)

## 설명
사용자별로 보폭 길이와 걸음 수가 주어질 때, 각 기록마다 이동 거리를 킬로미터 단위로 계산하여 출력하는 문제입니다.

<br>

## 접근법
먼저 보폭(cm)과 걸음 수를 곱한 뒤 100000으로 나누어 킬로미터로 변환합니다. 보폭이 0이면 입력이 종료됩니다.

이후 각 사용자마다 번호를 출력하고, 해당 사용자의 모든 기록을 소수점 다섯째 자리까지 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var user = 1;
    while (true) {
      var line = Console.ReadLine();
      if (line == null)
        break;

      var l = int.Parse(line);
      if (l == 0)
        break;

      var n = int.Parse(Console.ReadLine()!);
      Console.WriteLine($"User {user}");
      for (var i = 0; i < n; i++) {
        var steps = int.Parse(Console.ReadLine()!);
        var dist = steps * l / 100000.0;
        Console.WriteLine($"{dist:F5}");
      }
      user++;
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int user = 1;
  int l;
  while (cin >> l) {
    if (l == 0)
      break;

    int n; cin >> n;
    cout << "User " << user << "\n";
    for (int i = 0; i < n; i++) {
      int steps; cin >> steps;
      double dist = steps * (double)l / 100000.0;
      cout << fixed << setprecision(5) << dist << "\n";
    }
    user++;
  }

  return 0;
}
```
