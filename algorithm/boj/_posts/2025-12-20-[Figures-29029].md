## 문제 링크
[29029번 - Фигурки](https://www.acmicpc.net/problem/29029)

## 설명
여러 피규어가 각각 동서남북 중 한 방향을 보고 있을 때, 모든 피규어를 한 방향으로 통일하는 데 필요한 최소 변경 횟수를 구하는 문제입니다.

<br>

## 접근법
가장 많은 피규어가 보고 있는 방향으로 통일하면 변경 횟수가 최소가 됩니다.

각 방향의 개수를 세고, 전체 개수에서 가장 많은 방향의 개수를 빼면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;

    var cnt = new int[4];
    foreach (var c in s) {
      switch (c) {
        case 'N': cnt[0]++; break;
        case 'S': cnt[1]++; break;
        case 'W': cnt[2]++; break;
        case 'E': cnt[3]++; break;
      }
    }

    var maxCnt = Math.Max(Math.Max(cnt[0], cnt[1]), Math.Max(cnt[2], cnt[3]));
    Console.WriteLine(n - maxCnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  int cnt[4] = {0, 0, 0, 0};
  for (char c : s) {
    if (c == 'N') cnt[0]++;
    else if (c == 'S') cnt[1]++;
    else if (c == 'W') cnt[2]++;
    else if (c == 'E') cnt[3]++;
  }

  int maxCnt = max(max(cnt[0], cnt[1]), max(cnt[2], cnt[3]));
  cout << n - maxCnt << "\n";

  return 0;
}
```
