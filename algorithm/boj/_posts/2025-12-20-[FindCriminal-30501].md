## 문제 링크
[30501번 - 관공... 어찌하여 목만 오셨소...](https://www.acmicpc.net/problem/30501)

## 설명
용의자들의 이름이 주어지고, 이름에 S가 들어가는 용의자가 단 하나일 때 그 이름을 찾아 출력하는 문제입니다.

<br>

## 접근법
모든 이름을 순회하며 S 문자가 포함되어 있는지 확인합니다. 유일성이 보장되므로 발견 즉시 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < n; i++) {
      var name = Console.ReadLine()!;
      if (name.Contains('S')) {
        Console.WriteLine(name);
        return;
      }
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  for (int i = 0; i < n; i++) {
    string name; cin >> name;
    if (name.find('S') != string::npos) {
      cout << name << "\n";
      return 0;
    }
  }

  return 0;
}
```
