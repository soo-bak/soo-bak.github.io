## 문제 링크
[30310번 - Finding Forks](https://www.acmicpc.net/problem/30310)

## 설명
포크 종류별 개수가 주어질 때, 최소 두 자리를 비우기 위해 치워야 하는 포크의 최소 개수를 구하는 문제입니다.

한 자리를 비우려면 해당 종류의 포크를 모두 치워야 합니다.

<br>

## 접근법
두 자리를 비우려면 두 종류의 포크를 모두 치워야 합니다. 치우는 포크 수를 최소화하려면 개수가 가장 적은 두 종류를 선택하면 됩니다.

배열을 한 번 순회하면서 가장 작은 값과 두 번째로 작은 값을 찾아 합하면 답이 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Console.ReadLine()!.Split();
    var min1 = long.MaxValue;
    var min2 = long.MaxValue;

    for (var i = 0; i < n; i++) {
      var v = long.Parse(parts[i]);
      if (v < min1) { min2 = min1; min1 = v; }
      else if (v < min2) min2 = v;
    }

    Console.WriteLine(min1 + min2);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  cin >> n;

  ll min1 = LLONG_MAX, min2 = LLONG_MAX;
  for (int i = 0; i < n; i++) {
    ll v; cin >> v;
    if (v < min1) { min2 = min1; min1 = v; }
    else if (v < min2) min2 = v;
  }

  cout << min1 + min2 << "\n";

  return 0;
}
```
