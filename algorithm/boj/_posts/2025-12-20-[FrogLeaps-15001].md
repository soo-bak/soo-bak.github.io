## 문제 링크
[15001번 - Frog Leaps](https://www.acmicpc.net/problem/15001)

## 설명
점프 거리의 제곱만큼 에너지가 드는 개구리가 정거장들을 순서대로 방문할 때 최소 에너지를 구하는 문제입니다. 제곱 비용은 볼록하므로 거리를 쪼갤수록 비용이 줄어듭니다.

<br>

## 접근법
한 번에 긴 거리를 뛰는 것보다 짧게 여러 번 뛰는 것이 제곱 비용에서 유리합니다. 따라서 모든 정거장을 순서대로 밟으며 인접한 정거장 사이의 거리를 제곱해 더하면 최소 에너지가 됩니다.

정거장 좌표가 오름차순으로 주어지므로 인접 차이를 계산해 제곱 합을 구합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var pos = new int[n];
    for (var i = 0; i < n; i++)
      pos[i] = int.Parse(Console.ReadLine()!);

    var energy = 0L;
    for (var i = 1; i < n; i++) {
      var d = (long)(pos[i] - pos[i - 1]);
      energy += d * d;
    }

    Console.WriteLine(energy);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef vector<ll> vll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vll x(n);
  for (int i = 0; i < n; i++)
    cin >> x[i];

  ll energy = 0;
  for (int i = 1; i < n; i++) {
    ll d = x[i] - x[i - 1];
    energy += d * d;
  }

  cout << energy << "\n";

  return 0;
}
```
