## 문제 링크
[31009번 - 진주로 가자! (Easy)](https://www.acmicpc.net/problem/31009)

## 설명
각 교통편의 도착지와 요금이 주어질 때, 진주로 가는 교통편의 요금과 그보다 비싼 교통편의 개수를 구하는 문제입니다.

<br>

## 접근법
모든 교통편 정보를 저장한 뒤 도착지가 jinju인 노선의 요금을 찾습니다. 그 후 전체 교통편을 다시 순회하면서 진주 요금보다 비싼 교통편의 개수를 셉니다.

진주행 노선은 반드시 하나 존재하므로 찾으면 바로 반복을 종료해도 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var dest = new string[n];
    var cost = new int[n];

    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      dest[i] = parts[0];
      cost[i] = int.Parse(parts[1]);
    }

    var jinju = 0;
    for (var i = 0; i < n; i++) {
      if (dest[i] == "jinju") {
        jinju = cost[i];
        break;
      }
    }

    var cnt = 0;
    for (var i = 0; i < n; i++)
      if (cost[i] > jinju) cnt++;

    Console.WriteLine(jinju);
    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vector<string> d(n);
  vi c(n);
  for (int i = 0; i < n; i++)
    cin >> d[i] >> c[i];

  int jinju = 0;
  for (int i = 0; i < n; i++) {
    if (d[i] == "jinju") {
      jinju = c[i];
      break;
    }
  }

  int cnt = 0;
  for (int i = 0; i < n; i++)
    if (c[i] > jinju) cnt++;

  cout << jinju << "\n" << cnt << "\n";

  return 0;
}
```
