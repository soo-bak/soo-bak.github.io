## 문제 링크
[30008번 - 준영이의 등급](https://www.acmicpc.net/problem/30008)

## 설명
학생 수 n, 과목 수 k, 각 과목 등수가 주어질 때, 등수를 백분율로 환산하여 1~9등급을 매기는 문제입니다.

<br>
백분율은 등수 × 100 ÷ n으로 계산하며, 4 이하면 1등급, 11 이하면 2등급, ... , 96 초과면 9등급입니다.

<br>

## 접근법
각 과목의 등수에 대해 백분율을 구한 뒤, 조건문으로 구간을 판정합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static int GetGrade(int p) {
    if (p <= 4) return 1;
    if (p <= 11) return 2;
    if (p <= 23) return 3;
    if (p <= 40) return 4;
    if (p <= 60) return 5;
    if (p <= 77) return 6;
    if (p <= 89) return 7;
    if (p <= 96) return 8;
    return 9;
  }

  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var k = int.Parse(first[1]);
    var g = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var ans = new int[k];
    for (var i = 0; i < k; i++) {
      var p = g[i] * 100 / n;
      ans[i] = GetGrade(p);
    }

    Console.WriteLine(string.Join(" ", ans));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int getGrade(int p) {
  if (p <= 4) return 1;
  if (p <= 11) return 2;
  if (p <= 23) return 3;
  if (p <= 40) return 4;
  if (p <= 60) return 5;
  if (p <= 77) return 6;
  if (p <= 89) return 7;
  if (p <= 96) return 8;
  return 9;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, k; cin >> n >> k;
  vi g(k);
  for (int i = 0; i < k; i++)
    cin >> g[i];

  for (int i = 0; i < k; i++) {
    int p = g[i] * 100 / n;
    cout << getGrade(p);
    if (i + 1 != k) cout << ' ';
  }
  cout << "\n";

  return 0;
}
```
