## 문제 링크
[13617번 - Handebol](https://www.acmicpc.net/problem/13617)

## 설명
각 선수의 경기별 득점이 주어질 때, 모든 경기에서 한 골 이상 넣은 선수의 수를 구하는 문제입니다.

<br>

## 접근법
각 선수에 대해 모든 경기의 득점이 0보다 큰지 확인합니다. 조건을 만족하면 카운트를 증가시킵니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);

    var ans = 0;
    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var allPositive = true;
      for (var j = 0; j < m; j++) {
        var x = int.Parse(parts[j]);
        if (x == 0) { allPositive = false; break; }
      }
      if (allPositive) ans++;
    }

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;

  int ans = 0;
  for (int i = 0; i < n; i++) {
    bool allPositive = true;
    for (int j = 0; j < m; j++) {
      int x; cin >> x;
      if (x == 0) allPositive = false;
    }
    if (allPositive) ans++;
  }

  cout << ans << "\n";

  return 0;
}
```
