## 문제 링크
[25814번 - Heavy Numbers](https://www.acmicpc.net/problem/25814)

## 설명
양의 정수의 무게를 자릿수 개수와 자릿수 합의 곱으로 정의할 때, 두 수 중 더 무거운 수를 찾는 문제입니다.

첫 번째 수가 더 무거우면 1, 두 번째 수가 더 무거우면 2, 같으면 0을 출력합니다.

<br>

## 접근법
각 수를 문자열로 변환하면 자릿수 개수는 문자열의 길이가 되고, 각 문자에서 숫자를 추출해 더하면 자릿수 합을 구할 수 있습니다. 두 값을 곱하면 무게가 됩니다.

두 수의 무게를 각각 계산한 뒤 비교하여 결과를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static int Weight(int x) {
    var s = x.ToString();
    var sum = 0;
    foreach (var ch in s)
      sum += ch - '0';
    return s.Length * sum;
  }

  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);

    var wa = Weight(a);
    var wb = Weight(b);

    var ans = 0;
    if (wa > wb) ans = 1;
    else if (wb > wa) ans = 2;

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int weight(int x) {
  string s = to_string(x);
  int sum = 0;
  for (char c : s)
    sum += c - '0';
  return (int)s.size() * sum;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b; cin >> a >> b;

  int wa = weight(a);
  int wb = weight(b);

  int ans = 0;
  if (wa > wb) ans = 1;
  else if (wb > wa) ans = 2;

  cout << ans << "\n";

  return 0;
}
```
