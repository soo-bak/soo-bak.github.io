## 문제 링크
[9907번 - ID](https://www.acmicpc.net/problem/9907)

## 설명
7자리 숫자가 주어질 때 각 자리수에 정해진 가중치(2,7,6,5,4,3,2)를 곱해 합을 구하고, 11로 나눈 나머지를 대응 표(0=J, …, 9=I, 10=Z)에 따라 문자로 변환해 출력하는 문제입니다.

<br>

## 접근법
7자리 숫자의 각 자리에 정해진 가중치를 곱해 합을 구합니다. 이 합을 11로 나눈 나머지가 0이면 J, 1부터 9까지는 A부터 I, 10이면 Z에 대응됩니다.

매핑 문자열을 미리 정의해두면 나머지 값을 인덱스로 바로 문자를 찾을 수 있습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var w = new[] { 2, 7, 6, 5, 4, 3, 2 };
    var sum = 0;
    for (var i = 0; i < 7; i++)
      sum += (s[i] - '0') * w[i];

    var r = sum % 11;
    var map = "JABCDEFGHIZ";
    Console.WriteLine(map[r]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>

}
```
