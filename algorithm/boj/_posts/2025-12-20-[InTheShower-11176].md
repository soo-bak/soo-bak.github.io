## 문제 링크
[11176번 - In the Shower](https://www.acmicpc.net/problem/11176)

## 설명
샴푸 병의 시도 횟수가 임계값을 초과하면 빈 병으로 간주하여 그 개수를 세는 문제입니다.

<br>

## 접근법
각 테스트케이스마다 임계값과 병의 개수를 입력받고, 각 병의 시도 횟수를 순회합니다.

시도 횟수가 임계값보다 크면 빈 병으로 판정하여 카운트를 증가시킵니다. 모든 병을 확인한 후 카운트를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var tc = 0; tc < t; tc++) {
      var first = Console.ReadLine()!.Split();
      var e = int.Parse(first[0]);
      var n = int.Parse(first[1]);

      var cnt = 0;
      for (var i = 0; i < n; i++) {
        var v = int.Parse(Console.ReadLine()!);
        if (v > e) cnt++;
      }

      Console.WriteLine(cnt);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;

  while (t--) {
    int e, n; cin >> e >> n;

    int cnt = 0;
    for (int i = 0; i < n; i++) {
      int v; cin >> v;
      if (v > e) cnt++;
    }

    cout << cnt << "\n";
  }

  return 0;
}
```
