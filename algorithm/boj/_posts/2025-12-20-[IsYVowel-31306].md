## 문제 링크
[31306번 - Is Y a Vowel?](https://www.acmicpc.net/problem/31306)

## 설명
소문자 단어가 주어질 때, y를 모음으로 보지 않을 때와 모음으로 볼 때의 모음 개수를 각각 세어 출력하는 문제입니다. 기본 모음은 a, e, i, o, u입니다.

<br>

## 접근법
기본 모음은 a, e, i, o, u이고, y를 모음으로 볼 때는 여기에 y가 추가됩니다. 문자열을 순회하면서 기본 모음이면 두 카운터 모두 증가시키고, y이면 y를 포함하는 카운터만 증가시킵니다.

이렇게 하면 한 번의 순회로 두 경우의 모음 개수를 동시에 계산할 수 있습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var cnt1 = 0;
    var cnt2 = 0;

    foreach (var ch in s) {
      if ("aeiou".IndexOf(ch) >= 0) { cnt1++; cnt2++; }
      else if (ch == 'y') cnt2++;
    }

    Console.WriteLine($"{cnt1} {cnt2}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;

  int cnt1 = 0, cnt2 = 0;
  for (char c : s) {
    if (c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u') { cnt1++; cnt2++; }
    else if (c == 'y') cnt2++;
  }

  cout << cnt1 << " " << cnt2 << "\n";

  return 0;
}
```
