## 문제 링크
[27333번 - JOI エディタ (JOI Editor)](https://www.acmicpc.net/problem/27333)

## 설명
j, o, i 키를 순서대로 입력하며, 같은 소문자가 연속으로 두 글자가 되면 즉시 대문자로 치환하는 에디터의 결과를 구하는 문제입니다.

<br>

## 접근법
문자를 하나씩 추가하면서 매번 마지막 두 글자를 확인합니다. 두 글자가 같은 소문자이면 둘 다 대문자로 치환합니다. 한 번 대문자가 된 글자는 이후 치환 대상이 아닙니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    _ = Console.ReadLine();
    var s = Console.ReadLine()!;

    var sb = new StringBuilder();
    foreach (var ch in s) {
      sb.Append(ch);
      var len = sb.Length;
      if (len >= 2) {
        var a = sb[len - 2];
        var b = sb[len - 1];
        if (char.IsLower(a) && a == b) {
          sb[len - 2] = char.ToUpper(a);
          sb[len - 1] = char.ToUpper(b);
        }
      }
    }

    Console.WriteLine(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  string out;
  out.reserve(n);
  for (char c : s) {
    out.push_back(c);
    int len = out.size();
    if (len >= 2) {
      char a = out[len - 2], b = out[len - 1];
      if (islower(a) && a == b) {
        out[len - 2] = toupper(a);
        out[len - 1] = toupper(b);
      }
    }
  }

  cout << out << "\n";

  return 0;
}
```
