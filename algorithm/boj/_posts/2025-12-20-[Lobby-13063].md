## 문제 링크
[13063번 - Lobby](https://www.acmicpc.net/problem/13063)

## 설명
전체 의원 수와 찬성/반대 인원이 주어질 때, 과반을 넘기기 위해 설득해야 하는 무소속의 최소 인원수를 구하는 문제입니다. 불가능하면 -1을 출력합니다.

<br>

## 접근법
과반을 넘기려면 전체의 절반에 1을 더한 표가 필요합니다. 이미 찬성하는 보수당 인원을 빼면 추가로 필요한 표 수가 나옵니다. 이 값이 음수면 이미 과반이므로 0으로 처리합니다.

무소속 수는 전체에서 보수당과 개혁당을 뺀 값입니다. 필요한 표가 무소속 수 이하면 그만큼 설득하면 되고, 초과하면 불가능하므로 -1을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    while (true) {
      var line = Console.ReadLine();
      if (line == null) break;
      var parts = line.Split();
      var n = int.Parse(parts[0]);
      var m = int.Parse(parts[1]);
      var k = int.Parse(parts[2]);
      if (n == 0 && m == 0 && k == 0) break;

      var need = (n / 2 + 1) - m;
      if (need < 0) need = 0;
      var indep = n - m - k;

      var ans = need <= indep ? need : -1;
      Console.WriteLine(ans);
    }
  }
}
```
