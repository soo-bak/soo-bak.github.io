## 문제 링크
[17884번 - Lost Lineup](https://www.acmicpc.net/problem/17884)

## 설명
지미를 포함한 n명이 줄을 섰을 때, 각 사람은 자신과 지미 사이에 몇 명이 있었는지만 기억합니다. 지미는 항상 맨 앞에 있습니다. 이 정보를 이용해 원래 줄 서기 순서를 출력하는 문제입니다.

<br>

## 접근법
지미는 항상 맨 앞이므로 1번 위치에 고정됩니다. 각 친구가 기억하는 값은 지미와 자신 사이에 있는 인원 수이므로, 그 값에 2를 더하면 해당 친구의 위치가 됩니다.

배열을 만들어 지미를 첫 칸에 넣고, 각 친구를 계산된 위치에 배치하면 원래 줄 서기 순서가 복원됩니다.

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var dist = Console.ReadLine()!.Split().Select(int.Parse).ToArray();

    var order = new int[n];
    order[0] = 1;
    for (var i = 0; i < dist.Length; i++) {
      var pos = dist[i] + 1;
      order[pos] = i + 2;
    }

    Console.WriteLine(string.Join(" ", order));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi order(n, 0);
  order[0] = 1;

  for (int i = 0; i < n - 1; i++) {
    int d; cin >> d;
    order[d + 1] = i + 2;
  }

  for (int i = 0; i < n; i++) {
    if (i) cout << ' ';
    cout << order[i];
  }
  cout << "\n";

  return 0;
}
```
