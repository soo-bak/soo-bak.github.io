## 문제 링크
[30664번 - Loteria Falha](https://www.acmicpc.net/problem/30664)

## 설명
최대 30자리 정수가 주어질 때, 42의 배수인지 판별하는 문제입니다. 64비트 범위를 넘을 수 있으므로 문자열로 나머지를 계산해야 합니다.

<br>

## 접근법
문자열로 읽은 뒤 각 자리를 순회하며 나머지를 갱신합니다. 최종 나머지가 0이면 배수입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    while (true) {
      var s = Console.ReadLine()!;
      if (s == "0") break;

      var rem = 0;
      foreach (var c in s)
        rem = (rem * 10 + (c - '0')) % 42;

      Console.WriteLine(rem == 0 ? "PREMIADO" : "TENTE NOVAMENTE");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s;
  while (cin >> s) {
    if (s == "0") break;
    int rem = 0;
    for (char c : s)
      rem = (rem * 10 + (c - '0')) % 42;
    if (rem == 0) cout << "PREMIADO\n";
    else cout << "TENTE NOVAMENTE\n";
  }

  return 0;
}
```
