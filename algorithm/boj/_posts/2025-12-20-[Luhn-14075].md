## 문제 링크
[14075번 - Luhn](https://www.acmicpc.net/problem/14075)

## 설명
16자리 카드 번호가 루한 알고리즘을 만족하는지 판정하는 문제입니다.

끝자리부터 한 자리 건너 하나씩 숫자를 두 배로 만들고, 두 배가 10 이상이면 9를 뺍니다. 모든 자릿수의 합이 10으로 나누어떨어지면 유효합니다.

<br>

## 접근법
오른쪽 끝부터 순회하며 위치에 따라 처리합니다. 오른쪽에서 짝수 번째 위치의 숫자는 두 배로 만들고, 10 이상이면 9를 빼서 자릿수 합과 같은 효과를 냅니다. 모든 자릿수를 더한 후 10으로 나눈 나머지가 0인지 확인합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var sum = 0;
    var len = s.Length;

    for (var i = len - 1; i >= 0; i--) {
      var pos = len - 1 - i;
      var d = s[i] - '0';
      if ((pos & 1) == 1) {
        d *= 2;
        if (d >= 10) d -= 9;
      }
      sum += d;
    }

    Console.WriteLine(sum % 10 == 0 ? "DA" : "NE");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;

  int sum = 0;
  int len = s.size();
  for (int i = len - 1; i >= 0; i--) {
    int pos = len - 1 - i;
    int d = s[i] - '0';
    if (pos % 2 == 1) {
      d *= 2;
      if (d >= 10) d -= 9;
    }
    sum += d;
  }

  cout << (sum % 10 == 0 ? "DA" : "NE") << "\n";

  return 0;
}
```
