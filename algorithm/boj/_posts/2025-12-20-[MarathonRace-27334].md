## 문제 링크
[27334번 - Marathon Race](https://www.acmicpc.net/problem/27334)

## 설명
각 학생의 마라톤 기록이 주어질 때, 자신보다 기록이 빠른(시간이 더 작은) 학생 수에 1을 더해 순위를 구하는 문제입니다.

<br>

## 접근법
순위는 자신보다 빠른 사람의 수에 1을 더한 값입니다. 따라서 각 학생에 대해 모든 학생의 기록과 비교하여 자신보다 작은 기록의 개수를 세면 순위를 구할 수 있습니다.

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var arr = Console.ReadLine()!.Split().Select(int.Parse).ToArray();

    for (var i = 0; i < n; i++) {
      var faster = 0;
      for (var j = 0; j < n; j++) {
        if (arr[j] < arr[i]) faster++;
      }
      Console.WriteLine(faster + 1);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi a(n);
  for (int i = 0; i < n; i++)
    cin >> a[i];

  for (int i = 0; i < n; i++) {
    int faster = 0;
    for (int j = 0; j < n; j++) {
      if (a[j] < a[i]) faster++;
    }
    cout << faster + 1 << "\n";
  }

  return 0;
}
```
