## 문제 링크
[9776번 - Max Volume](https://www.acmicpc.net/problem/9776)

## 설명
원뿔, 원기둥, 구 중 하나인 도형들이 주어질 때, 각 도형의 부피를 계산하여 최대값을 구하는 문제입니다.

<br>

## 접근법
먼저 각 도형의 부피 공식을 적용합니다. 원뿔은 (1/3)πr²h, 원기둥은 πr²h, 구는 (4/3)πr³ 입니다.

이후 모든 도형의 부피를 계산하며 최대값을 갱신합니다. 출력 시 소수점 셋째 자리까지 고정 포맷을 사용합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    const double PI = 3.14159;
    var n = int.Parse(Console.ReadLine()!);
    var maxV = 0.0;

    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var type = parts[0][0];
      if (type == 'C') {
        var r = double.Parse(parts[1]);
        var h = double.Parse(parts[2]);
        var v = (PI * r * r * h) / 3.0;
        if (v > maxV)
          maxV = v;
      } else if (type == 'L') {
        var r = double.Parse(parts[1]);
        var h = double.Parse(parts[2]);
        var v = PI * r * r * h;
        if (v > maxV)
          maxV = v;
      } else if (type == 'S') {
        var r = double.Parse(parts[1]);
        var v = (4.0 * PI * r * r * r) / 3.0;
        if (v > maxV)
          maxV = v;
      }
    }

    Console.WriteLine("{0:F3}", maxV);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  const double PI = 3.14159;
  int n; cin >> n;
  double maxV = 0.0;

  for (int i = 0; i < n; i++) {
    char type; cin >> type;
    if (type == 'C') {
      double r, h; cin >> r >> h;
      double v = (PI * r * r * h) / 3.0;
      maxV = max(maxV, v);
    } else if (type == 'L') {
      double r, h; cin >> r >> h;
      double v = PI * r * r * h;
      maxV = max(maxV, v);
    } else if (type == 'S') {
      double r; cin >> r;
      double v = (4.0 * PI * r * r * r) / 3.0;
      maxV = max(maxV, v);
    }
  }

  cout << fixed << setprecision(3) << maxV << "\n";

  return 0;
}
```
