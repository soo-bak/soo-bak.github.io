## 문제 링크
[29731번 - 2033년 밈 투표](https://www.acmicpc.net/problem/29731)

## 설명
허용된 문장 목록이 주어지고, 입력된 문장들이 모두 목록에 속하는지 판별하는 문제입니다. 하나라도 목록에 없으면 Yes, 전부 있으면 No를 출력합니다.

<br>

## 접근법
먼저 허용 문장들을 집합에 저장합니다.

다음으로 입력 문장을 한 줄씩 검사해 집합에 없으면 플래그를 세웁니다.

- - -

## Code

### C#
```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var valid = new HashSet<string> {
      "Never gonna give you up",
      "Never gonna let you down",
      "Never gonna run around and desert you",
      "Never gonna make you cry",
      "Never gonna say goodbye",
      "Never gonna tell a lie and hurt you",
      "Never gonna stop"
    };

    var n = int.Parse(Console.ReadLine()!);
    var hacked = false;
    for (var i = 0; i < n; i++) {
      var s = Console.ReadLine()!;
      if (!valid.Contains(s)) hacked = true;
    }

    Console.WriteLine(hacked ? "Yes" : "No");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  unordered_set<string> valid = {
    "Never gonna give you up",
    "Never gonna let you down",
    "Never gonna run around and desert you",
    "Never gonna make you cry",
    "Never gonna say goodbye",
    "Never gonna tell a lie and hurt you",
    "Never gonna stop"
  };

  int n; cin >> n;
  string line;
  getline(cin, line);

  bool hacked = false;
  for (int i = 0; i < n; i++) {
    getline(cin, line);
    if (!valid.count(line)) hacked = true;
  }

  cout << (hacked ? "Yes" : "No") << "\n";

  return 0;
}
```
