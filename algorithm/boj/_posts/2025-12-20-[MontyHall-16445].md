## 문제 링크
[16445번 - Desvendando Monty Hall](https://www.acmicpc.net/problem/16445)

## 설명
플레이어가 처음에 항상 문 1을 고르고, 진행자가 염소가 있는 문을 공개한 뒤 항상 문을 바꿀 때, 차를 얻는 횟수를 세는 문제입니다.

<br>
차가 문 1에 없으면 스위치 전략으로 승리하고, 문 1에 있으면 패배합니다.

<br>

## 접근법
각 게임에서 차가 있는 문 번호가 1이 아니면 승리입니다. 1이 아닌 경우를 세어 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var win = 0;
    for (var i = 0; i < n; i++) {
      var door = int.Parse(Console.ReadLine()!);
      if (door != 1) win++;
    }
    Console.WriteLine(win);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);

  int n; cin >> n;

  for (int i = 0; i < n; i++) {
    int door; cin >> door;
    if (door != 1) win++;
  }
  cout << win << "\n";

  return 0;
}
```
