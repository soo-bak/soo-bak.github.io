## 문제 링크
[24198번 - Muffinspelet](https://www.acmicpc.net/problem/24198)

## 설명
머핀 더미를 번갈아 절반씩 나누고 먹는 게임을 시뮬레이션하여 두 플레이어가 먹는 머핀 개수를 구하는 문제입니다.

<br>

## 접근법
먼저 더미를 반으로 나누면 큰 쪽을 먹고 작은 쪽이 남습니다. 두 사람이 번갈아 큰 더미를 선택하므로 매번 올림 나눗셈으로 먹는 양을 계산합니다.

이후 더미가 0이 될 때까지 반복하면 각자가 먹는 총량을 구할 수 있습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);

    var alf = 0; var beata = 0;
    var pile = n;
    var beataTurn = true;

    while (pile > 0) {
      var eat = (pile + 1) / 2;
      if (beataTurn) beata += eat;
      else alf += eat;
      pile -= eat;
      beataTurn = !beataTurn;
    }

    Console.WriteLine($"{alf} {beata}");
  }
}
```
