## 문제 링크
[24084번 - 次の文字 (Next Character)](https://www.acmicpc.net/problem/24084)

## 설명
문자열에서 J가 나타나는 위치의 직전 문자를 차례대로 출력하는 문제입니다.

<br>

## 접근법
문자열을 순회하며 현재 문자가 J이면 바로 앞 문자를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;
    for (var i = 1; i < n; i++) {
      if (s[i] == 'J') Console.WriteLine(s[i - 1]);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;
  for (int i = 1; i < n; i++) {
    if (s[i] == 'J') cout << s[i - 1] << "\n";
  }

  return 0;
}
```
