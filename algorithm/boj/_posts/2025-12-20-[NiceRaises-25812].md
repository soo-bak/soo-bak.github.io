## 문제 링크
[25812번 - Nice Raises](https://www.acmicpc.net/problem/25812)

## 설명
고정 인상액과 현재 연봉의 2배 중 더 큰 인상안을 각자 선택할 때, 어느 쪽이 더 많이 선택되었는지 판별하는 문제입니다.

<br>

## 접근법
고정 인상액이 현재 연봉보다 크면 고정 인상안이 유리하고, 작으면 2배 인상안이 유리합니다. 같으면 인상액이 동일하므로 집계하지 않습니다.

각 사람의 연봉을 순회하며 두 인상안의 선택 인원을 각각 세고, 더 많은 쪽에 따라 결과를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var r = int.Parse(first[1]);

    var cntFixed = 0;
    var cntDouble = 0;
    for (var i = 0; i < n; i++) {
      var s = int.Parse(Console.ReadLine()!);
      if (r > s) cntFixed++;
      else if (r < s) cntDouble++;
    }

    var ans = 0;
    if (cntFixed > cntDouble) ans = 1;
    else if (cntDouble > cntFixed) ans = 2;

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, r; cin >> n >> r;
  int cntFixed = 0, cntDouble = 0;

  for (int i = 0; i < n; i++) {
    int s; cin >> s;
    if (r > s) cntFixed++;
    else if (r < s) cntDouble++;
  }

  int ans = 0;
  if (cntFixed > cntDouble) ans = 1;
  else if (cntDouble > cntFixed) ans = 2;

  cout << ans << "\n";

  return 0;
}
```
