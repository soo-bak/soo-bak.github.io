## 문제 링크
[26560번 - Periods](https://www.acmicpc.net/problem/26560)

## 설명
문장이 주어질 때, 끝이 마침표로 끝나지 않으면 마침표를 하나 붙여서 출력하는 문제입니다.

<br>

## 접근법
한 줄씩 읽어 마지막 문자가 마침표인지 확인합니다. 마침표가 아니면 마침표를 추가하여 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!;
      if (line[^1] == '.') Console.WriteLine(line);
      else Console.WriteLine(line + ".");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string line;
  getline(cin, line);
  for (int i = 0; i < n; i++) {
    getline(cin, line);
    if (!line.empty() && line.back() == '.') cout << line << "\n";
    else cout << line << ".\n";
  }

  return 0;
}
```
