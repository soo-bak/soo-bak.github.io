## 문제 링크
[30007번 - 라면 공식](https://www.acmicpc.net/problem/30007)

## 설명
라면을 끓일 때 필요한 물의 양을 공식에 따라 계산하는 문제입니다.

<br>

## 접근법
주어진 공식 W = A × (X - 1) + B를 그대로 계산하여 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = int.Parse(parts[0]);
      var b = int.Parse(parts[1]);
      var x = int.Parse(parts[2]);
      var w = a * (x - 1) + b;
      Console.WriteLine(w);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  for (int i = 0; i < n; i++) {
    int a, b, x; cin >> a >> b >> x;
    int w = a * (x - 1) + b;
    cout << w << "\n";
  }

  return 0;
}
```
