## 문제 링크
[31613번 - 繰り返し (Repetition)](https://www.acmicpc.net/problem/31613)

## 설명
초기 값에서 시작해 3으로 나눈 나머지에 따라 값을 갱신하며, 목표값 이상이 될 때까지 반복 횟수를 구하는 문제입니다.

나머지가 0이면 1을 더하고, 1이면 2를 곱하고, 2이면 3을 곱합니다.

<br>

## 접근법
값이 빠르게 증가하므로 단순 시뮬레이션으로 충분합니다. 값이 목표 이상이 될 때까지 나머지에 따라 분기 처리하고 횟수를 세어 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var x = int.Parse(Console.ReadLine()!);
    var n = int.Parse(Console.ReadLine()!);

    var cnt = 0;
    while (x < n) {
      var r = x % 3;
      if (r == 0) x += 1;
      else if (r == 1) x *= 2;
      else x *= 3;
      cnt++;
    }

    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>

    else if (r == 1) x *= 2;

  }

  cout << cnt << "\n";

  return 0;
}
```
