## 문제 링크
[23925번 - Retype](https://www.acmicpc.net/problem/23925)

## 설명
게임에서 현재 진행 상황과 저장 지점이 주어질 때, 재시작하거나 되돌아가서 클리어하는 두 선택지 중 더 적은 시간을 구하는 문제입니다.

<br>

## 접근법
재시작하면 처음부터 끝까지 가야 하므로 이미 쓴 시간에 전체 레벨 수를 더합니다. 되돌아가면 저장 지점까지 뒤로 간 뒤 거기서 끝까지 다시 진행해야 합니다.

두 선택지의 총 시간을 각각 계산해서 더 작은 값을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var i = 1; i <= t; i++) {
      var p = Console.ReadLine()!.Split();
      var n = long.Parse(p[0]);
      var k = long.Parse(p[1]);
      var s = long.Parse(p[2]);

      var option1 = k - 1 + 1 + n;
      var option2 = k - 1 + (k - s) + (n - s + 1);
      var result = Math.Min(option1, option2);

      Console.WriteLine($"Case #{i}: {result}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;

  for (int i = 1; i <= t; i++) {
    ll n, k, s; cin >> n >> k >> s;

    ll option1 = k - 1 + 1 + n;
    ll option2 = k - 1 + (k - s) + (n - s + 1);
    ll result = min(option1, option2);

    cout << "Case #" << i << ": " << result << "\n";
  }

  return 0;
}
```
