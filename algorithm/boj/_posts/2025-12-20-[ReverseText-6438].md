## 문제 링크
[6438번 - Reverse Text](https://www.acmicpc.net/problem/6438)

## 설명
주어진 문장을 거꾸로 출력하는 문제입니다.

<br>

## 접근법
먼저 각 줄을 입력받습니다.

이후 문자열을 뒤집어서 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < t; i++) {
      var s = Console.ReadLine()!;
      var arr = s.ToCharArray();
      Array.Reverse(arr);
      Console.WriteLine(new string(arr));
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  cin.ignore();

  for (int i = 0; i < t; i++) {
    string line;
    getline(cin, line);
    reverse(line.begin(), line.end());
    cout << line << "\n";
  }

  return 0;
}
```
