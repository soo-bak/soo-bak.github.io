## 문제 링크
[30033번 - Rust Study](https://www.acmicpc.net/problem/30033)

## 설명
n일 동안 계획한 페이지 수와 실제 공부한 페이지 수가 주어질 때, 계획 이상으로 공부한 날의 개수를 출력하는 문제입니다.

<br>

## 접근법
각 날에 대해 실제 공부한 페이지가 계획 이상인지 확인하며 개수를 셉니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var a = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var b = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var cnt = 0;
    for (var i = 0; i < n; i++) {
      if (b[i] >= a[i]) cnt++;
    }

    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi a(n), b(n);
  for (int i = 0; i < n; i++)
    cin >> a[i];
  for (int i = 0; i < n; i++)
    cin >> b[i];

  int cnt = 0;
  for (int i = 0; i < n; i++) {
    if (b[i] >= a[i]) cnt++;
  }

  cout << cnt << "\n";

  return 0;
}
```
