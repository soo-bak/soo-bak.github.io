## 문제 링크
[29699번 - Welcome to SMUPC!](https://www.acmicpc.net/problem/29699)

## 설명
문자열 WelcomeToSMUPC를 무한 반복한다고 할 때, n번째 문자를 출력하는 문제입니다.

<br>

## 접근법
패턴 길이가 14이므로, n - 1을 14로 나눈 나머지가 0부터 시작하는 인덱스가 됩니다. 해당 위치의 문자를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var pat = "WelcomeToSMUPC";
    var idx = (n - 1) % pat.Length;
    Console.WriteLine(pat[idx]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  long long n; cin >> n;
  string pat = "WelcomeToSMUPC";
  int idx = (int)((n - 1) % pat.size());
  cout << pat[idx] << "\n";

  return 0;
}
```
