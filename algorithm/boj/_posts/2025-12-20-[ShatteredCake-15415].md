## 문제 링크
[15415번 - Shattered Cake](https://www.acmicpc.net/problem/15415)

## 설명
폭이 주어진 직사각형 케이크가 조각났을 때, 각 조각의 정보로 원래 케이크의 길이를 구하는 문제입니다. 모든 조각의 넓이 합은 원래 케이크 넓이와 같습니다.

<br>

## 접근법
각 조각의 폭과 길이를 곱해 넓이를 구하고 모두 더합니다. 전체 넓이를 케이크 폭으로 나누면 원래 케이크의 길이가 됩니다.

조각 수와 크기가 클 수 있으므로 넓이 합은 64비트 정수로 저장합니다.

- - -

## Code

### C#
```csharp
using System;
using System.IO;

class Program {
  static void Main() {
    var sr = new StreamReader(Console.OpenStandardInput());
    var width = int.Parse(sr.ReadLine()!);
    var n = int.Parse(sr.ReadLine()!);

    var area = 0L;
    for (var i = 0; i < n; i++) {
      var parts = sr.ReadLine()!.Split();
      var w = int.Parse(parts[0]);
      var l = int.Parse(parts[1]);
      area += (long)w * l;
    }

    Console.WriteLine(area / width);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll width; cin >> width;
  int n; cin >> n;

  ll area = 0;
  for (int i = 0; i < n; i++) {
    ll w, l; cin >> w >> l;
    area += w * l;
  }

  cout << area / width << "\n";

  return 0;
}
```
