## 문제 링크
[30791번 - gahui and sousenkyo 1](https://www.acmicpc.net/problem/30791)

## 설명
16위부터 20위까지의 득표수가 주어질 때, 16위와 표 차이가 1000 이하인 하위 캐릭터 수를 구하는 문제입니다.

<br>

## 접근법
먼저 16위의 득표수를 기준으로 저장합니다.

이후 17위부터 20위까지 순회하며 기준과의 차이가 1000 이하인 경우를 세어 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var v0 = int.Parse(parts[0]);
    var cnt = 0;

    for (var i = 1; i < 5; i++) {
      var v = int.Parse(parts[i]);
      if (v0 - v <= 1000) cnt++;
    }

    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  vi v(5);
  for (int i = 0; i < 5; i++)
    cin >> v[i];

  int v0 = v[0];
  int cnt = 0;
  for (int i = 1; i < 5; i++) {
    if (v0 - v[i] <= 1000) cnt++;
  }

  cout << cnt << "\n";

  return 0;
}
```
