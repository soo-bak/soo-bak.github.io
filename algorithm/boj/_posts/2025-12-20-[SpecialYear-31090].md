## 문제 링크
[31090번 - 2023은 무엇이 특별할까?](https://www.acmicpc.net/problem/31090)

## 설명
연도 N이 주어질 때, N+1이 N의 끝 두 자리(0이 아님)로 나누어떨어지는지 판별해 조건을 만족하면 Good, 아니면 Bye를 출력하는 문제입니다.

<br>

## 접근법
연도 N의 끝 두 자리는 N을 100으로 나눈 나머지로 구할 수 있습니다. 이 값을 k라 하면, N + 1이 k로 나누어떨어지는지 확인하면 됩니다.

나누어떨어지면 Good, 그렇지 않으면 Bye를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < t; i++) {
      var n = int.Parse(Console.ReadLine()!);
      var k = n % 100;
      Console.WriteLine((n + 1) % k == 0 ? "Good" : "Bye");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    int n; cin >> n;
    int k = n % 100;
    cout << ((n + 1) % k == 0 ? "Good" : "Bye") << "\n";
  }

  return 0;
}
```
