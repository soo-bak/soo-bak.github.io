## 문제 링크
[30821번 - 별자리가 될 수 있다면](https://www.acmicpc.net/problem/30821)

## 설명
정N각형의 꼭짓점 중 5개를 골라 오각별을 만들 수 있는 경우의 수를 구하는 문제입니다.

<br>

## 접근법
오각별은 5개의 꼭짓점을 하나 건너뛰며 연결한 형태입니다. 어떤 5개를 고르든 그 순서대로 연결하면 유일한 별이 되므로, 서로 다른 별의 수는 N개 중 5개를 고르는 조합의 수와 같습니다. N × (N-1) × (N-2) × (N-3) × (N-4) / 120으로 계산합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = long.Parse(Console.ReadLine()!);
    var ans = n * (n - 1) * (n - 2) * (n - 3) * (n - 4) / 120;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll n; cin >> n;
  ll ans = n * (n - 1) * (n - 2) * (n - 3) * (n - 4) / 120;
  cout << ans << "\n";

  return 0;
}
```
