## 문제 링크
[31612번 - 画数数え (Stroke Count)](https://www.acmicpc.net/problem/31612)

## 설명
문자열에 포함된 j, o, i 문자의 획수를 합산하는 문제입니다.

각 문자의 획수는 j는 2획, o는 1획, i는 2획입니다.

<br>

## 접근법
문자열을 순회하며 각 문자에 해당하는 획수를 더합니다. o이면 1을 더하고, j나 i이면 2를 더합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    _ = Console.ReadLine();
    var s = Console.ReadLine()!;

    var sum = 0;
    foreach (var ch in s) {
      if (ch == 'o') sum += 1;
      else sum += 2;
    }

    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  int sum = 0;
  for (char c : s) {
    if (c == 'o') sum += 1;
    else sum += 2;
  }

  cout << sum << "\n";

  return 0;
}
```
