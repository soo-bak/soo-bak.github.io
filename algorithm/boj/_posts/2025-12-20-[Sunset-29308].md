## 문제 링크
[29308번 - Закат](https://www.acmicpc.net/problem/29308)

## 설명
러시아 국적 선수들 중 요구 연봉이 가장 큰 선수를 찾아 제외하는 문제입니다.

<br>

## 접근법
각 선수의 정보를 입력받으면서 국적이 Russia인 선수들만 고려합니다. 그 중 요구 연봉이 최대인 선수의 이름을 추적하여 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var best = -1;
    var bestName = "";

    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var salary = int.Parse(parts[0]);
      var name = parts[1];
      var country = parts[2];
      if (country == "Russia" && salary > best) {
        best = salary;
        bestName = name;
      }
    }

    Console.WriteLine(bestName);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {

  string ans;
  for (int i = 0; i < n; i++) {
    int salary; string name, country;
    cin >> salary >> name >> country;
    if (country == "Russia" && salary > best) {
      best = salary;
      ans = name;
    }
  }

  cout << ans << "\n";

  return 0;
}
```
