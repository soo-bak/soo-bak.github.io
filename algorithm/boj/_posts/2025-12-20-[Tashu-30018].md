## 문제 링크
[30018번 - 타슈](https://www.acmicpc.net/problem/30018)

## 설명
각 대여소의 목표 자전거 수와 현재 자전거 수가 주어질 때, 목표 상태로 만들기 위한 최소 이동 횟수를 구하는 문제입니다.

<br>
총합은 동일하므로 초과분을 부족한 곳으로 옮기면 됩니다.

<br>

## 접근법
각 대여소에서 현재 수에서 목표 수를 뺀 값이 양수이면 초과분입니다. 모든 초과분의 합이 최소 이동 횟수입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var a = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var b = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var moves = 0;
    for (var i = 0; i < n; i++) {
      var diff = b[i] - a[i];
      if (diff > 0) moves += diff;
    }

    Console.WriteLine(moves);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi a(n), b(n);
  for (int i = 0; i < n; i++)
    cin >> a[i];
  for (int i = 0; i < n; i++)
    cin >> b[i];

  int moves = 0;
  for (int i = 0; i < n; i++) {
    int diff = b[i] - a[i];
    if (diff > 0) moves += diff;
  }

  cout << moves << "\n";

  return 0;
}
```
