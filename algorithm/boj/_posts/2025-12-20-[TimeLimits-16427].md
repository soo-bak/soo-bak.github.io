## 문제 링크
[16427번 - Time Limits](https://www.acmicpc.net/problem/16427)

## 설명
판정용 해답들의 실행 시간이 밀리초 단위로 주어질 때, 가장 느린 시간의 s배 이상이 되도록 시간 제한을 설정하는 문제입니다.

<br>
시간 제한은 초 단위 정수이며, 가능한 한 작아야 합니다.

<br>

## 접근법
먼저 최대 실행 시간을 구합니다. 이후 최대 실행 시간에 s를 곱한 값을 1000으로 나누어 올림하면 초 단위 시간 제한이 됩니다.

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var s = int.Parse(first[1]);

    var arr = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var maxTime = arr.Max();
    var need = maxTime * s;
    var limitSec = (need + 999) / 1000;

    Console.WriteLine(limitSec);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, s; cin >> n >> s;
  int maxTime = 0;
  for (int i = 0; i < n; i++) {
    int x; cin >> x;
    maxTime = max(maxTime, x);
  }
  int need = maxTime * s;
  int limitSec = (need + 999) / 1000;
  cout << limitSec << "\n";

  return 0;
}
```
