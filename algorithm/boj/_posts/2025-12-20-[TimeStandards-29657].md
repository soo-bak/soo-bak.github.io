## 문제 링크
[29657번 - Стандарты времени](https://www.acmicpc.net/problem/29657)

## 설명
서로 다른 시간 체계가 주어지고, 첫 번째 체계의 시각을 두 번째 체계로 변환하는 문제입니다. 각 체계는 하루의 시간 수, 한 시간의 분 수, 한 분의 초 수로 정의됩니다.

<br>

## 접근법
먼저 첫 번째 체계에서 주어진 시각을 총 초 단위로 변환합니다.

다음으로 두 번째 체계의 단위로 분해합니다. 총 초를 시간당 초로 나누어 시간을 구하고, 나머지를 분당 초로 나누어 분과 초를 구합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var p1 = Console.ReadLine()!.Split();
    var p2 = Console.ReadLine()!.Split();
    var t1 = Console.ReadLine()!.Split();

    var a1 = int.Parse(p1[0]); var b1 = int.Parse(p1[1]); var c1 = int.Parse(p1[2]);
    var a2 = int.Parse(p2[0]); var b2 = int.Parse(p2[1]); var c2 = int.Parse(p2[2]);
    var h1 = int.Parse(t1[0]); var m1 = int.Parse(t1[1]); var s1 = int.Parse(t1[2]);

    var total = h1 * (b1 * c1) + m1 * c1 + s1;

    var secPerHour2 = b2 * c2;
    var h2 = total / secPerHour2;
    var rem = total % secPerHour2;
    var m2 = rem / c2;
    var s2 = rem % c2;

    Console.WriteLine($"{h2} {m2} {s2}");
  }
}
```

### C++

  int total = h1 * (b1 * c1) + m1 * c1 + s1;

  int secPerHour2 = b2 * c2;
  int h2 = total / secPerHour2;

  int m2 = rem / c2;

  return 0;
}
