## 문제 링크
[30876번 - Tren del Fin del Mundo](https://www.acmicpc.net/problem/30876)

## 설명
여러 역의 좌표가 순서대로 주어질 때, 가장 남쪽에 있는 점의 좌표를 찾는 문제입니다.

선분 위의 점은 양 끝점 사이의 값을 가지므로, 최소 y 좌표는 반드시 역 좌표 중 하나에서 나타납니다.

<br>

## 접근법
먼저 모든 역 좌표를 순회하며 y 좌표가 가장 작은 점을 찾습니다.

이후 해당 점의 좌표를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var bestX = 0;
    var bestY = int.MaxValue;

    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var x = int.Parse(parts[0]);
      var y = int.Parse(parts[1]);
      if (y < bestY) {
        bestY = y;
        bestX = x;
      }
    }

    Console.WriteLine($"{bestX} {bestY}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int bestX = 0, bestY = INT_MAX;

  for (int i = 0; i < n; i++) {
    int x, y; cin >> x >> y;
    if (y < bestY) {
      bestY = y;
      bestX = x;
    }
  }

  cout << bestX << " " << bestY << "\n";

  return 0;
}
```
