## 문제 링크
[31611번 - 火曜日 (Tuesday)](https://www.acmicpc.net/problem/31611)

## 설명
오늘이 일요일일 때 X일 뒤가 화요일인지 판별하는 문제입니다.

<br>

## 접근법
일요일을 0으로 보면 화요일은 2입니다. X를 7로 나눈 나머지가 2이면 화요일이므로 1을 출력하고, 그렇지 않으면 0을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var x = int.Parse(Console.ReadLine()!);

    Console.WriteLine(x % 7 == 2 ? 1 : 0);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int x; cin >> x;

  cout << ((x % 7 == 2) ? 1 : 0) << "\n";

  return 0;
}
```
