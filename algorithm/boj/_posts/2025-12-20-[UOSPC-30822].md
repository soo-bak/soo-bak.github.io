## 문제 링크
[30822번 - UOSPC 세기](https://www.acmicpc.net/problem/30822)

## 설명
문자열의 문자들을 재배열해 'uospc'를 최대 몇 번 만들 수 있는지 묻는 문제입니다. 각 패턴에 필요한 문자는 u, o, s, p, c가 하나씩입니다.

<br>

## 접근법
문자 빈도를 센 뒤 u, o, s, p, c의 등장 횟수 중 최솟값이 만들 수 있는 패턴의 개수입니다. O(n)으로 빈도를 계산하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;

    var cnt = new int[256];
    foreach (var ch in s) cnt[ch]++;

    var chars = new[] { 'u', 'o', 's', 'p', 'c' };
    var ans = int.MaxValue;
    foreach (var ch in chars)
      ans = Math.Min(ans, cnt[ch]);

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  vector<int> cnt(256, 0);
  for (char c : s) cnt[(unsigned char)c]++;

  string need = "uospc";
  int ans = INT_MAX;
  for (char c : need) ans = min(ans, cnt[(unsigned char)c]);

  cout << ans << "\n";
  return 0;
}
```
