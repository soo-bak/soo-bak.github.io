## 문제 링크
[30868번 - 개표](https://www.acmicpc.net/problem/30868)

## 설명
득표수를 5개 단위로 ++++와 나머지는 |로 표현하는 문자열을 출력하는 문제입니다.

<br>

## 접근법
먼저 득표수를 5로 나눈 몫만큼 ++++와 공백을 붙입니다.

다음으로 나머지만큼 |를 붙입니다.

이후 끝에 불필요한 공백이 있으면 제거하고 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < t; i++) {
      var n = int.Parse(Console.ReadLine()!);
      var k = n / 5;
      var r = n % 5;
      var sb = new StringBuilder();

      for (var j = 0; j < k; j++) {
        sb.Append("++++");
        sb.Append(' ');
      }
      for (var j = 0; j < r; j++)
        sb.Append('|');

      var result = sb.ToString();
      if (result.Length > 0 && result[^1] == ' ')
        result = result.TrimEnd();

      Console.WriteLine(result);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    int n; cin >> n;
    int k = n / 5;
    int r = n % 5;
    string res;

    for (int i = 0; i < k; i++) {
      res += "++++";
      res += " ";
    }
    for (int i = 0; i < r; i++)
      res += "|";

    if (!res.empty() && res.back() == ' ')
      res.pop_back();

    cout << res << "\n";
  }

  return 0;
}
```
