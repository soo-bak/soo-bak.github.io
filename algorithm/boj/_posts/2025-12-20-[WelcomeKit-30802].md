## 문제 링크
[30802번 - 웰컴 키트](https://www.acmicpc.net/problem/30802)

## 설명
사이즈별 티셔츠 신청 수와 펜 총 개수가 주어질 때, 티셔츠 묶음 수와 펜 묶음 및 낱개 수를 구하는 문제입니다. 티셔츠는 같은 사이즈끼리만 묶음 주문이 가능하고, 펜은 묶음을 최대한 많이 주문한 뒤 나머지를 낱개로 주문합니다.

<br>

## 접근법
각 사이즈별로 필요한 티셔츠 묶음 수는 신청 수를 묶음 크기로 나눈 뒤 올림한 값입니다. 정수 나눗셈에서 올림은 (신청 수 + 묶음 크기 - 1)을 묶음 크기로 나누면 됩니다. 모든 사이즈의 묶음 수를 더하면 전체 티셔츠 묶음 수가 됩니다.

펜은 총 인원수를 묶음 크기로 나눈 몫이 묶음 수, 나머지가 낱개 수입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = long.Parse(Console.ReadLine()!);
    var sizes = Array.ConvertAll(Console.ReadLine()!.Split(), long.Parse);
    var tp = Console.ReadLine()!.Split();
    var t = long.Parse(tp[0]);
    var p = long.Parse(tp[1]);

    var shirts = 0L;
    foreach (var cnt in sizes)
      shirts += (cnt + t - 1) / t;

    var penBundles = n / p;
    var penSingles = n % p;

    Console.WriteLine(shirts);
    Console.WriteLine($"{penBundles} {penSingles}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll n; cin >> n;
  ll sz[6];
  for (int i = 0; i < 6; i++)
    cin >> sz[i];
  ll t, p; cin >> t >> p;

  ll shirts = 0;
  for (int i = 0; i < 6; i++)
    shirts += (sz[i] + t - 1) / t;

  ll penBundles = n / p;
  ll penSingles = n % p;

  cout << shirts << "\n";
  cout << penBundles << " " << penSingles << "\n";

  return 0;
}
```
