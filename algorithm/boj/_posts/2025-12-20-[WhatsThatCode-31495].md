## 문제 링크
[31495번 - 그게 무슨 코드니..](https://www.acmicpc.net/problem/31495)

## 설명
입력 문자열이 양 끝 큰따옴표로 올바르게 감싸져 있고 내부가 비어있지 않은지 판정하는 문제입니다.

조건을 만족하면 큰따옴표를 제거한 내용을 출력하고, 그렇지 않으면 CE를 출력합니다.

<br>

## 접근법
문자열 길이가 3 이상이고, 첫 글자와 마지막 글자가 모두 큰따옴표인지 확인합니다. 길이가 2이면 내부가 비어있는 것이므로 유효하지 않습니다.

조건을 만족하면 첫 글자와 마지막 글자를 제외한 부분을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;

    if (s.Length < 2 || s[0] != '"' || s[^1] != '"' || s.Length == 2) Console.WriteLine("CE");
    else Console.WriteLine(s.Substring(1, s.Length - 2));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s;
  getline(cin, s);

  if (s.size() < 2 || s.front() != '"' || s.back() != '"' || s.size() == 2) cout << "CE\n";
  else cout << s.substr(1, s.size() - 2) << "\n";

  return 0;
}
```
