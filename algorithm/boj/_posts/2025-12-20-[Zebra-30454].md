## 문제 링크
[30454번 - 얼룩말을 찾아라!](https://www.acmicpc.net/problem/30454)

## 설명
각 얼룩말의 털 상태가 0/1 문자열로 주어질 때, 검은 줄 개수가 최대인 얼룩말의 줄 개수와 해당 개체 수를 구하는 문제입니다.

<br>

## 접근법
먼저 각 문자열을 순회하며 연속된 1의 그룹 개수를 셉니다.

다음으로 그룹 수가 현재 최대보다 크면 최대값과 개체 수를 갱신하고, 같으면 개체 수만 증가시킵니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var l = int.Parse(first[1]);

    var mx = 0; var num = 0;
    for (var i = 0; i < n; i++) {
      var s = Console.ReadLine()!;
      var cnt = 0;
      for (var j = 0; j < l; j++) {
        if (s[j] == '1') {
          cnt++;
          while (j + 1 < l && s[j + 1] == '1') j++;
        }
      }
      if (cnt > mx) { mx = cnt; num = 1; }
      else if (cnt == mx) num++;
    }

    Console.WriteLine($"{mx} {num}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, l; cin >> n >> l;

  int mx = 0, num = 0;
  for (int i = 0; i < n; i++) {
    string s; cin >> s;
    int cnt = 0;
    for (int j = 0; j < l; j++) {
      if (s[j] == '1') {
        cnt++;
        while (j + 1 < l && s[j + 1] == '1') j++;
      }
    }
    if (cnt > mx) { mx = cnt; num = 1; }
    else if (cnt == mx) num++;
  }

  cout << mx << " " << num << "\n";

  return 0;
}
```
