## 문제 링크
[30215번 - Age Expression](https://www.acmicpc.net/problem/30215)

## 설명
세 나이가 주어졌을 때 D = a × A + k × K를 만족하는 양의 정수 a, k가 존재하면 1, 아니면 0을 출력하는 문제입니다.

<br>

## 접근법
첫 번째 나이의 배수를 1배부터 순회하며, 목표 나이에서 뺀 값이 두 번째 나이의 양의 배수인지 확인합니다. 조건을 만족하는 경우가 있으면 1, 없으면 0을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var d = int.Parse(parts[0]);
    var aAge = int.Parse(parts[1]);
    var kAge = int.Parse(parts[2]);

    var ok = false;
    for (var a = 1; a <= d / aAge; a++) {
      var remain = d - a * aAge;
      if (remain > 0 && remain % kAge == 0) {
        ok = true;
        break;
      }
    }

    Console.WriteLine(ok ? 1 : 0);
  }
}
```
