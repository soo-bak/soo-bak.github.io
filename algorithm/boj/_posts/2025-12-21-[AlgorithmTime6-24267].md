## 문제 링크
[24267번 - 알고리즘 수업 - 알고리즘의 수행 시간 6](https://www.acmicpc.net/problem/24267)

## 설명
주어진 n에 대해 삼중 반복문의 실행 횟수와 최고차항 차수를 출력하는 문제입니다.

<br>

## 접근법
코드1은 모든 i < j < k 조합을 한 번씩 방문하므로 실행 횟수는 조합의 개수인 `nC3 = n(n-1)(n-2)/6`입니다. 이는 3차 다항식이므로 최고차항 차수는 3을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var cnt = n * (n - 1) * (n - 2) / 6;
    Console.WriteLine(cnt);
    Console.WriteLine(3);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll n; cin >> n;
  ll cnt = n * (n - 1) * (n - 2) / 6;
  cout << cnt << "\n";
  cout << 3 << "\n";

  return 0;
}
```
