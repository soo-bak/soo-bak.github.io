## 문제 링크
[30306번 - Better Dice](https://www.acmicpc.net/problem/30306)

## 설명
두 주사위의 눈이 주어질 때, 더 큰 눈이 나올 확률이 더 높은 쪽을 판단하는 문제입니다.

<br>

## 접근법
두 주사위의 모든 눈 조합에 대해 첫 번째가 더 큰 경우와 두 번째가 더 큰 경우를 각각 셉니다.

이후 더 많이 이긴 쪽을 출력하고, 같으면 tie를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var aParts = Console.ReadLine()!.Split();
    var bParts = Console.ReadLine()!.Split();

    var a = new long[n];
    var b = new long[n];
    for (var i = 0; i < n; i++) a[i] = long.Parse(aParts[i]);
    for (var i = 0; i < n; i++) b[i] = long.Parse(bParts[i]);

    long winA = 0, winB = 0;
    for (var i = 0; i < n; i++) {
      for (var j = 0; j < n; j++) {
        if (a[i] > b[j]) winA++;
        else if (a[i] < b[j]) winB++;
      }
    }

    if (winA > winB) Console.WriteLine("first");
    else if (winB > winA) Console.WriteLine("second");
    else Console.WriteLine("tie");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef vector<ll> vll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vll a(n), b(n);
  for (int i = 0; i < n; i++)
    cin >> a[i];
  for (int i = 0; i < n; i++)
    cin >> b[i];

  ll winA = 0, winB = 0;
  for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
      if (a[i] > b[j]) winA++;
      else if (a[i] < b[j]) winB++;
    }
  }

  if (winA > winB) cout << "first\n";
  else if (winB > winA) cout << "second\n";
  else cout << "tie\n";

  return 0;
}
```
