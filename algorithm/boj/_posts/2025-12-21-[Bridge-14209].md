## 문제 링크
[14209번 - Bridž](https://www.acmicpc.net/problem/14209)

## 설명
각 카드 패에서 A, K, Q, J에 부여된 점수를 합산해 전체 점수를 출력하는 문제입니다.

<br>

## 접근법
각 카드 패의 문자를 순회하며 에이스는 4점, 킹은 3점, 퀸은 2점, 잭은 1점을 더합니다.

이후 모든 패의 합계를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sum = 0;

    for (var i = 0; i < n; i++) {
      var s = Console.ReadLine()!;
      foreach (var ch in s) {
        if (ch == 'A') sum += 4;
        else if (ch == 'K') sum += 3;
        else if (ch == 'Q') sum += 2;
        else if (ch == 'J') sum += 1;
      }
    }

    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int sum = 0;
  for (int i = 0; i < n; i++) {
    string s; cin >> s;
    for (char ch : s) {
      if (ch == 'A') sum += 4;
      else if (ch == 'K') sum += 3;
      else if (ch == 'Q') sum += 2;
      else if (ch == 'J') sum += 1;
    }
  }

  cout << sum << "\n";

  return 0;
}
```
