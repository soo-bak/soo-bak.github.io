## 문제 링크
[20017번 - Топот котов](https://www.acmicpc.net/problem/20017)

## 설명
바로 위층 집의 고양이 수가 자기 집의 두 배를 초과하면 벌금이 부과될 때, 총 벌금을 계산하는 문제입니다.

<br>

## 접근법
아파트는 한 층에 m세대씩 n층으로 구성되어 있습니다. 각 집에서 바로 위층 집은 인덱스가 m만큼 차이 납니다. 1층부터 n-1층까지의 각 집에 대해 위층 집의 고양이 수가 자기 집의 두 배를 초과하는지 확인합니다. 조건을 만족하면 벌금을 누적해 총합을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);
    var a = int.Parse(first[2]);
    var parts = Console.ReadLine()!.Split();
    var total = n * m;
    var b = new int[total];
    for (var i = 0; i < total; i++)
      b[i] = int.Parse(parts[i]);

    var fine = 0L;
    for (var i = 0; i + m < total; i++) {
      if (b[i + m] > 2 * b[i]) fine += a;
    }

    Console.WriteLine(fine);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m, a; cin >> n >> m >> a;
  int total = n * m;
  vi b(total);
  for (int i = 0; i < total; i++)
    cin >> b[i];

  ll fine = 0;
  for (int i = 0; i + m < total; i++) {
    if (b[i + m] > 2 * b[i]) fine += a;
  }

  cout << fine << "\n";

  return 0;
}
```
