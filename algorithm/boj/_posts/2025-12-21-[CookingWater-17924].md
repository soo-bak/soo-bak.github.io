## 문제 링크
[17924번 - Cooking Water](https://www.acmicpc.net/problem/17924)

## 설명
여러 번의 실험에서 물이 끓기 시작한 시각이 동일할 수 있는지 판정하는 문제입니다.

<br>

## 접근법
각 실험에서 관찰하지 않은 구간이 주어지고, 끓기 시작한 시각은 그 구간 안에 있습니다. 동일한 시각이 존재하려면 모든 구간의 교집합이 비어 있지 않아야 합니다.

교집합은 모든 구간 시작점의 최댓값과 모든 구간 끝점의 최솟값으로 구할 수 있습니다. 시작점의 최댓값이 끝점의 최솟값 이하라면 공통 시각이 존재합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var lo = 0;
    var hi = 1000;

    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = int.Parse(parts[0]);
      var b = int.Parse(parts[1]);
      if (a > lo) lo = a;
      if (b < hi) hi = b;
    }

    Console.WriteLine(lo <= hi ? "gunilla has a point" : "edward is right");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int lo = 0, hi = 1000;

  for (int i = 0; i < n; i++) {
    int a, b; cin >> a >> b;
    lo = max(lo, a);
    hi = min(hi, b);
  }

  cout << (lo <= hi ? "gunilla has a point" : "edward is right") << "\n";

  return 0;
}
```
