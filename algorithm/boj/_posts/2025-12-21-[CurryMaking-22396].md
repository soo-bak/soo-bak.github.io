## 문제 링크
[22396번 - カレー作り](https://www.acmicpc.net/problem/22396)

## 설명
현재 농도의 카레에 루우와 물을 더해 농도를 C로 만들 때, 추가해야 하는 루우 개수의 최솟값을 구하는 문제입니다.

<br>

## 접근법
목표 농도를 만족하려면 루우 총량이 물 총량과 목표 농도의 곱 이상이어야 합니다. 물은 더 넣을 수 있으므로, 부족한 루우 양을 채우는 최소 개수를 올림하여 계산합니다.

이미 조건을 만족하면 0을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    string? line;
    while ((line = Console.ReadLine()) != null) {
      var parts = line.Split();
      if (parts.Length == 0) continue;
      var r0 = int.Parse(parts[0]);
      var w0 = int.Parse(parts[1]);
      var c = int.Parse(parts[2]);
      var r = int.Parse(parts[3]);

      if (r0 == 0 && w0 == 0 && c == 0 && r == 0) break;

      var need = c * w0 - r0;
      var x = 0;
      if (need > 0) x = (need + r - 1) / r;

      Console.WriteLine(x);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>

    if (r0 == 0 && w0 == 0 && c == 0 && r == 0) break;

    int need = c * w0 - r0;
    int x = 0;
    if (need > 0) x = (need + r - 1) / r;

    cout << x << "\n";
  }

  return 0;
}
```
