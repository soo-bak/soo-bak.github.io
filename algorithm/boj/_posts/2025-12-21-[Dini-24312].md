## 문제 링크
[24312번 - ДИНИ](https://www.acmicpc.net/problem/24312)

## 설명
네 개의 무게를 두 손에 나눠 담을 때, 양손 무게 합의 차이 최소값을 구하는 문제입니다.

<br>

## 접근법
4개의 원소를 두 그룹으로 나누는 모든 경우를 탐색합니다. 한 그룹의 합을 전체에서 빼면 다른 그룹의 합이 되므로, 두 합의 차이가 최소인 경우를 찾아 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var w = new int[4];
    var total = 0;
    for (var i = 0; i < 4; i++) {
      w[i] = int.Parse(parts[i]);
      total += w[i];
    }

    var best = int.MaxValue;
    for (var mask = 0; mask < 16; mask++) {
      var sum = 0;
      for (var i = 0; i < 4; i++) {
        if ((mask & (1 << i)) != 0) sum += w[i];
      }
      var diff = Math.Abs(total - 2 * sum);
      if (diff < best) best = diff;
    }

    Console.WriteLine(best);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int w[4];
  int total = 0;
  for (int i = 0; i < 4; i++) {
    cin >> w[i];
    total += w[i];
  }

  int best = 1e9;
  for (int mask = 0; mask < 16; mask++) {
    int sum = 0;
    for (int i = 0; i < 4; i++) {
      if (mask & (1 << i)) sum += w[i];
    }
    int diff = abs(total - 2 * sum);
    if (diff < best) best = diff;
  }

  cout << best << "\n";

  return 0;
}
```
