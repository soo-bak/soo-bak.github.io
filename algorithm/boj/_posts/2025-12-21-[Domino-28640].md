## 문제 링크
[28640번 - Домино](https://www.acmicpc.net/problem/28640)

## 설명
두 도미노가 주어질 때, 맞닿을 수 있는 면이 있는지 판정하는 문제입니다. 각 도미노는 왼쪽과 오른쪽 절반으로 나뉘며, 두 도미노의 어떤 절반끼리든 점 개수가 같으면 맞출 수 있습니다.

<br>

## 접근법
각 도미노에서 구분자를 기준으로 왼쪽과 오른쪽의 점 개수를 셉니다. 첫 번째 도미노의 두 값과 두 번째 도미노의 두 값 중 하나라도 같은 것이 있으면 Yes, 없으면 No를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s1 = Console.ReadLine()!;
    var s2 = Console.ReadLine()!;

    var a1 = 0;
    var a2 = 0;
    var sep = false;
    foreach (var ch in s1) {
      if (ch == '|') sep = true;
      else if (!sep) a1++;
      else a2++;
    }

    var b1 = 0;
    var b2 = 0;
    sep = false;
    foreach (var ch in s2) {
      if (ch == '|') sep = true;
      else if (!sep) b1++;
      else b2++;
    }

    var ok = a1 == b1 || a1 == b2 || a2 == b1 || a2 == b2;
    Console.WriteLine(ok ? "Yes" : "No");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s1, s2; cin >> s1 >> s2;

  int a1 = 0, a2 = 0;
  bool sep = false;
  for (char c : s1) {
    if (c == '|') sep = true;
    else if (!sep) a1++;
    else a2++;
  }

  int b1 = 0, b2 = 0;
  sep = false;
  for (char c : s2) {
    if (c == '|') sep = true;
    else if (!sep) b1++;
    else b2++;
  }

  bool ok = (a1 == b1) || (a1 == b2) || (a2 == b1) || (a2 == b2);
  cout << (ok ? "Yes" : "No") << "\n";

  return 0;
}
```
