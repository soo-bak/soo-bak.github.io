## 문제 링크
[31610번 - 飴の袋詰め (Drops Packing)](https://www.acmicpc.net/problem/31610)

## 설명
사탕 가격 A, 개수 B, 봉지 가격 C가 주어질 때 총 금액을 출력하는 문제입니다.

<br>

## 접근법
총 금액은 `A * B + C`이므로 그대로 계산해 출력하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var a = int.Parse(Console.ReadLine()!);
    var b = int.Parse(Console.ReadLine()!);
    var c = int.Parse(Console.ReadLine()!);

    Console.WriteLine(a * b + c);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, c; cin >> a >> b >> c;
  
  cout << a * b + c << "\n";

  return 0;
}
```
