## 문제 링크
[20473번 - Гостиница](https://www.acmicpc.net/problem/20473)

## 설명
2인실과 3인실만 있는 호텔에서 n명을 빈자리 없이 배치할 때, 필요한 최소 방 개수와 방 종류 개수를 구하는 문제입니다.

<br>

## 접근법
방 개수를 최소화하려면 3인실을 최대한 많이 쓰는 것이 유리합니다. 3인실 수를 큰 값부터 줄여가며 남은 인원이 2로 나누어떨어지는지 확인하고, 최초로 성립하는 경우의 2인실과 3인실 개수를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var a2 = 0;
    var a3 = 0;

    for (var t3 = n / 3; t3 >= 0; t3--) {
      var remain = n - 3 * t3;
      if (remain % 2 == 0) {
        a3 = t3;
        a2 = remain / 2;
        break;
      }
    }

    Console.WriteLine($"{a2} {a3}");
  }
}
```

### C++
