## 문제 링크
[31831번 - 과민성 대장 증후군](https://www.acmicpc.net/problem/31831)

## 설명
매일 스트레스 변화량이 주어질 때 복통을 겪는 날의 개수를 구하는 문제입니다.

<br>

## 접근법
스트레스는 0에서 시작하며, 매일 변화량만큼 증가하거나 감소합니다. 단, 스트레스는 음수가 될 수 없으므로 0 미만으로 떨어지면 0으로 유지합니다. 각 날마다 스트레스를 갱신한 뒤, 임계값 이상이면 복통을 겪는 날로 카운트합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = long.Parse(first[1]);
    var arr = Console.ReadLine()!.Split().Select(long.Parse).ToArray();

    var stress = 0L;
    var days = 0;
    for (var i = 0; i < n; i++) {
      stress += arr[i];
      if (stress < 0) stress = 0;
      if (stress >= m) days++;
    }

    Console.WriteLine(days);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef vector<ll> vll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; ll m;
  cin >> n >> m;
  vll a(n);
  for (int i = 0; i < n; i++)
    cin >> a[i];

  ll stress = 0;
  int days = 0;
  for (int i = 0; i < n; i++) {
    stress += a[i];
    if (stress < 0) stress = 0;
    if (stress >= m) days++;
  }

  cout << days << "\n";

  return 0;
}
```
