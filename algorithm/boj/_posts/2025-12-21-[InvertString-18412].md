## 문제 링크
[18412번 - 文字列の反転 (Inversion of a String)](https://www.acmicpc.net/problem/18412)

## 설명
문자열에서 지정된 구간을 뒤집어 출력하는 문제입니다.

<br>

## 접근법
문제에서 주어지는 인덱스는 1부터 시작하므로, 0 기준으로 변환해서 처리합니다. 시작 위치 앞부분은 그대로 두고, 지정 구간만 뒤집은 뒤, 끝 위치 뒷부분을 이어붙이면 됩니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var a = int.Parse(first[1]);
    var b = int.Parse(first[2]);
    var s = Console.ReadLine()!;

    var sb = new StringBuilder();
    sb.Append(s.Substring(0, a - 1));
    var mid = s.Substring(a - 1, b - a + 1).ToCharArray();
    Array.Reverse(mid);
    sb.Append(new string(mid));
    sb.Append(s.Substring(b));

    Console.WriteLine(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, a, b; cin >> n >> a >> b;
  string s; cin >> s;

  reverse(s.begin() + (a - 1), s.begin() + b);
  cout << s << "\n";

  return 0;
}
```
