## 문제 링크
[14039번 - Magic Squares](https://www.acmicpc.net/problem/14039)

## 설명
4×4 정수 배열이 마방진인지 판별하는 문제입니다.

<br>

## 접근법
마방진은 모든 행과 열의 합이 동일한 정사각 배열입니다. 첫 번째 행의 합을 기준값으로 설정한 뒤, 나머지 행들의 합과 모든 열의 합이 이 기준값과 같은지 확인합니다. 하나라도 다르면 마방진이 아닙니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var a = new int[4, 4];
    for (var i = 0; i < 4; i++) {
      var parts = Console.ReadLine()!.Split();
      for (var j = 0; j < 4; j++)
        a[i, j] = int.Parse(parts[j]);
    }

    var target = 0;
    for (var j = 0; j < 4; j++)
      target += a[0, j];

    var ok = true;
    for (var i = 1; i < 4 && ok; i++) {
      var sum = 0;
      for (var j = 0; j < 4; j++)
        sum += a[i, j];
      if (sum != target) ok = false;
    }
    for (var j = 0; j < 4 && ok; j++) {
      var sum = 0;
      for (var i = 0; i < 4; i++)
        sum += a[i, j];
      if (sum != target) ok = false;
    }

    Console.WriteLine(ok ? "magic" : "not magic");
  }
}
```

      cin >> a[i][j];
  }

  int target = 0;
  for (int j = 0; j < 4; j++)
    target += a[0][j];

  bool ok = true;
  for (int i = 1; i < 4 && ok; i++) {
    int sum = 0;
    for (int j = 0; j < 4; j++)
      sum += a[i][j];
    if (sum != target) ok = false;
  }
  for (int j = 0; j < 4 && ok; j++) {
    int sum = 0;
    for (int i = 0; i < 4; i++)
      sum += a[i][j];
    if (sum != target) ok = false;
  }

  cout << (ok ? "magic" : "not magic") << "\n";

  return 0;
}
