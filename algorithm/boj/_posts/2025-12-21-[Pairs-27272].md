## 문제 링크
[27272번 - Пары](https://www.acmicpc.net/problem/27272)

## 설명
네 수를 두 쌍으로 나눠 각 쌍의 곱을 더한 값이 최대가 되도록 하는 문제입니다.

<br>

## 접근법
네 수를 두 쌍으로 묶는 방법은 세 가지뿐입니다. 첫째와 둘째를 묶거나, 첫째와 셋째를 묶거나, 첫째와 넷째를 묶는 경우입니다. 세 경우의 값을 모두 계산해 가장 큰 값을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);
    var c = int.Parse(parts[2]);
    var d = int.Parse(parts[3]);

    var s1 = a * b + c * d;
    var s2 = a * c + b * d;
    var s3 = a * d + b * c;

    Console.WriteLine(Math.Max(s1, Math.Max(s2, s3)));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, c, d; cin >> a >> b >> c >> d;

  int s1 = a * b + c * d;
  int s2 = a * c + b * d;
  int s3 = a * d + b * c;

  cout << max(s1, max(s2, s3)) << "\n";

  return 0;
}
```
