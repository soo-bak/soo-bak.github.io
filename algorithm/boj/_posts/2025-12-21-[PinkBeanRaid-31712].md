## 문제 링크
[31712번 - 핑크빈 레이드](https://www.acmicpc.net/problem/31712)

## 설명
세 캐릭터가 0초에 동시에 공격을 시작하고, 이후 각자의 주기마다 공격을 반복할 때, 핑크빈의 체력이 0 이하가 되는 가장 빠른 시간을 구하는 문제입니다.

<br>

## 접근법
시간 t에서 각 캐릭터의 공격 횟수는 t를 주기로 나눈 몫에 1을 더한 값입니다. 0초에 한 번 공격하고, 이후 주기마다 공격하기 때문입니다. 세 캐릭터의 누적 대미지를 모두 더하면 시간 t까지의 총 대미지가 됩니다.

시간이 지날수록 대미지는 늘어나기만 하므로, 체력 이상의 대미지를 처음 달성하는 시간을 찾으면 됩니다. 상한을 1부터 시작해 조건을 만족할 때까지 두 배씩 늘려가며 범위를 정하고, 그 안에서 조건을 만족하는 최소 시간을 찾습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static long Calc(long t, int[] c, int[] d) {
    var sum = 0L;
    for (var i = 0; i < 3; i++)
      sum += (t / c[i] + 1) * d[i];
    return sum;
  }

  static void Main() {
    var c = new int[3];
    var d = new int[3];
    for (var i = 0; i < 3; i++) {
      var parts = Console.ReadLine()!.Split();
      c[i] = int.Parse(parts[0]);
      d[i] = int.Parse(parts[1]);
    }
    var h = long.Parse(Console.ReadLine()!);

    var lo = 0L;
    var hi = 1L;
    while (Calc(hi, c, d) < h)
      hi <<= 1;

    while (lo < hi) {
      var mid = (lo + hi) / 2;
      if (Calc(mid, c, d) >= h) hi = mid;
      else lo = mid + 1;
    }

    Console.WriteLine(lo);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

ll calc(ll t, int c[3], int d[3]) {
  ll sum = 0;
  for (int i = 0; i < 3; i++)
    sum += (t / c[i] + 1) * 1LL * d[i];
  return sum;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int c[3], d[3];
  for (int i = 0; i < 3; i++)
    cin >> c[i] >> d[i];

  ll h; cin >> h;

  ll lo = 0, hi = 1;
  while (calc(hi, c, d) < h)
    hi <<= 1;

  while (lo < hi) {
    ll mid = (lo + hi) / 2;
    if (calc(mid, c, d) >= h) hi = mid;
    else lo = mid + 1;
  }

  cout << lo << "\n";

  return 0;
}
```
