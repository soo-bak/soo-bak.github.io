## 문제 링크
[18964번 - Questionnaire](https://www.acmicpc.net/problem/18964)

## 설명
각 참가자가 고른 수들로부터, 특정 m과 k를 정해 k mod m에 해당하는 사람 수가 절반 이상이 되도록 하는 문제입니다.

<br>

## 접근법
2로 나눈 나머지로 분류하면 모든 수는 짝수 또는 홀수로 나뉩니다. 둘 중 더 많은 쪽은 항상 절반 이상이므로, 짝수가 더 많으면 0을, 홀수가 더 많으면 1을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Console.ReadLine()!.Split();

    var even = 0;
    var odd = 0;
    for (var i = 0; i < n; i++) {
      var val = int.Parse(parts[i]);
      if (val % 2 == 0) even++;
      else odd++;
    }

    var k = even >= odd ? 0 : 1;
    Console.WriteLine($"2 {k}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi a(n);
  for (int i = 0; i < n; i++)
    cin >> a[i];

  int even = 0, odd = 0;
  for (int i = 0; i < n; i++) {
    if (a[i] % 2 == 0) even++;
    else odd++;
  }

  int k = (even >= odd) ? 0 : 1;
  cout << 2 << " " << k << "\n";

  return 0;
}
```
