## 문제 링크
[21645번 - Ролевая игра](https://www.acmicpc.net/problem/21645)

## 설명
어떤 레벨 배정이 오더라도 n명 모두에게 배지를 맞춰줄 수 있도록 준비해야 하는 최소 배지 총합을 구하는 문제입니다.

<br>

## 접근법
흰색 배지는 1레벨, 빨간 배지는 k레벨을 의미하고, 흰색은 k-1개 이하만 사용할 수 있습니다. 레벨 x를 만들려면 흰색 x % k개와 빨간색 x / k개가 필요합니다.

1부터 m까지의 레벨 중 흰색이 가장 많이 필요한 경우는 m이 k보다 작으면 m개, 그렇지 않으면 k-1개입니다. 빨간색이 가장 많이 필요한 경우는 m / k개입니다. 최악의 경우 n명 모두가 이 최대치를 요구할 수 있으므로, 준비해야 할 총 배지 수는 n × (흰색 최대 + 빨간색 최대)입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = int.Parse(parts[0]);
    var m = int.Parse(parts[1]);
    var k = int.Parse(parts[2]);

    var wmax = m < k ? m : k - 1;
    var rmax = m / k;
    var total = n * (wmax + rmax);

    Console.WriteLine(total);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m, k; cin >> n >> m >> k;

  int wmax = (m < k) ? m : (k - 1);
  int rmax = m / k;
  int total = n * (wmax + rmax);

  cout << total << "\n";

  return 0;
}
```
