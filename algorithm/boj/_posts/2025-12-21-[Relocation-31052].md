## 문제 링크
[31052번 - Relocation](https://www.acmicpc.net/problem/31052)

## 설명
회사들의 위치가 주어질 때, 위치 변경과 두 회사 사이의 거리를 묻는 질의를 처리하는 문제입니다.

<br>

## 접근법
각 회사의 현재 위치를 배열에 저장합니다. 업데이트 요청은 해당 인덱스 값을 갱신하고, 거리 질의는 두 좌표의 차이 절댓값을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var q = int.Parse(first[1]);

    var parts = Console.ReadLine()!.Split();
    var pos = new int[n + 1];
    for (var i = 1; i <= n; i++)
      pos[i] = int.Parse(parts[i - 1]);

    for (var i = 0; i < q; i++) {
      var line = Console.ReadLine()!.Split();
      var type = int.Parse(line[0]);
      var a = int.Parse(line[1]);
      var b = int.Parse(line[2]);
      if (type == 1) pos[a] = b;
      else {
        var diff = pos[a] - pos[b];
        Console.WriteLine(diff < 0 ? -diff : diff);
      }
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, q; cin >> n >> q;
  vi pos(n + 1);
  for (int i = 1; i <= n; i++)
    cin >> pos[i];

  for (int i = 0; i < q; i++) {
    int type, a, b; cin >> type >> a >> b;
    if (type == 1) pos[a] = b;
    else cout << abs(pos[a] - pos[b]) << "\n";
  }

  return 0;
}
```
