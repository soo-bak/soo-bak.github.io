## 문제 링크
[26583번 - Scale](https://www.acmicpc.net/problem/26583)

## 설명
정수 리스트에서 각 원소를 이웃 값들과 곱한 결과를 출력하는 문제입니다.

<br>

## 접근법
각 원소의 스케일링 규칙은 위치에 따라 다릅니다. 첫 원소는 오른쪽 이웃과 곱하고, 마지막 원소는 왼쪽 이웃과 곱합니다. 중간 원소들은 자신과 양쪽 이웃을 모두 곱합니다. 원소가 하나뿐이면 그대로 출력합니다.

입력이 여러 줄일 수 있으므로 입력이 끝날 때까지 각 줄을 처리하고 결과를 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    string? line;
    while ((line = Console.ReadLine()) != null) {
      var v = line.Split(' ', StringSplitOptions.RemoveEmptyEntries)
                  .Select(int.Parse)
                  .ToArray();
      if (v.Length == 0) continue;

      var n = v.Length;
      var res = new int[n];
      if (n == 1) {
        res[0] = v[0];
      } else {
        res[0] = v[0] * v[1];
        res[n - 1] = v[n - 2] * v[n - 1];
        for (var i = 1; i < n - 1; i++)
          res[i] = v[i - 1] * v[i] * v[i + 1];
      }

      Console.WriteLine(string.Join(" ", res));
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef vector<ll> vll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string line;
  while (getline(cin, line)) {
    if (line.empty()) continue;

    stringstream ss(line);
    vll v;
    ll x;
    while (ss >> x)
      v.push_back(x);
    if (v.empty()) continue;

    int n = v.size();
    vll res(n);
    if (n == 1) {
      res[0] = v[0];
    } else {
      res[0] = v[0] * v[1];
      res[n - 1] = v[n - 2] * v[n - 1];
      for (int i = 1; i < n - 1; i++)
        res[i] = v[i - 1] * v[i] * v[i + 1];
    }

    for (int i = 0; i < n; i++) {
      if (i) cout << " ";
      cout << res[i];
    }
    cout << "\n";
  }

  return 0;
}
```
