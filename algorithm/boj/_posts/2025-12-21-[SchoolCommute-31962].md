## 문제 링크
[31962번 - 등교](https://www.acmicpc.net/problem/31962)

## 설명
각 버스의 출발까지 남은 시간 S와 이동 시간 T가 주어질 때, S+T가 X 이내인 버스 중 가장 늦게 출발하는 S를 출력하는 문제입니다.

<br>

## 접근법
모든 버스를 확인하며 S+T가 X 이하인 경우만 후보로 보고, 그 중 최대 S를 갱신합니다. 후보가 없으면 -1을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var x = int.Parse(first[1]);

    var best = -1;
    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var s = int.Parse(parts[0]);
      var t = int.Parse(parts[1]);
      if (s + t <= x && s > best) best = s;
    }

    Console.WriteLine(best);
  }
}
```

### C++
