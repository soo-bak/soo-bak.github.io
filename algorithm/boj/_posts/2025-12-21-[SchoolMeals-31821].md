## 문제 링크
[31821번 - 학식 사주기](https://www.acmicpc.net/problem/31821)

## 설명
각 코너의 메뉴 가격과 새내기들이 선택한 코너 번호가 주어질 때, 결제해야 하는 총 금액을 구하는 문제입니다.

<br>

## 접근법
코너 번호는 1부터 시작하므로 가격을 배열에 저장한 뒤, 각 새내기의 번호를 읽어 해당 가격을 합산합니다. 합계를 출력하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var price = new int[n + 1];
    for (var i = 1; i <= n; i++)
      price[i] = int.Parse(Console.ReadLine()!);

    var m = int.Parse(Console.ReadLine()!);
    var sum = 0;
    for (var i = 0; i < m; i++) {
      var idx = int.Parse(Console.ReadLine()!);
      sum += price[idx];
    }

    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi price(n + 1);
  for (int i = 1; i <= n; i++)
    cin >> price[i];

  int m; cin >> m;
  int sum = 0;
  for (int i = 0; i < m; i++) {
    int idx; cin >> idx;
    sum += price[idx];
  }

  cout << sum << "\n";

  return 0;
}
```
