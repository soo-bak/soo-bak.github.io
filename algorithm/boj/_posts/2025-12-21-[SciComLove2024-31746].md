## 문제 링크
[31746번 - SciComLove (2024)](https://www.acmicpc.net/problem/31746)

## 설명
문자열 SciComLove를 N번 뒤집었을 때의 결과를 출력하는 문제입니다.

<br>

## 접근법
문자열을 한 번 뒤집으면 방향이 바뀌고, 두 번 뒤집으면 원래대로 돌아옵니다. 따라서 N이 홀수면 뒤집힌 문자열을, 짝수면 원문을 출력하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = "SciComLove";
    if (n % 2 == 0) Console.WriteLine(s);
    else {
      var arr = s.ToCharArray();
      Array.Reverse(arr);
      Console.WriteLine(new string(arr));
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s = "SciComLove";
  if (n % 2 == 0) cout << s << "\n";
  else {
    reverse(s.begin(), s.end());
    cout << s << "\n";
  }

  return 0;
}
```
