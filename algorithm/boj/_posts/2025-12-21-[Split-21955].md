## 문제 링크
[21955번 - Split](https://www.acmicpc.net/problem/21955)

## 설명
짝수 자릿수로 이루어진 정수를 두 부분으로 나눠 각각 출력하는 문제입니다.

<br>

## 접근법
입력을 문자열로 받아 길이를 절반으로 나눕니다. 앞 절반과 뒤 절반을 분리해 그대로 출력하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var half = s.Length / 2;
    var a = s.Substring(0, half);
    var b = s.Substring(half);
    Console.WriteLine($"{a} {b}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  int half = s.size() / 2;
  cout << s.substr(0, half) << " " << s.substr(half) << "\n";

  return 0;
}
```
