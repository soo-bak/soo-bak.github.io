## 문제 링크
[31656번 - Sticky Keys](https://www.acmicpc.net/problem/31656)

## 설명
메시지에서 연속으로 같은 문자가 반복되면 한 글자만 남겨서 출력하는 문제입니다.

<br>

## 접근법
문자열을 한 글자씩 순회하며 직전 문자와 다를 때만 결과에 추가합니다. 공백도 문자로 취급하므로 그대로 비교하면 됩니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var sb = new StringBuilder();
    var prev = '\0';

    foreach (var ch in s) {
      if (ch != prev) sb.Append(ch);
      prev = ch;
    }

    Console.WriteLine(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s;
  getline(cin, s);

  string res;
  char prev = '\0';
  for (char ch : s) {
    if (ch != prev) res.push_back(ch);
    prev = ch;
  }

  cout << res << "\n";

  return 0;
}
```
