## 문제 링크
[31694번 - Kto wygrał?](https://www.acmicpc.net/problem/31694)

## 설명
두 사람의 18개 문제 점수가 주어질 때, 총점이 높은 사람이 이기고, 총점이 같으면 10점 개수가 많은 쪽, 그래도 같으면 9점 개수 순으로 비교해 승자를 결정하는 문제입니다.

<br>

## 접근법
각 사람의 총점과 점수별 개수를 셉니다. 총점이 다르면 높은 쪽이 승자입니다. 총점이 같으면 10점부터 0점까지 차례로 개수를 비교해 처음 차이가 나는 점수에서 많은 쪽이 승자입니다. 끝까지 같으면 무승부입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var a = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var b = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);

    var sumA = 0;
    var sumB = 0;
    var cntA = new int[11];
    var cntB = new int[11];

    for (var i = 0; i < 18; i++) {
      sumA += a[i]; cntA[a[i]]++;
      sumB += b[i]; cntB[b[i]]++;
    }

    if (sumA != sumB) {
      Console.WriteLine(sumA > sumB ? "Algosia" : "Bajtek");
      return;
    }

    for (var s = 10; s >= 0; s--) {
      if (cntA[s] != cntB[s]) {
        Console.WriteLine(cntA[s] > cntB[s] ? "Algosia" : "Bajtek");
        return;
      }
    }

    Console.WriteLine("remis");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  vi a(18), b(18);
  for (int i = 0; i < 18; i++) cin >> a[i];
  for (int i = 0; i < 18; i++) cin >> b[i];

  int sumA = 0, sumB = 0;
  int cntA[11] = {}, cntB[11] = {};
  for (int i = 0; i < 18; i++) {
    sumA += a[i]; cntA[a[i]]++;
    sumB += b[i]; cntB[b[i]]++;
  }

  if (sumA != sumB) {
    cout << (sumA > sumB ? "Algosia" : "Bajtek") << "\n";
    return 0;
  }

  for (int s = 10; s >= 0; s--) {
    if (cntA[s] != cntB[s]) {
      cout << (cntA[s] > cntB[s] ? "Algosia" : "Bajtek") << "\n";
      return 0;
    }
  }

  cout << "remis\n";

  return 0;
}
```
