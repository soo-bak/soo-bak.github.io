## 문제 링크
[29070번 - Ремонт от BB-8](https://www.acmicpc.net/problem/29070)

## 설명
높이 a, 너비 b인 판으로 높이 h, 너비 w인 구멍을 덮을 때 필요한 최소 개수를 구하는 문제입니다.

<br>

## 접근법
판은 회전할 수 없으므로 높이 방향과 너비 방향을 각각 판의 크기로 나눕니다.

이후 각 방향에서 필요한 판 개수를 올림하여 구하고, 두 값을 곱해 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);
    var h = int.Parse(parts[2]);
    var w = int.Parse(parts[3]);

    var needH = (h + a - 1) / a;
    var needW = (w + b - 1) / b;

    Console.WriteLine(needH * needW);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, h, w; cin >> a >> b >> h >> w;

  int needH = (h + a - 1) / a;
  int needW = (w + b - 1) / b;

  cout << needH * needW << "\n";

  return 0;
}
```
