## 문제 링크
[1718번 - 암호](https://www.acmicpc.net/problem/1718)

## 설명
평문과 키가 주어질 때, 키의 알파벳 순서를 평문에서 빼서 암호문을 만드는 문제입니다.

<br>

## 접근법
평문을 왼쪽부터 순회하며 같은 위치의 키 문자를 사용합니다. 공백은 그대로 두고 인덱스만 진행합니다.

알파벳은 키 값만큼 앞당기되, 범위를 벗어나면 알파벳 끝으로 순환시켜 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var text = Console.ReadLine()!;
    var key = Console.ReadLine()!;
    var klen = key.Length;

    var sb = new StringBuilder(text.Length);
    for (var i = 0; i < text.Length; i++) {
      var ch = text[i];
      if (ch == ' ') {
        sb.Append(' ');
        continue;
      }
      var shift = key[i % klen] - 'a' + 1;
      var val = (ch - 'a' + 1) - shift;
      if (val <= 0) val += 26;
      sb.Append((char)('a' + val - 1));
    }

    Console.WriteLine(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string text, key;
  getline(cin, text);
  getline(cin, key);
  int klen = key.size();

  string res;
  res.reserve(text.size());
  for (int i = 0; i < (int)text.size(); i++) {
    char ch = text[i];
    if (ch == ' ') {
      res.push_back(' ');
      continue;
    }
    int shift = key[i % klen] - 'a' + 1;
    int val = (ch - 'a' + 1) - shift;
    if (val <= 0) val += 26;
    res.push_back(char('a' + val - 1));
  }

  cout << res << "\n";

  return 0;
}
```
