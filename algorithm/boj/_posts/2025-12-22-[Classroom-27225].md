## 문제 링크
[27225번 - Класс](https://www.acmicpc.net/problem/27225)

## 설명
남학생 n명과 여학생 m명을 두 명씩 앉힐 때, 집중할 수 있는 학생 수의 최댓값을 구하는 문제입니다.

<br>

## 접근법
남녀가 짝을 이루면 둘 다 집중하므로, 짝을 이룰 수 있는 최대 인원의 2배가 기본 집중 인원입니다.

이후 남는 학생 수가 홀수면 짝을 이루지 못한 1명이 추가로 집중할 수 있으므로, 기본 인원에 1을 더합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var m = int.Parse(Console.ReadLine()!);

    var min = n < m ? n : m;
    var diff = n - m;
    if (diff < 0) diff = -diff;

    var ans = 2 * min + (diff % 2);
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;

  int mn = min(n, m);
  int diff = n - m;
  if (diff < 0) diff = -diff;

  int ans = 2 * mn + (diff % 2);
  cout << ans << "\n";

  return 0;
}
```
