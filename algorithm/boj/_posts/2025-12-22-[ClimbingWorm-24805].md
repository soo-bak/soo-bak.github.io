## 문제 링크
[24805번 - Climbing Worm](https://www.acmicpc.net/problem/24805)

## 설명
한 번 오르고 미끄러지는 과정을 반복할 때 정상에 도달하기 위한 횟수를 구하는 문제입니다.

<br>

## 접근법
정상에 도달하는 마지막 이동은 미끄러짐이 없으므로, 한 번에 도달하지 못한다면 첫 이동 이후부터는 매번 오르는 거리에서 미끄러지는 거리를 뺀 만큼 높이가 증가합니다.

첫 이동으로 도달 가능하면 1회이고, 그렇지 않으면 남은 거리를 한 사이클당 순증가량으로 나눈 올림값에 1을 더해 계산합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);
    var h = int.Parse(parts[2]);

    if (h <= a) {
      Console.WriteLine(1);
      return;
    }

    var step = a - b;
    var remain = h - a;
    var cnt = 1 + (remain + step - 1) / step;
    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, h; cin >> a >> b >> h;
  if (h <= a) {
    cout << 1 << "\n";
    return 0;
  }

  int step = a - b;
  int remain = h - a;
  int cnt = 1 + (remain + step - 1) / step;
  cout << cnt << "\n";

  return 0;
}
```
