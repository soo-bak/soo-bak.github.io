## 문제 링크
[29220번 - Свидание](https://www.acmicpc.net/problem/29220)

## 설명
지불한 금액에서 어떤 한 장을 빼도 식사값 이상이면 벨라까지 계산하려 했다고 판단하는 문제입니다.

<br>

## 접근법
모든 지폐의 합에서 가장 작은 지폐를 뺀 값이 식사값 이상인지 확인합니다.

이 조건을 만족하면 YES, 아니면 NO를 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    var k = long.Parse(Console.ReadLine()!);
    var n = int.Parse(Console.ReadLine()!);
    var bills = Console.ReadLine()!.Split().Select(int.Parse).ToArray();

    var total = bills.Sum(b => (long)b) - bills.Min();

    Console.WriteLine(total >= k ? "YES" : "NO");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll k; int n; cin >> k >> n;

  vi bills(n);
  for (int& bill : bills)
    cin >> bill;

  ll total = accumulate(bills.begin(), bills.end(), 0LL);
  total -= *min_element(bills.begin(), bills.end());

  cout << (total >= k ? "YES" : "NO") << "\n";

  return 0;
}
```
