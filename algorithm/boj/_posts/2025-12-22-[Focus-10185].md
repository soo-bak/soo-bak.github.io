## 문제 링크
[10185번 - Focus](https://www.acmicpc.net/problem/10185)

## 설명
물체 거리 p와 상 거리 q가 주어질 때 초점거리 f를 계산해 출력하는 문제입니다.

<br>

## 접근법
렌즈 방정식을 정리하면 초점거리는 물체 거리와 상 거리의 곱을 두 값의 합으로 나눈 값입니다.

이를 실수로 계산해 소수 첫째 자리까지 반올림하여 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < t; i++) {
      var parts = Console.ReadLine()!.Split();
      var p = double.Parse(parts[0]);
      var q = double.Parse(parts[1]);
      var f = p * q / (p + q);
      Console.WriteLine($"f = {f:F1}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  for (int i = 0; i < t; i++) {
    double p, q; cin >> p >> q;
    double f = p * q / (p + q);
    cout << "f = " << fixed << setprecision(1) << f << "\n";
  }

  return 0;
}
```
