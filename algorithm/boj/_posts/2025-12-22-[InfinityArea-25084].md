## 문제 링크
[25084번 - Infinity Area](https://www.acmicpc.net/problem/25084)

## 설명
주어진 규칙대로 반지름이 변하는 원들을 그릴 때 넓이의 총합을 구하는 문제입니다.

<br>

## 접근법
오른쪽 원을 그린 뒤 반지름을 A배로 키워 왼쪽 원을 그리고, 이후 반지름을 B로 나눈 뒤 다시 같은 과정을 반복합니다.

반지름이 0이 되면 종료되므로, 반복하면서 각 원의 넓이를 누적하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var tc = 1; tc <= t; tc++) {
      var parts = Console.ReadLine()!.Split();
      var r = long.Parse(parts[0]);
      var a = int.Parse(parts[1]);
      var b = int.Parse(parts[2]);

      double sum = 0;
      while (r > 0) {
        sum += Math.PI * r * (double)r;
        r *= a;
        sum += Math.PI * r * (double)r;
        r /= b;
      }

      Console.WriteLine($"Case #{tc}: {sum:F6}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  const double pi = acos(-1.0);

  for (int tc = 1; tc <= t; tc++) {
    long long r; int a, b;
    cin >> r >> a >> b;

    double sum = 0;
    while (r > 0) {
      sum += pi * (double)r * r;
      r *= a;
      sum += pi * (double)r * r;
      r /= b;
    }

    cout << "Case #" << tc << ": " << fixed << setprecision(6) << sum << "\n";
  }

  return 0;
}
```
