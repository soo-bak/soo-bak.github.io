## 문제 링크
[32025번 - 체육은 수학과목 입니다](https://www.acmicpc.net/problem/32025)

## 설명
운동장 직사각형 안에 그릴 수 있는 가장 큰 원의 반지름을 구하는 문제입니다.

<br>

## 접근법
가장 큰 원은 직사각형의 짧은 변에 내접합니다. 반지름은 짧은 변의 절반입니다.

이후 미터를 센티미터로 변환하면 짧은 변에 50을 곱한 값이 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var h = int.Parse(Console.ReadLine()!);
    var w = int.Parse(Console.ReadLine()!);
    var min = h < w ? h : w;
    Console.WriteLine(min * 50);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int h, w; cin >> h >> w;
  int mn = min(h, w);
  cout << mn * 50 << "\n";

  return 0;
}
```
