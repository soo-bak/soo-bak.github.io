## 문제 링크
[32193번 - 승강장의 깊이](https://www.acmicpc.net/problem/32193)

## 설명
각 역으로 이동할 때 지표면과 승강장 높이가 변할 때, 각 역의 깊이를 구하는 문제입니다.

<br>

## 접근법
지표면 높이와 승강장 높이를 각각 누적합니다.

매 역마다 지표면 높이에서 승강장 높이를 뺀 값을 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var surf = 0;
    var plat = 0;
    var sb = new StringBuilder();

    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = int.Parse(parts[0]);
      var b = int.Parse(parts[1]);
      surf += a;
      plat += b;
      sb.AppendLine((surf - plat).ToString());
    }

    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int surf = 0, plat = 0;
  for (int i = 0; i < n; i++) {
    int a, b; cin >> a >> b;
    surf += a;
    plat += b;
    cout << (surf - plat) << "\n";
  }

  return 0;
}
```
