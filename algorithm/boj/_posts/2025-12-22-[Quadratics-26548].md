## 문제 링크
[26548번 - Quadratics](https://www.acmicpc.net/problem/26548)

## 설명
이차방정식의 해를 공식으로 계산해 두 값을 출력하는 문제입니다.

<br>

## 접근법
판별식의 제곱근을 구해 근의 공식에 대입합니다.

이후 더하는 해를 먼저, 빼는 해를 나중에 출력하며 소수 셋째 자리까지 반올림합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < t; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = double.Parse(parts[0]);
      var b = double.Parse(parts[1]);
      var c = double.Parse(parts[2]);

      var disc = Math.Sqrt(b * b - 4 * a * c);
      var x1 = (-b + disc) / (2 * a);
      var x2 = (-b - disc) / (2 * a);

      Console.WriteLine($"{x1:F3}, {x2:F3}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  for (int i = 0; i < t; i++) {
    double a, b, c; cin >> a >> b >> c;

    double disc = sqrt(b * b - 4 * a * c);
    double x1 = (-b + disc) / (2 * a);
    double x2 = (-b - disc) / (2 * a);

    cout << fixed << setprecision(3) << x1 << ", " << x2 << "\n";
  }

  return 0;
}
```
