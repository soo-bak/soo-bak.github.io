## 문제 링크
[3282번 - ROOMS](https://www.acmicpc.net/problem/3282)

## 설명
손님 그룹이 도착하는 순서가 주어질 때 규칙에 따라 방을 배정하고 각 방의 인원 수를 출력하는 문제입니다.

<br>

## 접근법
빈 방이 있으면 두 명씩 가장 번호가 작은 빈 방에 배정하고, 홀수가 남으면 한 명을 같은 방식으로 배정합니다.

빈 방이 없으면 한 명만 있는 방 중 번호가 가장 작은 방부터 한 명씩 채우며, 각 그룹마다 이 과정을 반복합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static int Find(int[] rooms, int target) {
    for (var i = 0; i < rooms.Length; i++) {
      if (rooms[i] == target) return i;
    }
    return -1;
  }

  static void Main() {
    var data = Console.In.ReadToEnd();
    var parts = data.Split(new[] { ' ', '\n', '\r', '\t' }, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var g = int.Parse(parts[idx++]);

    var rooms = new int[n];
    for (var t = 0; t < g; t++) {
      var cnt = int.Parse(parts[idx++]);

      while (cnt >= 2) {
        var pos = Find(rooms, 0);
        if (pos == -1) break;
        rooms[pos] = 2;
        cnt -= 2;
      }

      if (cnt == 1) {
        var pos = Find(rooms, 0);
        if (pos != -1) {
          rooms[pos] = 1;
          cnt = 0;
        }
      }

      while (cnt > 0) {
        var pos = Find(rooms, 1);
        rooms[pos] = 2;
        cnt--;
      }
    }

    for (var i = 0; i < n; i++)
      Console.WriteLine(rooms[i]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int findRoom(const vi& rooms, int target) {
  for (int i = 0; i < (int)rooms.size(); i++) {
    if (rooms[i] == target) return i;
  }
  return -1;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, g; cin >> n >> g;
  vi rooms(n, 0);

  for (int t = 0; t < g; t++) {
    int cnt; cin >> cnt;

    while (cnt >= 2) {
      int pos = findRoom(rooms, 0);
      if (pos == -1) break;
      rooms[pos] = 2;
      cnt -= 2;
    }

    if (cnt == 1) {
      int pos = findRoom(rooms, 0);
      if (pos != -1) {
        rooms[pos] = 1;
        cnt = 0;
      }
    }

    while (cnt > 0) {
      int pos = findRoom(rooms, 1);
      rooms[pos] = 2;
      cnt--;
    }
  }

  for (int i = 0; i < n; i++)
    cout << rooms[i] << "\n";

  return 0;
}
```
