## 문제 링크
[30281번 - Saldainiai](https://www.acmicpc.net/problem/30281)

## 설명
상자들을 골라 두 사람이 똑같이 나눌 수 있는 최대 사탕 개수를 구하는 문제입니다.

<br>

## 접근법
모든 상자를 다 고른 합이 짝수면 그 절반이 답입니다.

합이 홀수면 한 개의 홀수 상자를 제외해야 짝수가 되므로, 가장 작은 홀수 값을 빼고 절반을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var input = Console.In.ReadToEnd();
    var parts = input.Split(new[] { ' ', '\n', '\r' }, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var n = int.Parse(parts[idx++]);

    var sum = 0;
    var minOdd = int.MaxValue;
    for (var i = 0; i < n; i++) {
      var v = int.Parse(parts[idx++]);
      sum += v;
      if (v % 2 == 1 && v < minOdd) minOdd = v;
    }

    var ans = sum % 2 == 0 ? sum / 2 : (sum - minOdd) / 2;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int sum = 0;
  int minOdd = 1e9;
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    sum += v;
    if (v % 2 == 1 && v < minOdd) minOdd = v;
  }

  int ans = (sum % 2 == 0) ? (sum / 2) : ((sum - minOdd) / 2);
  cout << ans << "\n";

  return 0;
}
```
