## 문제 링크
[27251번 - Звездочки](https://www.acmicpc.net/problem/27251)

## 설명
1부터 n까지 각 줄에 k^2개의 별을 출력하되, 100개를 넘으면 100개만 출력하고 뒤에 ...을 붙이는 문제입니다.

<br>

## 접근법
각 줄 번호에 대해 그 수의 제곱을 계산합니다.

이후 100 이하이면 그 수만큼 별을 출력하고, 100을 넘으면 별 100개와 ...을 이어서 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    for (var k = 1; k <= n; k++) {
      var cnt = k * k;
      if (cnt <= 100) Console.WriteLine(new string('*', cnt));
      else Console.WriteLine(new string('*', 100) + "...");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  for (int k = 1; k <= n; k++) {
    int cnt = k * k;
    if (cnt <= 100) cout << string(cnt, '*') << "\n";
    else cout << string(100, '*') << "...\n";
  }

  return 0;
}
```
