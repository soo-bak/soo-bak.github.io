## 문제 링크
[30917번 - A+B - 10 (제1편)](https://www.acmicpc.net/problem/30917)

## 설명
질문을 통해 A, B를 알아낸 뒤 A+B를 출력하는 인터랙티브 문제입니다.

<br>

## 접근법
두 수는 각각 1에서 9 사이의 값이므로, 순차적으로 질문하여 채점기가 1을 반환하는 값을 찾으면 됩니다.

이후 각 질문마다 출력 버퍼를 비우고, 찾은 두 값의 합을 느낌표와 함께 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static int Ask(char who, int x) {
    Console.WriteLine($"? {who} {x}");
    Console.Out.Flush();
    return int.Parse(Console.ReadLine()!);
  }

  static void Main() {
    var a = 0;
    var b = 0;

    for (var i = 1; i <= 9; i++) {
      if (Ask('A', i) == 1) { a = i; break; }
    }
    for (var i = 1; i <= 9; i++) {
      if (Ask('B', i) == 1) { b = i; break; }
    }

    Console.WriteLine($"! {a + b}");
    Console.Out.Flush();
  }
}
```

### C++
```cpp

    if (ask('B', i) == 1) { b = i; break; }
  }

  return 0;
}
```
