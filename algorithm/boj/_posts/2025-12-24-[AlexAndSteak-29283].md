## 문제 링크
[29283번 - Алекс и стейк](https://www.acmicpc.net/problem/29283)

## 설명
버튼을 k번 눌렀을 때 누적되는 가열 시간을 계산하는 문제입니다.

<br>

## 접근법
버튼을 5회씩 묶으면 각 묶음에서 늘어나는 시간은 30, 60, 90초처럼 등차수열을 이룹니다.

따라서 완전한 묶음 개수와 나머지 횟수를 구한 뒤, 완전 묶음들의 합과 나머지 횟수만큼의 추가 시간을 더하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var k = long.Parse(Console.ReadLine()!);
    var g = k / 5;
    var r = k % 5;

    var total = 75 * g * (g + 1) + r * 30 * (g + 1);
    Console.WriteLine(total);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll k; cin >> k;
  ll g = k / 5, r = k % 5;
  ll total = 75 * g * (g + 1) + r * 30 * (g + 1);
  cout << total << "\n";

  return 0;
}
```
