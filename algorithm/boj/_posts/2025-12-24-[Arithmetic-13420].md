## 문제 링크
[13420번 - 사칙연산](https://www.acmicpc.net/problem/13420)

## 설명
입력으로 주어진 사칙연산 수식이 올바른지 판별하는 문제입니다.

<br>

## 접근법
연산자에 따라 값을 계산해 우변과 비교합니다.

나눗셈은 항상 나누어떨어지는 경우만 주어집니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < t; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = long.Parse(parts[0]);
      var op = parts[1];
      var b = long.Parse(parts[2]);
      var c = long.Parse(parts[4]);

      long val;
      if (op == "+") val = a + b;
      else if (op == "-") val = a - b;
      else if (op == "*") val = a * b;
      else val = a / b;

      Console.WriteLine(val == c ? "correct" : "wrong answer");
    }
  }
}
```

### C++

```cpp
#include <bits/stdc++.h>

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    ll a, b, c;
    char op, eq;
    cin >> a >> op >> b >> eq >> c;

    ll val;
    if (op == '+') val = a + b;
    else if (op == '-') val = a - b;
    else if (op == '*') val = a * b;
    else val = a / b;

    cout << (val == c ? "correct" : "wrong answer") << "\n";
  }

  return 0;
}
```
