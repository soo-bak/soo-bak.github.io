## 문제 링크
[32515번 - BB84](https://www.acmicpc.net/problem/32515)

## 설명
같은 기저에서 측정값이 일치하는지 확인해 도청 여부를 판단하고, 새 키를 출력하는 문제입니다.

<br>

## 접근법
두 사람이 같은 기저를 선택한 자리만 확인합니다.

이후 같은 기저인데 비트가 하나라도 다르면 도청이 확실하므로 htg!를 출력하고, 모두 일치하면 해당 자리의 비트를 이어 새 키를 만듭니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var baseJ = Console.ReadLine()!;
    var keyJ = Console.ReadLine()!;
    var baseI = Console.ReadLine()!;
    var measI = Console.ReadLine()!;

    var sb = new StringBuilder();
    for (var i = 0; i < n; i++) {
      if (baseJ[i] != baseI[i]) continue;
      if (keyJ[i] != measI[i]) {
        Console.WriteLine("htg!");
        return;
      }
      sb.Append(keyJ[i]);
    }

    Console.WriteLine(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string baseJ, keyJ, baseI, measI; cin >> baseJ >> keyJ >> baseI >> measI;

  string res;
  res.reserve(n);
  for (int i = 0; i < n; i++) {
    if (baseJ[i] != baseI[i]) continue;
    if (keyJ[i] != measI[i]) {
      cout << "htg!\n";
      return 0;
    }
    res.push_back(keyJ[i]);
  }

  cout << res << "\n";

  return 0;
}
```
