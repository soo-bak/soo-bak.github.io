## 문제 링크
[14915번 - 진수 변환기](https://www.acmicpc.net/problem/14915)

## 설명
10진수 m을 n진수로 변환해 출력하는 문제입니다.

<br>

## 접근법
0이면 그대로 0을 출력합니다.

그 외에는 m을 n으로 나누며 나머지를 모아 뒤집어 출력합니다.

나머지가 10 이상이면 A~F로 변환합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var m = int.Parse(parts[0]);
    var n = int.Parse(parts[1]);

    if (m == 0) {
      Console.WriteLine(0);
      return;
    }

    var sb = new StringBuilder();
    while (m > 0) {
      var r = m % n;
      if (r < 10) sb.Append((char)('0' + r));
      else sb.Append((char)('A' + (r - 10)));
      m /= n;
    }

    var arr = sb.ToString().ToCharArray();
    Array.Reverse(arr);
    Console.WriteLine(new string(arr));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int m, n; cin >> m >> n;
  if (m == 0) {
    cout << 0 << "\n";
    return 0;
  }

  string res;
  while (m > 0) {
    int r = m % n;
    if (r < 10) res.push_back('0' + r);
    else res.push_back('A' + (r - 10));
    m /= n;
  }

  reverse(res.begin(), res.end());
  cout << res << "\n";

  return 0;
}
```
