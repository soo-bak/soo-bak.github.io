## 문제 링크
[3595번 - 맥주 냉장고](https://www.acmicpc.net/problem/3595)

## 설명
부피가 n인 직육면체 중 겉넓이가 최소인 크기 a, b, c를 구하는 문제입니다.

<br>

## 접근법
부피가 n이 되는 세 변의 조합을 찾기 위해 가능한 약수 조합을 확인합니다.

이후 각 조합에서 겉넓이를 계산해 최솟값을 갱신하고, 해당 조합을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var best = int.MaxValue;
    var ansA = 1;
    var ansB = 1;
    var ansC = n;

    for (var a = 1; a * a * a <= n; a++) {
      if (n % a != 0) continue;
      var n1 = n / a;
      for (var b = a; b * b <= n1; b++) {
        if (n1 % b != 0) continue;
        var c = n1 / b;
        var area = 2 * (a * b + b * c + c * a);
        if (area < best) {
          best = area;
          ansA = a;
          ansB = b;
          ansC = c;
        }
      }
    }

    Console.WriteLine($"{ansA} {ansB} {ansC}");
  }
}
```
