## 문제 링크
[10834번 - 벨트](https://www.acmicpc.net/problem/10834)

## 설명
벨트로 연결된 바퀴들의 회전수 비와 꼬임 정보를 이용해 마지막 바퀴의 방향과 회전수를 구하는 문제입니다.

<br>

## 접근법
바퀴 1의 회전수는 1이고 방향은 시계 방향입니다.

각 벨트마다 회전수는 `회전수 × b / a`로 갱신하고, 꼬임이 있으면 방향을 뒤집습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var m = int.Parse(Console.ReadLine()!);
    long rpm = 1;
    var dir = 0;

    for (var i = 0; i < m; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = long.Parse(parts[0]);
      var b = long.Parse(parts[1]);
      var s = int.Parse(parts[2]);

      rpm = rpm * b / a;
      if (s == 1) dir ^= 1;
    }

    Console.WriteLine($"{dir} {rpm}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int m; cin >> m;
  ll rpm = 1;
  int dir = 0;

  for (int i = 0; i < m; i++) {
    long long a, b; int s;
    cin >> a >> b >> s;
    rpm = rpm * b / a;
    if (s == 1) dir ^= 1;
  }

  cout << dir << " " << rpm << "\n";

  return 0;
}
```
