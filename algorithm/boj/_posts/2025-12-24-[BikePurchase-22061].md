## 문제 링크
[22061번 - Покупка велосипеда](https://www.acmicpc.net/problem/22061)

## 설명
1원 동전 a개와 2원 동전 b개로 정확히 c원을 만들 수 있는지 판단하는 문제입니다.

<br>

## 접근법
2원 동전을 최대한 사용하면 남는 금액이 최소가 됩니다.

남은 금액이 1원 동전 개수 이하라면 채울 수 있으므로 가능하고, 그렇지 않으면 불가능합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var data = Console.In.ReadToEnd();
    var parts = data.Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);
    var sb = new StringBuilder();

    for (var i = 0; i < t; i++) {
      var a = int.Parse(parts[idx++]);
      var b = int.Parse(parts[idx++]);
      var c = int.Parse(parts[idx++]);

      var use2 = Math.Min(b, c / 2);
      var remain = c - 2 * use2;
      sb.AppendLine(remain <= a ? "YES" : "NO");
    }

    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    int a, b, c; cin >> a >> b >> c;
    int use2 = min(b, c / 2);
    int remain = c - 2 * use2;
    cout << (remain <= a ? "YES" : "NO") << "\n";
  }

  return 0;
}
```
