## 문제 링크
[32246번 - 빙고 막기](https://www.acmicpc.net/problem/32246)

## 설명
N×N 격자에서 모든 행과 열, 두 대각선에 최소 한 칸씩 칠해야 할 때 필요한 최소 칸 수를 구하는 문제입니다.

<br>

## 접근법
모든 행과 열을 채우려면 최소 N칸이 필요하며, 대부분의 경우 대각선까지 포함하는 배치가 가능합니다.

다만 N이 2일 때는 대각선을 모두 포함하는 배치가 불가능하므로 3칸이 필요하고, 그 외에는 N칸이 정답입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    Console.WriteLine(n == 2 ? 3 : n);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  cout << (n == 2 ? 3 : n) << "\n";

  return 0;
}
```
