## 문제 링크
[21918번 - 전구](https://www.acmicpc.net/problem/21918)

## 설명
전구 상태 배열에 네 가지 명령을 순서대로 적용한 결과를 출력하는 문제입니다.

<br>

## 접근법
전구 상태를 배열로 저장해 각 명령을 그대로 시뮬레이션합니다.

2번 명령은 토글, 3번은 끄기, 4번은 켜기로 처리합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);

    var parts = Console.ReadLine()!.Split();
    var s = new int[n];
    for (var i = 0; i < n; i++)
      s[i] = int.Parse(parts[i]);

    for (var t = 0; t < m; t++) {
      var cmd = Console.ReadLine()!.Split();
      var a = int.Parse(cmd[0]);
      var b = int.Parse(cmd[1]) - 1;
      var c = int.Parse(cmd[2]);

      if (a == 1) {
        s[b] = c;
      } else {
        var r = c - 1;
        for (var i = b; i <= r; i++) {
          if (a == 2) s[i] = 1 - s[i];
          else if (a == 3) s[i] = 0;
          else s[i] = 1;
        }
      }
    }

    Console.WriteLine(string.Join(" ", s));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;
  vector<int> s(n);
  for (int i = 0; i < n; i++) cin >> s[i];

  for (int t = 0; t < m; t++) {
    int a, b, c; cin >> a >> b >> c;
    b--;
    if (a == 1) {
      s[b] = c;
    } else {
      int r = c - 1;
      for (int i = b; i <= r; i++) {
        if (a == 2) s[i] = 1 - s[i];
        else if (a == 3) s[i] = 0;
        else s[i] = 1;
      }
    }
  }

  for (int i = 0; i < n; i++) {
    if (i) cout << " ";
    cout << s[i];
  }
  cout << "\n";

  return 0;
}
```
