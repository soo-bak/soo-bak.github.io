## 문제 링크
[32215번 - 코드마스터 2024](https://www.acmicpc.net/problem/32215)

## 설명
직접 설치한 횟수와 매크로용 노트북 설치 횟수의 합을 구하는 문제입니다.

<br>

## 접근법
도훈이는 여러 대의 컴퓨터에 각각 모든 프로그램을 직접 설치하고, 선생님은 노트북에 모든 프로그램을 한 번씩 설치합니다.

따라서 총 설치 횟수는 프로그램 수에 컴퓨터 대수와 노트북 1대를 더한 값을 곱하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = int.Parse(parts[0]);
    var m = int.Parse(parts[1]);
    var k = int.Parse(parts[2]);

    Console.WriteLine(m * (k + 1));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m, k; cin >> n >> m >> k;
  cout << m * (k + 1) << "\n";

  return 0;
}
```
