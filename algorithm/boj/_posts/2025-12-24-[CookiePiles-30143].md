## 문제 링크
[30143번 - Cookie Piles](https://www.acmicpc.net/problem/30143)

## 설명
최소 높이와 공차가 주어진 N개의 더미의 합을 계산하는 문제입니다.

<br>

## 접근법
각 더미의 높이가 공차만큼 증가하는 등차수열을 이룹니다.

따라서 등차수열의 합 공식을 적용하면 전체 쿠키 수를 바로 구할 수 있습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var i = 0; i < t; i++) {
      var parts = Console.ReadLine()!.Split();
      var n = int.Parse(parts[0]);
      var a = int.Parse(parts[1]);
      var d = int.Parse(parts[2]);

      var sum = n * (2 * a + (n - 1) * d) / 2;
      Console.WriteLine(sum);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    int n, a, d; cin >> n >> a >> d;
    int sum = n * (2 * a + (n - 1) * d) / 2;
    cout << sum << "\n";
  }

  return 0;
}
```
