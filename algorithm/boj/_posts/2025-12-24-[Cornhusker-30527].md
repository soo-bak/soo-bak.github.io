## 문제 링크
[30527번 - Cornhusker](https://www.acmicpc.net/problem/30527)

## 설명
다섯 개 옥수수 귀의 알 수를 평균낸 뒤 수확량을 정수로 계산하는 문제입니다.

<br>

## 접근법
각 옥수수 귀의 알 수는 줄 수와 줄당 알 수를 곱해서 구하고, 다섯 귀의 평균을 계산합니다.

이후 평균에 에이커당 개체 수를 곱하고 커널 무게 계수로 나눈 몫이 수확량입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var sum = 0;
    for (var i = 0; i < 5; i++) {
      var a = int.Parse(first[2 * i]);
      var l = int.Parse(first[2 * i + 1]);
      sum += a * l;
    }

    var second = Console.ReadLine()!.Split();
    var n = int.Parse(second[0]);
    var kwf = int.Parse(second[1]);

    var avg = sum / 5;
    var ans = (avg * n) / kwf;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int sum = 0;
  for (int i = 0; i < 5; i++) {
    int a, l; cin >> a >> l;
    sum += a * l;
  }
  int n, kwf; cin >> n >> kwf;

  int avg = sum / 5;
  int ans = (avg * n) / kwf;
  cout << ans << "\n";

  return 0;
}
```
