## 문제 링크
[21167번 - Curve Speed](https://www.acmicpc.net/problem/21167)

## 설명
반경 R과 superelevation S가 주어졌을 때 최대 속도를 계산해 반올림 출력하는 문제입니다.

<br>

## 접근법
주어진 공식에 반경과 경사를 대입하여 최대 속도를 계산합니다.

각 테스트케이스마다 계산 결과를 반올림해 정수로 출력하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var data = Console.In.ReadToEnd();
    var parts = data.Split();
    var idx = 0;

    while (idx + 1 < parts.Length) {
      var r = double.Parse(parts[idx++]);
      var s = double.Parse(parts[idx++]);
      var v = Math.Sqrt((r * (s + 0.16)) / 0.067);
      Console.WriteLine((int)Math.Round(v));
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  double r, s;
  while (cin >> r >> s) {
    double v = sqrt((r * (s + 0.16)) / 0.067);
    cout << (int)llround(v) << "\n";
  }

  return 0;
}
```
