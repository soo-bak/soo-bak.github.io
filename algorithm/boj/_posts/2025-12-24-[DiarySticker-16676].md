## 문제 링크
[16676번 - 근우의 다이어리 꾸미기](https://www.acmicpc.net/problem/16676)

## 설명
0부터 N까지 모든 수를 표현하려면 각 숫자 스티커를 몇 장씩 준비해야 하는지 구하는 문제입니다.

<br>

## 접근법
필요한 스티커 팩 수는 1~9 중 가장 많이 필요한 숫자의 개수와 같습니다.

0부터 N까지의 숫자 중 1~9가 각각 몇 번 등장하는지 생각하면, 최소 팩 수는 N의 자릿수와 관련됩니다.

결과적으로, N이 9, 99, 999처럼 모든 자릿수가 9인 경우는 자릿수와 같고, 그 외에는 자릿수 + 1이 필요합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = long.Parse(Console.ReadLine()!);
    if (n <= 9) {
      Console.WriteLine(1);
      return;
    }

    var s = n.ToString();
    var allNine = true;
    for (var i = 0; i < s.Length; i++) {
      if (s[i] != '9') { allNine = false; break; }
    }

    Console.WriteLine(allNine ? s.Length : s.Length + 1);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  long long n; cin >> n;
  if (n <= 9) {
    cout << 1 << "\n";
    return 0;
  }

  string s = to_string(n);
  bool allNine = true;
  for (char ch : s) {
    if (ch != '9') { allNine = false; break; }
  }

  cout << (allNine ? (int)s.size() : (int)s.size() + 1) << "\n";

  return 0;
}
```
