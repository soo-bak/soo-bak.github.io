## 문제 링크
[21756번 - 지우개](https://www.acmicpc.net/problem/21756)

## 설명
홀수 위치를 반복해서 지우고 남은 수를 왼쪽으로 모을 때 마지막으로 남는 값을 구하는 문제입니다.

<br>

## 접근법
작업 한 번마다 남는 값은 이전 값들의 2배가 됩니다.

따라서 마지막 값은 N 이하인 가장 큰 2의 거듭제곱입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var ans = 1;
    while (ans * 2 <= n) ans *= 2;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int ans = 1;
  while (ans * 2 <= n) ans *= 2;
  cout << ans << "\n";

  return 0;
}
```
