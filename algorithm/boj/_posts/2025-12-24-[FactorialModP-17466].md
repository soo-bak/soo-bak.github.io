## 문제 링크
[17466번 - N! mod P (1)](https://www.acmicpc.net/problem/17466)

## 설명
소수 P에 대해 N!을 P로 나눈 나머지를 구하는 문제입니다.

<br>

## 접근법
먼저 N부터 1까지 순서대로 곱하면서 각 단계마다 P로 나눈 나머지를 계산합니다.

다음으로 결과값을 P로 나눈 나머지가 정답입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = long.Parse(parts[0]);
    var p = long.Parse(parts[1]);

    var r = 1L;
    while (n > 0) {
      r = r * n % p;
      n--;
    }

    Console.WriteLine(r);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll n, p; cin >> n >> p;

  ll r = 1;
  while (n) {
    r = r * n % p;
    n--;
  }

  cout << r << "\n";

  return 0;
}
```
