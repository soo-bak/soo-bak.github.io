## 문제 링크
[16283번 - Farm](https://www.acmicpc.net/problem/16283)

## 설명
총 마리 수와 사료 소비량이 주어졌을 때 양과 염소의 수를 구하는 문제입니다.

<br>

## 접근법
양의 수를 x라 두면 염소의 수는 n-x입니다.

`a*x + b*(n-x) = w`를 만족하는 x를 모두 확인해 해가 유일한지 판단합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);
    var n = int.Parse(parts[2]);
    var w = int.Parse(parts[3]);

    var found = 0;
    var sheep = 0;
    var goat = 0;

    for (var x = 1; x <= n - 1; x++) {
      var y = n - x;
      if (a * x + b * y == w) {
        found++;
        sheep = x;
        goat = y;
      }
    }

    if (found == 1) Console.WriteLine($"{sheep} {goat}");
    else Console.WriteLine(-1);
  }
}
```
