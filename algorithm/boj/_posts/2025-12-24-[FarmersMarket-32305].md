## 문제 링크
[32305번 - Farmers’ Market](https://www.acmicpc.net/problem/32305)

## 설명
사과 개수를 12개 단위로 올림해 비용을 계산하는 문제입니다.

<br>

## 접근법
총 사과 수는 상자 수와 상자당 개수를 곱해서 구합니다.

이후 12개 단위로 올림한 개수에 단가를 곱하면 총 비용이 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);
    var d = int.Parse(Console.ReadLine()!);

    var total = a * b;
    var dozens = (total + 11) / 12;
    Console.WriteLine(dozens * d);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b; cin >> a >> b;
  int d; cin >> d;
  int total = a * b;
  int dozens = (total + 11) / 12;
  cout << dozens * d << "\n";

  return 0;
}
```
