## 문제 링크
[22864번 - 피로도](https://www.acmicpc.net/problem/22864)

## 설명
피로도 제한을 넘지 않도록 24시간 동안 일할 때 최대 처리량을 구하는 문제입니다.

<br>

## 접근법
매 시간마다 일할 수 있으면 일하고, 아니면 쉬어 피로도를 줄입니다.

24시간을 그대로 시뮬레이션하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);
    var c = int.Parse(parts[2]);
    var m = int.Parse(parts[3]);

    var fatigue = 0;
    var work = 0;
    for (var t = 0; t < 24; t++) {
      if (fatigue + a <= m) {
        fatigue += a;
        work += b;
      } else {
        fatigue -= c;
        if (fatigue < 0) fatigue = 0;
      }
    }

    Console.WriteLine(work);
  }
}
```
