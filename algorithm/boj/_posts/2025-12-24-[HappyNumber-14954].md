## 문제 링크
[14954번 - Happy Number](https://www.acmicpc.net/problem/14954)

## 설명
자릿수 제곱합을 반복했을 때 1에 도달하면 HAPPY, 아니면 UNHAPPY를 출력하는 문제입니다.

<br>

## 접근법
행복하지 않은 수는 4를 포함한 고정 사이클로 들어간다는 성질을 사용합니다.

n이 1이거나 4가 될 때까지 자릿수 제곱합을 반복하고 결과에 따라 판정합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static int Next(int n) {
    var sum = 0;
    while (n > 0) {
      var d = n % 10;
      sum += d * d;
      n /= 10;
    }
    return sum;
  }

  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    while (n != 1 && n != 4)
      n = Next(n);
      
    Console.WriteLine(n == 1 ? "HAPPY" : "UNHAPPY");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int nextVal(int n) {
  int sum = 0;
  while (n > 0) {
    int d = n % 10;
    sum += d * d;
    n /= 10;
  }
  return sum;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  while (n != 1 && n != 4)
    n = nextVal(n);

  cout << (n == 1 ? "HAPPY" : "UNHAPPY") << "\n";

  return 0;
}
```
