## 문제 링크
[29097번 - Короли](https://www.acmicpc.net/problem/29097)

## 설명
세 왕의 병력 수를 계산해 가장 큰 값을 가진 왕의 이름을 사전순으로 출력하는 문제입니다.

<br>

## 접근법
각 왕의 병력 수는 대대 수와 대대당 병사 수를 곱해서 구합니다.

이후 최대값을 찾고, 해당 값을 가진 왕의 이름을 사전순으로 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = int.Parse(parts[0]);
    var m = int.Parse(parts[1]);
    var k = int.Parse(parts[2]);
    var a = int.Parse(parts[3]);
    var b = int.Parse(parts[4]);
    var c = int.Parse(parts[5]);

    var j = n * a;
    var r = m * b;
    var s = k * c;
    var mx = Math.Max(j, Math.Max(r, s));

    var first = true;
    if (j == mx) { Console.Write("Joffrey"); first = false; }
    if (r == mx) {
      if (!first) Console.Write(" ");
      Console.Write("Robb");
      first = false;
    }
    if (s == mx) {
      if (!first) Console.Write(" ");
      Console.Write("Stannis");
    }
    Console.WriteLine();
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);

  int n, m, k, a, b, c; cin >> n >> m >> k >> a >> b >> c;

  int j = n * a;
  int r = m * b;
  int s = k * c;
  int mx = max(j, max(r, s));

  bool first = true;
  if (j == mx) { cout << "Joffrey"; first = false; }
  if (r == mx) {
    if (!first) cout << " ";
    cout << "Robb";
    first = false;
  }
  if (s == mx) {
    if (!first) cout << " ";
    cout << "Stannis";
  }
  cout << "\n";

  return 0;
}
```
