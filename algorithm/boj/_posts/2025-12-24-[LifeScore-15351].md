## 문제 링크
[15351번 - 인생 점수](https://www.acmicpc.net/problem/15351)

## 설명
각 문장을 알파벳 점수 합으로 환산해 100이면 PERFECT LIFE, 아니면 점수를 출력하는 문제입니다.

<br>

## 접근법
문장에 포함된 대문자 알파벳만 합산합니다.

합이 100이면 PERFECT LIFE를 출력하고, 아니면 합을 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();
    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!;
      var sum = 0;
      foreach (var ch in line) {
        if (ch == ' ') continue;
        sum += ch - 'A' + 1;
      }
      if (sum == 100) sb.AppendLine("PERFECT LIFE");
      else sb.AppendLine(sum.ToString());
    }
    Console.Write(sb.ToString());
  }
}
```

### C++

      sum += ch - 'A' + 1;

    if (sum == 100) cout << "PERFECT LIFE\n";

  }
