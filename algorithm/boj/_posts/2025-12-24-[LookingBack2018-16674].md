## 문제 링크
[16674번 - 2018년을 되돌아보며](https://www.acmicpc.net/problem/16674)

## 설명
정수가 2,0,1,8로만 이루어졌는지, 네 숫자를 모두 포함하는지, 개수가 같은지를 판별하는 문제입니다.

<br>

## 접근법
문자열로 읽어 각 자리 숫자를 확인합니다.

2,0,1,8 외의 숫자가 있으면 0을 출력합니다.

모두 포함하지 못하면 1, 모두 포함했지만 개수가 모두 같지 않으면 2, 모두 같으면 8을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var cnt2 = 0;
    var cnt0 = 0;
    var cnt1 = 0;
    var cnt8 = 0;

    foreach (var ch in s) {
      if (ch == '2') cnt2++;
      else if (ch == '0') cnt0++;
      else if (ch == '1') cnt1++;
      else if (ch == '8') cnt8++;
      else {
        Console.WriteLine(0);
        return;
      }
    }

    if (cnt2 == 0 || cnt0 == 0 || cnt1 == 0 || cnt8 == 0) {
      Console.WriteLine(1);
      return;
    }

    if (cnt2 == cnt0 && cnt0 == cnt1 && cnt1 == cnt8) Console.WriteLine(8);
    else Console.WriteLine(2);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  int c2 = 0, c0 = 0, c1 = 0, c8 = 0;
  for (char ch : s) {
    if (ch == '2') c2++;
    else if (ch == '0') c0++;
    else if (ch == '1') c1++;
    else if (ch == '8') c8++;
    else {
      cout << 0 << "\n";
      return 0;
    }
  }

  if (c2 == 0 || c0 == 0 || c1 == 0 || c8 == 0) {
    cout << 1 << "\n";
    return 0;
  }

  if (c2 == c0 && c0 == c1 && c1 == c8) cout << 8 << "\n";
  else cout << 2 << "\n";

  return 0;
}
```
