## 문제 링크
[32297번 - 문자열을 만들어요](https://www.acmicpc.net/problem/32297)

## 설명
주어진 문자열에 gori가 연속으로 포함되는지 판별하는 문제입니다.

<br>

## 접근법
문자열 내에 gori가 연속으로 포함되어 있는지 확인합니다.

포함되어 있으면 YES, 그렇지 않으면 NO를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;
    Console.WriteLine(s.Contains("gori") ? "YES" : "NO");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;
  cout << (s.find("gori") != string::npos ? "YES" : "NO") << "\n";

  return 0;
}
```
