## 문제 링크
[13366번 - Math Contest](https://www.acmicpc.net/problem/13366)

## 설명
주어진 수가 9의 배수인지 판별해 YES/NO를 출력하는 문제입니다.

<br>

## 접근법
어떤 수가 9의 배수인지 확인하려면 모든 자릿수의 합이 9의 배수인지 확인하면 됩니다.

입력 크기가 매우 크므로 문자열로 받아 자릿수 합을 구한 뒤 판별합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();
    for (var i = 0; i < t; i++) {
      var s = Console.ReadLine()!;
      var sum = 0;
      foreach (var ch in s)
        sum += ch - '0';
      sb.AppendLine(sum % 9 == 0 ? "YES" : "NO");
    }
    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    string s; cin >> s;
    int sum = 0;
    for (char ch : s) sum += ch - '0';
    cout << (sum % 9 == 0 ? "YES" : "NO") << "\n";
  }

  return 0;
}
```
