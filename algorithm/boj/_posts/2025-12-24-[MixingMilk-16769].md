## 문제 링크
[16769번 - Mixing Milk](https://www.acmicpc.net/problem/16769)

## 설명
세 양동이에 대해 1→2→3→1 순으로 100번 붓는 과정을 시뮬레이션하는 문제입니다.

<br>

## 접근법
각 양동이의 용량과 현재 양을 배열로 관리합니다.

100번 반복하며 현재 양동이에서 다음 양동이로 가능한 만큼 붓습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var c = new int[3];
    var m = new int[3];
    for (var i = 0; i < 3; i++) {
      var parts = Console.ReadLine()!.Split();
      c[i] = int.Parse(parts[0]);
      m[i] = int.Parse(parts[1]);
    }

    for (var t = 0; t < 100; t++) {
      var from = t % 3;
      var to = (t + 1) % 3;
      var pour = Math.Min(m[from], c[to] - m[to]);
      m[from] -= pour;
      m[to] += pour;
    }

    Console.WriteLine(m[0]);
    Console.WriteLine(m[1]);
    Console.WriteLine(m[2]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int c[3], m[3];
  for (int i = 0; i < 3; i++) {
    cin >> c[i] >> m[i];
  }

  for (int t = 0; t < 100; t++) {
    int from = t % 3;
    int to = (t + 1) % 3;
    int pour = min(m[from], c[to] - m[to]);
    m[from] -= pour;
    m[to] += pour;
  }

  cout << m[0] << "\n";
  cout << m[1] << "\n";
  cout << m[2] << "\n";

  return 0;
}
```
