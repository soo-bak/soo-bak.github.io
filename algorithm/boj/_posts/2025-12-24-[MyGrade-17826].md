## 문제 링크
[17826번 - 나의 학점은?](https://www.acmicpc.net/problem/17826)

## 설명
정렬된 점수 목록에서 본인의 순위를 찾아 학점을 출력하는 문제입니다.

<br>

## 접근법
50개의 점수에서 본인의 점수가 몇 등인지 찾습니다.

등수 구간에 맞춰 학점을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static string Grade(int rank) {
    if (rank <= 5) return "A+";
    if (rank <= 15) return "A0";
    if (rank <= 30) return "B+";
    if (rank <= 35) return "B0";
    if (rank <= 45) return "C+";
    if (rank <= 48) return "C0";
    return "F";
  }

  static void Main() {
    var scores = Console.ReadLine()!.Split();
    var me = Console.ReadLine()!;

    var rank = 0;
    for (var i = 0; i < 50; i++) {
      if (scores[i] == me) {
        rank = i + 1;
        break;
      }
    }

    Console.WriteLine(Grade(rank));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<string> vs;

string grade(int rank) {
  if (rank <= 5) return "A+";
  if (rank <= 15) return "A0";
  if (rank <= 30) return "B+";
  if (rank <= 35) return "B0";
  if (rank <= 45) return "C+";
  if (rank <= 48) return "C0";
  return "F";
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  vs scores(50);
  for (int i = 0; i < 50; i++) cin >> scores[i];
  string me; cin >> me;

  int rank = 0;
  for (int i = 0; i < 50; i++) {
    if (scores[i] == me) {
      rank = i + 1;
      break;
    }
  }

  cout << grade(rank) << "\n";

  return 0;
}
```
