## 문제 링크
[15813번 - 너의 이름은 몇 점이니?](https://www.acmicpc.net/problem/15813)

## 설명
대문자 이름의 알파벳 점수 합을 구하는 문제입니다.

<br>

## 접근법
문자마다 `A=1, ..., Z=26`으로 환산해 합산합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;

    var sum = 0;
    for (var i = 0; i < n; i++)
      sum += s[i] - 'A' + 1;

    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  int sum = 0;
  for (int i = 0; i < n; i++)
    sum += s[i] - 'A' + 1;

  cout << sum << "\n";

  return 0;
}
```
