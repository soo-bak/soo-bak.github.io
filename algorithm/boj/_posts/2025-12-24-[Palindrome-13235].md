## 문제 링크
[13235번 - 팰린드롬](https://www.acmicpc.net/problem/13235)

## 설명
입력 문자열이 팰린드롬인지 판단하는 문제입니다.

<br>

## 접근법
양 끝에서 가운데로 좁혀가며 문자를 비교합니다.

하나라도 다르면 false, 끝까지 같으면 true를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var l = 0;
    var r = s.Length - 1;

    var ok = true;
    while (l < r) {
      if (s[l] != s[r]) { ok = false; break; }
      l++;
      r--;
    }

    Console.WriteLine(ok ? "true" : "false");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  int l = 0, r = (int)s.size() - 1;
  bool ok = true;

  while (l < r) {
    if (s[l] != s[r]) { ok = false; break; }
    l++;
    r--;
  }

  cout << (ok ? "true" : "false") << "\n";

  return 0;
}
```
