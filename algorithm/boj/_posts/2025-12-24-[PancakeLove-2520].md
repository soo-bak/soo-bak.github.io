## 문제 링크
[2520번 - 팬케이크 사랑](https://www.acmicpc.net/problem/2520)

## 설명
반죽 재료와 토핑 재료로 만들 수 있는 팬케이크 최대 개수를 구하는 문제입니다.

<br>

## 접근법
반죽은 각 기본 재료를 비율로 나눈 값 중 최솟값만큼 만들 수 있습니다.

이후 토핑은 종류별로 만들 수 있는 개수를 합산하고, 최종 답은 반죽 수와 토핑 수 중 작은 값입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split(default(char[]), StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var t = int.Parse(parts[idx++]);

    for (var tc = 0; tc < t; tc++) {
      var cm = int.Parse(parts[idx++]);
      var y = int.Parse(parts[idx++]);
      var ssu = int.Parse(parts[idx++]);
      var ssa = int.Parse(parts[idx++]);
      var f = int.Parse(parts[idx++]);

      var b = int.Parse(parts[idx++]);
      var gs = int.Parse(parts[idx++]);
      var gc = int.Parse(parts[idx++]);
      var w = int.Parse(parts[idx++]);

      var batter = Math.Min(
        Math.Min(cm * 16 / 8, y * 16 / 8),
        Math.Min(ssu * 16 / 4, Math.Min(ssa * 16, f * 16 / 9))
      );

      var topping = b + gs / 30 + gc / 25 + w / 10;
      Console.WriteLine(Math.Min(batter, topping));
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    int cm, y, ssu, ssa, f; cin >> cm >> y >> ssu >> ssa >> f;
    int b, gs, gc, w; cin >> b >> gs >> gc >> w;

    int batter = min(
      min(cm * 16 / 8, y * 16 / 8),
      min(ssu * 16 / 4, min(ssa * 16, f * 16 / 9))
    );

    int topping = b + gs / 30 + gc / 25 + w / 10;
    cout << min(batter, topping) << "\n";
  }

  return 0;
}
```
