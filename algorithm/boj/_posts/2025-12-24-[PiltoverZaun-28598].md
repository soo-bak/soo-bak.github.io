## 문제 링크
[28598번 - Пилтовер и Заун](https://www.acmicpc.net/problem/28598)

## 설명
두 도시의 현재 상태와 연도 수가 주어졌을 때, 같은 시작값과 연도별 양의 변화로 도달 가능한지 판별하는 문제입니다.

<br>

## 접근법
두 도시의 상태 합이 짝수여야 시작값이 정수가 되고, 차이의 절반이 증가량 합이 됩니다.

증가량 합이 음수가 아니어야 하고, 연도 수만큼의 양의 정수를 더한 합 이상이어야 합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var x1 = int.Parse(parts[0]);
    var x2 = int.Parse(parts[1]);
    var n = int.Parse(parts[2]);

    if ((x1 + x2) % 2 != 0) {
      Console.WriteLine("NO");
      return;
    }

    var s = (x1 - x2) / 2;
    if (s < 0) Console.WriteLine("NO");
    else if (n == 0) Console.WriteLine(s == 0 ? "YES" : "NO");
    else Console.WriteLine(s >= n ? "YES" : "NO");
  }
}
```

### C++
