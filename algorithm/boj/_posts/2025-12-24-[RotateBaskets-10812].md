## 문제 링크
[10812번 - 바구니 순서 바꾸기](https://www.acmicpc.net/problem/10812)

## 설명
지정된 구간을 기준 바구니부터 시작하도록 회전시키는 문제입니다.

<br>

## 접근법
배열을 1부터 N까지 초기화합니다.

각 연산마다 지정된 구간을 기준 위치부터 끝까지, 그 다음 시작부터 기준 직전까지 순서로 재배치합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);

    var arr = new int[n];
    for (var i = 0; i < n; i++)
      arr[i] = i + 1;

    for (var t = 0; t < m; t++) {
      var parts = Console.ReadLine()!.Split();
      var i = int.Parse(parts[0]) - 1;
      var j = int.Parse(parts[1]) - 1;
      var k = int.Parse(parts[2]) - 1;

      var tmp = new int[j - i + 1];
      var idx = 0;
      for (var p = k; p <= j; p++)
        tmp[idx++] = arr[p];
      for (var p = i; p < k; p++)
        tmp[idx++] = arr[p];

      for (var p = 0; p < tmp.Length; p++)
        arr[i + p] = tmp[p];
    }

    Console.WriteLine(string.Join(" ", arr));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;
  vi arr(n);
  for (int i = 0; i < n; i++)
    arr[i] = i + 1;

  for (int t = 0; t < m; t++) {
    int i, j, k; cin >> i >> j >> k;
    i--; j--; k--;

    vi tmp;
    for (int p = k; p <= j; p++)
      tmp.push_back(arr[p]);
    for (int p = i; p < k; p++)
      tmp.push_back(arr[p]);

    for (int p = 0; p < (int)tmp.size(); p++)
      arr[i + p] = tmp[p];
  }

  for (int i = 0; i < n; i++) {
    if (i) cout << " ";
    cout << arr[i];
  }
  cout << "\n";

  return 0;
}
```
