## 문제 링크
[21734번 - SMUPC의 등장](https://www.acmicpc.net/problem/21734)

## 설명
문자열의 각 문자에 대해 ASCII 코드의 자릿수 합만큼 해당 문자를 반복 출력하는 문제입니다.

<br>

## 접근법
각 문자의 ASCII 값을 구한 뒤, 자릿수 합을 계산합니다.

계산한 횟수만큼 해당 문자를 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static int DigitSum(int v) {
    var sum = 0;
    while (v > 0) {
      sum += v % 10;
      v /= 10;
    }
    return sum;
  }

  static void Main() {
    var s = Console.ReadLine()!;
    var sb = new StringBuilder();
    foreach (var ch in s) {
      var cnt = DigitSum(ch);
      for (var i = 0; i < cnt; i++) sb.Append(ch);
      sb.AppendLine();
    }
    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int digitSum(int v) {
  int sum = 0;
  while (v > 0) {
    sum += v % 10;
    v /= 10;
  }
  return sum;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  for (char ch : s) {
    int cnt = digitSum((int)ch);
    for (int i = 0; i < cnt; i++) cout << ch;
    cout << "\n";
  }

  return 0;
}
```
