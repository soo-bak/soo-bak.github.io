## 문제 링크
[31394번 - Scholarship](https://www.acmicpc.net/problem/31394)

## 설명
시험 점수 목록이 주어질 때 장학금 종류를 출력하는 문제입니다.

<br>

## 접근법
3점이 하나라도 있으면 장학금을 받을 수 없고, 모든 점수가 5점이면 Named 장학금을 받습니다.

이외의 경우에는 평균이 4.5 이상이면 High, 그렇지 않으면 Common 장학금을 받습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sum = 0;
    var has3 = false;
    var all5 = true;

    for (var i = 0; i < n; i++) {
      var v = int.Parse(Console.ReadLine()!);
      sum += v;
      if (v == 3) has3 = true;
      if (v != 5) all5 = false;
    }

    if (has3) Console.WriteLine("None");
    else if (all5) Console.WriteLine("Named");
    else if (sum * 2 >= n * 9) Console.WriteLine("High");
    else Console.WriteLine("Common");
  }
}
```
