## 문제 링크
[30775번 - Рассадка](https://www.acmicpc.net/problem/30775)

## 설명
참가자 총합을 M개의 강의실에 1명 차이 이내로 배치할 때, 한 강의실에 들어갈 수 있는 최대 인원을 구하는 문제입니다.

<br>

## 접근법
컴팩트 배치는 인원 차이가 1 이하이므로, 한 강의실의 최대 인원은 총합을 강의실 수로 나눈 올림값입니다.

참가자 수를 모두 더한 뒤 올림 나눗셈으로 계산하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var m = long.Parse(first[0]);
    var k = int.Parse(first[1]);

    var parts = Console.ReadLine()!.Split();
    long sum = 0;
    for (var i = 0; i < k; i++)
      sum += long.Parse(parts[i]);

    var ans = (sum + m - 1) / m;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef vector<ll> vll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  ll m; int k; cin >> m >> k;
  vll n(k);
  for (int i = 0; i < k; i++)
    cin >> n[i];

  ll sum = 0;
  for (int i = 0; i < k; i++)
    sum += n[i];

  cout << (sum + m - 1) / m << "\n";

  return 0;
}
```
