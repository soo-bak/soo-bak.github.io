## 문제 링크
[28939번 - Шкаф для обуви](https://www.acmicpc.net/problem/28939)

## 설명
각 선반에 놓인 신발이 관찰된 조건을 만족하는지 확인해 불가능한 신발의 개수를 구하는 문제입니다.

<br>

## 접근법
선반 높이와 신발 크기 각각에서 키의 가능한 범위를 계산합니다.

두 범위가 겹치지 않으면 해당 신발은 놓일 수 없으므로 그 개수를 세면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var data = Console.In.ReadToEnd();
    var parts = data.Split();
    var idx = 0;

    var n = int.Parse(parts[idx++]);
    var k = int.Parse(parts[idx++]);
    var m1 = int.Parse(parts[idx++]);
    var m2 = int.Parse(parts[idx++]);

    var bad = 0;
    for (var i = 0; i < n; i++) {
      var h = int.Parse(parts[idx++]);
      var cnt = int.Parse(parts[idx++]);
      var hMax = h * k;

      for (var j = 0; j < cnt; j++) {
        var s = int.Parse(parts[idx++]);
        var low = Math.Max(h, s * m2);
        var high = Math.Min(hMax, s * m1);
        if (low > high) bad++;
      }
    }

    Console.WriteLine(bad);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, k, m1, m2; cin >> n >> k >> m1 >> m2;
  int bad = 0;

  for (int i = 0; i < n; i++) {
    int h, cnt; cin >> h >> cnt;
    int hMax = h * k;
    for (int j = 0; j < cnt; j++) {
      int s; cin >> s;
      int low = max(h, s * m2);
      int high = min(hMax, s * m1);
      if (low > high) bad++;
    }
  }

  cout << bad << "\n";

  return 0;
}
```
