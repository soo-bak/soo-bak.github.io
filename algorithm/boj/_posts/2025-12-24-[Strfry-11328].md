## 문제 링크
[11328번 - Strfry](https://www.acmicpc.net/problem/11328)

## 설명
두 문자열이 같은 문자 개수를 가지는지 확인하는 문제입니다.

<br>

## 접근법
각 문자열의 알파벳 빈도를 세어 비교합니다.

모든 문자의 개수가 같으면 Possible, 아니면 Impossible입니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();

    for (var tc = 0; tc < t; tc++) {
      var parts = Console.ReadLine()!.Split();
      var a = parts[0];
      var b = parts[1];

      var cnt = new int[26];
      foreach (var ch in a) cnt[ch - 'a']++;
      foreach (var ch in b) cnt[ch - 'a']--;

      var ok = true;
      for (var i = 0; i < 26; i++) {
        if (cnt[i] != 0) { ok = false; break; }
      }

      sb.AppendLine(ok ? "Possible" : "Impossible");
    }

    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    string a, b; cin >> a >> b;
    int cnt[26] = {};
    for (char ch : a) cnt[ch - 'a']++;
    for (char ch : b) cnt[ch - 'a']--;

    bool ok = true;
    for (int i = 0; i < 26; i++) {
      if (cnt[i] != 0) { ok = false; break; }
    }

    cout << (ok ? "Possible" : "Impossible") << "\n";
  }

  return 0;
}
```
