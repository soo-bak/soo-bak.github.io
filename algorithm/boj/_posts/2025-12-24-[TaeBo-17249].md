## 문제 링크
[17249번 - 태보태보 총난타](https://www.acmicpc.net/problem/17249)

## 설명
얼굴 문자열을 기준으로 왼쪽과 오른쪽의 @ 개수를 출력하는 문제입니다.

<br>

## 접근법
문자열을 (^0^) 기준으로 나눕니다.

왼쪽과 오른쪽에서 @의 개수를 각각 세어 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var parts = s.Split(new[] { "(^0^)" }, StringSplitOptions.None);
    var left = parts[0];
    var right = parts[1];

    var lcnt = 0;
    foreach (var ch in left) {
      if (ch == '@') lcnt++;
    }
    var rcnt = 0;
    foreach (var ch in right) {
      if (ch == '@') rcnt++;
    }

    Console.WriteLine($"{lcnt} {rcnt}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  string face = "(^0^)";
  int pos = s.find(face);
  string left = s.substr(0, pos);
  string right = s.substr(pos + face.size());

  int lcnt = 0, rcnt = 0;
  for (char ch : left) {
    if (ch == '@') lcnt++;
  }
  for (char ch : right) {
    if (ch == '@') rcnt++;
  }

  cout << lcnt << " " << rcnt << "\n";

  return 0;
}
```
