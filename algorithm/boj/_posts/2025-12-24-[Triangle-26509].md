## 문제 링크
[26509번 - Triangle](https://www.acmicpc.net/problem/26509)

## 설명
직사각형을 한 번 잘라 얻을 수 있는 두 삼각형인지 판별하는 문제입니다.

<br>

## 접근법
직사각형을 한 번 잘라 두 삼각형이 되려면 대각선으로 잘라야 하므로, 두 삼각형은 서로 합동인 직각삼각형이어야 합니다.

각 삼각형의 세 변을 정렬해 직각 여부를 확인하고, 두 삼각형의 변 구성이 동일한지 비교합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static bool IsRight(int[] v) {
    Array.Sort(v);
    return v[0] * v[0] + v[1] * v[1] == v[2] * v[2];
  }

  static void Main() {
    var data = Console.In.ReadToEnd();
    var parts = data.Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);

    for (var tc = 0; tc < t; tc++) {
      var a = new int[3];
      var b = new int[3];
      for (var i = 0; i < 3; i++) a[i] = int.Parse(parts[idx++]);
      for (var i = 0; i < 3; i++) b[i] = int.Parse(parts[idx++]);

      var okA = IsRight(a);
      var okB = IsRight(b);
      var same = a[0] == b[0] && a[1] == b[1] && a[2] == b[2];

      Console.WriteLine(okA && okB && same ? "YES" : "NO");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

bool isRight(array<int, 3>& v) {
  sort(v.begin(), v.end());
  return v[0] * v[0] + v[1] * v[1] == v[2] * v[2];
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    array<int, 3> a, b;
    for (int i = 0; i < 3; i++) cin >> a[i];
    for (int i = 0; i < 3; i++) cin >> b[i];

    bool okA = isRight(a);
    bool okB = isRight(b);
    bool same = (a == b);

    cout << (okA && okB && same ? "YES" : "NO") << "\n";
  }

  return 0;
}
```
