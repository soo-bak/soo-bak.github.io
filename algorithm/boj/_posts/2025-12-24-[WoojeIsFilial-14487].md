## 문제 링크
[14487번 - 욱제는 효도쟁이야!!](https://www.acmicpc.net/problem/14487)

## 설명
원형으로 연결된 마을을 모두 방문할 때 필요한 최소 이동비용을 구하는 문제입니다.

<br>

## 접근법
모든 마을을 방문하려면 원형 경로에서 한 구간만 제외한 경로를 따라가면 됩니다.

따라서 전체 이동비용 합에서 가장 큰 구간 하나를 빼는 값이 최소가 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Console.ReadLine()!.Split();

    var sum = 0;
    var max = 0;
    for (var i = 0; i < n; i++) {
      var v = int.Parse(parts[i]);
      sum += v;
      if (v > max) max = v;
    }

    Console.WriteLine(sum - max);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int sum = 0, mx = 0;
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    sum += v;
    if (v > mx) mx = v;
  }

  cout << sum - mx << "\n";

  return 0;
}
```
