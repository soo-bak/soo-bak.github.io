## 문제 링크
[18238번 - ZOAC 2](https://www.acmicpc.net/problem/18238)

## 설명
원판이 A에서 시작할 때, 목표 문자열을 앞에서부터 출력하기 위한 최소 회전 시간을 계산하는 문제입니다.

<br>

## 접근법
현재 위치와 다음 문자 사이의 시계방향/반시계방향 거리 중 작은 값을 더합니다.

문자 위치는 A를 0으로 두고 0~25로 변환해 계산합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var cur = 0;
    var total = 0;
    foreach (var ch in s) {
      var next = ch - 'A';
      var diff = Math.Abs(next - cur);
      total += Math.Min(diff, 26 - diff);
      cur = next;
    }
    Console.WriteLine(total);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  int cur = 0;
  int total = 0;
  for (char ch : s) {
    int next = ch - 'A';
    int diff = abs(next - cur);
    total += min(diff, 26 - diff);
    cur = next;
  }

  cout << total << "\n";

  return 0;
}
```
