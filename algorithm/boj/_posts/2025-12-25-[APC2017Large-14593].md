## 문제 링크
[14593번 - 2017 아주대학교 프로그래밍 경시대회 (Large)](https://www.acmicpc.net/problem/14593)

## 설명
점수 합이 크고, 제출 횟수가 적고, 마지막 업로드 시간이 빠른 참가자가 높은 순위를 차지합니다.  
이 기준으로 1등 참가자의 번호를 출력하는 문제입니다.

<br>

## 접근법
참가자를 순서대로 보면서 현재 1등과 비교합니다.  
점수는 큰 값 우선, 제출 횟수는 작은 값 우선, 마지막 업로드 시간은 작은 값 우선으로 갱신합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);

    var bestIdx = 1;
    var bestS = -1;
    var bestC = int.MaxValue;
    var bestL = int.MaxValue;

    for (var i = 1; i <= n; i++) {
      var parts = Console.ReadLine()!.Split();
      var s = int.Parse(parts[0]);
      var c = int.Parse(parts[1]);
      var l = int.Parse(parts[2]);

      var better = false;
      if (s > bestS) better = true;
      else if (s == bestS) {
        if (c < bestC) better = true;
        else if (c == bestC && l < bestL) better = true;
      }

      if (better) {
        bestIdx = i;
        bestS = s;
        bestC = c;
        bestL = l;
      }
    }

    Console.WriteLine(bestIdx);
  }
}
```
