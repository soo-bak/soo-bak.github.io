## 문제 링크
[10420번 - 기념일 1](https://www.acmicpc.net/problem/10420)

## 설명
기준 날짜 2014-04-02에서 시작해 N일째 날짜를 구하는 문제입니다.

<br>

## 접근법
N이 1이면 그대로 기준 날짜가 정답이므로, 날짜를 하루씩 N-1번 증가시키면 됩니다.

윤년 규칙을 이용해 해당 월의 날짜 수를 계산하고, 날짜가 넘치면 월과 연도를 순서대로 갱신합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static bool IsLeap(int y) {
    if (y % 400 == 0) return true;
    if (y % 100 == 0) return false;
    return y % 4 == 0;
  }

  static int DaysInMonth(int y, int m) {
    var days = new[] { 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31 };
    if (m == 2 && IsLeap(y)) return 29;
    return days[m - 1];
  }

  static void Main() {
    var n = int.Parse(Console.ReadLine()!);

    var year = 2014;
    var month = 4;
    var day = 2;

    for (var i = 1; i < n; i++) {
      day++;
      if (day > DaysInMonth(year, month)) {
        day = 1;
        month++;
        if (month == 13) {
          month = 1;
          year++;
        }
      }
    }

    Console.WriteLine($"{year:D4}-{month:D2}-{day:D2}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

bool isLeap(int y) {
  if (y % 400 == 0) return true;
  if (y % 100 == 0) return false;
  return y % 4 == 0;
}

int daysInMonth(int y, int m) {
  int days[12] = {31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
  if (m == 2 && isLeap(y)) return 29;
  return days[m - 1];
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int year = 2014, month = 4, day = 2;

  for (int i = 1; i < n; i++) {
    day++;
    if (day > daysInMonth(year, month)) {
      day = 1;
      month++;
      if (month == 13) {
        month = 1;
        year++;
      }
    }
  }

  cout << setw(4) << setfill('0') << year << "-"
       << setw(2) << setfill('0') << month << "-"
       << setw(2) << setfill('0') << day << "\n";

  return 0;
}
```
