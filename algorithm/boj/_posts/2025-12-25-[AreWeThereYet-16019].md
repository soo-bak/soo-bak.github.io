## 문제 링크
[16019번 - Are we there yet?](https://www.acmicpc.net/problem/16019)

## 설명
연속한 도시 사이 거리로부터 모든 도시 쌍의 거리를 표 형태로 출력하는 문제입니다.

<br>

## 접근법
도시 1을 기준으로 누적 거리를 만들고, 두 도시 사이 거리는 누적 거리의 차로 구합니다.  
각 행마다 1~5번 도시까지의 거리를 순서대로 출력하면 됩니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var d = new int[4];
    for (var i = 0; i < 4; i++) d[i] = int.Parse(parts[i]);

    var pos = new int[5];
    for (var i = 1; i < 5; i++)
      pos[i] = pos[i - 1] + d[i - 1];

    var sb = new StringBuilder();
    for (var i = 0; i < 5; i++) {
      for (var j = 0; j < 5; j++) {
        if (j > 0) sb.Append(' ');
        sb.Append(Math.Abs(pos[i] - pos[j]));
      }
      sb.AppendLine();
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int d[4];
  for (int i = 0; i < 4; i++) cin >> d[i];

  int pos[5] = {};
  for (int i = 1; i < 5; i++)
    pos[i] = pos[i - 1] + d[i - 1];

  for (int i = 0; i < 5; i++) {
    for (int j = 0; j < 5; j++) {
      if (j > 0) cout << ' ';
      cout << abs(pos[i] - pos[j]);
    }
    cout << "\n";
  }

  return 0;
}
```
