## 문제 링크
[15449번 - Art](https://www.acmicpc.net/problem/15449)

## 설명
다섯 개의 막대 중 세 개를 골라 삼각형이 되는 경우의 수를 구하는 문제입니다.

<br>

## 접근법
세 막대를 고르는 모든 조합을 확인하고, 두 길이의 합이 가장 긴 길이보다 큰 경우만 셉니다.  
총 조합 수가 10개뿐이라 전부 확인해도 충분합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = new int[5];
    for (var i = 0; i < 5; i++) a[i] = int.Parse(parts[i]);

    var cnt = 0;
    for (var i = 0; i < 5; i++) {
      for (var j = i + 1; j < 5; j++) {
        for (var k = j + 1; k < 5; k++) {
          var x = a[i];
          var y = a[j];
          var z = a[k];
          var max = Math.Max(x, Math.Max(y, z));
          var sum = x + y + z - max;
          if (sum > max) cnt++;
        }
      }
    }

    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a[5];
  for (int i = 0; i < 5; i++) cin >> a[i];

  int cnt = 0;
  for (int i = 0; i < 5; i++) {
    for (int j = i + 1; j < 5; j++) {
      for (int k = j + 1; k < 5; k++) {
        int x = a[i], y = a[j], z = a[k];
        int mx = max(x, max(y, z));
        int sum = x + y + z - mx;
        if (sum > mx) cnt++;
      }
    }
  }

  cout << cnt << "\n";

  return 0;
}
```
