## 문제 링크
[6246번 - 풍선 놀이](https://www.acmicpc.net/problem/6246)

## 설명
지정한 시작 슬롯과 간격으로 풍선을 꽂은 뒤, 비어 있는 슬롯의 개수를 구하는 문제입니다.

<br>

## 접근법
슬롯마다 풍선이 꽂혔는지 기록하는 배열을 준비합니다.

각 규칙마다 시작 슬롯부터 간격만큼 증가시키며 표시하고, 마지막에 표시되지 않은 슬롯을 세면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = int.Parse(parts[0]);
    var q = int.Parse(parts[1]);
    var used = new bool[n + 1];

    for (var i = 0; i < q; i++) {
      var rule = Console.ReadLine()!.Split();
      var l = int.Parse(rule[0]);
      var step = int.Parse(rule[1]);
      for (var pos = l; pos <= n; pos += step)
        used[pos] = true;
    }

    var empty = 0;
    for (var i = 1; i <= n; i++) {
      if (!used[i]) empty++;
    }

    Console.WriteLine(empty);
  }
}
```

### C++

  for (int i = 1; i <= n; i++) {

  }
