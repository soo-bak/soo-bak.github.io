## 문제 링크
[11213번 - Black Friday](https://www.acmicpc.net/problem/11213)

## 설명
주사위 결과 중에서 유일하게 가장 큰 값을 가진 참가자의 번호를 출력하는 문제입니다.

<br>

## 접근법
1~6의 등장 횟수를 센 뒤, 가장 큰 값부터 유일하게 한 번만 나온 값을 찾습니다.  
그 값을 가진 참가자의 인덱스를 출력하고, 없으면 none을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Console.ReadLine()!.Split();
    var rolls = new int[n];
    var cnt = new int[7];

    for (var i = 0; i < n; i++) {
      rolls[i] = int.Parse(parts[i]);
      cnt[rolls[i]]++;
    }

    var target = -1;
    for (var v = 6; v >= 1; v--) {
      if (cnt[v] == 1) { target = v; break; }
    }

    if (target == -1) {
      Console.WriteLine("none");
      return;
    }

    for (var i = 0; i < n; i++) {
      if (rolls[i] == target) {
        Console.WriteLine(i + 1);
        return;
      }
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vector<int> rolls(n);
  int cnt[7] = {};
  for (int i = 0; i < n; i++) {
    cin >> rolls[i];
    cnt[rolls[i]]++;
  }

  int target = -1;
  for (int v = 6; v >= 1; v--) {
    if (cnt[v] == 1) { target = v; break; }
  }

  if (target == -1) {
    cout << "none\n";
    return 0;
  }

  for (int i = 0; i < n; i++) {
    if (rolls[i] == target) {
      cout << i + 1 << "\n";
      return 0;
    }
  }
}
```
