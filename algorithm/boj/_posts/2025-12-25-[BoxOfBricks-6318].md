## 문제 링크
[6318번 - Box of Bricks](https://www.acmicpc.net/problem/6318)

## 설명
여러 스택을 같은 높이로 만들 때 필요한 최소 이동 횟수를 구하는 문제입니다.

<br>

## 접근법
전체 벽돌 수는 고정이므로 목표 높이는 전체 합을 스택 수로 나눈 값입니다.  
목표보다 높은 스택에서 초과분만 옮기면 되며, 그 초과분의 합이 최소 이동 횟수입니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var sb = new StringBuilder();
    var caseNo = 1;

    while (true) {
      var line = Console.ReadLine();
      if (line == null) break;
      var n = int.Parse(line);
      if (n == 0) break;

      var parts = Console.ReadLine()!.Split();
      var h = new int[n];
      var sum = 0;
      for (var i = 0; i < n; i++) {
        h[i] = int.Parse(parts[i]);
        sum += h[i];
      }
      var avg = sum / n;

      var moves = 0;
      for (var i = 0; i < n; i++)
        if (h[i] > avg) moves += h[i] - avg;

      sb.AppendLine($"Set #{caseNo}");
      sb.AppendLine($"The minimum number of moves is {moves}.");
      sb.AppendLine();
      caseNo++;
    }

    Console.Write(sb);
  }
}
```

  while (cin >> n) {

    vector<int> h(n);
    int sum = 0;
    for (int i = 0; i < n; i++) {
      cin >> h[i];
      sum += h[i];
    }
    int avg = sum / n;

    int moves = 0;
    for (int i = 0; i < n; i++)
      if (h[i] > avg) moves += h[i] - avg;

    cout << "Set #" << caseNo << "\n";
    cout << "The minimum number of moves is " << moves << ".\n\n";
    caseNo++;
  }

  return 0;
}
