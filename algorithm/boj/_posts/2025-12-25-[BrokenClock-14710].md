## 문제 링크
[14710번 - 고장난 시계](https://www.acmicpc.net/problem/14710)

## 설명
시침 각도와 분침 각도가 주어졌을 때 정상적인 아날로그 시계의 시간으로 만들 수 있는지 판별하는 문제입니다.

<br>

## 접근법
시간이 t분일 때 분침 각도는 6t, 시침 각도는 0.5t입니다. 분침 각도 θ2가 주어지면 t = θ2/6 + 60h (h=0..11)로만 가능합니다. 이를 시침 각도 θ1에 대입해 12θ1 = θ2 + 360h가 성립하는 h가 있는지 확인합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);

    var ok = false;
    for (var h = 0; h < 12; h++) {
      if (12 * a == b + 360 * h) {
        ok = true;
        break;
      }
    }

    Console.WriteLine(ok ? "O" : "X");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>

```
