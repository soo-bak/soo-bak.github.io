## 문제 링크
[26529번 - Bunnies](https://www.acmicpc.net/problem/26529)

## 설명
개월 수가 주어질 때 F(0)=1, F(1)=1인 피보나치 수열로 토끼 수를 출력하는 문제입니다.

<br>

## 접근법
최대 45까지이므로 미리 피보나치 값을 배열로 계산해 두고 바로 출력합니다.  
F(0)=1, F(1)=1 기준으로 누적합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);

    var fib = new int[46];
    fib[0] = 1;
    fib[1] = 1;
    for (var i = 2; i <= 45; i++)
      fib[i] = fib[i - 1] + fib[i - 2];

    var sb = new StringBuilder();
    for (var i = 0; i < t; i++) {
      var x = int.Parse(parts[idx++]);
      sb.AppendLine(fib[x].ToString());
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  int fib[46];
  fib[0] = 1;
  fib[1] = 1;
  for (int i = 2; i <= 45; i++)
    fib[i] = fib[i - 1] + fib[i - 2];

  while (t--) {
    int x; cin >> x;
    cout << fib[x] << "\n";
  }

  return 0;
}
```
