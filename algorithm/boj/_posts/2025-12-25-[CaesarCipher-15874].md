## 문제 링크
[15874번 - Caesar Cipher](https://www.acmicpc.net/problem/15874)

## 설명
문자열 s에서 알파벳만 k만큼 밀어 암호문을 만들고, 대소문자와 공백/문장부호는 그대로 유지하는 문제입니다.

<br>

## 접근법
k를 26으로 나눈 나머지만큼만 회전하면 됩니다.  
각 문자를 확인해 대문자/소문자 범위일 때만 회전시키고, 나머지는 그대로 둡니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var k = int.Parse(first[0]);
    var s = Console.ReadLine()!;
    var shift = k % 26;

    var arr = s.ToCharArray();
    for (var i = 0; i < arr.Length; i++) {
      var c = arr[i];
      if (c >= 'A' && c <= 'Z') arr[i] = (char)('A' + (c - 'A' + shift) % 26);
      else if (c >= 'a' && c <= 'z') arr[i] = (char)('a' + (c - 'a' + shift) % 26);
    }

    Console.WriteLine(new string(arr));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int k, n; cin >> k >> n;
  string s;
  getline(cin, s);
  getline(cin, s);
  int shift = k % 26;

  for (char& c : s) {
    if ('A' <= c && c <= 'Z') c = char('A' + (c - 'A' + shift) % 26);
    else if ('a' <= c && c <= 'z') c = char('a' + (c - 'a' + shift) % 26);
  }

  cout << s << "\n";

  return 0;
}
```
