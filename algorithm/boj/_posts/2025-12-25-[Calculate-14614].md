## 문제 링크
[14614번 - Calculate!](https://www.acmicpc.net/problem/14614)

## 설명
A에 B를 C번 XOR한 결과를 출력하는 문제입니다.

<br>

## 접근법
XOR은 같은 값을 두 번 적용하면 원래 값으로 돌아옵니다.  
따라서 C가 짝수면 결과는 A, 홀수면 A XOR B입니다.  
C는 매우 큰 수이므로 문자열로 읽고 마지막 자리의 홀짝만 확인합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);
    var c = parts[2];

    var last = c[c.Length - 1] - '0';
    var result = (last % 2 == 0) ? a : (a ^ b);
    Console.WriteLine(result);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b;
  string c;
  cin >> a >> b >> c;

  int last = c.back() - '0';
  int result = (last % 2 == 0) ? a : (a ^ b);
  cout << result << "\n";

  return 0;
}
```
