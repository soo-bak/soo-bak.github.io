## 문제 링크
[32141번 - 카드 게임 (Easy)](https://www.acmicpc.net/problem/32141)

## 설명
정렬된 공격력을 가진 카드로 상대를 처치할 때, 사용할 수 있는 카드의 최대 개수를 구하는 문제입니다.

<br>

## 접근법
약한 카드부터 순서대로 사용하면 사용 개수를 최대화할 수 있습니다.  
누적 합이 H 이상이 되는 순간의 카드 개수를 출력하고, 끝까지 부족하면 -1을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var h = int.Parse(parts[idx++]);

    var sum = 0;
    var cnt = 0;
    for (var i = 0; i < n; i++) {
      sum += int.Parse(parts[idx++]);
      cnt++;
      if (sum >= h) {
        Console.WriteLine(cnt);
        return;
      }
    }

    Console.WriteLine(-1);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, h; cin >> n >> h;
  int sum = 0;
  for (int i = 0; i < n; i++) {
    int d; cin >> d;
    sum += d;
    if (sum >= h) {
      cout << i + 1 << "\n";
      return 0;
    }
  }

  cout << -1 << "\n";

  return 0;
}
```
