## 문제 링크
[9947번 - Coin tossing](https://www.acmicpc.net/problem/9947)

## 설명
각 게임에서 호출과 실제 결과가 주어질 때, 두 사람의 점수를 계산해 출력하는 문제입니다.

<br>

## 접근법
먼저 게임 이름을 읽고 종료 표식이면 입력을 끝냅니다.

다음으로 각 기록에서 호출과 결과가 같은지 확인해 첫 번째 사람의 점수를 셉니다.

이후 전체 횟수에서 첫 번째 점수를 제외한 값이 두 번째 사람의 점수입니다.

마지막으로 요구 형식에 맞춰 이름과 점수를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split((char[])null, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var sb = new StringBuilder();

    while (idx < parts.Length) {
      var name1 = parts[idx++];
      var name2 = parts[idx++];
      if (name1 == "#" && name2 == "#") break;

      var n = int.Parse(parts[idx++]);
      var score1 = 0;
      for (var i = 0; i < n; i++) {
        var call = parts[idx++];
        var res = parts[idx++];
        if (call == res) score1++;
      }

      var score2 = n - score1;
      sb.Append(name1).Append(' ').Append(score1).Append(' ')
        .Append(name2).Append(' ').Append(score2).AppendLine();
    }

    Console.Write(sb);
  }
}
```

### C++

#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);

  string name1, name2;
  while (cin >> name1 >> name2) {
    if (name1 == "#" && name2 == "#") break;

    int n; cin >> n;
    int score1 = 0;
    for (int i = 0; i < n; i++) {
      string call, res;
      cin >> call >> res;
      if (call == res) score1++;
    }

    int score2 = n - score1;
    cout << name1 << " " << score1 << " " << name2 << " " << score2 << "\n";
  }

  return 0;
}
