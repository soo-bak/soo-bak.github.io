## 문제 링크
[31945번 - 정육면체의 네 꼭짓점](https://www.acmicpc.net/problem/31945)

## 설명
정육면체의 네 꼭짓점 번호가 주어질 때, 같은 면 위에 있는지 판별하는 문제입니다.

<br>

## 접근법
각 꼭짓점 번호는 좌표의 3비트 표현과 대응됩니다.  
세 비트 중 하나가 네 꼭짓점 모두에서 같다면 같은 면에 있습니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);

    var sb = new StringBuilder();
    for (var tc = 0; tc < t; tc++) {
      var a = int.Parse(parts[idx++]);
      var b = int.Parse(parts[idx++]);
      var c = int.Parse(parts[idx++]);
      var d = int.Parse(parts[idx++]);

      var ok = false;
      for (var bit = 0; bit < 3; bit++) {
        var v = (a >> bit) & 1;
        if (v == ((b >> bit) & 1) && v == ((c >> bit) & 1) && v == ((d >> bit) & 1)) {
          ok = true;
          break;
        }
      }

      sb.AppendLine(ok ? "YES" : "NO");
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    int a, b, c, d; cin >> a >> b >> c >> d;
    bool ok = false;
    for (int bit = 0; bit < 3; bit++) {
      int v = (a >> bit) & 1;
      if (v == ((b >> bit) & 1) && v == ((c >> bit) & 1) && v == ((d >> bit) & 1)) {
        ok = true;
        break;
      }
    }
    cout << (ok ? "YES" : "NO") << "\n";
  }

  return 0;
}
```
