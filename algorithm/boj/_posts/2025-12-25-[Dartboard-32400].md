## 문제 링크
[32400번 - 다트판](https://www.acmicpc.net/problem/32400)

## 설명
다트판 점수 배열이 주어질 때 Alice와 Bob의 기대 점수를 비교하는 문제입니다.

<br>

## 접근법
Alice는 20점 영역과 양옆 두 영역의 평균을, Bob은 전체 20개 점수의 평균을 사용합니다.  

두 평균을 비교해 더 큰 쪽의 이름을 출력하고, 같으면 Tie를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var scores = new int[20];
    var idx = 0;
    for (var i = 0; i < 20; i++)
      scores[i] = int.Parse(parts[idx++]);

    var pos20 = 0;
    for (var i = 0; i < 20; i++)
      if (scores[i] == 20) pos20 = i;

    var left = (pos20 + 19) % 20;
    var right = (pos20 + 1) % 20;
    var aliceSum = scores[left] + scores[pos20] + scores[right];

    var total = 0;
    for (var i = 0; i < 20; i++) total += scores[i];

    if (aliceSum * 20 > total * 3) Console.WriteLine("Alice");
    else if (aliceSum * 20 < total * 3) Console.WriteLine("Bob");
    else Console.WriteLine("Tie");
  }
}
```

  int aliceSum = scores[left] + scores[pos20] + scores[right];

  int total = 0;
  for (int i = 0; i < 20; i++) total += scores[i];
