## 문제 링크
[10643번 - FUNGHI](https://www.acmicpc.net/problem/10643)

## 설명
원형으로 배치된 8조각 피자에서 연속한 4조각의 합이 최대가 되는 값을 구하는 문제입니다.

<br>

## 접근법
처음 4조각의 합을 만든 뒤 한 칸씩 이동하며 윈도우 합을 갱신합니다.  
마지막 조각 다음은 다시 처음으로 돌아가므로 인덱스는 `mod 8`로 처리합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var a = new int[8];
    for (var i = 0; i < 8; i++)
      a[i] = int.Parse(Console.ReadLine()!);

    var sum = 0;
    for (var i = 0; i < 4; i++) sum += a[i];
    var best = sum;

    for (var i = 1; i < 8; i++) {
      sum = sum - a[i - 1] + a[(i + 3) % 8];
      if (sum > best) best = sum;
    }

    Console.WriteLine(best);
  }
}
```

### C++
```cpp

  int best = sum;

    sum = sum - a[i - 1] + a[(i + 3) % 8];
    if (sum > best) best = sum;
  }

  return 0;
}
```
