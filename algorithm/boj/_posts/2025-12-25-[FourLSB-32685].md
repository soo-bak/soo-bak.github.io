## 문제 링크
[32685번 - 4-LSB](https://www.acmicpc.net/problem/32685)

## 설명
세 정수의 하위 4비트를 순서대로 이어 붙여 만든 값을 4자리로 출력하는 문제입니다.

<br>

## 접근법
각 수에서 하위 4비트를 꺼내고, 순서대로 이어 붙인 값이 비밀번호입니다.  
즉 첫 값의 4비트가 가장 앞에 오므로 `(a & 15) << 8`, 두 번째는 `(b & 15) << 4`, 세 번째는 `(c & 15)`로 합칩니다.  
결과를 4자리로 맞춰 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var a = int.Parse(Console.ReadLine()!);
    var b = int.Parse(Console.ReadLine()!);
    var c = int.Parse(Console.ReadLine()!);

    var pass = ((a & 15) << 8) | ((b & 15) << 4) | (c & 15);
    Console.WriteLine(pass.ToString("D4"));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, c; cin >> a >> b >> c;
  int pass = ((a & 15) << 8) | ((b & 15) << 4) | (c & 15);
  cout << setw(4) << setfill('0') << pass << "\n";

  return 0;
}
```
