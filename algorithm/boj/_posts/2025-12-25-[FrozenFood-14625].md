## 문제 링크
[14625번 - 냉동식품](https://www.acmicpc.net/problem/14625)

## 설명
시작 시각부터 종료 시각까지의 모든 분에 대해, 시각 표시(HH:MM)에 숫자 N이 포함된 횟수를 세는 문제입니다.

<br>

## 접근법
시각을 분 단위로 변환해 시작부터 종료까지 순회합니다.  
각 분마다 시와 분의 두 자리 숫자에서 N이 하나라도 있으면 카운트합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var sh = int.Parse(first[0]);
    var sm = int.Parse(first[1]);

    var second = Console.ReadLine()!.Split();
    var eh = int.Parse(second[0]);
    var em = int.Parse(second[1]);

    var n = int.Parse(Console.ReadLine()!);

    var start = sh * 60 + sm;
    var end = eh * 60 + em;

    var cnt = 0;
    for (var t = start; t <= end; t++) {
      var h = t / 60;
      var m = t % 60;
      if (h / 10 == n || h % 10 == n || m / 10 == n || m % 10 == n)
        cnt++;
    }

    Console.WriteLine(cnt);
  }
}
```
