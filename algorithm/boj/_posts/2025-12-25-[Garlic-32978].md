## 문제 링크
[32978번 - 아 맞다 마늘](https://www.acmicpc.net/problem/32978)

## 설명
전체 재료 목록과 실제 사용한 재료 목록이 주어질 때 빠진 재료를 찾는 문제입니다.

<br>

## 접근법
먼저 전체 재료를 집합에 저장합니다.

다음으로 사용한 재료를 집합에서 제거합니다.

이후 집합에 남은 하나가 빠뜨린 재료입니다.

마지막으로 그 재료를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var n = int.Parse(parts[idx++]);

    var set = new HashSet<string>();
    for (var i = 0; i < n; i++) set.Add(parts[idx++]);
    for (var i = 0; i < n - 1; i++) set.Remove(parts[idx++]);

    foreach (var item in set) {
      Console.WriteLine(item);
      break;
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;

  unordered_set<string> s;
  s.reserve(n * 2);
  for (int i = 0; i < n; i++) {
    string x; cin >> x;
    s.insert(x);
  }
  for (int i = 0; i < n - 1; i++) {
    string x; cin >> x;
    s.erase(x);
  }

  if (!s.empty()) cout << *s.begin() << "\n";

  return 0;
}
```
