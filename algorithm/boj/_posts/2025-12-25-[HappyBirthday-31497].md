## 문제 링크
[31497번 - 생일 축하합니다~](https://www.acmicpc.net/problem/31497)

## 설명
오늘 생일인 사람을 질문으로 찾아내는 인터랙티브 문제입니다.

<br>

## 접근법
각 사람을 두 번씩 질문합니다. 전체에서 거짓말은 최대 한 번이므로, 생일인 사람은 두 번 모두 1이거나 한 번은 1을 받습니다.  
두 번 모두 1인 사람이 있으면 그 사람이 정답이고, 그렇지 않다면 1이 한 번이라도 나온 사람이 정답입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var nLine = Console.ReadLine();
    if (nLine == null) return;
    var n = int.Parse(nLine);

    var names = new string[n];
    for (var i = 0; i < n; i++)
      names[i] = Console.ReadLine()!;

    var cnt = new int[n];
    for (var i = 0; i < n; i++) {
      for (var k = 0; k < 2; k++) {
        Console.WriteLine("? " + names[i]);
        Console.Out.Flush();
        var ans = int.Parse(Console.ReadLine()!);
        if (ans == 1) cnt[i]++;
      }
    }

    var idx = -1;
    for (var i = 0; i < n; i++) {
      if (cnt[i] == 2) { idx = i; break; }
    }
    if (idx == -1) {
      for (var i = 0; i < n; i++) {
        if (cnt[i] == 1) { idx = i; break; }
      }
    }

    Console.WriteLine("! " + names[idx]);
    Console.Out.Flush();
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  if (!(cin >> n)) return 0;
  vector<string> names(n);
  for (int i = 0; i < n; i++)
    cin >> names[i];

  vector<int> cnt(n, 0);
  for (int i = 0; i < n; i++) {
    for (int k = 0; k < 2; k++) {
      cout << "? " << names[i] << "\n" << flush;
      int ans; cin >> ans;
      if (ans == 1) cnt[i]++;
    }
  }

  int idx = -1;
  for (int i = 0; i < n; i++) {
    if (cnt[i] == 2) { idx = i; break; }
  }
  if (idx == -1) {
    for (int i = 0; i < n; i++) {
      if (cnt[i] == 1) { idx = i; break; }
    }
  }

  cout << "! " << names[idx] << "\n" << flush;

  return 0;
}
```
