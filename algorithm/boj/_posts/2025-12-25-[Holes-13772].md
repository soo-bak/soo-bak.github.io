## 문제 링크
[13772번 - Holes](https://www.acmicpc.net/problem/13772)

## 설명
문장에 포함된 각 대문자 알파벳의 구멍 개수를 모두 합해 출력하는 문제입니다.

<br>

## 접근법
알파벳마다 구멍 개수가 정해져 있으므로, 줄을 한 글자씩 보면서 해당 값을 더합니다.  
구멍이 2개인 글자(B), 1개인 글자(A, D, O, P, Q, R), 나머지 글자와 공백은 0개로 처리하면 됩니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();

    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!;
      var sum = 0;
      foreach (var ch in line) {
        if (ch == 'B') sum += 2;
        else if (ch == 'A' || ch == 'D' || ch == 'O' || ch == 'P' || ch == 'Q' || ch == 'R')
          sum += 1;
      }
      sb.AppendLine(sum.ToString());
    }

    Console.Write(sb);
  }
}
```
