## 문제 링크
[21867번 - Java Bitecode](https://www.acmicpc.net/problem/21867)

## 설명
주어진 코드에서 알파벳 J, A, V를 모두 제거한 결과를 출력합니다.  
모두 제거되어 빈 문자열이 되면 `nojava`를 출력합니다.

<br>

## 접근법
문자열을 순회하며 J, A, V가 아닌 문자만 모아서 출력합니다.  
모아진 결과가 비었는지 확인해 `nojava`를 처리합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;

    var sb = new StringBuilder();
    for (var i = 0; i < n; i++) {
      var ch = s[i];
      if (ch == 'J' || ch == 'A' || ch == 'V') continue;
      sb.Append(ch);
    }

    Console.WriteLine(sb.Length == 0 ? "nojava" : sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  string res;
  res.reserve(n);
  for (char ch : s) {
    if (ch == 'J' || ch == 'A' || ch == 'V') continue;
    res.push_back(ch);
  }

  if (res.empty()) cout << "nojava\n";
  else cout << res << "\n";

  return 0;
}
```
