## 문제 링크
[23756번 - 노브 돌리기](https://www.acmicpc.net/problem/23756)

## 설명
각도를 순서대로 맞추기 위해 필요한 최소 회전 각도의 총합을 구하는 문제입니다.

<br>

## 접근법
현재 각도에서 다음 각도까지의 차이를 구합니다.  
시계/반시계 중 작은 회전값 `min(diff, 360 - diff)`을 더하면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var cur = int.Parse(Console.ReadLine()!);
    var sum = 0;
    for (var i = 0; i < n; i++) {
      var next = int.Parse(Console.ReadLine()!);
      var diff = Math.Abs(next - cur);
      sum += Math.Min(diff, 360 - diff);
      cur = next;
    }

    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int cur; cin >> cur;
  int sum = 0;
  for (int i = 0; i < n; i++) {
    int next; cin >> next;
    int diff = abs(next - cur);
    sum += min(diff, 360 - diff);
    cur = next;
  }

  cout << sum << "\n";

  return 0;
}
```
