## 문제 링크
[11117번 - Letter Cookies](https://www.acmicpc.net/problem/11117)

## 설명
쿠키 상자의 문자들로 각 단어를 만들 수 있는지 확인하는 문제입니다.

<br>

## 접근법
상자에 있는 문자 빈도를 센 뒤, 각 단어의 문자 빈도를 비교합니다.  
어떤 문자의 필요 개수가 상자 보유 개수를 넘으면 만들 수 없습니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);
    var sb = new StringBuilder();

    for (var tc = 0; tc < t; tc++) {
      var box = parts[idx++];
      var baseCnt = new int[26];
      foreach (var ch in box) baseCnt[ch - 'A']++;

      var w = int.Parse(parts[idx++]);
      for (var i = 0; i < w; i++) {
        var word = parts[idx++];
        var cnt = new int[26];
        foreach (var ch in word) cnt[ch - 'A']++;

        var ok = true;
        for (var k = 0; k < 26; k++) {
          if (cnt[k] > baseCnt[k]) { ok = false; break; }
        }

        sb.AppendLine(ok ? "YES" : "NO");
      }
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    string box; cin >> box;
    int baseCnt[26] = {};
    for (char ch : box) baseCnt[ch - 'A']++;

    int w; cin >> w;
    for (int i = 0; i < w; i++) {
      string word; cin >> word;
      int cnt[26] = {};
      for (char ch : word) cnt[ch - 'A']++;

      bool ok = true;
      for (int k = 0; k < 26; k++) {
        if (cnt[k] > baseCnt[k]) { ok = false; break; }
      }

      cout << (ok ? "YES" : "NO") << "\n";
    }
  }

  return 0;
}
```
