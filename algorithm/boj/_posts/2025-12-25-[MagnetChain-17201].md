## 문제 링크
[17201번 - 자석 체인](https://www.acmicpc.net/problem/17201)

## 설명
자석들의 현재 연결 상태가 주어질 때, 모든 자석이 끌어당겨 연결되어 있는지 판별하는 문제입니다.

<br>

## 접근법
각 자석은 두 자리로 표현되므로, 인접한 자석이 맞닿는 위치는 `a[2i+1]`과 `a[2i+2]`입니다.  
이 값이 같으면 서로 밀어내므로 연결이 끊어진 상태입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;

    var ok = true;
    for (var i = 0; i < n - 1; i++) {
      if (s[2 * i + 1] == s[2 * i + 2]) {
        ok = false;
        break;
      }
    }

    Console.WriteLine(ok ? "Yes" : "No");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  bool ok = true;
  for (int i = 0; i < n - 1; i++) {
    if (s[2 * i + 1] == s[2 * i + 2]) {
      ok = false;
      break;
    }
  }

  cout << (ok ? "Yes" : "No") << "\n";

  return 0;
}
```
