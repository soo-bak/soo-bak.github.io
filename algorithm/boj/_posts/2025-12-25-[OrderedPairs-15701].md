## 문제 링크
[15701번 - 순서쌍](https://www.acmicpc.net/problem/15701)

## 설명
자연수 N이 주어질 때 a×b=N을 만족하는 서로 다른 순서쌍의 개수를 구하는 문제입니다.

<br>

## 접근법
각 약수 d마다 (d, N/d) 한 쌍이 대응되므로 답은 N의 약수 개수입니다.  
소인수분해로 각 지수의 (지수+1)을 곱해 약수 개수를 계산합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var num = n;
    var ans = 1;

    for (var p = 2; p * p <= num; p++) {
      if (num % p != 0) continue;
      var cnt = 0;
      while (num % p == 0) {
        num /= p;
        cnt++;
      }
      ans *= cnt + 1;
    }

    if (num > 1) ans *= 2;
    Console.WriteLine(ans);
  }
}
```
