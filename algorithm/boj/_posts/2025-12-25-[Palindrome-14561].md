## 문제 링크
[14561번 - 회문](https://www.acmicpc.net/problem/14561)

## 설명
십진수 A를 n진수로 표현했을 때 회문인지 판별하는 문제입니다.

<br>

## 접근법
A를 n진수로 변환해 각 자리 숫자를 구합니다.  
양 끝에서부터 같은지 비교해 회문 여부를 판단합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Collections.Generic;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);

    var sb = new StringBuilder();
    for (var tc = 0; tc < t; tc++) {
      long a = long.Parse(parts[idx++]);
      var n = int.Parse(parts[idx++]);
      var digits = new List<int>();

      while (a > 0) {
        digits.Add((int)(a % n));
        a /= n;
      }

      var ok = true;
      var l = 0;
      var r = digits.Count - 1;
      while (l < r) {
        if (digits[l] != digits[r]) { ok = false; break; }
        l++; r--;
      }

      sb.AppendLine(ok ? "1" : "0");
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef vector<int> vi;
typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    ll a; int n;
    cin >> a >> n;

    vi digits;
    while (a > 0) {
      digits.push_back(a % n);
      a /= n;
    }

    bool ok = true;
    int l = 0, r = (int)digits.size() - 1;
    while (l < r) {
      if (digits[l] != digits[r]) { ok = false; break; }
      l++; r--;
    }

    cout << (ok ? 1 : 0) << "\n";
  }

  return 0;
}
```
