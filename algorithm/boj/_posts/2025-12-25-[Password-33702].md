## 문제 링크
[33702번 - 비밀번호](https://www.acmicpc.net/problem/33702)

## 설명
3x3 버튼에서 시작 위치가 주어질 때, 상하좌우로만 이동하며 모든 버튼을 한 번씩 누르는 경로의 개수를 구하는 문제입니다.

<br>

## 접근법
3x3 격자에서 모든 칸을 한 번씩 방문하는 경로 수는 시작 위치에 따라 결정됩니다.

모서리(1, 3, 7, 9)나 중앙(5)에서 시작하면 경로가 8개이고, 변의 중간(2, 4, 6, 8)에서 시작하면 경로가 없습니다.

이 패턴을 보면 홀수 번호에서 시작할 때만 경로가 존재하므로, k가 홀수면 8, 짝수면 0을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var k = int.Parse(Console.ReadLine()!);
    Console.WriteLine(k % 2 == 1 ? 8 : 0);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int k; cin >> k;
  cout << (k % 2 == 1 ? 8 : 0) << "\n";

  return 0;
}
```
