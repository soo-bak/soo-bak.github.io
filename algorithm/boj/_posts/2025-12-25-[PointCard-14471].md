## 문제 링크
[14471번 - 포인트 카드](https://www.acmicpc.net/problem/14471)

## 설명
각 카드에서 당첨 도장이 N개 이상이면 경품과 교환할 수 있을 때, M장 중 최소 M-1장을 만들기 위해 필요한 변경 비용의 최솟값을 구하는 문제입니다.

<br>

## 접근법
카드 i를 경품 교환 가능하게 만들기 위한 비용은 `max(0, N - A_i)`입니다.  
M-1장 이상이 필요하므로, 최대 한 장만 포기할 수 있습니다. 전체 비용의 합에서 가장 큰 비용 하나를 빼는 것이 최소 비용입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);

    var sum = 0;
    var maxNeed = 0;
    for (var i = 0; i < m; i++) {
      var parts = Console.ReadLine()!.Split();
      var a = int.Parse(parts[0]);
      var need = n - a;
      if (need < 0) need = 0;

      sum += need;
      if (need > maxNeed) maxNeed = need;
    }

    Console.WriteLine(sum - maxNeed);
  }
}
```

### C++
