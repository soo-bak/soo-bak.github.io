## 문제 링크
[8912번 - Sales](https://www.acmicpc.net/problem/8912)

## 설명
각 날의 판매량에 대해 이전 날들 중 현재 값 이하인 개수를 세고, 그 합을 출력하는 문제입니다.

<br>

## 접근법
하루씩 앞에서부터 보면서 그 이전 값들과 비교하면 됩니다.  
n이 1000이라 모든 쌍을 직접 비교해도 충분히 빠릅니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);
    var sb = new StringBuilder();

    for (var caseNo = 0; caseNo < t; caseNo++) {
      var n = int.Parse(parts[idx++]);
      var a = new int[n];
      for (var i = 0; i < n; i++)
        a[i] = int.Parse(parts[idx++]);

      var sum = 0;
      for (var i = 1; i < n; i++) {
        for (var j = 0; j < i; j++)
          if (a[j] <= a[i]) sum++;
      }

      sb.AppendLine(sum.ToString());
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    int n; cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++)
      cin >> a[i];

    int sum = 0;
    for (int i = 1; i < n; i++) {
      for (int j = 0; j < i; j++)
        if (a[j] <= a[i]) sum++;
    }

    cout << sum << "\n";
  }

  return 0;
}
```
