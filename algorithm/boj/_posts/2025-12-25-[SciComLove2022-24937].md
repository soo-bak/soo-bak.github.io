## 문제 링크
[24937번 - SciComLove (2022)](https://www.acmicpc.net/problem/24937)

## 설명
문자열 "SciComLove"를 왼쪽으로 N번 회전한 결과를 출력하는 문제입니다.

<br>

## 접근법
문자열 길이는 10이므로, 실제 회전 횟수는 `N % 10`입니다.  
앞에서 N%10만큼을 떼어 뒤에 붙여 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = "SciComLove";
    var k = n % s.Length;
    var res = s.Substring(k) + s.Substring(0, k);
    Console.WriteLine(res);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s = "SciComLove";
  int k = n % (int)s.size();
  cout << s.substr(k) + s.substr(0, k) << "\n";

  return 0;
}
```
