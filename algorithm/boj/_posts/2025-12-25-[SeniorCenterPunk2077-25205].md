## 문제 링크
[25205번 - 경로당펑크 2077](https://www.acmicpc.net/problem/25205)

## 설명
두벌식 자판으로 입력한 문자열이 주어질 때, 마지막 음절에 받침이 있는지 판별하는 문제입니다.

<br>

## 접근법
문제의 조건에서는 복합 모음이나 겹받침이 등장하지 않으므로, 한 음절은 초성+중성(+종성)으로만 구성됩니다.

따라서 문자열의 마지막 글자가 모음 키라면 받침이 없고, 자음 키라면 받침이 있다고 판단할 수 있습니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;

    var vowels = "koijpuhynbml";
    var last = s[s.Length - 1];
    Console.WriteLine(vowels.Contains(last) ? 0 : 1);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  string vowels = "koijpuhynbml";
  cout << (vowels.find(s.back()) != string::npos ? 0 : 1) << "\n";

  return 0;
}
```
