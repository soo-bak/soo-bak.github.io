## 문제 링크
[17027번 - Shell Game](https://www.acmicpc.net/problem/17027)

## 설명
초기 조개 위치를 알 수 없을 때, 가능한 초기 위치 중 Elsie가 얻을 수 있는 점수의 최댓값을 구하는 문제입니다.

<br>

## 접근법
초기 조개 위치가 1, 2, 3일 때를 각각 시뮬레이션합니다.  
매 단계에서 조개 위치를 스왑한 뒤, Elsie의 추측과 같으면 점수를 증가시킵니다.  
세 경우 중 최대 점수를 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var a = new int[n];
    var b = new int[n];
    var g = new int[n];

    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      a[i] = int.Parse(parts[0]);
      b[i] = int.Parse(parts[1]);
      g[i] = int.Parse(parts[2]);
    }

    var best = 0;
    for (var start = 1; start <= 3; start++) {
      var pos = start;
      var score = 0;
      for (var i = 0; i < n; i++) {
        if (pos == a[i]) pos = b[i];
        else if (pos == b[i]) pos = a[i];
        if (pos == g[i]) score++;
      }
      if (score > best) best = score;
    }

    Console.WriteLine(best);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi a(n), b(n), g(n);
  for (int i = 0; i < n; i++)
    cin >> a[i] >> b[i] >> g[i];

  int best = 0;
  for (int start = 1; start <= 3; start++) {
    int pos = start;
    int score = 0;
    for (int i = 0; i < n; i++) {
      if (pos == a[i]) pos = b[i];
      else if (pos == b[i]) pos = a[i];
      if (pos == g[i]) score++;
    }
    if (score > best) best = score;
  }

  cout << best << "\n";

  return 0;
}
```
