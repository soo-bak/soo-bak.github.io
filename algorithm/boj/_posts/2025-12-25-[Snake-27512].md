## 문제 링크
[27512번 - 스네이크](https://www.acmicpc.net/problem/27512)

## 설명
n×m 격자에서 머리와 꼬리를 인접하게 유지하는 전략으로 유지할 수 있는 뱀의 최대 길이를 구하는 문제입니다.

<br>

## 접근법
머리와 꼬리가 인접하면 뱀의 칸들은 사이클을 이룹니다.  
격자 그래프의 사이클 길이는 항상 짝수이므로, n과 m이 모두 홀수면 최대 길이는 `n * m - 1`이고 그 외에는 `n * m`입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = int.Parse(parts[0]);
    var m = int.Parse(parts[1]);

    var ans = (n % 2 == 1 && m % 2 == 1) ? n * m - 1 : n * m;
    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;
  int ans = n * m;
  if (n % 2 == 1 && m % 2 == 1) ans = n * m - 1;
  cout << ans << "\n";

  return 0;
}
```
