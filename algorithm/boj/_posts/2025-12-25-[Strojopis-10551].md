## 문제 링크
[10551번 - STROJOPIS](https://www.acmicpc.net/problem/10551)

## 설명
주어진 문자열을 올바른 타이핑 규칙으로 입력했을 때 각 손가락(엄지 제외)이 눌린 횟수를 출력하는 문제입니다.

<br>

## 접근법
손가락별로 담당하는 키 집합을 미리 정해 두고, 문자열의 각 문자가 어느 집합에 속하는지 확인해 카운트를 늘립니다.  
왼손 새끼손가락부터 오른손 새끼손가락까지 순서대로 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    string[] keys = {
      "1QAZ",
      "2WSX",
      "3EDC",
      "4RFV5TGB",
      "6YHN7UJM",
      "8IK,",
      "9OL.",
      "0P[]\\;'/=-"
    };

    var cnt = new int[8];
    foreach (var ch in s) {
      for (var i = 0; i < 8; i++) {
        if (keys[i].IndexOf(ch) >= 0) {
          cnt[i]++;
          break;
        }
      }
    }

    for (var i = 0; i < 8; i++)
      Console.WriteLine(cnt[i]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  string keys[8] = {
    "1QAZ",
    "2WSX",
    "3EDC",
    "4RFV5TGB",
    "6YHN7UJM",
    "8IK,",
    "9OL.",
    "0P[]\\;'/=-"
  };

  int cnt[8] = {};
  for (char ch : s) {
    for (int i = 0; i < 8; i++) {
      if (keys[i].find(ch) != string::npos) {
        cnt[i]++;
        break;
      }
    }
  }

  for (int i = 0; i < 8; i++)
    cout << cnt[i] << "\n";

  return 0;
}
```
