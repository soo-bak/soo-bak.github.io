## 문제 링크
[14686번 - Sum Game](https://www.acmicpc.net/problem/14686)

## 설명
두 팀의 누적 점수가 같은 마지막 날짜 K를 구하는 문제입니다.

<br>

## 접근법
날짜를 순서대로 보면서 두 팀의 누적 점수를 동시에 더합니다.  
누적 점수가 같아지는 날을 만나면 그 날짜를 기록해두고, 끝까지 갔을 때 가장 마지막으로 기록된 값을 출력합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var a = Console.ReadLine()!.Split();
    var b = Console.ReadLine()!.Split();

    var sumA = 0;
    var sumB = 0;
    var ans = 0;
    for (var i = 0; i < n; i++) {
      sumA += int.Parse(a[i]);
      sumB += int.Parse(b[i]);
      if (sumA == sumB) ans = i + 1;
    }

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vector<int> a(n), b(n);
  for (int i = 0; i < n; i++) cin >> a[i];
  for (int i = 0; i < n; i++) cin >> b[i];

  int sumA = 0, sumB = 0, ans = 0;
  for (int i = 0; i < n; i++) {
    sumA += a[i];
    sumB += b[i];
    if (sumA == sumB) ans = i + 1;
  }

  cout << ans << "\n";

  return 0;
}
```
