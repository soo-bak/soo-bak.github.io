## 문제 링크
[20494번 - 스시](https://www.acmicpc.net/problem/20494)

## 설명
각 손님의 스시 목록이 주어질 때, 모든 손님이 식사를 마치기 위해 셰프가 만들어야 하는 스시의 최소 개수를 구하는 문제입니다.

<br>

## 접근법
손님이 먹는 스시 수는 각 목록의 길이 합이며, 이보다 적게 만들 수는 없습니다.  
필요한 만큼만 만들면 결국 모두 먹게 되므로 최소 개수는 전체 길이 합입니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sum = 0;
    for (var i = 0; i < n; i++) {
      var s = Console.ReadLine()!;
      sum += s.Length;
    }

    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int sum = 0;
  for (int i = 0; i < n; i++) {
    string s; cin >> s;
    sum += s.size();
  }

  cout << sum << "\n";

  return 0;
}
```
