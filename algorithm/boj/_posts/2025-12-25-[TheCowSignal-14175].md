## 문제 링크
[14175번 - The Cow-Signal](https://www.acmicpc.net/problem/14175)

## 설명
주어진 M×N 신호를 가로와 세로로 각각 K배 확대한 결과를 출력하는 문제입니다.

<br>

## 접근법
각 줄을 읽으면서 문자 하나를 K번 반복해 확장한 줄을 만든 뒤, 그 줄을 K번 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var m = int.Parse(first[0]);
    var n = int.Parse(first[1]);
    var k = int.Parse(first[2]);

    var sb = new StringBuilder();
    for (var i = 0; i < m; i++) {
      var line = Console.ReadLine()!;
      var expanded = new StringBuilder();
      for (var j = 0; j < n; j++) {
        for (var t = 0; t < k; t++) expanded.Append(line[j]);
      }

      var row = expanded.ToString();
      for (var t = 0; t < k; t++)
        sb.AppendLine(row);
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int m, n, k; cin >> m >> n >> k;
  for (int i = 0; i < m; i++) {
    string line; cin >> line;
    string expanded;
    expanded.reserve(n * k);
    for (int j = 0; j < n; j++) {
      expanded.append(k, line[j]);
    }
    for (int t = 0; t < k; t++)
      cout << expanded << "\n";
  }

  return 0;
}
```
