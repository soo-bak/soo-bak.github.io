## 문제 링크
[14530번 - The Lost Cow](https://www.acmicpc.net/problem/14530)

## 설명
위치 x에서 시작해 1, 2, 4, 8...만큼 번갈아 이동할 때, y에 도달하기까지 이동한 총 거리를 구하는 문제입니다.

<br>

## 접근법
현재 위치에서 다음 목표 위치까지 이동할 때, 그 구간에 y가 포함되면 그 지점까지의 거리만 더하고 종료합니다.  
포함되지 않으면 해당 구간의 전체 거리를 더한 뒤, 이동 거리를 두 배로 키우고 방향을 반전합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var x = int.Parse(parts[0]);
    var y = int.Parse(parts[1]);

    int pos = x;
    int step = 1;
    int dist = 0;
    var dir = 1;

    while (true) {
      var target = x + dir * step;
      if (dir == 1) {
        if (pos <= y && y <= target) {
          dist += y - pos;
          break;
        }
      } else {
        if (target <= y && y <= pos) {
          dist += pos - y;
          break;
        }
      }

      dist += Math.Abs(target - pos);
      pos = target;
      step *= 2;
      dir *= -1;
    }

    Console.WriteLine(dist);
  }
}
```

### C++

  int pos = x;
  int step = 1;
  int dist = 0;
  int dir = 1;

  while (true) {
    int target = x + dir * step;
    if (dir == 1) {
      if (pos <= y && y <= target) {
        dist += y - pos;
        break;
      }
    } else {
      if (target <= y && y <= pos) {
        dist += pos - y;
        break;
      }
    }

    dist += llabs(target - pos);
    pos = target;
    step *= 2;
    dir *= -1;
  }

  cout << dist << "\n";

  return 0;
}
