## 문제 링크
[14409번 - Tuna](https://www.acmicpc.net/problem/14409)

## 설명
각 참치에 대해 두 추정치가 주어지며, 차이가 기준 이하면 큰 값을 쓰고 초과하면 세 번째 값을 쓰는 규칙으로 총합을 구하는 문제입니다.

<br>

## 접근법
먼저 각 참치의 두 추정치 차이가 기준 이내인지 확인합니다.

다음으로 기준 이내이면 큰 값을 더하고, 초과이면 입력에 따라 주어지는 세 번째 값을 더합니다.

마지막으로 모든 참치의 값을 더한 결과를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var x = int.Parse(parts[idx++]);

    var sum = 0;
    for (var i = 0; i < n; i++) {
      var p1 = int.Parse(parts[idx++]);
      var p2 = int.Parse(parts[idx++]);

      if (Math.Abs(p1 - p2) <= x) sum += Math.Max(p1, p2);
      else {
        var p3 = int.Parse(parts[idx++]);
        sum += p3;
      }
    }

    Console.WriteLine(sum);
  }
}
```

### C++
```cpp

  for (int i = 0; i < n; i++) {
    int p1, p2; cin >> p1 >> p2;
    if (abs(p1 - p2) <= x) sum += max(p1, p2);
    else {
      int p3; cin >> p3;
      sum += p3;
    }
  }

  cout << sum << "\n";
  return 0;
}
```
