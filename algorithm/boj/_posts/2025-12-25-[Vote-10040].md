## 문제 링크
[10040번 - 투표](https://www.acmicpc.net/problem/10040)

## 설명
각 위원이 비용 기준 이하에서 가장 재미있는 경기(번호가 가장 작은 경기)에 투표할 때, 최다 득표 경기를 찾는 문제입니다.

<br>

## 접근법
비용이 1부터 1000까지이므로 각 비용 한도에 대해 선택되는 경기 번호를 미리 구합니다.

그 다음 각 위원의 기준으로 해당 경기 번호에 표를 더하고, 최다 득표 경기 번호를 찾으면 됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);

    var cost = new int[n + 1];
    for (var i = 1; i <= n; i++)
      cost[i] = int.Parse(Console.ReadLine()!);

    var best = new int[1001];
    for (var b = 1; b <= 1000; b++) {
      for (var i = 1; i <= n; i++) {
        if (cost[i] <= b) {
          best[b] = i;
          break;
        }
      }
    }

    var cnt = new int[n + 1];
    for (var i = 0; i < m; i++) {
      var b = int.Parse(Console.ReadLine()!);
      cnt[best[b]]++;
    }

    var ans = 1;
    for (var i = 2; i <= n; i++) {
      if (cnt[i] > cnt[ans]) ans = i;
    }

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;
  vi cost(n + 1);
  for (int i = 1; i <= n; i++)
    cin >> cost[i];

  int best[1001] = {};
  for (int b = 1; b <= 1000; b++) {
    for (int i = 1; i <= n; i++) {
      if (cost[i] <= b) {
        best[b] = i;
        break;
      }
    }
  }

  vi cnt(n + 1, 0);
  for (int i = 0; i < m; i++) {
    int b; cin >> b;
    cnt[best[b]]++;
  }

  int ans = 1;
  for (int i = 2; i <= n; i++) {
    if (cnt[i] > cnt[ans]) ans = i;
  }

  cout << ans << "\n";

  return 0;
}
```
