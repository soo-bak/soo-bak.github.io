## 문제 링크
[13877번 - 이건 무슨 진법이지?](https://www.acmicpc.net/problem/13877)

## 설명
숫자 문자열이 주어질 때 이를 8진수, 10진수, 16진수로 해석한 값을 출력하는 문제입니다.

<br>

## 접근법
8진수는 0~7만 허용되므로 한 글자라도 8 이상이면 0을 출력합니다.  
그 외에는 각 진법으로 해석해 출력하면 됩니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);

    var sb = new StringBuilder();
    for (var i = 0; i < t; i++) {
      var k = int.Parse(parts[idx++]);
      var s = parts[idx++];

      var octOk = true;
      foreach (var c in s) {
        if (c > '7') { octOk = false; break; }
      }

      var oct = octOk ? Convert.ToInt32(s, 8) : 0;
      var dec = int.Parse(s);
      var hex = Convert.ToInt32(s, 16);

      sb.AppendLine($"{k} {oct} {dec} {hex}");
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  for (int i = 0; i < t; i++) {
    int k; string s;
    cin >> k >> s;

    bool octOk = true;
    for (char c : s) {
      if (c > '7') { octOk = false; break; }
    }

    int oct = octOk ? stoi(s, nullptr, 8) : 0;
    int dec = stoi(s);
    int hex = stoi(s, nullptr, 16);

    cout << k << " " << oct << " " << dec << " " << hex << "\n";
  }

  return 0;
}
```
