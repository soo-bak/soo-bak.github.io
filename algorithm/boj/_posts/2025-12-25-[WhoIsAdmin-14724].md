## 문제 링크
[14724번 - 관리자는 누구?](https://www.acmicpc.net/problem/14724)

## 설명
각 동아리에서 가장 많은 문제를 푼 사람을 후보로 뽑고, 후보 중 점수가 가장 높은 동아리의 이름을 출력하는 문제입니다.

<br>

## 접근법
동아리 9개의 점수를 차례대로 읽으며 각 줄의 최댓값을 구합니다.  
최댓값이 가장 큰 동아리의 이름을 저장해 두었다가 출력하면 됩니다.  
후보 점수는 모두 다르므로 최대값은 유일합니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var names = new[] {
      "PROBRAIN", "GROW", "ARGOS", "ADMIN", "ANT",
      "MOTION", "SPG", "COMON", "ALMIGHTY"
    };

    var best = -1;
    var bestIdx = 0;
    for (var i = 0; i < 9; i++) {
      var parts = Console.ReadLine()!.Split(' ', StringSplitOptions.RemoveEmptyEntries);
      var mx = 0;
      for (var j = 0; j < n; j++) {
        var v = int.Parse(parts[j]);
        if (v > mx) mx = v;
      }
      if (mx > best) {
        best = mx;
        bestIdx = i;
      }
    }

    Console.WriteLine(names[bestIdx]);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string names[9] = {
    "PROBRAIN", "GROW", "ARGOS", "ADMIN", "ANT",
    "MOTION", "SPG", "COMON", "ALMIGHTY"
  };

  int best = -1, bestIdx = 0;
  for (int i = 0; i < 9; i++) {
    int mx = 0;
    for (int j = 0; j < n; j++) {
      int v; cin >> v;
      if (v > mx) mx = v;
    }
    if (mx > best) {
      best = mx;
      bestIdx = i;
    }
  }

  cout << names[bestIdx] << "\n";

  return 0;
}
```
