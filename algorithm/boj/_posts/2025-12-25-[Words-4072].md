## 문제 링크
[4072번 - Words](https://www.acmicpc.net/problem/4072)

## 설명
여러 줄의 문자열이 주어질 때, 각 줄의 단어를 뒤집어 출력하는 문제입니다.

<br>

## 접근법
한 줄씩 읽어 `#`이면 종료합니다.  
공백으로 분리한 단어들을 각각 뒤집어 같은 줄에 다시 이어 붙입니다.

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    while (true) {
      var line = Console.ReadLine();
      if (line == null || line == "#") break;

      var parts = line.Split(' ');
      var sb = new StringBuilder();
      for (var i = 0; i < parts.Length; i++) {
        var arr = parts[i].ToCharArray();
        Array.Reverse(arr);
        if (i > 0) sb.Append(' ');
        sb.Append(arr);
      }

      Console.WriteLine(sb.ToString());
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string line;
  while (getline(cin, line)) {
    if (line == "#") break;
    stringstream ss(line);
    string word;
    bool first = true;
    while (ss >> word) {
      reverse(word.begin(), word.end());
      if (!first) cout << " ";
      first = false;
      cout << word;
    }
    cout << "\n";
  }

  return 0;
}
```
