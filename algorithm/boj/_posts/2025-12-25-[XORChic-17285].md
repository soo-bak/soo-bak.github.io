## 문제 링크
[17285번 - XORChic](https://www.acmicpc.net/problem/17285)

## 설명
XOR로 암호화된 문자열 T가 주어질 때, 원문이 "CHICKENS"로 시작한다는 정보를 이용해 원문을 복호화하는 문제입니다.

<br>

## 접근법
첫 글자에 대해 `key = T[0] XOR 'C'`로 키를 구합니다.  
모든 문자에 동일한 키를 XOR하면 원문이 복원됩니다.

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = Console.ReadLine()!;
    var key = t[0] ^ 'C';
    var arr = t.ToCharArray();
    for (var i = 0; i < arr.Length; i++)
      arr[i] = (char)(arr[i] ^ key);
    Console.WriteLine(new string(arr));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string t; cin >> t;
  int key = t[0] ^ 'C';
  for (int i = 0; i < (int)t.size(); i++)
    t[i] = char(t[i] ^ key);

  cout << t << "\n";

  return 0;
}
```
