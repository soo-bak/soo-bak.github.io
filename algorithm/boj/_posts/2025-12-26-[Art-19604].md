## 문제 링크
[19604번 - Art](https://www.acmicpc.net/problem/19604)

## 설명
모든 점이 경계가 아닌 내부에 들어가도록 하는 가장 작은 직사각형의 두 꼭짓점을 출력하는 문제입니다.

<br>

## 접근법
먼저 점들의 가장 작은 x와 y, 가장 큰 x와 y를 구합니다.

다음으로 모든 점이 경계에 닿지 않도록 범위를 한 칸씩 바깥으로 확장합니다.

마지막으로 확장된 좌하단과 우상단 좌표를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var minX = int.MaxValue;
    var minY = int.MaxValue;
    var maxX = int.MinValue;
    var maxY = int.MinValue;

    for (var i = 0; i < n; i++) {
      var line = Console.ReadLine()!;
      var parts = line.Split(',');
      var x = int.Parse(parts[0]);
      var y = int.Parse(parts[1]);
      if (x < minX) minX = x;
      if (y < minY) minY = y;
      if (x > maxX) maxX = x;
      if (y > maxY) maxY = y;
    }

    Console.WriteLine($"{minX - 1},{minY - 1}");
    Console.WriteLine($"{maxX + 1},{maxY + 1}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string line;
  getline(cin, line);

  int minX = 1000000000, minY = 1000000000;
  int maxX = -1, maxY = -1;

  for (int i = 0; i < n; i++) {
    getline(cin, line);
    int pos = (int)line.find(',');
    int x = stoi(line.substr(0, pos));
    int y = stoi(line.substr(pos + 1));

    if (x < minX) minX = x;
    if (y < minY) minY = y;
    if (x > maxX) maxX = x;
    if (y > maxY) maxY = y;
  }

  cout << minX - 1 << "," << minY - 1 << "\n";
  cout << maxX + 1 << "," << maxY + 1 << "\n";

  return 0;
}
```
