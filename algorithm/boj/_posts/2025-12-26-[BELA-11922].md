## 문제 링크
[11922번 - BELA](https://www.acmicpc.net/problem/11922)

## 설명
우세 무늬가 주어졌을 때 각 카드의 점수를 합산해 총점을 구하는 문제입니다.

<br>

## 접근법
먼저 카드 등급별 점수표를 우세 무늬와 일반 무늬로 나눠 준비합니다.

다음으로 각 카드의 무늬가 우세 무늬인지 확인해 해당 점수를 더합니다.

마지막으로 총합을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var dom = parts[idx++][0];

    var scoreDom = new int[128];
    var scoreNor = new int[128];
    scoreDom['A'] = 11; scoreNor['A'] = 11;
    scoreDom['K'] = 4;  scoreNor['K'] = 4;
    scoreDom['Q'] = 3;  scoreNor['Q'] = 3;
    scoreDom['J'] = 20; scoreNor['J'] = 2;
    scoreDom['T'] = 10; scoreNor['T'] = 10;
    scoreDom['9'] = 14; scoreNor['9'] = 0;
    scoreDom['8'] = 0;  scoreNor['8'] = 0;
    scoreDom['7'] = 0;  scoreNor['7'] = 0;

    var sum = 0;
    for (var i = 0; i < 4 * n; i++) {
      var card = parts[idx++];
      var val = card[0];
      var suit = card[1];
      sum += (suit == dom) ? scoreDom[val] : scoreNor[val];
    }

    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; char dom;
  cin >> n >> dom;

  int scoreDom[128] = {};
  int scoreNor[128] = {};
  scoreDom['A'] = 11; scoreNor['A'] = 11;
  scoreDom['K'] = 4;  scoreNor['K'] = 4;
  scoreDom['Q'] = 3;  scoreNor['Q'] = 3;
  scoreDom['J'] = 20; scoreNor['J'] = 2;
  scoreDom['T'] = 10; scoreNor['T'] = 10;
  scoreDom['9'] = 14; scoreNor['9'] = 0;
  scoreDom['8'] = 0;  scoreNor['8'] = 0;
  scoreDom['7'] = 0;  scoreNor['7'] = 0;

  int sum = 0;
  for (int i = 0; i < 4 * n; i++) {
    string card; cin >> card;
    char val = card[0];
    char suit = card[1];
    sum += (suit == dom) ? scoreDom[val] : scoreNor[val];
  }

  cout << sum << "\n";
  return 0;
}
```
