## 문제 링크
[11297번 - Cypher](https://www.acmicpc.net/problem/11297)

## 설명
날짜로부터 시프트 값을 구하고 시저 암호를 복호화하는 문제입니다.

<br>

## 접근법
먼저 날짜의 합으로 시프트 값을 계산합니다.

다음으로 문자열을 한 글자씩 보면서 알파벳만 뒤로 이동시킵니다.

이후 알파벳 범위를 벗어나면 뒤에서 앞으로 순환합니다.

마지막으로 0 0 0이 나올 때까지 반복해 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    while (true) {
      var line = Console.ReadLine();
      if (line == null) break;
      var parts = line.Split();
      var d = int.Parse(parts[0]);
      var m = int.Parse(parts[1]);
      var y = int.Parse(parts[2]);
      if (d == 0 && m == 0 && y == 0) break;

      var shift = (d + m + y) % 25 + 1;
      var s = Console.ReadLine()!;
      var arr = s.ToCharArray();
      for (var i = 0; i < arr.Length; i++) {
        var c = arr[i];
        if (c >= 'a' && c <= 'z') {
          var v = c - 'a' - shift;
          if (v < 0) v += 26;
          arr[i] = (char)('a' + v);
        }
      }
      Console.WriteLine(new string(arr));
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  while (true) {
    int d, m, y; cin >> d >> m >> y;
    if (d == 0 && m == 0 && y == 0) break;
    int shift = (d + m + y) % 25 + 1;

    string s;
    getline(cin, s);
    getline(cin, s);

    for (char &c : s) {
      if (c >= 'a' && c <= 'z') {
        int v = c - 'a' - shift;
        if (v < 0) v += 26;
        c = char('a' + v);
      }
    }

    cout << s << "\n";
  }

  return 0;
}
```
