## 문제 링크
[12353번 - Dr. Spaceman의 특별한 알고리즘](https://www.acmicpc.net/problem/12353)

## 설명
부모의 키와 아이의 성별이 주어질 때 알고리즘에 따라 아이 키의 범위를 구해 출력하는 문제입니다.

<br>

## 접근법
먼저 부모의 키를 인치 단위로 바꿔 합산하고 성별에 따라 5인치를 더하거나 뺍니다.

다음으로 그 값을 절반으로 한 기준에서 위아래 4인치 범위를 잡습니다.

이후 범위의 끝이 정수가 아니면 안쪽으로 당겨 정수 인치만 남깁니다.

마지막으로 피트와 인치 형식으로 변환해 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static int ToInches(string s) {
    var p = s.IndexOf('\'');
    var feet = int.Parse(s.Substring(0, p));
    var inch = int.Parse(s.Substring(p + 1, s.Length - p - 2));
    return feet * 12 + inch;
  }

  static string Format(int inches) {
    var feet = inches / 12;
    var inch = inches % 12;
    return $"{feet}'{inch}\"";
  }

  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);
    var sb = new StringBuilder();

    for (var caseNum = 1; caseNum <= t; caseNum++) {
      var gender = parts[idx++];
      var mom = ToInches(parts[idx++]);
      var dad = ToInches(parts[idx++]);

      var sum = mom + dad;
      if (gender == "B") sum += 5;
      else sum -= 5;

      var low2 = sum - 8;
      var high2 = sum + 8;
      var low = (low2 + 1) / 2;
      var high = high2 / 2;

      sb.Append("Case #").Append(caseNum).Append(": ")
        .Append(Format(low)).Append(" to ").Append(Format(high)).AppendLine();
    }

    Console.Write(sb);
  }
}
```

#include <bits/stdc++.h>
using namespace std;

int toInches(const string &s) {
  int p = (int)s.find('\'');
  int feet = stoi(s.substr(0, p));
  int inch = stoi(s.substr(p + 1, s.size() - p - 2));
  return feet * 12 + inch;
}

string formatHeight(int inches) {
  int feet = inches / 12;
  int inch = inches % 12;
  return to_string(feet) + "'" + to_string(inch) + "\"";
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  for (int caseNum = 1; caseNum <= t; caseNum++) {
    string gender, momStr, dadStr;
    cin >> gender >> momStr >> dadStr;

    int mom = toInches(momStr);
    int dad = toInches(dadStr);

    int sum = mom + dad;
    if (gender == "B") sum += 5;
    else sum -= 5;

    int low2 = sum - 8;
    int high2 = sum + 8;
    int low = (low2 + 1) / 2;
    int high = high2 / 2;

    cout << "Case #" << caseNum << ": "
         << formatHeight(low) << " to " << formatHeight(high) << "\n";
  }

  return 0;
}
