## 문제 링크
[32762번 - 더치 페이](https://www.acmicpc.net/problem/32762)

## 설명
주문된 음식의 총액을 참가자 수로 나눈 1인당 평균 지불 금액을 구하는 문제입니다.

<br>

## 접근법
주문 금액을 모두 합산한 후, 그 합을 사람 수로 나눠 평균을 구합니다.

결과를 소수점 이하 출력 형식에 맞춰 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var m = int.Parse(parts[idx++]);

    for (var i = 0; i < n; i++)
      idx += 2;

    var sum = 0;
    for (var i = 0; i < m; i++) {
      idx++;
      sum += int.Parse(parts[idx++]);
    }

    var avg = sum / (double)n;
    Console.WriteLine(avg.ToString("F5"));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;

  for (int i = 0; i < n; i++) {
    int s, e; cin >> s >> e;
  }

  int sum = 0;
  for (int i = 0; i < m; i++) {
    int t, p; cin >> t >> p;
    sum += p;
  }

  double avg = sum / (double)n;
  cout.setf(ios::fixed);
  cout << setprecision(5) << avg << "\n";
  return 0;
}
```
