## 문제 링크
[6081번 - Hay Expenses](https://www.acmicpc.net/problem/6081)

## 설명
여러 구간에 대해 해당 일수의 건초 개수 합을 출력하는 문제입니다.

<br>

## 접근법
먼저 날별 건초 개수를 누적해 두면 구간 합을 빠르게 구할 수 있습니다.

다음으로 각 질의의 시작과 끝을 받아 두 누적값의 차이로 결과를 계산합니다.

마지막으로 모든 질의의 답을 순서대로 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var q = int.Parse(parts[idx++]);

    var pref = new int[n + 1];
    for (var i = 1; i <= n; i++) {
      var v = int.Parse(parts[idx++]);
      pref[i] = pref[i - 1] + v;
    }

    var sb = new StringBuilder();
    for (var i = 0; i < q; i++) {
      var s = int.Parse(parts[idx++]);
      var e = int.Parse(parts[idx++]);
      sb.AppendLine((pref[e] - pref[s - 1]).ToString());
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, q; cin >> n >> q;
  vector<int> pref(n + 1, 0);
  for (int i = 1; i <= n; i++) {
    int v; cin >> v;
    pref[i] = pref[i - 1] + v;
  }

  for (int i = 0; i < q; i++) {
    int s, e; cin >> s >> e;
    cout << pref[e] - pref[s - 1] << "\n";
  }

  return 0;
}
```
