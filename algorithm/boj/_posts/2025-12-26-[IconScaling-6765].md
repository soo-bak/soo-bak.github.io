## 문제 링크
[6765번 - Icon Scaling](https://www.acmicpc.net/problem/6765)

## 설명
고정된 3×3 아이콘을 입력 배율 k에 맞춰 가로와 세로로 늘려 출력하는 문제입니다.

<br>

## 접근법
먼저 3줄의 원본 아이콘을 문자열로 준비합니다.

다음으로 각 줄을 가로로 k배 늘린 뒤, 세로로도 k번 반복해 출력합니다.

마지막으로 모든 줄을 순서대로 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var k = int.Parse(Console.ReadLine()!);
    var src = new[] { "*x*", " xx", "* *" };

    var sb = new StringBuilder();
    for (var i = 0; i < 3; i++) {
      var lineSb = new StringBuilder();
      for (var j = 0; j < 3; j++) {
        for (var t = 0; t < k; t++)
          lineSb.Append(src[i][j]);
      }

      var line = lineSb.ToString();
      for (var r = 0; r < k; r++)
        sb.AppendLine(line);
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int k; cin >> k;

  string src[3] = {"*x*", " xx", "* *"};

  for (int i = 0; i < 3; i++) {
    string line;
    for (int j = 0; j < 3; j++) {
      for (int t = 0; t < k; t++)
        line.push_back(src[i][j]);
    }

    for (int r = 0; r < k; r++)
      cout << line << "\n";
  }

  return 0;
}
```
