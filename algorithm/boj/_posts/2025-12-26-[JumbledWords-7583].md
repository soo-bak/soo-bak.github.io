## 문제 링크
[7583번 - Jumbled Words](https://www.acmicpc.net/problem/7583)

## 설명
여러 줄의 문장에서 각 단어의 첫 글자와 마지막 글자를 유지한 채 내부 글자를 뒤집어 출력하는 문제입니다.

<br>

## 접근법
먼저 한 줄씩 읽어 종료 표식이 나오면 중단합니다.

다음으로 각 단어를 분리해 길이가 2 이하이면 그대로, 그 이상이면 가운데 글자들을 뒤집습니다.

이후 단어 사이의 공백을 유지한 형태로 다시 이어 붙입니다.

마지막으로 변환된 줄을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static string Jumble(string w) {
    if (w.Length <= 2) return w;
    var arr = w.ToCharArray();
    var l = 1;
    var r = w.Length - 2;
    while (l < r) {
      var t = arr[l];
      arr[l] = arr[r];
      arr[r] = t;
      l++;
      r--;
    }
    return new string(arr);
  }

  static void Main() {
    var sb = new StringBuilder();
    while (true) {
      var line = Console.ReadLine();
      if (line == null || line == "#") break;

      var parts = line.Split(' ');
      for (var i = 0; i < parts.Length; i++) {
        if (i > 0) sb.Append(' ');
        sb.Append(Jumble(parts[i]));
      }
      sb.AppendLine();
    }
    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

string jumble(const string &w) {
  if ((int)w.size() <= 2) return w;
  string s = w;
  int l = 1;
  int r = (int)s.size() - 2;
  while (l < r) {
    char t = s[l];
    s[l] = s[r];
    s[r] = t;
    l++;
    r--;
  }
  return s;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string line;
  while (getline(cin, line)) {
    if (line == "#") break;
    stringstream ss(line);
    string word;
    bool first = true;
    while (ss >> word) {
      if (!first) cout << ' ';
      cout << jumble(word);
      first = false;
    }
    cout << "\n";
  }

  return 0;
}
```
