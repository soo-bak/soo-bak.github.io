## 문제 링크
[14183번 - Key Maker](https://www.acmicpc.net/problem/14183)

## 설명
고객 키와 버려진 키들의 깊이가 주어질 때, 목표 깊이로 맞출 수 있는 키 개수를 구하는 문제입니다.

<br>

## 접근법
먼저 고객 키의 깊이를 기준으로 각 위치의 최대 깊이를 정합니다.

다음으로 각 버려진 키가 모든 위치에서 그 깊이를 넘지 않는지 확인합니다.

이후 조건을 만족하는 키의 개수를 세어 출력합니다.

마지막으로 0 0이 나올 때까지 반복합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var sb = new StringBuilder();

    while (true) {
      var m = int.Parse(parts[idx++]);
      var n = int.Parse(parts[idx++]);
      if (m == 0 && n == 0) break;

      var target = new int[m];
      for (var i = 0; i < m; i++)
        target[i] = int.Parse(parts[idx++]);

      var cnt = 0;
      for (var i = 0; i < n; i++) {
        var ok = true;
        for (var j = 0; j < m; j++) {
          var v = int.Parse(parts[idx++]);
          if (v > target[j]) ok = false;
        }
        if (ok) cnt++;
      }

      sb.AppendLine(cnt.ToString());
    }

    Console.Write(sb);
  }
}
```

### C++

#include <bits/stdc++.h>

int main() {

  cin.tie(nullptr);

  while (true) {
    int m, n; cin >> m >> n;
    if (m == 0 && n == 0) break;

    vector<int> target(m);
    for (int i = 0; i < m; i++)
      cin >> target[i];

    int cnt = 0;
    for (int i = 0; i < n; i++) {
      bool ok = true;
      for (int j = 0; j < m; j++) {
        int v; cin >> v;
        if (v > target[j]) ok = false;
      }
      if (ok) cnt++;
    }

    cout << cnt << "\n";
  }

  return 0;
}
