## 문제 링크
[15322번 - Košnja](https://www.acmicpc.net/problem/15322)

## 설명
직사각형 격자에서 모든 칸을 최소한의 회전으로 방문할 때 필요한 회전 수를 구하는 문제입니다.

<br>

## 접근법
먼저 한 줄을 쭉 이동하는 동안에는 회전이 필요 없음을 이용합니다.

다음으로 한 줄을 끝낸 뒤 다음 줄로 넘어가려면 방향 전환이 두 번 필요합니다.

이후 짧은 변의 길이만큼 줄을 훑고 이동하게 되므로 줄을 바꾸는 횟수는 그 길이에서 1을 뺀 값입니다.

마지막으로 짧은 변이 1이면 0, 그 외에는 줄 전환 횟수의 두 배를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var k = int.Parse(parts[idx++]);
    var sb = new StringBuilder();

    for (var i = 0; i < k; i++) {
      var n = int.Parse(parts[idx++]);
      var m = int.Parse(parts[idx++]);
      var r = n < m ? n : m;
      var ans = (r - 1) * 2;
      sb.AppendLine(ans.ToString());
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);

  int k; cin >> k;
  while (k--) {
    int n, m; cin >> n >> m;
    int r = n < m ? n : m;
    int ans = (r - 1) * 2;
    cout << ans << "\n";
  }

  return 0;
}
```
