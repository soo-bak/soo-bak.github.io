## 문제 링크
[4841번 - Look and Say](https://www.acmicpc.net/problem/4841)

## 설명
숫자 문자열이 주어질 때 연속된 숫자의 개수와 값을 읽어 다음 문자열을 출력하는 문제입니다.

<br>

## 접근법
먼저 문자열을 왼쪽부터 훑으며 같은 숫자가 연속되는 구간을 찾습니다.

다음으로 각 구간에 대해 개수와 숫자를 차례로 기록합니다.

이후 모든 구간을 처리해 만들어진 문자열을 출력합니다.

마지막으로 각 테스트 케이스를 반복합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();

    for (var caseNum = 0; caseNum < t; caseNum++) {
      var s = Console.ReadLine()!;
      var outSb = new StringBuilder();
      var i = 0;
      while (i < s.Length) {
        var j = i + 1;
        while (j < s.Length && s[j] == s[i]) j++;
        outSb.Append(j - i).Append(s[i]);
        i = j;
      }
      sb.AppendLine(outSb.ToString());
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    string s; cin >> s;
    string out;
    for (int i = 0; i < (int)s.size();) {
      int j = i + 1;
      while (j < (int)s.size() && s[j] == s[i]) j++;
      out += to_string(j - i);
      out += s[i];
      i = j;
    }
    cout << out << "\n";
  }

  return 0;
}
```
