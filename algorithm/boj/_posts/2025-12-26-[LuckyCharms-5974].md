## 문제 링크
[5974번 - Lucky Charms](https://www.acmicpc.net/problem/5974)

## 설명
팔찌가 네일에 걸렸을 때 각 장식이 네일에서 얼마나 떨어지는지 구하는 문제입니다.

<br>

## 접근법
먼저 네일 위치와 장식이 달린 위치의 차이를 구합니다.

다음으로 그 값에 줄 길이를 더해 장식의 거리를 계산합니다.

마지막으로 각 장식의 결과를 순서대로 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var l = int.Parse(parts[idx++]);
    var c = int.Parse(parts[idx++]);
    var n = int.Parse(parts[idx++]);

    var sb = new StringBuilder();
    for (var i = 0; i < c; i++) {
      var s = int.Parse(parts[idx++]);
      var p = int.Parse(parts[idx++]);
      var dist = Math.Abs(p - n) + s;
      sb.AppendLine(dist.ToString());
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int l, c, n; cin >> l >> c >> n;

  for (int i = 0; i < c; i++) {
    int s, p; cin >> s >> p;
    int dist = abs(p - n) + s;
    cout << dist << "\n";
  }

  return 0;
}
```
