## 문제 링크
[7015번 - Millennium](https://www.acmicpc.net/problem/7015)

## 설명
특수 달력에서 주어진 생일부터 1000년 1월 1일까지의 일수를 구하는 문제입니다.

<br>

## 접근법
먼저 한 해의 일수는 평년과 3의 배수 해로 나뉘므로 누적 일수를 계산하는 함수를 만듭니다.

다음으로 생일의 누적 일수를 구해 1000년 1월 1일의 누적 일수에서 빼면 됩니다.

마지막으로 각 입력에 대해 결과를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static int DaysInYear(int y) {
    return (y % 3 == 0) ? 200 : 195;
  }

  static int DaysBeforeMonth(int y, int m) {
    var bigYear = (y % 3 == 0);
    var sum = 0;
    for (var i = 1; i < m; i++) {
      if (bigYear) sum += 20;
      else sum += (i % 2 == 1) ? 20 : 19;
    }
    return sum;
  }

  static int ToDays(int y, int m, int d) {
    var sum = 0;
    for (var i = 1; i < y; i++) sum += DaysInYear(i);
    sum += DaysBeforeMonth(y, m);
    sum += d - 1;
    return sum;
  }

  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var target = ToDays(1000, 1, 1);
    var sb = new StringBuilder();

    for (var i = 0; i < n; i++) {
      var y = int.Parse(parts[idx++]);
      var m = int.Parse(parts[idx++]);
      var d = int.Parse(parts[idx++]);
      var cur = ToDays(y, m, d);
      sb.AppendLine((target - cur).ToString());
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int daysInYear(int y) {
  return (y % 3 == 0) ? 200 : 195;
}

int daysBeforeMonth(int y, int m) {
  bool bigYear = (y % 3 == 0);
  int sum = 0;
  for (int i = 1; i < m; i++) {
    if (bigYear) sum += 20;
    else sum += (i % 2 == 1) ? 20 : 19;
  }
  return sum;
}

int toDays(int y, int m, int d) {
  int sum = 0;
  for (int i = 1; i < y; i++) sum += daysInYear(i);
  sum += daysBeforeMonth(y, m);
  sum += d - 1;
  return sum;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int target = toDays(1000, 1, 1);
  for (int i = 0; i < n; i++) {
    int y, m, d; cin >> y >> m >> d;
    int cur = toDays(y, m, d);
    cout << target - cur << "\n";
  }

  return 0;
}
```
