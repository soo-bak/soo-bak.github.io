## 문제 링크
[5489번 - Numbers](https://www.acmicpc.net/problem/5489)

## 설명
주어진 수열에서 가장 많이 등장하는 수들 중 가장 작은 값을 출력하는 문제입니다.

<br>

## 접근법
먼저 등장 횟수를 센 뒤 최댓값을 갱신합니다.

다음으로 같은 최댓값이 여러 개면 가장 작은 값을 선택합니다.

마지막으로 선택된 값을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var cnt = new int[10001];

    for (var i = 0; i < n; i++) {
      var v = int.Parse(parts[idx++]);
      cnt[v]++;
    }

    var bestCnt = -1;
    var bestVal = 0;
    for (var v = 1; v <= 10000; v++) {
      var c = cnt[v];
      if (c > bestCnt) {
        bestCnt = c;
        bestVal = v;
      }
    }

    Console.WriteLine(bestVal);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;

  int cnt[10001] = {};
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    cnt[v]++;
  }

  int bestCnt = -1;
  int bestVal = 0;
  for (int v = 1; v <= 10000; v++) {
    int c = cnt[v];
    if (c > bestCnt) {
      bestCnt = c;
      bestVal = v;
    }
  }

  cout << bestVal << "\n";
  return 0;
}
```
