## 문제 링크
[8295번 - Rectangles](https://www.acmicpc.net/problem/8295)

## 설명
격자에서 꼭짓점이 격자점에 놓이는 직사각형 중 둘레가 p 이상인 개수를 구하는 문제입니다.

<br>

## 접근법
먼저 가능한 가로 길이와 세로 길이를 모두 확인합니다.

다음으로 해당 길이의 직사각형이 놓일 수 있는 위치 수를 더합니다.

마지막으로 둘레 조건을 만족하는 경우만 합산해 결과를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = int.Parse(parts[0]);
    var m = int.Parse(parts[1]);
    var p = int.Parse(parts[2]);

    var ans = 0;
    for (var w = 1; w <= n; w++) {
      for (var h = 1; h <= m; h++) {
        var per = 2 * (w + h);
        if (per >= p) ans += (n - w + 1) * (m - h + 1);
      }
    }

    Console.WriteLine(ans);
  }
}
```
