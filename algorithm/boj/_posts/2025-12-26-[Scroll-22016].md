## 문제 링크
[22016번 - 巻物 (Scroll)](https://www.acmicpc.net/problem/22016)

## 설명
기록된 문자열의 K번째 이후 대소문자가 뒤집혀 있을 때 원래 문자열을 복원하는 문제입니다.

<br>

## 접근법
먼저 K 이전은 그대로 두고, K 이후 문자의 대소문자를 뒤집습니다.

마지막으로 복원된 문자열을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = int.Parse(parts[0]);
    var k = int.Parse(parts[1]);
    var t = Console.ReadLine()!;

    var arr = t.ToCharArray();
    for (var i = k - 1; i < n; i++) {
      var c = arr[i];
      if (c >= 'a' && c <= 'z') arr[i] = (char)(c - 'a' + 'A');
      else arr[i] = (char)(c - 'A' + 'a');
    }

    Console.WriteLine(new string(arr));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, k; cin >> n >> k;
  string t; cin >> t;

  for (int i = k - 1; i < n; i++) {
    char c = t[i];
    if (c >= 'a' && c <= 'z') t[i] = char(c - 'a' + 'A');
    else t[i] = char(c - 'A' + 'a');
  }

  cout << t << "\n";
  return 0;
}
```
