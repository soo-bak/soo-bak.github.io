## 문제 링크
[21613번 - Silent Auction](https://www.acmicpc.net/problem/21613)

## 설명
입찰 순서대로 주어진 이름과 금액 중 가장 큰 금액을 낸 사람을 찾는 문제입니다.

<br>

## 접근법
먼저 현재 최고 입찰 금액과 이름을 기록합니다.

다음으로 각 입찰을 확인하며 더 큰 금액이 나오면 갱신합니다.

마지막으로 기록된 이름을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var bestName = "";
    var bestBid = -1;

    for (var i = 0; i < n; i++) {
      var name = Console.ReadLine()!;
      var bid = int.Parse(Console.ReadLine()!);
      if (bid > bestBid) {
        bestBid = bid;
        bestName = name;
      }
    }

    Console.WriteLine(bestName);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;

  string bestName = "";
  int bestBid = -1;

  for (int i = 0; i < n; i++) {
    string name; int bid; cin >> name >> bid;
    if (bid > bestBid) {
      bestBid = bid;
      bestName = name;
    }
  }

  cout << bestName << "\n";
  return 0;
}
```
