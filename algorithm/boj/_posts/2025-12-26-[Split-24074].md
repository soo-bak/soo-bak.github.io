## 문제 링크
[24074번 - 分割 (Split)](https://www.acmicpc.net/problem/24074)

## 설명
수열의 최댓값을 기준으로 앞과 뒤의 합을 각각 출력하는 문제입니다.

<br>

## 접근법
먼저 최댓값과 그 위치를 찾습니다.

다음으로 그 위치 앞의 합과 뒤의 합을 각각 계산합니다.

마지막으로 두 값을 줄마다 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Console.ReadLine()!.Split();

    var maxVal = -1;
    var maxIdx = 0;
    var arr = new int[n];
    for (var i = 0; i < n; i++) {
      var v = int.Parse(parts[i]);
      arr[i] = v;
      if (v > maxVal) { maxVal = v; maxIdx = i; }
    }

    var left = 0;
    for (var i = 0; i < maxIdx; i++)
      left += arr[i];
    var right = 0;
    for (var i = maxIdx + 1; i < n; i++)
      right += arr[i];

    Console.WriteLine(left);
    Console.WriteLine(right);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi arr(n);
  int maxVal = -1;
  int maxIdx = 0;
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    arr[i] = v;
    if (v > maxVal) { maxVal = v; maxIdx = i; }
  }

  int left = 0;
  for (int i = 0; i < maxIdx; i++)
    left += arr[i];
  int right = 0;
  for (int i = maxIdx + 1; i < n; i++)
    right += arr[i];

  cout << left << "\n";
  cout << right << "\n";

  return 0;
}
```
