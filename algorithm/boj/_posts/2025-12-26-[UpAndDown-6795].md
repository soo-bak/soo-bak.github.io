## 문제 링크
[6795번 - Up and Down](https://www.acmicpc.net/problem/6795)

## 설명
니키와 바이런의 이동 규칙이 주어질 때 s걸음 뒤 더 멀리 간 사람을 출력하는 문제입니다.

<br>

## 접근법
먼저 각 사람의 한 주기 이동 패턴을 구성합니다.

다음으로 s걸음까지 시뮬레이션하며 이동 거리를 누적합니다.

마지막으로 두 거리를 비교해 결과를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static int Move(int forward, int back, int steps) {
    var pos = 0;
    var cycle = forward + back;
    for (var i = 1; i <= steps; i++) {
      var t = (i - 1) % cycle;
      if (t < forward) pos++;
      else pos--;
    }
    return pos;
  }

  static void Main() {
    var a = int.Parse(Console.ReadLine()!);
    var b = int.Parse(Console.ReadLine()!);
    var c = int.Parse(Console.ReadLine()!);
    var d = int.Parse(Console.ReadLine()!);
    var s = int.Parse(Console.ReadLine()!);

    var nPos = Move(a, b, s);
    var bPos = Move(c, d, s);

    if (nPos > bPos) Console.WriteLine("Nikky");
    else if (nPos < bPos) Console.WriteLine("Byron");
    else Console.WriteLine("Tied");
  }
}
```

### C++

#include <bits/stdc++.h>
using namespace std;

int movePos(int forward, int back, int steps) {
  int pos = 0;
  int cycle = forward + back;
  for (int i = 1; i <= steps; i++) {
    int t = (i - 1) % cycle;
    if (t < forward) pos++;
    else pos--;
  }
  return pos;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, c, d, s;
  cin >> a >> b >> c >> d >> s;

  int nPos = movePos(a, b, s);
  int bPos = movePos(c, d, s);

  if (nPos > bPos) cout << "Nikky\n";
  else if (nPos < bPos) cout << "Byron\n";
  else cout << "Tied\n";
  return 0;
}
