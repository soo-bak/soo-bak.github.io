## 문제 링크
[26552번 - Zero](https://www.acmicpc.net/problem/26552)

## 설명
0이 포함되지 않는 정수만 순서대로 적을 때, 주어진 수 다음에 오는 값을 구하는 문제입니다.

<br>

## 접근법
먼저 수를 1씩 증가시키며 0이 포함되어 있는지 확인합니다.

다음으로 0이 없는 가장 작은 수를 찾으면 바로 출력합니다.

마지막으로 각 테스트 케이스마다 위 과정을 반복합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static bool HasZero(int x) {
    while (x > 0) {
      if (x % 10 == 0) return true;
      x /= 10;
    }
    return false;
  }

  static void Main() {
    var parts = Console.In.ReadToEnd().Split();
    var idx = 0;
    var t = int.Parse(parts[idx++]);
    var sb = new StringBuilder();

    for (var tc = 0; tc < t; tc++) {
      var k = int.Parse(parts[idx++]);
      var cur = k + 1;
      while (HasZero(cur))
        cur++;
      sb.AppendLine(cur.ToString());
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

bool hasZero(int x) {
  while (x > 0) {
    if (x % 10 == 0) return true;
    x /= 10;
  }
  return false;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;

  for (int tc = 0; tc < t; tc++) {
    int k; cin >> k;
    int cur = k + 1;
    while (hasZero(cur))
      cur++;
    
    cout << cur << "\n";
  }

  return 0;
}
```
