## 문제 링크
[27335번 - 年齢の差 (Age Difference)](https://www.acmicpc.net/problem/27335)

## 설명
주민들의 나이가 주어질 때, 각 주민마다 다른 주민과의 나이 차이 중 최댓값을 구하는 문제입니다.

<br>

## 접근법
전체에서 가장 어린 나이와 가장 많은 나이를 먼저 구합니다.

각 주민의 답은 가장 어린 사람과의 차이 또는 가장 나이 많은 사람과의 차이 중 큰 값입니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Linq;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var arr = Console.ReadLine()!.Split().Select(long.Parse).ToArray();

    long minA = long.MaxValue, maxA = long.MinValue;
    foreach (var v in arr) {
      if (v < minA) minA = v;
      if (v > maxA) maxA = v;
    }

    var sb = new StringBuilder();
    foreach (var v in arr) {
      var diff = Math.Max(maxA - v, v - minA);
      sb.AppendLine(diff.ToString());
    }
    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef vector<ll> vll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vll a(n);
  ll mn = LLONG_MAX, mx = LLONG_MIN;
  for (int i = 0; i < n; i++) {
    cin >> a[i];
    mn = min(mn, a[i]);
    mx = max(mx, a[i]);
  }

  for (int i = 0; i < n; i++) {
    ll diff = max(mx - a[i], a[i] - mn);
    cout << diff << "\n";
  }

  return 0;
}
```
