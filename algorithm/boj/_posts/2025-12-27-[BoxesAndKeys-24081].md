## 문제 링크
[24081번 - 箱と鍵 (Boxes and Keys)](https://www.acmicpc.net/problem/24081)

## 설명
각 상자와 열쇠에 숫자가 적혀 있습니다. 상자에 적힌 숫자와 같은 열쇠가 있으면 그 상자를 열 수 있으며, 열쇠는 재사용 가능합니다. 열 수 있는 상자의 최대 개수를 구하는 문제입니다.

<br>

## 접근법
열쇠에 적힌 숫자를 집합으로 저장합니다.

각 상자의 숫자가 집합에 있으면 열 수 있으므로, 조건을 만족하는 상자 개수를 세어 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = int.Parse(parts[0]);
    var m = int.Parse(parts[1]);

    var boxes = Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse);
    var keys = new HashSet<int>(Array.ConvertAll(Console.ReadLine()!.Split(), int.Parse));

    var cnt = 0;
    foreach (var a in boxes) {
      if (keys.Contains(a)) cnt++;
    }

    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;
  vi a(n);
  for (int i = 0; i < n; i++)
    cin >> a[i];
  unordered_set<int> keys;
  for (int j = 0; j < m; j++) {
    int b; cin >> b;
    keys.insert(b);
  }

  int cnt = 0;
  for (int x : a) {
    if (keys.count(x)) cnt++;
  }
  cout << cnt << "\n";

  return 0;
}
```
