## 문제 링크
[14963번 - Cezar](https://www.acmicpc.net/problem/14963)

## 설명
블랙잭 덱에서 이미 뽑은 카드들의 합이 주어집니다. 남은 덱에서 한 장을 더 뽑았을 때 21을 초과할 확률이 높으면 뽑지 않고, 그렇지 않으면 뽑는 문제입니다.

<br>

## 접근법
각 카드 값의 남은 장수를 관리합니다. 이미 뽑은 카드를 반영해 장수를 감소시키고 합을 구합니다.

21에서 현재 합을 뺀 값보다 큰 카드 수와 이하인 카드 수를 비교해 결과를 출력합니다.

- - -

## Code

### C#
```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);

    var cnt = new Dictionary<int, int>();
    for (var v = 2; v <= 9; v++) cnt[v] = 4;
    cnt[10] = 16;
    cnt[11] = 4;

    var sum = 0;
    for (var i = 0; i < n; i++) {
      var v = int.Parse(Console.ReadLine()!);
      sum += v;
      cnt[v]--;
    }

    var x = 21 - sum;
    var bigger = 0;
    var smaller = 0;
    foreach (var kv in cnt) {
      if (kv.Key > x) bigger += kv.Value;
      else smaller += kv.Value;
    }

    Console.WriteLine(bigger >= smaller ? "DOSTA" : "VUCI");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;

  map<int, int> cnt;
  for (int v = 2; v <= 9; v++)
    cnt[v] = 4;
  cnt[10] = 16;
  cnt[11] = 4;

  int sum = 0;
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    sum += v;
    cnt[v]--;
  }

  int x = 21 - sum;
  int bigger = 0, smaller = 0;
  for (auto& [v, c] : cnt) {
    if (v > x) bigger += c;
    else smaller += c;
  }

  cout << (bigger >= smaller ? "DOSTA" : "VUCI") << "\n";

  return 0;
}
```
