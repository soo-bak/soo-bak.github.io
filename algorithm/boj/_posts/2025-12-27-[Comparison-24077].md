## 문제 링크
[24077번 - 比較 (Comparison)](https://www.acmicpc.net/problem/24077)

## 설명
길이 N의 수열 A와 길이 M의 수열 B가 주어집니다.

Ai ≤ Bj 를 만족하는 모든 (i, j) 쌍의 개수를 구하는 문제입니다.

<br>

## 접근법
먼저 A의 각 원소에 대해 B의 모든 원소를 비교하며, Ai ≤ Bj 인 경우에 카운트를 늘립니다.

모든 비교가 끝나면 누적된 카운트를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split((char[])null, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var m = int.Parse(parts[idx++]);

    var a = new int[n];
    for (var i = 0; i < n; i++) a[i] = int.Parse(parts[idx++]);

    var b = new int[m];
    for (var j = 0; j < m; j++) b[j] = int.Parse(parts[idx++]);

    long count = 0;
    for (var i = 0; i < n; i++) {
      for (var j = 0; j < m; j++) {
        if (a[i] <= b[j]) count++;
      }
    }

    Console.WriteLine(count);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;

  vi a(n), b(m);
  for (int i = 0; i < n; i++)
    cin >> a[i];
  for (int j = 0; j < m; j++)
    cin >> b[j];

  ll count = 0;
  for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
      if (a[i] <= b[j]) count++;
    }
  }

  cout << count << "\n";
  return 0;
}
```
