## 문제 링크
[20224번 - Count Down Up 2020](https://www.acmicpc.net/problem/20224)

## 설명
여러 수열이 주어질 때, 각 수열에서 2 0 2 0이 순서대로 연속 등장하는 횟수를 셉니다. 겹쳐도 각각 카운트합니다. 입력은 0이 나오면 종료합니다.

<br>

## 접근법
수열을 처음부터 끝까지 슬라이딩하며 연속한 네 원소가 2, 0, 2, 0인지 확인합니다. 겹치는 경우도 각각 카운트합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split((char[])null, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var sb = new StringBuilder();

    while (idx < parts.Length) {
      var n = int.Parse(parts[idx++]);
      if (n == 0) break;

      var arr = new int[n];
      for (var i = 0; i < n; i++)
        arr[i] = int.Parse(parts[idx++]);

      var cnt = 0;
      for (var i = 0; i + 3 < n; i++) {
        if (arr[i] == 2 && arr[i + 1] == 0 && arr[i + 2] == 2 && arr[i + 3] == 0)
          cnt++;
      }

      sb.AppendLine(cnt.ToString());
    }

    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  while (cin >> n && n) {
    vi a(n);
    for (int i = 0; i < n; i++)
      cin >> a[i];
    int cnt = 0;
    for (int i = 0; i + 3 < n; i++) {
      if (a[i] == 2 && a[i + 1] == 0 && a[i + 2] == 2 && a[i + 3] == 0)
        cnt++;
    }
    cout << cnt << "\n";
  }

  return 0;
}
```
