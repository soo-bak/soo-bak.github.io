## 문제 링크
[21617번 - Crazy Fencing](https://www.acmicpc.net/problem/21617)

## 설명
울타리가 여러 판자로 이루어져 있고, 각 판자의 왼쪽 높이, 오른쪽 높이, 폭이 주어집니다. 각 판자는 사다리꼴 모양이며, 모든 판자 넓이의 합을 구하는 문제입니다.

<br>

## 접근법
각 판자의 넓이는 사다리꼴 공식으로 계산합니다. 왼쪽 높이와 오른쪽 높이의 평균에 폭을 곱하면 됩니다.

모든 판자에 대해 이 값을 더하여 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var h = Console.ReadLine()!.Split().Select(int.Parse).ToArray();
    var w = Console.ReadLine()!.Split().Select(int.Parse).ToArray();

    var area = 0.0;
    for (var i = 0; i < n; i++) {
      area += (h[i] + h[i + 1]) * 0.5 * w[i];
    }

    Console.WriteLine(area.ToString("0.######"));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<double> vd;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vd h(n + 1), w(n);
  for (int i = 0; i <= n; i++)
    cin >> h[i];
  for (int i = 0; i < n; i++)
    cin >> w[i];

  double area = 0.0;
  for (int i = 0; i < n; i++) {
    area += (h[i] + h[i + 1]) * 0.5 * w[i];
  }

  cout << fixed << setprecision(6) << area << "\n";

  return 0;
}
```
