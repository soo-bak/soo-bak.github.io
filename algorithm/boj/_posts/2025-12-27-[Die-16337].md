## 문제 링크
[16337번 - Die](https://www.acmicpc.net/problem/16337)

## 설명
3x3 패턴이 주사위 윗면을 나타낼 때, 어떤 눈인지 판별하는 문제입니다. 회전된 상태일 수 있으며, 어떤 눈과도 맞지 않으면 unknown을 출력합니다.

<br>

## 접근법
각 눈의 가능한 모든 회전 패턴을 미리 문자열로 저장해둡니다.

입력 세 줄을 하나의 문자열로 합쳐 저장된 패턴과 비교해 일치하는 눈을 출력합니다.

<br>

## Code

### C#
```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var mp = new Dictionary<string, string> {
      {"::::o::::", "1"},
      {"o:::::::o", "2"},
      {"::o:::o::", "2"},
      {"o:::o:::o", "3"},
      {"::o:o:o::", "3"},
      {"o:o:::o:o", "4"},
      {"o:o:o:o:o", "5"},
      {"ooo:::ooo", "6"},
      {"o:oo:oo:o", "6"}
    };

    var a = Console.ReadLine()!;
    var b = Console.ReadLine()!;
    var c = Console.ReadLine()!;
    var key = a + b + c;

    Console.WriteLine(mp.TryGetValue(key, out var val) ? val : "unknown");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  unordered_map<string, string> mp = {
    {"::::o::::", "1"},
    {"o:::::::o", "2"},
    {"::o:::o::", "2"},
    {"o:::o:::o", "3"},
    {"::o:o:o::", "3"},
    {"o:o:::o:o", "4"},
    {"o:o:o:o:o", "5"},
    {"ooo:::ooo", "6"},
    {"o:oo:oo:o", "6"}
  };

  string a, b, c; cin >> a >> b >> c;
  string key = a + b + c;

  auto it = mp.find(key);
  cout << (it != mp.end() ? it->second : "unknown") << "\n";

  return 0;
}
```
