## 문제 링크
[5364번 - Escape Route](https://www.acmicpc.net/problem/5364)

## 설명
현재 위치와 여러 행성 좌표가 주어질 때, 가장 가까운 행성을 찾아 현재 좌표, 해당 행성 좌표, 거리를 출력하는 문제입니다.

<br>

## 접근법
각 행성까지의 거리 제곱을 계산해 최솟값을 갱신합니다.

제곱 비교로 최소를 찾고, 마지막에 제곱근을 적용해 거리를 구합니다.

<br>

## Code

### C#
```csharp
using System;
using System.Globalization;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var first = Console.ReadLine()!.Split();
    int x0 = int.Parse(first[0]), y0 = int.Parse(first[1]);

    int bestX = 0, bestY = 0;
    double bestDist2 = double.MaxValue;

    for (var i = 1; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      int x = int.Parse(parts[0]), y = int.Parse(parts[1]);
      var dx = x - x0;
      var dy = y - y0;
      double d2 = dx * (double)dx + dy * (double)dy;
      if (d2 < bestDist2) {
        bestDist2 = d2;
        bestX = x;
        bestY = y;
      }
    }

    var dist = Math.Sqrt(bestDist2);
    Console.WriteLine($"{x0} {y0}");
    Console.WriteLine($"{bestX} {bestY}");
    Console.WriteLine(dist.ToString("0.00", CultureInfo.InvariantCulture));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int x0, y0; cin >> x0 >> y0;

  int bestX = 0, bestY = 0;
  double bestD2 = numeric_limits<double>::max();

  for (int i = 1; i < n; i++) {
    int x, y; cin >> x >> y;
    int dx = x - x0;
    int dy = y - y0;
    double d2 = 1.0L * dx * dx + 1.0L * dy * dy;
    if (d2 < bestD2) {
      bestD2 = d2;
      bestX = x;
      bestY = y;
    }
  }

  double dist = sqrt(bestD2);
  cout << fixed << setprecision(2);
  cout << x0 << " " << y0 << "\n";
  cout << bestX << " " << bestY << "\n";
  cout << dist << "\n";

  return 0;
}
```
