## 문제 링크
[2609번 - 최대공약수와 최소공배수](https://www.acmicpc.net/problem/2609)

## 설명
두 자연수가 주어질 때 최대공약수와 최소공배수를 구하는 문제입니다.

<br>

## 접근법
유클리드 호제법을 사용하여 최대공약수를 구합니다. 두 수 a, b에서 a를 b로 나눈 나머지가 0이 될 때까지 재귀적으로 나누면 마지막 나눈 수가 최대공약수가 됩니다.

최소공배수는 두 수의 곱을 최대공약수로 나누어 구합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static int GetGcd(int a, int b) {
    if (b == 0) return a;
    return GetGcd(b, a % b);
  }

  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var nat1 = int.Parse(parts[0]);
    var nat2 = int.Parse(parts[1]);

    var gcd = GetGcd(nat1, nat2);
    var lcm = nat1 * nat2 / gcd;

    Console.WriteLine(gcd);
    Console.WriteLine(lcm);
  }
}
```

### C++
```cpp
#include <iostream>
using namespace std;

int getGcd(int a, int b) {
  if (b == 0) return a;
  return getGcd(b, a % b);
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int nat1, nat2;
  cin >> nat1 >> nat2;

  int gcd = getGcd(nat1, nat2);
  int lcm = nat1 * nat2 / gcd;

  cout << gcd << '\n' << lcm << '\n';

  return 0;
}
```
