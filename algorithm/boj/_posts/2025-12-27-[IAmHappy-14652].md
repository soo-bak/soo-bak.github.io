## 문제 링크
[14652번 - 나는 행복합니다~](https://www.acmicpc.net/problem/14652)

## 설명
관중석이 왼쪽 위부터 오른쪽으로 번호가 매겨져 있을 때, 좌석 번호로부터 좌표를 구하는 문제입니다.

<br>

## 접근법
좌석 번호를 열 개수로 나누면 행 번호, 나머지가 열 번호가 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var n = int.Parse(parts[0]);
    var m = int.Parse(parts[1]);
    var k = int.Parse(parts[2]);

    var r = k / m;
    var c = k % m;
    Console.WriteLine($"{r} {c}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m, k; cin >> n >> m >> k;

  int r = k / m;
  int c = k % m;
  cout << r << " " << c << "\n";

  return 0;
}
```
