## 문제 링크
[24076번 - IOI 文字列 (IOI String)](https://www.acmicpc.net/problem/24076)

## 설명
길이가 홀수인 문자열 S를, 1번째 문자가 I이고 I와 O가 교대로 나오는 IOI 패턴으로 만들기 위해 변경해야 하는 최소 문자 수를 구합니다. 각 문자는 임의의 대문자로 바꿀 수 있습니다.

<br>

## 접근법
목표 패턴은 인덱스가 짝수(0-based)면 I, 홀수면 O입니다. 각 위치를 비교해 다른 경우를 카운트하면 그 합이 최소 변경 횟수입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;

    var diff = 0;
    for (var i = 0; i < n; i++) {
      var expect = (i % 2 == 0) ? 'I' : 'O';
      if (s[i] != expect) diff++;
    }

    Console.WriteLine(diff);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  int diff = 0;
  for (int i = 0; i < n; i++) {
    char expect = (i % 2 == 0) ? 'I' : 'O';
    if (s[i] != expect) diff++;
  }

  cout << diff << "\n";
  return 0;
}
```
