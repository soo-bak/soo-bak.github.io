## 문제 링크
[3276번 - ICONS](https://www.acmicpc.net/problem/3276)

## 설명
N개의 조약돌을 행과 열로 배열할 때, 행의 수와 열의 수의 합이 최소가 되도록 하는 문제입니다.

<br>

## 접근법
행의 수 r을 1부터 N까지 순회하며 각 경우에 필요한 열의 수를 계산합니다. 열의 수는 N을 r로 나눈 값이며, 나머지가 있으면 1을 더해야 합니다.

이렇게 구한 r + c의 합이 최소가 되는 경우를 찾아 출력합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);

    var minSum = int.MaxValue;
    var ansR = 0;
    var ansC = 0;

    for (var r = 1; r <= n; r++) {
      var c = (n + r - 1) / r;
      if (r + c < minSum) {
        minSum = r + c;
        ansR = r;
        ansC = c;
      }
    }

    Console.WriteLine($"{ansR} {ansC}");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n;
  cin >> n;

  int cntSum = INT_MAX, cntR = 0, cntC = 0;
  for (int r = 1; r <= n; r++) {
    int c = n / r;
    if (n % r != 0) c++;

    if (r + c < cntSum) {
      cntSum = r + c;
      cntR = r;
      cntC = c;
    }
  }

  cout << cntR << " " << cntC << "\n";

  return 0;
}
```
