## 문제 링크
[9287번 - Jenga](https://www.acmicpc.net/problem/9287)

## 설명
젠가 보드가 주어질 때, 어떤 행에서든 연속한 두 칸이 비어 있으면 탑이 넘어진 것으로 판정하는 문제입니다.

<br>

## 접근법
모든 행을 순회하며 연속한 빈 칸이 있는지 확인합니다.

하나라도 발견되면 Fallen, 끝까지 없으면 Standing을 출력합니다.

<br>

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var cs = 1; cs <= t; cs++) {
      var h = int.Parse(Console.ReadLine()!);
      var fallen = false;
      for (var i = 0; i < h; i++) {
        var row = Console.ReadLine()!;
        if (!fallen && row.Contains("00")) fallen = true;
      }
      Console.WriteLine($"Case {cs}: {(fallen ? "Fallen" : "Standing")}");
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  for (int cs = 1; cs <= t; cs++) {
    int h; cin >> h;
    bool fallen = false;
    for (int i = 0; i < h; i++) {
      string row;
      cin >> row;
      if (!fallen && row.find("00") != string::npos) fallen = true;
    }
    cout << "Case " << cs << ": " << (fallen ? "Fallen" : "Standing") << "\n";
  }

  return 0;
}
```
