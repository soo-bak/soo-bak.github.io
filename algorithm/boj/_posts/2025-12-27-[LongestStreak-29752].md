## 문제 링크
[29752번 - 최장 스트릭](https://www.acmicpc.net/problem/29752)

## 설명
N일 동안 매일 푼 문제 수가 주어질 때, 연속으로 1문제 이상 푼 최대 일수를 구하는 문제입니다.

<br>

## 접근법
각 날의 문제 수를 순회하며 현재 스트릭을 관리합니다.

문제를 푼 날이면 현재 스트릭을 증가시키고, 문제를 안 푼 날이면 스트릭을 초기화합니다. 그 과정에서 최대 스트릭을 갱신합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var parts = Console.ReadLine()!.Split();

    var maxStreak = 0;
    var curStreak = 0;
    for (var i = 0; i < n; i++) {
      var s = int.Parse(parts[i]);
      if (s > 0) {
        curStreak++;
        maxStreak = Math.Max(maxStreak, curStreak);
      } else curStreak = 0;
    }

    Console.WriteLine(maxStreak);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;

  int maxStreak = 0, curStreak = 0;
  for (int i = 0; i < n; i++) {
    int s; cin >> s;
    if (s > 0) {
      curStreak++;
      maxStreak = max(maxStreak, curStreak);
    } else curStreak = 0;
  }

  cout << maxStreak << "\n";

  return 0;
}
```
