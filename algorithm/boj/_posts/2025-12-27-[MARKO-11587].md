## 문제 링크
[11587번 - MARKO](https://www.acmicpc.net/problem/11587)

## 설명
T9 키패드에서 각 문자는 특정 숫자에 대응될 때, 주어진 숫자열과 길이가 같고 숫자 매핑이 일치하는 사전 단어의 개수를 구하는 문제입니다.

<br>

## 접근법
문자에서 숫자로의 매핑 테이블을 미리 만들어 둡니다.

각 사전 단어를 숫자열로 변환하고, 주어진 숫자열과 같은지 비교하여 개수를 셉니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var words = new List<string>(n);
    for (var i = 0; i < n; i++) words.Add(Console.ReadLine()!);
    var target = Console.ReadLine()!;

    var map = new Dictionary<char, char>();
    foreach (var c in "abc")
      map[c] = '2';
    foreach (var c in "def")
      map[c] = '3';
    foreach (var c in "ghi")
      map[c] = '4';
    foreach (var c in "jkl")
      map[c] = '5';
    foreach (var c in "mno")
      map[c] = '6';
    foreach (var c in "pqrs")
      map[c] = '7';
    foreach (var c in "tuv")
      map[c] = '8';
    foreach (var c in "wxyz")
      map[c] = '9';

    var count = 0;
    foreach (var w in words) {
      if (w.Length != target.Length) continue;
      var ok = true;
      for (var i = 0; i < w.Length; i++) {
        if (map[w[i]] != target[i]) { ok = false; break; }
      }
      if (ok) count++;
    }

    Console.WriteLine(count);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<string> vs;
typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vs words(n);
  for (int i = 0; i < n; i++)
    cin >> words[i];
  string target; cin >> target;

  vi mp(26);
  auto setMap = [&](string letters, char digit) {
    for (char c : letters)
      mp[c - 'a'] = digit;
  };
  setMap("abc", '2');
  setMap("def", '3');
  setMap("ghi", '4');
  setMap("jkl", '5');
  setMap("mno", '6');
  setMap("pqrs", '7');
  setMap("tuv", '8');
  setMap("wxyz", '9');

  int ans = 0;
  for (auto &w : words) {
    if ((int)w.size() != (int)target.size()) continue;
    bool ok = true;
    for (int i = 0; i < (int)w.size(); i++) {
      if (mp[w[i] - 'a'] != target[i]) { ok = false; break; }
    }
    if (ok) ans++;
  }

  cout << ans << "\n";

  return 0;
}
```
