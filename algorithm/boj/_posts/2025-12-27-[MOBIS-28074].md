## 문제 링크
[28074번 - 모비스](https://www.acmicpc.net/problem/28074)

## 설명
주어진 문자열에 M, O, B, I, S가 각각 하나 이상 포함되어 있는지 확인하는 문제입니다.

<br>

## 접근법
필요한 문자들의 등장 횟수를 세고, 주어진 문자열에서 각 문자의 등장 횟수와 비교합니다.

필요한 문자가 모두 충분히 있으면 YES, 아니면 NO를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var mobis = "MOBIS";
    var need = new int[26];
    foreach (var c in mobis)
      need[c - 'A']++;

    var s = Console.ReadLine()!;
    var have = new int[26];
    foreach (var c in s)
      have[c - 'A']++;

    for (var i = 0; i < 26; i++) {
      if (need[i] > have[i]) {
        Console.WriteLine("NO");
        return;
      }
    }
    Console.WriteLine("YES");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  vi need(26, 0);
  string mobis = "MOBIS";
  for (char c : mobis)
    need[c - 'A']++;

  string s; cin >> s;
  vi have(26, 0);
  for (char c : s)
    have[c - 'A']++;

  for (int i = 0; i < 26; i++) {
    if (need[i] > have[i]) {
      cout << "NO\n";
      return 0;
    }
  }
  cout << "YES\n";

  return 0;
}
```
