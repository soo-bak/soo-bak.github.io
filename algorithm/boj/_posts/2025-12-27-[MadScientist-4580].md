## 문제 링크
[4580번 - Mad Scientist](https://www.acmicpc.net/problem/4580)

## 설명
길이 k인 누적 개수 수열이 주어집니다. 각 위치의 값은 해당 값 이하인 측정의 개수를 나타냅니다.

이를 이용해 원래의 비내림차순 측정 수열을 복원하는 문제입니다. 입력의 첫 값이 0이면 종료합니다.

<br>

## 접근법
연속한 두 누적값의 차이가 해당 값의 등장 횟수입니다. 이전 누적값을 0으로 시작해서, 현재 누적값에서 이전 값을 빼면 해당 값이 몇 번 나타나는지 알 수 있습니다.

등장 횟수만큼 해당 값을 출력 목록에 추가하고, 이전 값을 갱신하며 진행합니다.

<br>

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split((char[])null, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var sb = new StringBuilder();

    while (idx < parts.Length) {
      var k = int.Parse(parts[idx++]);
      if (k == 0)
        break;

      var p = new int[k + 1];
      for (var i = 1; i <= k; i++)
        p[i] = int.Parse(parts[idx++]);

      var prev = 0;
      var first = true;
      for (var j = 1; j <= k; j++) {
        var freq = p[j] - prev;
        for (var t = 0; t < freq; t++) {
          if (!first) sb.Append(' ');
          first = false;
          sb.Append(j);
        }
        prev = p[j];
      }
      sb.AppendLine();
    }

    Console.Write(sb);
  }
}
```

### C++

#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int k;
  while (cin >> k && k) {
    vi p(k + 1, 0);
    for (int i = 1; i <= k; i++)
      cin >> p[i];

    int prev = 0;
    bool first = true;
    for (int j = 1; j <= k; j++) {
      int freq = p[j] - prev;
      for (int t = 0; t < freq; t++) {
        if (!first)
          cout << ' ';
        first = false;
        cout << j;
      }
      prev = p[j];
    }
    cout << "\n";
  }

  return 0;
}
