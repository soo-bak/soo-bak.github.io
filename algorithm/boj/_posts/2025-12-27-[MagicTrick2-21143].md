## 문제 링크
[21143번 - Magic Trick](https://www.acmicpc.net/problem/21143)

## 설명
소문자가 적힌 카드들이 일렬로 놓여 있습니다. 관객이 두 카드의 위치를 바꾸거나 아무것도 하지 않았을 때, 최종 배치만 보고 관객이 무엇을 했는지 항상 맞힐 수 있는지 판별하는 문제입니다.

<br>

## 접근법
같은 문자가 두 개 이상 있으면 그 카드들을 서로 바꿔도 배치가 변하지 않아 아무것도 안 한 경우와 구분할 수 없습니다. 모든 문자가 서로 다르면 항상 맞힐 수 있습니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Collections.Generic;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    var seen = new HashSet<char>();
    foreach (var c in s) {
      if (!seen.Add(c)) {
        Console.WriteLine(0);
        return;
      }
    }
    Console.WriteLine(1);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s; cin >> s;
  vi cnt(26, 0);
  for (char c : s) {
    int idx = c - 'a';
    cnt[idx]++;
    if (cnt[idx] > 1) {
      cout << 0 << "\n";
      return 0;
    }
  }
  cout << 1 << "\n";

  return 0;
}
```
