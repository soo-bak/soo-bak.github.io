## 문제 링크
[18413번 - 最頻値 (Mode)](https://www.acmicpc.net/problem/18413)

## 설명
길이 N의 수열에서 1..M 값의 등장 횟수를 세어, 빈도 배열 B 중 최댓값을 출력하는 문제입니다.

<br>

## 접근법
크기 M+1의 배열에 등장 횟수를 누적하고, 그 중 최댓값을 출력합니다.

N, M이 100 이하이므로 단순 카운트로 충분합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split((char[])null, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    var m = int.Parse(parts[idx++]);

    var cnt = new int[m + 1];
    var best = 0;
    for (var i = 0; i < n; i++) {
      var v = int.Parse(parts[idx++]);
      cnt[v]++;
      if (cnt[v] > best) best = cnt[v];
    }

    Console.WriteLine(best);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;
  vi cnt(m + 1, 0);
  int best = 0;
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    cnt[v]++;
    if (cnt[v] > best) best = cnt[v];
  }

  cout << best << "\n";
  return 0;
}
```
