## 문제 링크
[21945번 - Palindromes](https://www.acmicpc.net/problem/21945)

## 설명
n개의 정수가 주어질 때, 팰린드롬인 수만 골라 모두 더한 값을 출력하는 문제입니다.

<br>

## 접근법
각 수를 문자열로 바꿔 앞뒤가 같은지 확인합니다.

팰린드롬이면 합계에 더하고, 마지막에 합을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static bool IsPal(int x) {
    var s = x.ToString();
    var len = s.Length;
    for (var i = 0; i < len / 2; i++) {
      if (s[i] != s[len - 1 - i]) return false;
    }
    return true;
  }

  static void Main() {
    var parts = Console.In.ReadToEnd().Split((char[])null, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var n = int.Parse(parts[idx++]);
    long sum = 0;
    for (var i = 0; i < n; i++) {
      var v = int.Parse(parts[idx++]);
      if (IsPal(v)) sum += v;
    }
    Console.WriteLine(sum);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

bool isPal(int x) {
  string s = to_string(x);
  int l = 0, r = (int)s.size() - 1;
  while (l < r) {
    if (s[l] != s[r]) return false;
    l++; r--;
  }
  return true;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  ll sum = 0;
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    if (isPal(v)) sum += v;
  }

  cout << sum << "\n";
  return 0;
}
```
