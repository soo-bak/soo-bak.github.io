## 문제 링크
[20660번 - Pizza](https://www.acmicpc.net/problem/20660)

## 설명
미르코가 싫어하는 토핑 목록이 주어질 때, 그 토핑이 하나도 없는 피자가 몇 개인지 구하는 문제입니다.

<br>

## 접근법
싫어하는 토핑을 집합으로 표시한 뒤, 각 피자의 토핑을 확인하며 하나라도 포함되면 제외합니다.

모두 통과하는 피자만 카운트하면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.In.ReadToEnd().Split((char[])null, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;

    var n = int.Parse(parts[idx++]);
    var dislike = new bool[101];
    for (var i = 0; i < n; i++)
      dislike[int.Parse(parts[idx++])] = true;

    var m = int.Parse(parts[idx++]);
    var ok = 0;

    for (var i = 0; i < m; i++) {
      var k = int.Parse(parts[idx++]);
      var good = true;
      for (var j = 0; j < k; j++) {
        var v = int.Parse(parts[idx++]);
        if (dislike[v]) good = false;
      }
      if (good) ok++;
    }

    Console.WriteLine(ok);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<bool> vb;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vb dislike(101, false);
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    dislike[v] = true;
  }

  int m; cin >> m;
  int ok = 0;
  for (int i = 0; i < m; i++) {
    int k; cin >> k;
    bool good = true;
    for (int j = 0; j < k; j++) {
      int v; cin >> v;
      if (dislike[v]) good = false;
    }
    
    if (good) ok++;
  }

  cout << ok << "\n";

  return 0;
}
```
