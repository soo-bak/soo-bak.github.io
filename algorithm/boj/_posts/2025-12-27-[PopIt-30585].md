## 문제 링크
[30585번 - Поп-ит](https://www.acmicpc.net/problem/30585)

## 설명
h x w 크기의 팝잇 장난감이 있습니다. 각 칸은 0 또는 1 상태이고, 모든 칸을 같은 상태로 만들기 위해 최소 몇 개를 눌러야 하는지 구하는 문제입니다.

<br>

## 접근법
0의 개수와 1의 개수를 각각 세고, 둘 중 작은 값을 출력합니다.

적은 쪽을 모두 바꾸는 것이 최소 횟수이기 때문입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var tokens = Console.ReadLine()!.Split();
    var h = int.Parse(tokens[0]);

    var count0 = 0;
    var count1 = 0;
    for (var i = 0; i < h; i++) {
      var row = Console.ReadLine()!;
      foreach (var c in row) {
        if (c == '0') count0++;
        else count1++;
      }
    }

    Console.WriteLine(Math.Min(count0, count1));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int h, w; cin >> h >> w;

  int count0 = 0, count1 = 0;
  for (int i = 0; i < h; i++) {
    string row; cin >> row;
    for (char c : row) {
      if (c == '0') count0++;
      else count1++;
    }
  }

  cout << min(count0, count1) << "\n";

  return 0;
}
```
