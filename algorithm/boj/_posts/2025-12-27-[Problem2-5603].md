## 문제 링크
[5603번 - 問題2](https://www.acmicpc.net/problem/5603)

## 설명
숫자 문자열에서 같은 숫자가 연속한 구간을 개수와 숫자로 바꾸는 변환을 n번 반복하는 문제입니다.

<br>

## 접근법
왼쪽부터 같은 문자가 연속하는 구간의 길이를 세고, 그 길이와 해당 문자를 이어 붙입니다.

이 과정을 n번 반복한 결과를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var s = Console.ReadLine()!;

    for (var iter = 0; iter < n; iter++) {
      var sb = new StringBuilder();
      var i = 0;
      while (i < s.Length) {
        var j = i;
        while (j < s.Length && s[j] == s[i]) j++;
        var cnt = j - i;
        sb.Append(cnt);
        sb.Append(s[i]);
        i = j;
      }
      s = sb.ToString();
    }

    Console.WriteLine(s);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  string s; cin >> s;

  for (int iter = 0; iter < n; iter++) {
    string res;
    res.reserve(s.size() * 2);
    int i = 0, len = s.size();
    while (i < len) {
      int j = i;
      while (j < len && s[j] == s[i]) j++;
      int cnt = j - i;
      res += to_string(cnt);
      res += s[i];
      i = j;
    }
    s.swap(res);
  }

  cout << s << "\n";

  return 0;
}
```
