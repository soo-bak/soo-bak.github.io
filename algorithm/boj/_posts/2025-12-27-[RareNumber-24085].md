## 문제 링크
[24085번 - 希少な数 (Rare Number)](https://www.acmicpc.net/problem/24085)

## 설명
수열에서 가장 드물게 등장하는 정수를 찾는 문제입니다. 그런 수가 여러 개면 가장 작은 값을 출력합니다.

<br>

## 접근법
각 수의 등장 횟수를 센 뒤, 가장 적게 등장한 횟수를 찾습니다. 그 횟수만큼 등장하는 수 중 가장 작은 값을 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var arr = Console.ReadLine()!.Split().Select(int.Parse).ToArray();

    var cnt = new int[2001];
    foreach (var v in arr) cnt[v]++;

    var minFreq = int.MaxValue;
    for (var v = 1; v <= 2000; v++) {
      if (cnt[v] > 0 && cnt[v] < minFreq) minFreq = cnt[v];
    }

    for (var v = 1; v <= 2000; v++) {
      if (cnt[v] == minFreq) {
        Console.WriteLine(v);
        break;
      }
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  vi cnt(2001, 0);
  for (int i = 0; i < n; i++) {
    int v; cin >> v;
    cnt[v]++;
  }

  int minFreq = INT_MAX;
  for (int v = 1; v <= 2000; v++) {
    if (cnt[v] > 0 && cnt[v] < minFreq) minFreq = cnt[v];
  }

  for (int v = 1; v <= 2000; v++) {
    if (cnt[v] == minFreq) {
      cout << v << "\n";
      break;
    }
  }

  return 0;
}
```
