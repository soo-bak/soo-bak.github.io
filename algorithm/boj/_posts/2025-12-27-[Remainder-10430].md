## 문제 링크
[10430번 - 나머지](https://www.acmicpc.net/problem/10430)

## 설명
세 수가 주어질 때, 나머지 연산의 분배법칙과 관련된 네 가지 값을 구하는 문제입니다.

<br>

## 접근법
덧셈과 곱셈에 대해 나머지 연산을 직접 적용한 값과, 각각에 먼저 나머지를 취한 뒤 연산한 값을 구해 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var parts = Console.ReadLine()!.Split();
    var a = int.Parse(parts[0]);
    var b = int.Parse(parts[1]);
    var c = int.Parse(parts[2]);

    Console.WriteLine((a + b) % c);
    Console.WriteLine(((a % c) + (b % c)) % c);
    Console.WriteLine((a * b) % c);
    Console.WriteLine(((a % c) * (b % c)) % c);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int a, b, c; cin >> a >> b >> c;

  cout << (a + b) % c << "\n"
       << ((a % c) + (b % c)) % c << "\n"
       << (a * b) % c << "\n"
       << ((a % c) * (b % c)) % c << "\n";

  return 0;
}
```
