## 문제 링크
[4678번 - Skew Binary](https://www.acmicpc.net/problem/4678)

## 설명
스큐 이진수의 각 자릿수(왼쪽이 가장 높은 자리) 가중치는 \(2^{k+1}-1\)입니다. 문자열로 주어진 스큐 이진수를 10진수로 변환합니다. 입력은 0이 나오면 종료합니다.

<br>

## 접근법
길이가 L인 문자열에서 i번째(0-based, 왼쪽부터) 문자의 값 d에 대해 가중치는 \(2^{L-i}-1\)입니다. 이를 모두 더하면 10진수 값이 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    string? s;
    while ((s = Console.ReadLine()) != null) {
      if (s == "0") break;
      long res = 0;
      int len = s.Length;
      for (int i = 0; i < len; i++) {
        int d = s[i] - '0';
        long weight = (1L << (len - i)) - 1;
        res += d * weight;
      }
      Console.WriteLine(res);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s;
  while (cin >> s) {
    if (s == "0") break;
    int len = s.size();
    ll res = 0;
    for (int i = 0; i < len; i++) {
      int d = s[i] - '0';
      ll weight = (1LL << (len - i)) - 1;
      res += d * weight;
    }
    cout << res << "\n";
  }
  return 0;
}
```
