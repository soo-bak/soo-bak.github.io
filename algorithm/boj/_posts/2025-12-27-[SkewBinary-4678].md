## 문제 링크
[4678번 - Skew Binary](https://www.acmicpc.net/problem/4678)

## 설명
Skew 이진수를 10진수로 변환하는 문제입니다.

<br>

## 접근법
왼쪽부터 각 자릿수를 순회하며, 해당 위치의 가중치를 계산하여 숫자와 곱한 값을 누적합니다. 가중치는 2를 자릿수만큼 왼쪽 시프트한 값에서 1을 뺀 것입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    string? s;
    while ((s = Console.ReadLine()) != null) {
      if (s == "0") break;
      var res = 0L;
      var len = s.Length;
      for (var i = 0; i < len; i++) {
        var d = s[i] - '0';
        var weight = (1L << (len - i)) - 1;
        res += d * weight;
      }
      Console.WriteLine(res);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s;
  while (cin >> s) {
    if (s == "0") break;
    int len = s.size();
    ll res = 0;
    for (int i = 0; i < len; i++) {
      int d = s[i] - '0';
      ll weight = (1LL << (len - i)) - 1;
      res += d * weight;
    }
    cout << res << "\n";
  }

  return 0;
}
```
