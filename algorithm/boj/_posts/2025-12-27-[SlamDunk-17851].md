## 문제 링크
[17851번 - This Problem’s a Slam Dunk](https://www.acmicpc.net/problem/17851)

## 설명
두 팀의 선수 키 5개씩이 주어질 때, 각 팀을 키 내림차순 정렬해 같은 순위끼리 매칭합니다. State U 선수가 더 큰 매칭의 개수를 구하는 문제입니다.

<br>

## 접근법
각 팀 키를 정렬한 뒤, 같은 인덱스끼리 비교하며 State U 키가 더 큰 경우를 카운트합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Linq;

class Program {
  static void Main() {
    var a = Console.ReadLine()!.Split().Select(int.Parse).OrderByDescending(x => x).ToArray();
    var b = Console.ReadLine()!.Split().Select(int.Parse).OrderByDescending(x => x).ToArray();

    var cnt = 0;
    for (var i = 0; i < 5; i++) {
      if (a[i] > b[i]) cnt++;
    }
    Console.WriteLine(cnt);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  vi a(5), b(5);
  for (int i = 0; i < 5; i++)
    cin >> a[i];
  for (int i = 0; i < 5; i++)
    cin >> b[i];

  sort(a.begin(), a.end(), greater<int>());
  sort(b.begin(), b.end(), greater<int>());

  int cnt = 0;
  for (int i = 0; i < 5; i++) {
    if (a[i] > b[i]) cnt++;
  }

  cout << cnt << "\n";

  return 0;
}
```
