## 문제 링크
[6929번 - Dressing Up](https://www.acmicpc.net/problem/6929)

## 설명
홀수 정수 h가 주어지면 높이 h, 너비 2h인 보타이 모양을 출력하는 문제입니다. 윗부분부터 별의 개수가 1, 3, 5로 늘어나다가 중간을 지나면 다시 줄어드는 패턴입니다.

<br>

## 접근법
먼저 중간 행 인덱스를 계산합니다. 중간까지는 별 개수가 1, 3, 5로 증가하고, 중간 이후에는 다시 감소합니다.

다음으로 각 행에서 왼쪽 별, 중간 공백, 오른쪽 별 순서로 출력합니다. 공백 개수는 전체 너비에서 양쪽 별 개수를 뺀 값이며, 중간 줄에서는 공백이 없습니다.

<br>

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var h = int.Parse(Console.ReadLine()!);
    var mid = h / 2;
    var sb = new StringBuilder();

    for (var i = 0; i < h; i++) {
      var s = (i <= mid) ? 2 * i + 1 : 2 * (h - 1 - i) + 1;
      var spaces = 2 * h - 2 * s;
      sb.Append(new string('*', s));
      sb.Append(new string(' ', spaces));
      sb.Append(new string('*', s));
      sb.Append('\n');
    }

    Console.Write(sb.ToString());
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int h; cin >> h;

  int mid = h / 2;
  for (int i = 0; i < h; i++) {
    int s = (i <= mid) ? 2 * i + 1 : 2 * (h - 1 - i) + 1;
    int spaces = 2 * h - 2 * s;
    cout << string(s, '*') << string(spaces, ' ') << string(s, '*') << '\n';
  }

  return 0;
}
```
