## 문제 링크
[34563번 - 궁핍한 모그](https://www.acmicpc.net/problem/34563)

## 설명
N개의 가로선과 M개의 세로선이 수직으로 교차하는 격자를 만들기 위해 필요한 최소 커넥터 개수를 구하는 문제입니다.

<br>

## 접근법
먼저 기준이 될 가로선 하나를 잡고, 이 가로선과 모든 세로선을 커넥터로 고정하면 M개의 세로선 각도가 결정됩니다.

다음으로 남은 가로선들은 각자 이미 고정된 세로선 중 하나와만 고정하면 각도가 결정됩니다. 이때 필요한 커넥터 수는 N - 1개입니다.

따라서 총 필요한 커넥터 수는 M + (N - 1) = N + M - 1개입니다. 이는 각 직선이 최소 한 번씩은 고정되어야 하는 조건을 만족하는 최소 간선 수입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var t = int.Parse(Console.ReadLine()!);
    for (var tc = 0; tc < t; tc++) {
      var parts = Console.ReadLine()!.Split();
      var n = long.Parse(parts[0]);
      var m = long.Parse(parts[1]);
      Console.WriteLine(n + m - 1);
    }
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t; cin >> t;
  while (t--) {
    ll n, m; cin >> n >> m;
    cout << (n + m - 1) << "\n";
  }

  return 0;
}
```
