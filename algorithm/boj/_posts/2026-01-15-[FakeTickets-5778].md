## 문제 링크
[5778번 - Fake tickets](https://www.acmicpc.net/problem/5778)

## 설명
중복된 티켓 번호가 몇 개인지를 구하는 문제입니다.

<br>

## 접근법
주어지는 티켓의 목록에서, 같은 번호가 두 번 이상 등장한 번호를 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;
using System.IO;

class Program {
  static void Main() {
    var reader = new StreamReader(Console.OpenStandardInput());
    var writer = new StreamWriter(Console.OpenStandardOutput());
    var parts = reader.ReadToEnd().Split((char[])null, StringSplitOptions.RemoveEmptyEntries);
    var idx = 0;
    var sb = new StringBuilder();

    while (idx < parts.Length) {
      var n = int.Parse(parts[idx++]);
      var m = int.Parse(parts[idx++]);
      if (n == 0 && m == 0)
        break;

      var count = new int[n + 1];
      for (var i = 0; i < m; i++) {
        var t = int.Parse(parts[idx++]);
        count[t]++;
      }

      var result = 0;
      for (var i = 1; i <= n; i++) {
        if (count[i] >= 2)
          result++;
      }

      sb.Append(result).Append('\n');
    }

    writer.Write(sb.ToString());
    writer.Flush();
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  while (true) {
    int n, m; cin >> n >> m;
    if (n == 0 && m == 0)
      break;

    vi count(n + 1, 0);
    for (int i = 0; i < m; i++) {
      int t; cin >> t;
      count[t]++;
    }

    int result = 0;
    for (int i = 1; i <= n; i++) {
      if (count[i] >= 2)
        result++;
    }

    cout << result << "\n";
  }


  return 0;
}
```
