## 문제 링크
[24089번 - ボールの移動 (Moving Balls)](https://www.acmicpc.net/problem/24089)

## 설명
1부터 N까지 번호가 붙은 공과 상자가 있고, 처음에는 공 i가 상자 i에 있습니다.

M번에 걸쳐 공을 이동시키는 작업이 주어질 때, 모든 작업 이후 각 공이 들어 있는 상자의 번호를 구하는 문제입니다.

<br>

## 접근법
먼저 각 공의 현재 위치를 저장하는 배열을 초기화합니다. 처음에는 공 i가 상자 i에 있으므로 위치 배열을 i로 설정합니다.

이후, 각 이동 작업을 순서대로 처리합니다. 공을 상자로 이동시킬 때마다 해당 공의 위치 정보를 갱신합니다.

모든 이동 작업을 완료한 후, 각 공의 최종 위치를 순서대로 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);

    var loc = new int[n + 1];
    for (var i = 1; i <= n; i++)
      loc[i] = i;

    for (var i = 0; i < m; i++) {
      var parts = Console.ReadLine()!.Split();
      var x = int.Parse(parts[0]);
      var y = int.Parse(parts[1]);
      loc[x] = y;
    }

    var sb = new StringBuilder();
    for (var i = 1; i <= n; i++)
      sb.Append(loc[i]).Append('\n');
    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m; cin >> n >> m;

  vi loc(n + 1);
  for (int i = 1; i <= n; i++)
    loc[i] = i;

  for (int i = 0; i < m; i++) {
    int x, y; cin >> x >> y;
    loc[x] = y;
  }

  for (int i = 1; i <= n; i++)
    cout << loc[i] << "\n";

  return 0;
}
```
