## 문제 링크
[34237번 - 호참전](https://www.acmicpc.net/problem/34237)

## 설명
각 호참전 기록에서 성립 가능한 베팅의 개수를 구하는 문제입니다.

<br>

## 접근법
먼저 모든 베팅 정보를 배열에 저장합니다.

이후 각 호참전 기록에 대해 모든 베팅을 순회하며 성립 가능 여부를 확인합니다.

베팅 a:b가 성립하려면 시작 점수 x:y에서 경기 진행 중 a:b가 되는 순간이 있어야 합니다.

따라서 호랑이 점수 a가 시작 점수의 호랑이 점수 x 이상이고, 참새 점수 b가 시작 점수의 참새 점수 y 이상이어야 합니다.

또한 두 점수의 합 a+b가 총 경기 수 g 이하여야 합니다.

이러한 조건들을 모두 만족하는 베팅의 개수를 세어 출력합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var first = Console.ReadLine()!.Split();
    var n = int.Parse(first[0]);
    var m = int.Parse(first[1]);

    var a = new int[n];
    var b = new int[n];
    for (var i = 0; i < n; i++) {
      var parts = Console.ReadLine()!.Split();
      a[i] = int.Parse(parts[0]);
      b[i] = int.Parse(parts[1]);
    }

    var sb = new StringBuilder();
    for (var j = 0; j < m; j++) {
      var parts = Console.ReadLine()!.Split();
      var g = int.Parse(parts[0]);
      var x = int.Parse(parts[1]);
      var y = int.Parse(parts[2]);

      var cnt = 0;
      for (var i = 0; i < n; i++) {
        if (a[i] >= x && b[i] >= y && a[i] + b[i] <= g)
          cnt++;
      }
      sb.Append(cnt).Append('\n');
    }
    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

typedef vector<int> vi;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n, m;
  cin >> n >> m;
  vi a(n), b(n);
  for (int i = 0; i < n; i++) {
    cin >> a[i] >> b[i];
  }

  for (int j = 0; j < m; j++) {
    int g, x, y;
    cin >> g >> x >> y;
    int cnt = 0;
    for (int i = 0; i < n; i++) {
      if (a[i] >= x && b[i] >= y && a[i] + b[i] <= g) {
        cnt++;
      }
    }
    cout << cnt << "\n";
  }

  return 0;
}
```
