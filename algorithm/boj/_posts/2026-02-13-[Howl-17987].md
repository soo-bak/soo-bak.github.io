## 문제 링크
[17987번 - Howl](https://www.acmicpc.net/problem/17987)

## 설명
울음은 A, H, O, W 네 글자가 모두 한 번 이상 들어가고, W나 H가 연속되면 안 되며, H 다음에 W·A가 바로 올 수 없고, 첫 O 이후에는 A가 더 나오면 안 됩니다.

입력이 이미 이 조건을 만족하므로, 끝에 `O` 하나를 붙여 길이만 늘려 출력하면 되는 문제입니다.

<br>

## 접근법
입력이 이미 유효하므로 뒤에 `O` 하나만 덧붙이면 조건을 유지한 채 길이만 늘릴 수 있습니다. 한 줄을 읽어 `O`를 붙여 출력하면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var s = Console.ReadLine()!;
    Console.WriteLine(s + "O");
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string s;
  if (getline(cin, s))
    cout << s << 'O' << "\n";
  return 0;
}
```
