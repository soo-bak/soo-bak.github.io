## 문제 링크
[28702번 - FizzBuzz](https://www.acmicpc.net/problem/28702)

## 설명
FizzBuzz 출력에서 연속으로 나온 세 문자열이 주어질 때, 그 다음에 출력될 문자열을 구하는 문제입니다.

<br>

## 접근법
세 문자열 중 숫자가 하나 이상은 반드시 포함됩니다. 숫자 줄이 i번째 출력에 해당한다는 점을 이용해 해당 i를 복원한 뒤, i+3번째의 FizzBuzz 값을 출력하면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static string FizzBuzz(int x) {
    if (x % 15 == 0) return "FizzBuzz";
    if (x % 3 == 0) return "Fizz";
    if (x % 5 == 0) return "Buzz";
    return x.ToString();
  }

  static void Main() {
    var a = Console.ReadLine()!;
    var b = Console.ReadLine()!;
    var c = Console.ReadLine()!;

    int n;
    if (int.TryParse(a, out var v1)) n = v1 + 3;
    else if (int.TryParse(b, out var v2)) n = v2 + 2;
    else n = int.Parse(c) + 1;

    Console.WriteLine(FizzBuzz(n));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

string fizzbuzz(int x) {
  if (x % 15 == 0) return "FizzBuzz";
  if (x % 3 == 0) return "Fizz";
  if (x % 5 == 0) return "Buzz";
  return to_string(x);
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string a, b, c;
  getline(cin, a);
  getline(cin, b);
  getline(cin, c);

  int n;
  if (all_of(a.begin(), a.end(), ::isdigit)) n = stoi(a) + 3;
  else if (all_of(b.begin(), b.end(), ::isdigit)) n = stoi(b) + 2;
  else n = stoi(c) + 1;

  cout << fizzbuzz(n) << "\n";
  return 0;
}
```
