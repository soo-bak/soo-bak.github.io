## 문제 링크
[33515번 - 노트북 세 대를 가지고 왔다](https://www.acmicpc.net/problem/33515)

## 설명
다른 두 팀원의 풀이 시간이 주어질 때, 가장 빠른 사람이 쓰고 동률이면 형진이가 우선인 규칙에서 형진이가 자신의 노트북을 쓰기 위해 허용되는 최대 풀이 시간을 구하는 문제입니다.

<br>

## 접근법
형진이가 노트북을 쓰려면 자신의 시간이 두 팀원 중 최소 시간 이하이면 됩니다. 따라서 가능한 최대 시간은 `min(T1, T2)`입니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    var p = Console.ReadLine()!.Split();
    var t1 = int.Parse(p[0]);
    var t2 = int.Parse(p[1]);
    Console.WriteLine(Math.Min(t1, t2));
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int t1, t2; cin >> t1 >> t2;
  cout << min(t1, t2) << "\n";

  return 0;
}
```
