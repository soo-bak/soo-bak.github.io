## 문제 링크
[14659번 - 한조서열정리하고옴ㅋㅋ](https://www.acmicpc.net/problem/14659)

## 설명
각 봉우리의 활잡이는 오른쪽으로만 이동하며 자신보다 낮은 봉우리의 적만 처치할 수 있고, 처음 출발한 높이보다 높은 봉우리를 만나면 멈춥니다. 이때 한 활잡이가 처치할 수 있는 적의 최대 수를 구하는 문제입니다.

<br>

## 접근법
왼쪽부터 보면서 현재까지의 최고 높이를 가진 활잡이만 추적하면 됩니다. 현재 봉우리가 최고 높이보다 낮으면 그 활잡이는 처치 대상이므로 연속 처치 수를 1 늘리고, 더 높으면 새로운 활잡이가 시작되므로 연속 처치 수를 0으로 초기화합니다. 순회 중 최대 연속 처치 수를 답으로 갱신하면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;

class Program {
  static void Main() {
    int n = int.Parse(Console.ReadLine()!);
    var parts = Console.ReadLine()!.Split();

    int maxHeight = 0;
    int cur = 0;
    int ans = 0;

    for (int i = 0; i < n; i++) {
      int h = int.Parse(parts[i]);
      if (h > maxHeight) {
        maxHeight = h;
        cur = 0;
      } else {
        cur++;
        if (cur > ans)
          ans = cur;
      }
    }

    Console.WriteLine(ans);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  int max_height = 0;
  int cur = 0;
  int ans = 0;

  for (int i = 0; i < n; i++) {
    int h; cin >> h;
    if (h > max_height) {
      max_height = h;
      cur = 0;
    } else {
      cur++;
      ans = max(ans, cur);
    }
  }

  cout << ans << "\n";

  return 0;
}
```
