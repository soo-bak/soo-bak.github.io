## 문제 링크
[15612번 - Cube Bits](https://www.acmicpc.net/problem/15612)

## 설명
여러 개의 10진수가 주어질 때, 각 수를 앞의 0 없이 3진수로 바꾸어 같은 순서대로 출력하는 문제입니다.

<br>

## 접근법
각 수를 3으로 계속 나누면서 나머지를 모으면 3진수 표현을 얻을 수 있습니다. 나머지는 뒤에서부터 결정되므로 거꾸로 모은 뒤 뒤집어 출력하면 됩니다.

값이 0인 경우에는 그대로 `0`을 출력하면 됩니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static string ToTernary(long x) {
    if (x == 0)
      return "0";

    var sb = new StringBuilder();
    while (x > 0) {
      sb.Append((char)('0' + (x % 3)));
      x /= 3;
    }

    var arr = sb.ToString().ToCharArray();
    Array.Reverse(arr);
    return new string(arr);
  }

  static void Main() {
    var n = int.Parse(Console.ReadLine()!);
    var sb = new StringBuilder();
    for (var i = 0; i < n; i++) {
      var v = long.Parse(Console.ReadLine()!);
      sb.Append(ToTernary(v)).Append('\n');
    }
    Console.Write(sb);
  }
}
```

### C++
```cpp
#include <bits/stdc++.h>
using namespace std;

string toTernary(long long x) {
  if (x == 0)
    return "0";

  string s;
  while (x > 0) {
    s.push_back(char('0' + (x % 3)));
    x /= 3;
  }
  reverse(s.begin(), s.end());
  return s;
}

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  int n; cin >> n;
  while (n--) {
    long long v; cin >> v;
    cout << toTernary(v) << "\n";
  }

  return 0;
}
```
