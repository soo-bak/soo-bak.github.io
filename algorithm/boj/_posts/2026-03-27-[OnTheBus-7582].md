## 문제 링크
[7582번 - On the Bus](https://www.acmicpc.net/problem/7582)

## 설명
여러 개의 버스 운행 시나리오가 주어질 때, 각 정류장에서 승객이 내리고 남은 좌석만큼만 새 승객을 태운 뒤 마지막 승객 수를 구하는 문제입니다.

<br>

## 접근법
시나리오마다 현재 버스에 타고 있는 승객 수를 하나의 변수로 관리하면 됩니다.

각 정류장에서 먼저 내리는 승객 수를 뺍니다. 그 다음 남은 좌석 수를 계산하고, 기다리는 승객 중에서 그 좌석 수만큼만 태웁니다.

이 과정을 모든 정류장에 대해 반복한 뒤, 마지막 승객 수를 노선 번호와 함께 출력하면 됩니다. 입력은 `# 0`이 나오면 종료합니다.

<br>

- - -

## Code

### C#
```csharp
using System;
using System.Text;

class Program {
  static void Main() {
    var sb = new StringBuilder();

    while (true) {
      string[] first = Console.ReadLine()!.Split();
      string route = first[0];
      int capacity = int.Parse(first[1]);

      if (route == "#" && capacity == 0)
        break;

      int passengers = int.Parse(Console.ReadLine()!);
      int stops = int.Parse(Console.ReadLine()!);

      for (int i = 0; i < stops; i++) {
        string[] line = Console.ReadLine()!.Split();
        int off = int.Parse(line[0]);
        int on = int.Parse(line[1]);

        passengers -= off;
        int seats = capacity - passengers;
        passengers += Math.Min(seats, on);
      }

      sb.AppendLine(route + " " + passengers);
    }

    Console.Write(sb.ToString());
  }
}
```

### C++

#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(nullptr);

  string route;
  int capacity;

  while (cin >> route >> capacity) {
    if (route == "#" && capacity == 0)
      break;

    int passengers, stops;
    cin >> passengers >> stops;

    for (int i = 0; i < stops; i++) {
      int off, on;
      cin >> off >> on;

      passengers -= off;
      int seats = capacity - passengers;
      passengers += min(seats, on);
    }

    cout << route << " " << passengers << "\n";
  }

  return 0;
}
